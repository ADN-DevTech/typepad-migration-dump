# Typepad Blog Migration Dump

This repository contains exported content from various Autodesk  TypePad blogs. Due to TypePad's announced shutdown, these blogs have been migrated to WordPress and are now available at **https://blog.autodesk.io**.

This repository serves as a complete dump and archive of the original TypePad content for reference and preservation.


## Contents

- **Blog exports** from multiple APS technical blogs
- **Migration guides** and documentation 
- **Scripts** for data processing and migration

## Blog Collections

- **AEC** - Architecture, Engineering & Construction
- **Around The Corner** - Maya development by Cyrille Fauvel
- **AutoCAD** - AutoCAD development blog
- **Cloud & Mobile** - Cloud and mobile development
- **Manufacturing** - Manufacturing Products API blog
- **Infrastructure Modeling** - Infrastructure development
- **Through the Interface** - Kean Walmsley's blog
- **The Building Coder** - Jeremy Tammik's Revit API blog
- **Mod the Machine** - Fusion 360 and Inventor development
- **Just Ones and Zeros** - Vault technical blog
- **Get Core Interface** - 3ds Max development
