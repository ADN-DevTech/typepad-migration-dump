<p>I recently mentioned the availability of the 

<a href="http://thebuildingcoder.typepad.com/blog/2011/06/subscription-packs.html">
subscription packs</a> for 

Revit 2012.

<p>Yesterday, another important set of subscription packs were released: the Revit Extensions:

<ul>
<li>Revit Extensions for Revit Architecture 2012
<li>Revit Extensions for Revit MEP 2012
<li>Revit Extensions for Revit Structure 2012
</ul>

<p>As always, they are available through the 

<a href="http://subscription.autodesk.com">Autodesk subscription center</a>,

to which ADN members also have access by going to the

<a href="http://adn.autodesk.com/adn/servlet/index?siteID=4814862&id=5396918">ADN home page</a> and selecting the

<a href="http://adn.autodesk.com/adn/servlet/item?siteID=4814862&id=11950902">Autodesk Subscription Center</a> link.

<p>Once again, even though all of this is "user stuff", many of the tools included in these extensions address workflows you might also be tempted to tackle through programming macros, so it is important for product users and application developers alike to be aware of them.
