<p>Here is a product related news item, although obviously of interest to developers as well.

<p>Revit product help has been 

<a href="http://thebuildingcoder.typepad.com/blog/2009/08/online-revit-help.html">
available on the web</a> for 

a while now, which provides a number of 

<a href="http://thebuildingcoder.typepad.com/blog/2009/08/online-revit-help.html">
benefits</a>.

<p>Some 

<a href="http://thebuildingcoder.typepad.com/blog/2009/08/online-revit-api-help.html ">
online Revit API help</a> is

also available, although the full Revit SDK documentation is not there yet.

<p>Now the product help system has taken the next step by moving to a 

<a href="http://en.wikipedia.org/wiki/Wiki">
wiki-based system</a> which provides support for a full community learning experience.

<p>Here is a

<span class="asset  asset-generic at-xid-6a00e553e16897883301538df84c82970b"><a href="http://thebuildingcoder.typepad.com/files/revit-2012-wikihelp-overview.pptx">Revit 2012 WikiHelp overview</a></span> which 

includes a summary of WikiHelp features and benefits, highlights key features and lists the benefits of this new community platform.
