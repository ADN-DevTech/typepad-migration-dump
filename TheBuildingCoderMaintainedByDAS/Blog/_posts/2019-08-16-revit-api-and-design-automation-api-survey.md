<p>The Revit product team is conducting a survey to improve the functionality of the Revit services.</p>

<p>Please help them prioritise new features and upcoming enhancements to future releases of the Autodesk Revit API and the Forge Design Automation for Revit API by taking
a <a href="https://autodeskfeedback.az1.qualtrics.com/jfe/form/SV_0fFVmiYvIuqDAJT">brief survey</a>:</p>

<blockquote>
  <p>We would like your help prioritizing API enhancements for future releases of the Autodesk Revit API and the Forge Design Automation for Revit API. Please answer the following 18 questions. It will take about 5-10 minutes to complete. Please help us help you by completing it. We appreciate your time and cooperation:</p>
</blockquote>

<blockquote>
<center>
<a href="https://autodeskfeedback.az1.qualtrics.com/jfe/form/SV_0fFVmiYvIuqDAJT">Revit API and Forge Design Automation for Revit API survey</a>
