<h4><a name="2"></a> RevitLookup for Revit 2020</h4>

<p>I migrated <a href="https://github.com/jeremytammik/RevitLookup">RevitLookup</a> to Revit 2020.</p>

<p>No source code changes at all were required; I just updated the Revit API DLL assembly references and edited the readme file to
create <a href="https://github.com/jeremytammik/RevitLookup/releases/tag/2020.0.0.0">RevitLookup release 2020.0.0.0</a>.</p>

<p>In addition, Harry Mattison of <a href="https://boostyourbim.wordpress.com">Boost your BIM</a> very
kindly provided a ready-built new installer in his post
on <a href="https://boostyourbim.wordpress.com/2019/04/16/revit-lookup-for-revit-2020">RevitLookup for Revit 2020</a>, saying:</p>

<blockquote>
  <p>Iâ€™ve added some conditional compilation and multiple configurations so that the single solution can be used to build against any version of Revit 2017-2020.</p>
</blockquote>

<p>For manual installation, all releases are also provided as <a href="https://github.com/jeremytammik/RevitLookup#-builds">pre-built versions generated by CI</a>
at <a href="https://lookupbuilds.com">lookupbuilds.com</a>.</p>

<h4><a name="3"></a> Revit 2020 SDK posted</h4>

<p>Last week, I described how
I <a href="https://thebuildingcoder.typepad.com/blog/2019/04/the-revit-2020-fcs-api-and-sdk.html">installed the Revit 2020 SDK</a> from
the Revit 2020 installation <code>Setup.exe</code>
and <a href="https://thebuildingcoder.typepad.com/blog/2019/04/the-revit-2020-fcs-api-and-sdk.html#8">set up RvtSamples</a> for it.</p>

<p>Now, in addition, the Revit 2020 SDK has also been posted to
the <a href="https://www.autodesk.com/developer-network/platform-technologies/revit">Revit developer centre public web page</a>,
so you can access it from there independently of the Revit product installation.</p>
