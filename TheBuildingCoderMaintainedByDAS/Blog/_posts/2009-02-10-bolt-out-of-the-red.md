<p>Here is a short note to point out that Guy Robinson has posted some interesting Revit API solutions on his blog 

<a href="http://redbolts.com/blog">
Bolt out of the Red</a>.

One that I found especially useful is on how to get a 

<a href="http://redbolts.com/blog/post/2008/12/01/Getting-your-Revit-thumbnails.aspx">
bitmap or thumbnail from a Revit project or family</a>.

Now he started a series on 

<a href="http://redbolts.com/blog/post/2009/02/10/Static-in-your-commands-User-settings-part-1.aspx">
saving user specific configuration data</a>

for Revit commands using standard .NET functionality from the System.Configuration namespace, which has a number of pitfalls and requires some understanding of how Revit add-ins are loaded into the Revit AppDomain etc.</p>
