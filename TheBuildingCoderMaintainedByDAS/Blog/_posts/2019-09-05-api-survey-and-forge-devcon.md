<h4><a name="2"></a> Revit API and Design Automation API Survey</h4>

<p>Please help prioritise new features and upcoming enhancements to future releases of the Revit API and the <a href="https://forge.autodesk.com/en/docs/design-automation/v3/developers_guide/overview">Forge Design Automation for Revit API</a> by participating in
a <a href="https://autodeskfeedback.az1.qualtrics.com/jfe/form/SV_0fFVmiYvIuqDAJT">brief survey</a>:</p>

<blockquote>
<center>
<a href="https://autodeskfeedback.az1.qualtrics.com/jfe/form/SV_0fFVmiYvIuqDAJT">Revit API and Forge Design Automation for Revit API survey</a>
</center>
</blockquote>

<p>It consists of 18 questions and will take about 5-10 minutes to complete.</p>

<p>The survey deadline has been extended and it now remains open until September 15th.</p>

<p>Thank you in advance! We appreciate your time and cooperation.</p>

<h4><a name="3"></a> Forge DevCon 2019 at AU in Las Vegas</h4>

<p>The <a href="https://forge.autodesk.com/devcon-2019">Forge DevCon</a> is returning
to <a href="https://www.autodesk.com/autodesk-university/conference/las-vegas/overview">Autodesk University in Las Vegas</a>,
on November 18th, and
the <a href="https://forge.autodesk.com/devcon-2019">registration is now open</a>!</p>

<p>About the conferences:​</p>

<ul>
<li>Forge DevCon focuses on cloud-based tools that enhance Autodesk products and help to expand or streamline customer applications.  ​</li>
<li>Autodesk University in Las Vegas is the annual flagship event for those who design and make the world around us.​</li>
<li>Combined, they offer hundreds of focused sessions with over 10,000 professionals. Therefore, this experience can be customized in many ways, creating a focused and hugely beneficial learning and networking opportunity.​</li>
</ul>

<p>What to Expect:​</p>

<ul>
<li><a href="https://autodeskuniversity.smarteventscloud.com/connect/sessionDetail.ww?SESSION_ID=332182">Industry focused keynotes</a> and
future plans for the Forge Platform​</li>
<li>Classes and demos
on <a href="https://autodeskuniversity.smarteventscloud.com/connect/search.ww?pass=forgeDevCon#loadSearch-searchPhrase=&amp;searchType=session&amp;tc=0&amp;sortBy=dayTime&amp;i(77460)=720719&amp;i(72466)=697601&amp;p=">data visualization</a>,
<a href="https://autodeskuniversity.smarteventscloud.com/connect/search.ww?pass=forgeDevCon#loadSearch-searchPhrase=&amp;searchType=session&amp;tc=0&amp;sortBy=dayTime&amp;i(77460)=720719&amp;i(72466)=697582&amp;p=">process automation</a>,
and <a href="https://autodeskuniversity.smarteventscloud.com/connect/search.ww?pass=forgeDevCon#loadSearch-searchPhrase=&amp;searchType=session&amp;tc=0&amp;sortBy=dayTime&amp;i(77460)=720719&amp;i(72466)=697573&amp;p=">cloud-based collaboration</a> for connecting teams and workflows​</li>
<li>Expanded networking Village in partnership with Connect and Construct​</li>
</ul>

<p>So how can you get involved?​</p>

<p><a href="https://autodeskuniversity.smarteventscloud.com/portal/registration.ww?pass=forgeDevCon">Register today for Forge DevCon</a> to
reserve your space, or select Forge DevCon when registering
for <a href="https://www.autodesk.com/autodesk-university/conference/las-vegas/overview?pass=forgeDevCon?utm_campaign=DevCon&amp;utm_medium=Presentation&amp;utm_source=SalesAsset&amp;utm_content=Ent&amp;utm_term=Forge">Autodesk University</a>​.</p>

<p>If you are interested in sponsoring the event or would like more information, please reach out to <a href="mailto:forge.devcon@autodesk.com">forge.devcon@autodesk.com​</a>.</p>

<p>We hope to see you in November.</p>
