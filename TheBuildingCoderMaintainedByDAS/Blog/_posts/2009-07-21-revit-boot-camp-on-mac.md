<p>Autodesk is now officially supporting Apple Boot Camp for users of AutoCAD, 3ds Max, Inventor and last but not least the three flavours of Revit.
Boot Camp is a component of the Mac OSX operating system that enables customers to run Windows on the Mac.
See 

<a href="http://www.autodesk.com/mac">www.autodesk.com/mac</a> 

for more general details and this 

<a href="http://usa.autodesk.com/adsk/servlet/ps/dl/item?siteID=123112&id=13578805&linkID=9243099">FAQ</a>

for some frequently asked Revit specific questions and hardware requirements.</p>
