<h4><a name="2"></a>Revit 2019 SDK Published</h4>

<p>The Revit 2019 SDK has now been properly and completely published.</p>

<p>I pointed out
the <a href="http://download.autodesk.com/us/revit-sdk/REVIT_2019_SDK.msi">direct <code>REVIT_2019_SDK.msi</code> download link</a> for the SDK when
describing <a href="http://thebuildingcoder.typepad.com/blog/2018/05/installing-the-revit-2019-sdk-april-update.html">installing the Revit 2019 SDK April update</a></p>

<p>Now it also appears in its proper location in
the <a href="http://autodesk.com/developrevit">Revit Developer Centre</a>
at <a href="http://autodesk.com/developrevit">autodesk.com/developrevit</a>,
which nowadays points to a new location
at <a href="https://www.autodesk.com/developer-network/platform-technologies/revit">www.autodesk.com/developer-network/platform-technologies/revit</a>.</p>

<h4><a name="3"></a>My First Revit Plugin Update</h4>

<p>Another important update for getting started with the Revit API is
the <a href="http://www.autodesk.com/myfirstrevitplugin">My First Revit Plug-In tutorial</a>
at <a href="http://www.autodesk.com/myfirstrevitplugin">www.autodesk.com/myfirstrevitplugin</a>.</p>

<blockquote>
  <p>It is a self-paced tutorial guide for a smooth introduction into the programming world. This is a 'one-stop shop' learning path for users who know Revit but are new to programming and thinking about taking the plunge. It introduces the Revit .NET API using the C# programming language.</p>
</blockquote>
