<p>Let's share a snippet of AutoCAD Architecture information here as well for a change, answering a question that just came up on that:</p>

<p><strong>Question:</strong> Nous avons besoin de basculer nos outils spécifiques sur AutoCAD Architecture 2019.</p>

<p>Nous ne trouvons pas la documentation sur la nouvelle API qui a visiblement été modifiée.</p>

<p>Pouvez-vous nous indiquer un lien?</p>

<p>Pour Revit nous utilisons <a href="http://www.revitapidocs.com">revitapidocs.com</a>, mais nous ne trouvons pas l'équivalent pour AutoCAD Architecture 2019.</p>

<p><em>We need to migrate our tools to AutoCAD Architecture 2019.
I see no information on the new API, although it has obviously been modified.
Can you share a link?
For Revit, we use <a href="http://www.revitapidocs.com">revitapidocs.com</a>, but I cannot find anything equivalent for AutoCAD Architecture 2019.</em></p>

<p><strong>Answer:</strong> The AutoCAD Architecture .NET reference guide in included in the product install, as explained in 
the <a href="https://www.autodesk.com/developer-network/platform-technologies/autocad-architecture">AutoCAD Architecture and MEP Developer Centre</a>:</p>

<blockquote>
  <p>Because AutoCAD Architecture is based on AutoCAD, you can take advantage of
  all the AutoCAD source code samples for .NET that are included with the installation
  of AutoCAD Architecture software in the Samples directory. There are also additional
  AutoCAD Architecture specific samples included in the installation.</p>
</blockquote>

<p>The developer centre also points to the relevant blogs and discussion groups for discussing news and raising questions.</p>

<p>Finally,
the <a href="https://adndevblog.typepad.com/aec/2013/05/api-webcast-archive.html">API webcast archive</a> provides
a host of further slightly dated information in different languages &ndash; English, Chinese and Japanese.</p>
