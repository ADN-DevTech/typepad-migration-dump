<h4><a name="2"></a> Revit 2021.1 Questions</h4>

<p>Yesterday, I mentioned <a href="https://thebuildingcoder.typepad.com/blog/2020/07/revit-20211-update-and-normalising-custom-export-uv.html#2">the Revit 2021.1 update</a>.</p>

<p>Two questions arose on that:</p>

<p>Vincent Cadoret <a href="https://twitter.com/vincentcadoret">@vincentcadoret</a>
<a href="https://twitter.com/vincentcadoret/status/1286292927573090305">asks</a>:</p>

<p>Q: Any changes in the Revit API in this 2021.1 update vs 2021.0?</p>

<p>A: Please read
the <a href="https://help.autodesk.com/view/RVT/2021/ENU/?guid=RevitReleaseNotes_2021updates_2021_1_Resolved_Issues_2021_1_html">Revit 2021.1 release notes</a> and
let us know what you find out &ndash; I see some API enhancements and fixes, so there is a slight change, indeed.</p>

<p>E B <a href="https://twitter.com/truevis">@truevis</a>
<a href="https://twitter.com/truevis/status/1286301553096179712">asks</a>:</p>

<p>Q: Any API access to Shared Views available?</p>

<p>A: Again, check out the release notes and let us know what you find out.
For example, I do see an entry saying, <em>Shared View: Provided an easy way to share 3D views in Revit</em>.</p>
