<P>The Revit 2009 web update 2 has been posted to the Autodesk product site for the English versions of Revit <A href="http://usa.autodesk.com/adsk/servlet/item?siteID=123112&amp;id=11017599">Architecture</A>, <A href="http://usa.autodesk.com/adsk/servlet/item?siteID=123112&amp;id=11273489">MEP</A> and <A href="http://usa.autodesk.com/adsk/servlet/item?siteID=123112&amp;id=9281007">Structure</A>. Unlike previous versions, this update makes use of patching technology and does not require a complete reinstallation. For this reason, the SDK is not included in the update, like it used to be in previous versions. Therefore, the Revit 2009 SDK web update 2 has been posted separately. It is available under the link <A href="http://images.autodesk.com/adsk/files/revit_2009_sdk.zip">Revit 2009 SDK (Update 2) (zip - 61330Kb)</A> in the <A href="http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=2484975">Revit Developer Center</A> as well as on <A href="http://adn.autodesk.com/adn/servlet/index?siteID=4814862&amp;id=5017413&amp;linkID=4901650">ADN</A>. Among other things, the updated version includes the <A href="http://thebuildingcoder.typepad.com/blog/2008/09/roomsroofs-sdk.html">RoofsRooms</A> SDK sample, the <a href="http://thebuildingcoder.typepad.com/blog/files/Revit_2009_API_Developer_Guide.zip">Revit 2009 API Developer Guide</a> and the 'Guide to placing Family Instances with the API.doc' mentioned in <A href="http://thebuildingcoder.typepad.com/blog/2008/08/the-revit-sdk-c.html">The Revit SDK Contents</A>.</P>
