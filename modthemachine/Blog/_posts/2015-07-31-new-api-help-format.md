<p>With the latest release of Fusion, a new format for the syntax in the help has been introduced.&#160; Although this change has been made for the Fusion help, the same help system is also used for Inventor so these changes can also be fairly easily made for the Inventor API help too.&#160; Because of that I would like to get feedback from both Inventor and Fusion users about this new format.</p>  <p>In the old help, the syntax was shown using a pseudo Python or Javascript syntax that represented that function as if you were writing the function.&#160; In Inventor it uses a VBA style syntax.&#160; It was pseudo code because it also included argument types which aren’t actually needed in Python or JavaScript but were include because they help to provide the full description.&#160;
