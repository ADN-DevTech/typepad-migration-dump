<p>In <a href="https://youtu.be/YLc7OwZ6Hfc" rel="noopener" target="_blank">this week’s episode</a> we look at 5 more Commodore 64 games loaded from floppy disk. Things kick off with <a href="https://en.wikipedia.org/wiki/Shadow_Dancer_(1989_video_game)" rel="noopener" target="_blank">Shadow Dancer</a>, which seems like a very playable side-scrolling fighting game. We then try a logic/puzzle game called <a href="https://wowroms.com/en/roms/commodore-64/sliding-skill/140191.html" rel="noopener" target="_blank">Sliding Skill</a> before loading a German-language steam engine simulator (how cool is that?) called <a href="https://www.c64games.de/phpseiten/spieledetail.php?filnummer=4787" rel="noopener" target="_blank">Dampfmaschine</a>. We then play a game called <a href="https://www.lemon64.com/games/details.php?ID=3974" rel="noopener" target="_blank">Billiards</a> or Pool (not sure which, to be honest) and then wrap up with a fun little game called <a href="https://en.wikipedia.org/wiki/Lazy_Jones" rel="noopener" target="_blank">Lazy Jones</a>.</p>
