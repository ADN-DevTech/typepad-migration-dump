<p>For the first time ever (apparently – I don’t remember this kind of detail) there has been a mid-cycle update to <a href="http://docs.autodesk.com/ACD/2013/ENU/" target="_blank">AutoCAD 2013’s online Help system</a> (comparable updates have been made for <a href="http://docs.autodesk.com/ACDLT/2013/ENU/" target="_blank">AutoCAD LT 2013</a> and <a href="http://docs.autodesk.com/AMECH_PP/2013/ENU/" target="_blank">AutoCAD Mechanical 2013</a>).</p>  <p>The updates include more than just new content – there have been significant enhancements made to the search mechanism, for instance.</p>  <p>One feature I think readers of this blog will find particularly helpful is the ability to filter for developer-oriented content:</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e2017c31d5fe47970b-pi" target="_blank"><img style="background-image: none; border-bottom: 0px; border-left: 0px; margin: 20px auto; padding-left: 0px; padding-right: 0px; display: block; float: none; border-top: 0px; border-right: 0px; padding-top: 0px" title="Filter for developer content in AutoCAD 2013&#39;s updated online Help" border="0" alt="Filter for developer content in AutoCAD 2013&#39;s updated online Help" src="/assets/image_130259.jpg" width="454" height="352" /></a></p>  <p>And be sure to “Include Web Results” if you want the results to include content from this blog, of course. :-)</p>  <p>You’ll find there’s a really handy filtering mechanism you can use to just pull up user documentation on commands and/or system variables, or – and this is where things get even cooler – once you’ve selected “Developer” you can, for instance, just choose to get information on .NET:</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e2017c31d5fe62970b-pi" target="_blank"><img style="background-image: none; border-bottom: 0px; border-left: 0px; margin: 20px auto; padding-left: 0px; padding-right: 0px; display: block; float: none; border-top: 0px; border-right: 0px; padding-top: 0px" title="And with additional API-centric filters" border="0" alt="And with additional API-centric filters" src="/assets/image_870714.jpg" width="454" height="352" /></a></p>  <p>For more information on this update, check out <a href="http://docs.autodesk.com/ACD/2013/ENU/index.html?url=files/GUID-60C67B97-FC11-4FCD-BE40-A16E82D55076.htm" target="_blank">this video</a> or head on over to <a href="http://autocadinsider.autodesk.com/my_weblog/2012/09/help-update-for-autocad-2013-and-autocad-lt-2013.html" target="_blank">this AutoCAD Insider blog post</a>.</p>  <p>I’m not sure whether this is really related, as such, but it’s certainly on the same topic: there has also been an update to the AutoCAD 2013 ActiveX Developer Documentation (which for me means the documentation we provide on our COM Automation API, but even that term may well now have been superseded), this time made available via <a href="http://autodesk.com/developautocad" target="_blank">the AutoCAD Developer Center</a>. Find out more on <a href="http://hyperpics.blogs.com/beyond_the_ui/2012/09/updated-autocad-2013-activex-developer-documentation.html" target="_blank">Lee Ambrosius’ blog post</a>.</p>
