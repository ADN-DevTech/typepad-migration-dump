<p>In <a href="https://youtu.be/1NiF8k1rJwg" rel="noopener" target="_blank">this week’s episode</a> we once again play some more games on a Commodore 64 that we find on random floppies. We start with a game I remembered we hadn’t had time to try last week, which turned out to be <a href="https://en.wikipedia.org/wiki/International_Karate" rel="noopener" target="_blank">International Karate</a>. As the directory listing stated “Seite A” (“Side A” in German), I decided to flip the floppy and see what was on Side B. Well, it had some real crackers (mostly cracked crackers, in fact) such as <a href="https://en.wikipedia.org/wiki/Enduro_Racer" rel="noopener" target="_blank">Enduro Racer</a>, <a href="https://en.wikipedia.org/wiki/Yie_Ar_Kung-Fu" rel="noopener" target="_blank">Yie Ar Kung-Fu</a>, <a href="https://en.wikipedia.org/wiki/Yie_Ar_Kung-Fu_II" rel="noopener" target="_blank">Yie Ar Kung-Fu II</a> and <a href="https://en.wikipedia.org/wiki/Tetris" rel="noopener" target="_blank">Tetris</a>. A fun episode!</p>
