<p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201b8d0921397970c-pi" target="_blank"><img title="Retrospecs" style="border-top: 0px; border-right: 0px; background-image: none; border-bottom: 0px; float: right; padding-top: 0px; padding-left: 0px; margin: 0px 0px 10px 20px; border-left: 0px; display: inline; padding-right: 0px" border="0" alt="Retrospecs" src="/assets/image_215564.jpg" width="103" align="right" height="103" /></a>Every so often I get an attack of nostalgia for <a href="http://through-the-interface.typepad.com/through_the_interface/2012/10/retro-computing-with-the-raspberry-pi.html" target="_blank">the early days</a> of <a href="http://through-the-interface.typepad.com/through_the_interface/2012/10/my-new-zx-spectrum.html" target="_blank">personal</a> <a href="http://through-the-interface.typepad.com/through_the_interface/2012/12/loading-games-on-my-zx-spectrum-from-an-ipod-photo.html" target="_blank">computing</a>. The latest bout was triggered by the discovery of an app called <a href="http://8bitartwork.co.uk" target="_blank">Retrospecs</a>, an iOS-based image processing app that transforms photos to use the colour palettes of 8- and 16-bit computers.</p>  <p>The latest release – v1.7 – supports the following emulations:</p>  <ul>   <li>Teletext </li>    <li>Apple ][ (Low res) </li>    <li>Atari 2600 (NTSC) </li>    <li>Intellivision </li>    <li>IBM CGA (6 variations) </li>    <li>BBC Micro (Mode 1 &amp; 2) </li>    <li>Sinclair ZX Spectrum </li>    <li>Commodore 64 (Low res &amp; high res modes) </li>    <li>Colecovision </li>    <li>Dragon 32 (PMODE 3) </li>    <li>Thomson TO7 </li>    <li>MSX (Screen mode 2 and (for comedy value) 3) </li>    <li>Sinclair QL (Low res &amp; high res modes) </li>    <li>Apple Macintosh (Original 1984 model) </li>    <li>Thomson MO5 </li>    <li>Amstrad CPC (Colour &amp; green screen versions) </li>    <li>Commodore C16/+4 (Low res &amp; high res modes) </li>    <li>IBM EGA </li>    <li>Commodore Amiga (OCS - 320x256 in 32 colours and 640x256 in 16 colours) </li>    <li>Atari ST (320x200 in 16 colours and 640x200 in 4 colours) </li>    <li>Sega Master System </li>    <li>IBM VGA (Mode 13h) </li>    <li>Sega Mega Drive </li>    <li>Nintendo Game Boy </li>    <li>Amstrad 464/6128 plus </li>    <li>Super Nintendo Entertainment System </li> </ul>  <p>You can also apply filters and tweak the dithering to get better results – very handy if the image is dark and needs a vibrancy boost (this is especially important if choosing one of the more gaudy colour palettes).</p>  <p>I’ve just upgraded to the latest release on my iPad 2 (the only iOS device I own) and took the new modes for a spin. Here’s a GIF of the whole set being applied to a fairly boring photo of my neighbour’s house. [Click the image for the full-size, 3MB version where you can read the names of the various modes.]</p>  <p><a href="http://through-the-interface.typepad.com/files/Retrospecs-big.gif" target="_blank"><img title="Retrospecs" style="float: none; margin: 20px auto; display: block" alt="Retrospecs" src="/assets/image_620301.jpg" width="300" height="400" /></a></p>  <p>This app is really fun: a great way to create retro images for profile pics or greeting cards.</p>  <p>It’s also a real bargain, weighing in at just <a href="https://itunes.apple.com/us/app/retrospecs/id887031094" target="_blank">$0.99 in the US store</a>.
