<p>In this week’s episode we load (or try to, anyway) a number of games from one floppy. We start with a game called <a href="https://en.wikipedia.org/wiki/Up%27n_Down" rel="noopener" target="_blank">Up ’n’ Down</a> (mis-labelled as Bump ‘n’ Jump) and then try to play <a href="https://www.lemon64.com/games/details.php?ID=1392" rel="noopener" target="_blank">Jump Jet</a> before moving onto <a href="https://en.wikipedia.org/wiki/BMX_Simulator" rel="noopener" target="_blank">BMX Simulator</a>. Finally I try to load <a href="https://en.wikipedia.org/wiki/The_Goonies#Video_games" rel="noopener" target="_blank">The Goonies</a>, but fail miserably. I’ll have to see if I have it elsewhere, or can copy it to a floppy from a .d64 image.</p>
