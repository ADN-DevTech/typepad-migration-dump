<p>There’s been an interesting project on Autodesk Labs for some time, now, called Project Chronicle. It allows you to document use of Autodesk software, whether to provide support, training or – for that matter – to blog about it.</p>  <p>Anyway, this technology has now <a href="http://labs.blogs.com/its_alive_in_the_lab/2014/06/autodesk-researchs-project-chronicle-graduates-to-autodesk-screencast.html" target="_blank">graduated from Autodesk Labs</a> and is available as <a href="https://screencast.autodesk.com" target="_blank">Autodesk Screencast</a>. It seemed like a good time to take it for a spin.</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201a73dd1dc6b970d-pi" target="_blank"><img title="Screencast Logo" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 20px auto; display: block; padding-right: 0px; border-top-width: 0px" border="0" alt="Screencast Logo" src="/assets/image_226288.jpg" width="390" height="50" /></a></p>  <p>Autodesk Screencast is about both creating recordings but also the hosting mechanism required to serve these up. If you go to the web-site you can download the recording tool but also see a gallery of existing recordings:</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201a3fd17424d970b-pi" target="_blank"><img title="Autodesk Screencast site" style="border-top: 0px; border-right: 0px; background-image: none; border-bottom: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 20px auto; border-left: 0px; display: block; padding-right: 0px" border="0" alt="Autodesk Screencast site" src="/assets/image_897749.jpg" width="387" height="311" /></a></p>  <p>Autodesk Screencast will capture recordings of pretty much any software, but to get command-level information you need to have a special plug-in installed. Just make sure the AutoCAD plug-in is selected when you install Screencast:</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201a73dd1e6bf970d-pi" target="_blank"><img title="Installing Screencast" style="border-top: 0px; border-right: 0px; background-image: none; border-bottom: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 20px auto; border-left: 0px; display: block; padding-right: 0px" border="0" alt="Installing Screencast" src="/assets/image_579379.jpg" width="263" height="204" /></a></p>  <p>Once installed, the simplest way to launch Screencast is from within AutoCAD, from the <strong>Add-Ins</strong> ribbon tab:</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201a511c6a73a970c-pi" target="_blank"><img title="Launching Screencast from within AutoCAD" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 20px auto; display: block; padding-right: 0px; border-top-width: 0px" border="0" alt="Launching Screencast from within AutoCAD" src="/assets/image_81958.jpg" width="390" height="204" /></a></p>  <p>Once logged into your Autodesk account, you should be able to use the dialog interface to record working inside AutoCAD:</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201a3fd17384d970b-pi" target="_blank"><img title="Recording dialog" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 20px auto; display: block; padding-right: 0px; border-top-width: 0px" border="0" alt="Recording dialog" src="/assets/image_941653.jpg" width="275" height="264" /></a></p>  <p>The dialog has noticed that the recording dimensions aren’t in the ideal 16x10 ratio, so it will offer to fix this before you record. Clicking the button next to the application name will do so before getting a warning.</p>  <p>Once you’ve completed your recording, edited and uploaded it, you’ll be prompted with a link to it on the Screencast site:</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201a3fd173854970b-pi" target="_blank"><img title="Recording completed and uploaded" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 20px auto; display: block; padding-right: 0px; border-top-width: 0px" border="0" alt="Recording completed and uploaded" src="/assets/image_490985.jpg" width="254" height="264" /></a></p>  <p>Once processed, you can get an embedding code from the site (although I had to calculate the dimensions to keep the aspect ratio manually for the width of my blog, something I’ve gotten used to not doing with YouTube).</p>  <p>Here’s a quick demo video of me using AutoCAD with a custom command to jig an associative hatch using code we saw in <a href="http://through-the-interface.typepad.com/through_the_interface/2011/04/jigging-an-autocad-hatch-using-net.html" target="_blank">this recent post</a>:</p>  <br /><iframe height="294" src="https://screencast.autodesk.com/Embed/e5cd5751-0b91-4d6f-824d-ae0cdda9986d" frameborder="0" width="470" webkitallowfullscreen="webkitallowfullscreen" allowfullscreen="allowfullscreen"></iframe>  <br />  <br />  <p>There you have it. I can see myself using this tool more often when sharing videos of using Autodesk software in the future.</p>
