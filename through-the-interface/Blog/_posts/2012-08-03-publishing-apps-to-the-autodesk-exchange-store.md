submit apps for hosting by Autodesk.</p>  <p>[<strong>*</strong> Hopefully this point is obvious: anyone can submit apps, but Autodesk clearly reserves the right to reject inappropriate submissions.]</p>  <p>Exchange Stores are available for the following products:</p>  <ul>   <li><a href="http://apps.exchange.autodesk.com/ACD/Home" target="_blank">AutoCAD</a> </li>    <li><a href="http://apps.exchange.autodesk.com/ARCH/Home" target="_blank">AutoCAD Architecture</a> </li>    <li><a href="http://apps.exchange.autodesk.com/CIV3D/Home" target="_blank">AutoCAD Civil 3D</a> </li>    <li><a href="http://apps.exchange.autodesk.com/ECSCAD/Home" target="_blank">AutoCAD ecscad</a> </li>    <li><a href="http://apps.exchange.autodesk.com/ACAD_E/Home" target="_blank">AutoCAD Electrical</a> </li>    <li><a href="http://apps.exchange.autodesk.com/AMECH/Home" target="_blank">AutoCAD Mechanical</a> </li>    <li><a href="http://apps.exchange.autodesk.com/MEP/Home" target="_blank">AutoCAD MEP</a> </li>    <li><a href="http://apps.exchange.autodesk.com/ALIAS/Home" target="_blank">Autodesk Alias</a> </li>    <li><a href="http://apps.exchange.autodesk.com/INVNTOR/Home" target="_blank">Autodesk Inventor</a> </li>    <li><a href="http://apps.exchange.autodesk.com/INVPUB/Home" target="_blank">Autodesk Inventor Publisher</a> </li>    <li><a href="http://apps.exchange.autodesk.com/RVT/Home" target="_blank">Autodesk Revit</a> </li>    <li><a href="http://apps.exchange.autodesk.com/SCFD/Home" target="_blank">Autodesk Simulation</a> </li>    <li><a href="http://apps.exchange.autodesk.com/VLTC/Home" target="_blank">Autodesk Vault</a> </li> </ul>  <p>As I’ve probably said before, these stores are a huge opportunity for developers to gain direct access to Autodesk customers, greatly reducing costs related sales &amp; marketing. When someone downloads an app you’ve published to the store – even if it’s for free – then you, as publisher, will get notified by email (and will then be able to maintain a database of the downloaders’ email addresses). </p>  <p>More detailed information on the submission process is available on <a href="http://autodesk.com/developapps" target="_blank">the Autodesk Exchange Developer Center</a> (including <a href="http://usa.autodesk.com/adsk/servlet/item?siteID=123112&amp;id=20143020" target="_blank">AutoCAD-specific information</a>).</p>  <p>I haven’t yet compared the new process with the information I’ve posted previously to see what’s different.
