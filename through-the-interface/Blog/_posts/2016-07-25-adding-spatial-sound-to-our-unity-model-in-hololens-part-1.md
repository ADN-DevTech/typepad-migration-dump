<p>As mentioned <a href="http://through-the-interface.typepad.com/through_the_interface/2016/07/developing-for-hololens-on-windows-10.html" target="_blank">last time</a>, upgrading to Windows 10 has opened the door to spatial sound in our HoloLens model.</p> <p>The foundational step is to set our Unity project’s audio settings to have the Spatializer Plugin as the “MS HRTF Spatializer”: if you don’t see this option then you may need to upgrade your OS (as I did).</p> <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201b7c87f54fe970b-pi"><img title="Spatial sound" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 30px auto; display: block; padding-right: 0px; border-top-width: 0px" border="0" alt="Spatial sound" src="/assets/image_194714.jpg" width="382" height="288"></a></p> <p>At this point we need to think a little about how best to implement spatial sound in this project, particularly as it literally contains a number of moving parts. I see three main alternatives:</p> <ol> <li>A single sound is assigned to our robot  <ul> <li>When the robot stops completely, so does the sound</li></ul> <li>The same sound is assigned to each of the robot’s parts  <ul> <li>When each part stops moving, so does the sound for that part</li></ul> <li>A different sound is assigned to each of the robot’s parts  <ul> <li>When each part stops moving, so does the sound for that part</li></ul></li></ol> <p>In some ways option 1 is simplest – we only need to attach a single sound to the root of the robot – but in others it’s the most complex, as we have to keep track of which parts are moving. But we’ll start with that and see how it goes: I have a sneaking suspicion that adding a sound for each part (i.e. options 2 &amp; 3) add unnecessary (and unwelcome) processing overhead for the device.</p> <p>The first order of business is to add the <a href="https://www.freesound.org/people/jessepash/sounds/139969/" target="_blank">chosen sound</a> as an AudioSource to our root object (by dragging and dropping the project asset) and set its properties according to the <a href="https://developer.microsoft.com/en-us/windows/holographic/holograms_101#chapter_5_-_spatial_sound" target="_blank">Holograms 101</a> tutorial:</p> <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201b7c87f5504970b-pi"><img title="Sound settings" style="border-left-width: 0px;
