<p>Colleagues in Autodesk’s Office of the CTO (OCTO) – who work in the <a href="http://through-the-interface.typepad.com/through_the_interface/2015/10/a-quick-trip-to-the-octo-robotics-lab-on-pier-9.html" target="_blank">robotics lab on Pier 9</a> – have just released the code for a new tool called <a href="https://mimicformaya.com" target="_blank">Mimic</a>.<br></p><p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201b7c95c459b970b-pi" target="_blank"><img width="500" height="187" title="Mimic" style="margin: 30px auto; float: none; display: block;" alt="Mimic" src="/assets/image_553378.jpg"></a>Mimic is an open source plugin – with the <a href="https://github.com/AutodeskRoboticsLab/Mimic" target="_blank">Python code posted on GitHub</a> – that allows you to simulate and control the use of a number of industrial robots (including <a href="https://github.com/AutodeskRoboticsLab/Mimic/blob/master/mimic/rigs/rigs.md" target="_blank">various robots from ABB and KUKA</a>).</p><p>Here’s a screenshot:<br></p><p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201b8d2e68395970c-pi" target="_blank"><img width="500" height="358" title="Mimic for Maya" style="margin: 30px auto; border: 0px currentcolor; border-image: none; float: none; display: block; background-image: none;" alt="Mimic for Maya" src="/assets/image_780912.jpg" border="0"></a>I’m really happy to see OCTO releasing open source projects in this way. If you’re interested in contributing to the project, be sure to check out the <a href="https://github.com/AutodeskRoboticsLab/Mimic/blob/master/mimic/docs/devnotes.md" target="_blank">developer notes</a>. There’s also a <a href="https://www.mimicformaya.com/#community-section" target="_blank">Slack channel</a> you can join, to see what’s happening in the community.</p><p>Maybe Mimic might even be used to <a href="https://www.youtube.com/watch?v=kO2toZcsevI&amp;list=PLMPZ0Qz6RKGqhrrLKOLebfE-AzcJUPtsb&amp;index=11" target="_blank">make robots dance</a>? :-)</p>
