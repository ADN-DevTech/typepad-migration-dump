<p>Here’s an idea I’ve been playing around with for some time: say you want to capture geometry as code for pasting back into your application, how do you do it? For instance, sometimes you might want to model geometry using AutoCAD and then capture it as code for later generation at runtime.</p>  <p>I have a specific example in mind, of course: I have some boundary loops that can be used to generate the outer shell of a space shuttle using the surfacing capabilities introduced in AutoCAD 2010. I’d really like to use code to store these loops and – in due course – generate the various surfaces using these loops to define the “lofts”. It may even be that the loops don’t have to be database-resident, but we’ll see that when we come to the actual surface generation.</p>  <p>Here are the loops:</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e20133f5b04c30970b-pi"><img style="border-bottom: 0px;
