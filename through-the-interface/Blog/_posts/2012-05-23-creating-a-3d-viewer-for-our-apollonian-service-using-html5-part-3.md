Many thanks to Jeff Geer for once again pointing me in the right direction on this. :-)</p>  <p>The trackball capability allows you not to worry about manual implementation of 3D navigation inside your viewer: you simply set up some basic parameters to indicate the size of your model and the speed with which you want navigation to occur – as well as the keys for rotate, zoom and pan – and then you just let it fly.</p>  <p><a href="http://through-the-interface.typepad.com/files/ApollonianViewerTrackball.html" target="_blank">Here’s the updated file for you to try</a> in either Chrome or Firefox (or elsewhere with WebGL enabled), with the A, S and D keys mapped to rotate, zoom and pan, respectively.</p>  <p>Here’s a video of the trackball capability in action:</p> <object id="Object1" width="470" height="470" type="application/x-shockwave-flash"
