<p>There’s a great opportunity coming up in March to learn about building software that works with Autodesk Construction Cloud: <a href="https://forge.autodesk.com/blog/acs-integration-partner-summit-2022" target="_blank">the ACS Integration Partner Summit 2022</a> is being held on St Patrick’s Day, March 17th.</p><p><a href="https://through-the-interface.typepad.com/.a/6a00d83452464869e202942f9a0303200c-pi" target="_blank"><img width="500" height="281" title="ACS Integration Partner Summit 2022" style="margin: 30px auto; border: 0px currentcolor; border-image: none; float: none; display: block; background-image: none;" alt="ACS Integration Partner Summit 2022" src="/assets/image_13983.jpg" border="0"></a></p><p>Here’s some information on what to expect from this event:</p><blockquote><ul><li>Autodesk Construction Overview</li><ul><li>Hear about Autodesk's construction platform vision, including recent business updates and a technical overview of the breadth of supported construction workflows.</li></ul><li>Autodesk Construction Cloud Platform, product, and Forge API updates</li><ul><li>A large focus will be on Forge APIs. We will discuss recently released APIs that allow you to deliver new features and workflows, review the API roadmap, and provide other Forge resources so you can start taking advantage of the platform now.</li></ul><li>Go-to-market best practices with co-marketing resources</li><ul><li>Want to connect to Autodesk’s construction customers? Learn about our sales alignment best practices, common marketing motions with partners, and how to utilize different toolkits for social media, blogs, and other promotional tactics.</li></ul></ul></blockquote><p>So don’t be green with envy (groan) of others building on top of this platform: attend yourself to find out more!</p>
