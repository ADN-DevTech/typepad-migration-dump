<p>I’ve been working with <a href="https://developer.autodesk.com/api/autocadio" target="_blank">AutoCAD I/O</a> quite heavily over the last week or two – mainly spending time fleshing out my <a href="http://jigsawify.com" target="_blank">Jigsawify</a> web application well in advance of <a href="http://through-the-interface.typepad.com/through_the_interface/2015/05/autodesk-university-2015-class-proposals.html" target="_blank">AU2015</a> – and came across this helpful trick for visualizing an <a href="http://www.odata.org" target="_blank">OData</a>-based REST API graphically: you can copy and paste the <a href="https://developer.autodesk.com/api/autocadio/#odata-interface" target="_blank">XML schema for the AutoCAD I/O API</a> into a file and then load it into the handy <a href="http://pragmatiqa.com/xodata" target="_blank">XOData tool</a>.</p>  <p>Here’s what was generated (after a little editing from my side for layout):</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201bb0856251c970d-pi" target="_blank"><img alt="AutoCAD IO OData API" border="0" height="255" src="/assets/image_562763.jpg" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 20px auto; display: block; padding-right: 0px; border-top-width: 0px" title="AutoCAD IO OData API" width="390" /></a></p>  <p>I’m making pretty good progress using AutoCAD I/O from Node.js – which is interesting as I can now run my edge-detection code both in the browser and on the server – but I still have a few issues to iron out before I can complete the series I <a href="http://through-the-interface.typepad.com/through_the_interface/2015/06/running-custom-net-code-in-the-cloud-using-autocad-io-part-1.html" target="_blank">started</a> <a href="http://through-the-interface.typepad.com/through_the_interface/2015/06/running-custom-net-code-in-the-cloud-using-autocad-io-part-2.html" target="_blank">some</a> <a href="http://through-the-interface.typepad.com/through_the_interface/2015/06/running-custom-net-code-in-the-cloud-using-autocad-io-part-3.html" target="_blank">time</a> <a href="http://through-the-interface.typepad.com/through_the_interface/2015/06/running-custom-net-code-in-the-cloud-using-autocad-io-part-4.html" target="_blank">ago</a>.
