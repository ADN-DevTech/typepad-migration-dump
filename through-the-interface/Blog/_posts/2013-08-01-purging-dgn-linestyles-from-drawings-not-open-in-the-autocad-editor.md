<p>This request came in a few weeks ago regarding <a href="http://through-the-interface.typepad.com/through_the_interface/2013/07/minor-update-to-the-dgn-purge-command.html" target="_blank">the source code</a> for <a href="http://through-the-interface.typepad.com/through_the_interface/2013/07/autocad-dgn-hotfix-now-available.html" target="_blank">the recently posted DGN clean-up tool</a>:</p>
<blockquote>
<p><em>Would you modify the tool so that it can be processed by the equivalent figure ObjectDBX (of course for AutoCAD, not RealDWG)? I would like to give the Database object to the function, and the tool would have to process them in batch (a tool for loading drawings on my side).</em></p>
</blockquote>
<p>It seemed like a reasonable enough request, so I spent some time putting together a version of the code that can be used on non-editor resident drawings. I changed the existing DGNPURGE command to make use of a helper function that takes two arguments – the Database to work on and the Editor to display information to the user – and then used that same function to purge unwanted DGN linestyles from a drawing selected by the user.</p>
<p>I took some file selection code from <a href="http://through-the-interface.typepad.com/through_the_interface/2010/08/importing-the-output-from-photofly-via-the-photo-scene-editor-into-autocad.html" target="_blank">this post</a> and some bits and pieces from <a href="http://through-the-interface.typepad.com/through_the_interface/2007/08/purging-regis-1.html" target="_blank">this post</a>, as well as picking up <a href="http://www.theswamp.org/index.php?topic=36671.0" target="_blank">this helpful tip</a> from our friends at The Swamp to open the drawing for read in order for the thumbnail bitmap to be retained properly in the output drawing.</p>
<p>Here’s the resultant C# code. It should be trivial to either build this into a RealDWG app or simply extend it to work on a folder of drawings. As it stands the DGNPURGE command and its new sibling, DGNPURGEEXT, should be scriptable using <a href="http://through-the-interface.typepad.com/through_the_interface/2010/10/novembers-plugin-of-the-month-live-on-autodesk-labs-scriptpro-20-for-autocad.html" target="_blank">ScriptPro 2.0</a> or <a href="http://through-the-interface.typepad.com/through_the_interface/2011/02/customizing-scriptpro-20-to-process-drawings-not-present-in-the-autocad-editor.html" target="_blank">a custom build of it</a> (using either a full AutoCAD-based product or <a href="http://through-the-interface.typepad.com/through_the_interface/2012/02/the-autocad-2013-core-console.html" target="_blank">the Core Console</a>).</p>
