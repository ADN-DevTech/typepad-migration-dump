<p>In today’s post I thought I’d quickly mention a couple of useful resources.</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e20120a52076f7970c-pi"><img style="border-right-width: 0px; margin: 0px 20px 0px 5px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px" title="labs.autodesk.com" border="0" alt="labs.autodesk.com" align="left" src="/assets/image_423850.jpg" width="176" height="130" /></a> The first is the newly made-over <a href="http://labs.autodesk.com">Autodesk Labs</a> site. You’ll find lots of cool, new technology hosted on the site, including – in the next month or so – the <a href="http://labs.autodesk.com/utilities/ADN_plugins">ADN Plugin of the Month</a>, as mentioned in <a href="http://through-the-interface.typepad.com/through_the_interface/2009/07/coming-soon-to-autodesk-labs-plugin-of-the-month.html">this previous post</a>.</p>  <p>The Labs site is well worth checking out: there’s plenty there that’s genuinely useful as well as stuff that’s just plain fun to play around with. :-)</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e20120a4c934ac970b-pi"><img style="border-right-width: 0px; margin: 0px 5px 0px 20px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px" title="docs.autodesk.com" border="0" alt="docs.autodesk.com" align="right" src="/assets/image_24059.jpg" width="173" height="128" /></a> The second resource is our <a href="http://docs.autodesk.com">online documentation portal</a>. This includes online product documentation (via the “online help” links) for an increasing number of our products.</p>  <p>If it’s familiar to some of you, it’s probably because it’s the infrastructure used for the recently-posted <a href="http://www.autodesk.com/autocad-net-developers-guide">AutoCAD .NET Developer’s Guide</a> (something else mentioned in <a href="http://through-the-interface.typepad.com/through_the_interface/2009/07/autocad-net-developers-guide-now-available-online.html">a recent post</a>).</p>
