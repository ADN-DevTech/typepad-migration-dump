<p>I stumbled across <a href="http://forums.autodesk.com/t5/AutoCAD-2012/AutoCAD-2013-Annotative-Objects-Wish-List/td-p/3169446" target="_blank">this wishlist request on the forums</a>, the other day, and thought it worth covering in a blog post.</p>
<p>It’s an interesting little piece of code – nothing very startling, but obviously whenever removing data you have to be a bit careful: in fact I would probably add an “are you sure?” question prior to actually removing the various annotation scales, even if the operation participates in the undo mechanism and can therefore be rolled back easily by the user.</p>
