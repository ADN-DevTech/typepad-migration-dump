<p>During the AutoCAD team’s efforts to introduce the dark theme in the 2015 version, work was done to more easily support additional themes. A lot of work was needed to create the underlying mechanism but now it’s really easy to add your own themes.</p>  <p>To test the mechanism, the AutoCAD team has created a sample theme that turns AutoCAD into various shades of pink. It works programmatically by adjusting the various UI elements, so you really don’t need much code of your own to implement it.</p>  <p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201a73d9c6091970d-pi" target="_blank"><img alt="Pink theme" border="0" height="281" src="/assets/image_707015.jpg" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; float: none; padding-top: 0px; padding-left: 0px; margin: 20px auto; display: block; padding-right: 0px; border-top-width: 0px" title="Pink theme" width="464" /></a></p>  <p>My 5-year old daughter absolutely loves it – she’s extremely interested in what I do for work, all of a sudden. I wouldn’t go as far as saying it’s a way to increase female representation in <a href="http://en.wikipedia.org/wiki/STEM_fields" target="_blank">STEM</a> fields – there are lots of women who detest pink just as there are plenty of men who like it – but I dare say it won’t hurt.</p>  <p>To make it really easy to install this new theme on your system, <a href="http://apps.exchange.autodesk.com/ACD/en/Detail/Index?id=appstore.exchange.autodesk.com%3aprettyinpink%3aen" target="_blank">we’ve added it to the Exchange Apps Store</a> (it may take a day or two to appear up there – we have a backlog of 2015-compatible apps to publish, it seems).</p>  <p>If you can’t wait for the app to be posted, here’s the C# code that implements it. Once compiled and NETLOADed, simply run the PRETTYINPINK command to enable the new theme.</p>  <p>It should be simple enough to take this code and adjust the various parameters to support a different colour, should you want to see AutoCAD in red or blue, for instance.</p>  <div style="font-size: 8pt;
