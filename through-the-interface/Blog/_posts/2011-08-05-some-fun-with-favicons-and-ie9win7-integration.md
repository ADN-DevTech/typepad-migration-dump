<p>Here’s something that’s a little off-topic – relative to my other posts – but potentially of interest to fellow bloggers, people who have their own websites and even readers who’re using Internet Explorer 9 and Windows 7.</p>
<p>Since attending <a href="http://blogs.msdn.com/b/swiss_dpe_team/archive/2011/04/12/techdays-11-gen-232-ve-screencasts-and-presentations-available-online.aspx" target="_blank">the Microsoft TechDays 11 event in Geneva</a>, back in April, I’ve been meaning to fool around with enabling this site for <a href="http://windows.microsoft.com/en-US/windows7/pin-a-website-to-your-taskbar" target="_blank">pinning to the Windows 7 taskbar</a>, with some special customizations for IE9. And now that I actually have a Windows 7 installation (albeit inside Parallels on my Mac), I’ve been able to do just that.</p>
<p>At the root of customizing IE9 – and ultimately other browsers – is providing a custom <em>favicon.ico</em> file at the root of your site (at least that’s the default location – you can point browsers to look for it in other places, simply enough). Most browsers look for a 16x16 version inside this file, but these days there are advantages to including larger formats, too.</p>
<p>My first step when creating a favicon for this blog was to convert a standard image into an icon. I ended up looking at a number of tools, for this:</p>
<ul>
<li>Online      
<ul>
<li><a href="http://favicon.co.uk" target="_blank">favicon.co.uk</a> </li>
<li><a href="http://www.rw-designer.com/image-to-icon" target="_blank">RealWorld Graphics Online Icon Creator</a> </li>
</ul>
</li>
<li>Offline      
<ul>
<li><a href="http://icofx.ro/" target="_blank">IcoFX</a> </li>
</ul>
</li>
</ul>
<p>I also used Visual Studio 2010 to do some simple manipulation, after the fact. One thing I did notice is that copying &amp; pasting between icons is often very dependent on that icon’s colour palette: I found I needed to save colour palettes for particular icons as .pal files to then load in for other icons (especially when moving between tools), increasing the chances that the colours would come across properly.</p>

The base image was created from the blog banner with a simple image editor (I think I ended up using the one from <a href="http://www.techsmith.com/snagit/" target="_blank">the excellent SnagIt tool from TechSmith</a>):</p>
