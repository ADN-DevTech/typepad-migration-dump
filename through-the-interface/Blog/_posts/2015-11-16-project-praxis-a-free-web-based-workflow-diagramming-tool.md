<p>The <a href="https://knowledge.autodesk.com" target="_blank">Autodesk Knowledge Network</a> team has created a new, web-based tool for diagramming Autodesk product workflows called <a href="https://praxis.autodesk.com" target="_blank">Project PRAXIS</a>. This <u>free</u> tool is currently in pre-release and allows you to document and share product workflows within your organisation or with your customers.</p>
<p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201bb0888f46c970d-pi" target="_blank"><img alt="Project PRAXIS" border="0" height="294" src="/assets/image_862208.jpg" style="background-image: none; float: none; padding-top: 0px; padding-left: 0px; margin: 50px auto; display: block; padding-right: 0px; border: 0px;" title="Project PRAXIS" width="504" /></a></p>
<p>I took it for a spin and quickly documented the Eat, Sleep, Code, Repeat cycle (inspired by Fatboy Slim’s <a href="https://youtu.be/wBoRkg5-Ieg" target="_blank">Eat, Sleep, Rave, Repeat</a>).</p>
<p><a href="http://through-the-interface.typepad.com/.a/6a00d83452464869e201b8d16eed5d970c-pi" target="_blank"><img alt="Eat Sleep Code Repeat" border="0" height="450" src="/assets/image_372494.jpg" style="background-image: none; float: none; padding-top: 0px; padding-left: 0px; margin: 50px auto; display: block; padding-right: 0px; border: 0px;" title="Eat Sleep Code Repeat" width="500" /></a></p>
<p>The tool has been used extensively by our Global Services division, and I’m sure that many external consultants and channel partners will find it&#0160;useful for mapping various&#0160;activities. Give it a try and <a href="http://forums.autodesk.com/t5/project-praxis-general/bd-p/215" target="_blank">let us know what you think</a>!</p>
