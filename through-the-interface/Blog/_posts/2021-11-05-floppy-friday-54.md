<p>In <a href="https://youtu.be/Z6MfkAOoizM" rel="noopener" target="_blank">this week’s episode</a> we play five more games loaded from a single floppy onto a Commodore 64. This week’s selection was actually really interesting, I found: we started with an action adventure called <a href="https://www.mobygames.com/game/zx-spectrum/olli-lissa-the-ghost-of-shilmore-castle" rel="noopener" target="_blank">Olli &amp; Lissa: The Ghost of Shilmore Castle</a>, part of a series that apparently <a href="https://en.wikipedia.org/wiki/Olli_%26_Lissa" rel="noopener" target="_blank">did quite well when released as budget titles</a>. From there we loaded a game called Toddler, which I believe was also released as <a href="https://www.mobygames.com/game/equalizer" rel="noopener" target="_blank">The Equalizer</a> (I guess it sounds more compelling… it’s certainly a much more 80s name!). We loaded a <a href="https://en.wikipedia.org/wiki/Breakout_(video_game)" rel="noopener" target="_blank">Breakout</a> clone called <a href="https://en.wikipedia.org/wiki/Krakout" rel="noopener" target="_blank">Krakout</a>, which had fun floating faces. Next was a rather fun-looking game called <a href="https://www.c64-wiki.com/wiki/Magic_Madness" rel="noopener" target="_blank">Magic Madness</a> – this is definitely one I’d play again. We wrapped up with a game called <a href="https://en.wikipedia.org/wiki/Agent_Orange_(video_game)" rel="noopener" target="_blank">Agent Orange</a>.</p>
