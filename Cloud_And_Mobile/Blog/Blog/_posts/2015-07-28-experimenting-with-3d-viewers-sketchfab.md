<p>By <a href="http://adndevblog.typepad.com/autocad/announcing-apphack-20.html">Stephen Preston</a></p>
<p>I&#39;m putting together some material for a presentation on simple 3D web publishing tools, so I&#39;m going to be using this blog for a few experimental posts. My first test is with SketchFab. First impressions - easy to get a ready made 3D model published on the web, basic settings are simple to control, can add very simple annotations to the model. Embedding is the usual iframe embed model.</p>
<p>This test model is the 3D printed trophy we created for the AngelHack Silicon Valley 2015 hackathon. The trophy was designed using Fusion 360. Read more about it here - http://forums.autodesk.com/t5/design-differently/3d-printed-trophies-with-fusion-360/ba-p/5707177.</p>
