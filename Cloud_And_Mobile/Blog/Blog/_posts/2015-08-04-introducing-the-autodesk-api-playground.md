<p>By&#0160;<a href="http://adndevblog.typepad.com/cloud_and_mobile/philippe-leefsma.html">Philippe Leefsma</a></p>
<p>A new tool is now available in order to quickly test and take a look at our web APIs:</p>
<p><a href="http://www.autodesk.help/" target="_self" title="">Autodesk API Playground</a></p>
<p>Without any registration or credentials needed, users are now able through this portal to try each of the available end points of our public web APIs and see the response they generate.</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d1437e55970c-pi" style="display: inline;"><img alt="Screen Shot 2015-08-04 at 10.11.29 AM" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d1437e55970c image-full img-responsive" src="/assets/image_6cf15e.jpg" title="Screen Shot 2015-08-04 at 10.11.29 AM" /></a></p>
<p>This is a work in progress, so not every feature is available yet. For example at the time of this writing AutoCAD IO is not enabled and you can&#39;t view directly your models in the viewer but we are getting there...</p>
<p>Definitely a link you should bookmark if you are using Autodesk web APIs and are willing to save some time with your developments.</p>
