<p>By <a href="http://adndevblog.typepad.com/cloud_and_mobile/daniel-du.html">Daniel Du</a></p> <p>If you have not heard of View and Data API, here is the idea, the View &amp; Data API enables web developers to very easily display 3D (and 2D) models on a WebGL-enabled browser. please read <a href="http://adndevblog.typepad.com/cloud_and_mobile/2015/01/what-is-this-view-data-api-thing-anyway.html">this one</a> first and get a key from <a href="http://developer.autodesk.com">http://developer.autodesk.com</a> and start playing with the API. In this blog post, I will paste some code snippet of making the viewer into full screen mode. Hope it is helpful :</p> <p>Following code snippet make the viewer full browser. </p><pre style="font-size: 13px; font-family: consolas; background: white; color: black">$(<span style="color: #a31515">&#39;#btnFullBrowser&#39;</span>).click(<span style="color: blue">function</span> () {<br /> <br />&#0160;&#0160;&#0160; <span style="color: blue">var</span> vsmd = <span style="color: blue">new</span> Autodesk.Viewing.ViewerScreenModeDelegate(viewer);<br />&#0160;&#0160;&#0160; <span style="color: blue">var</span> oldMode = vsmd.getMode();<br />&#0160;&#0160;&#0160; console.log(oldMode);<span style="color: green">//kFullScreen, kFullBrowser, kNormal</span><br /> <br />&#0160;&#0160;&#0160; <span style="color: blue">if</span> (vsmd.isModeSupported(Autodesk.Viewing.Viewer<br />                .ScreenMode.kFullBrowser)) {<br />&#0160;&#0160;&#0160;&#0160;&#0160;&#0160;&#0160; <span style="color: blue">var</span> newMode = Autodesk.Viewing.Viewer.ScreenMode.kFullBrowser;<br />&#0160;&#0160;&#0160;&#0160;&#0160;&#0160;&#0160; vsmd.doScreenModeChange(oldMode, newMode)<br />&#0160;&#0160;&#0160;&#0160;&#0160;&#0160;&#0160; <span style="color: green">//vsmd.setMode(newMode);</span><br /> <br />&#0160;&#0160;&#0160; }<br />&#0160;&#0160;&#0160; <span style="color: blue">else</span> {<br />&#0160;&#0160;&#0160;&#0160;&#0160;&#0160;&#0160; console.log(<span style="color: #a31515">&#39;ScreenMode.kFullBrowser not supported&#39;</span>);<br />&#0160;&#0160;&#0160; }<br /> <br /> <br />});<br /></pre>
<p>What exactly “full browser” means? it hides any other html elements and enlarges the viewer to make it fulfill the whole browser in current tab. For example, I have a test application like below, please note that I have a title in my application, and some buttons, also note that this application is just one of many browser tabs. </p>
