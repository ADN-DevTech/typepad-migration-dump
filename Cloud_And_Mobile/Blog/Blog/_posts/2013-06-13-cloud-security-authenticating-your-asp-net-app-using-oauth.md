<p>By <a href="http://adndevblog.typepad.com/cloud_and_mobile/gopinath-taget.html">Gopinath Taget</a></p>  <p><a href="http://en.wikipedia.org/wiki/OAuth">OAuth</a> is one of the popular Authentication protocols that is used to provides a layer of security to your web applications using login/password. One of the main uses of OAuth is to allow access to one website using Authentication with another. </p>  <p>For instance, let us say you have a custom website with its own login and password (<em>As a side note, the infrastructure for Authentication is referred to as </em><a href="http://msdn.microsoft.com/en-us/library/yh26yfzy(v=vs.100).aspx"><em>membership</em></a><em> in <a href="https://en.wikipedia.org/wiki/ASP.NET">ASP .NET</a>. You will also find information about it </em><a href="http://msdn.microsoft.com/en-us/library/6e9y4s5t(v=vs.100).aspx"><em>here</em></a><em> among many other places on the web. <a href="http://www.troyhunt.com/2011/10/5-minute-wonders-aspnet-membership.html">Here</a> is another video talking about it.</em>). </p>  <p>Back on the topic, now let us say you would like to give access to a user of this website using the user’s facebook login. You can do it using OAuth. i.e., you can let users login to your website with their facebook account. Under the hood, your facebook account is associated with the users local membership account.</p>  <p>How difficult is this to implement? With ASP .NET 4.5, it is ridiculously easy. If you want to authenticate your ASP .NET application with users’ <a href="https://www.facebook.com/">Facebook</a>, <a href="https://twitter.com/">Twitter</a>, <a href="microsoft.live.com">Microsoft</a> or <a href="http://www.google.com">Google</a> accounts have. <a href="http://www.youtube.com/watch?v=bLyCHhm2H0g">Here</a> is a video about using it with ASP .NET Forms application and <a href="http://www.codeproject.com/Articles/577384/Introduction-to-OAuth-in-ASP-NET-MVC">here</a> is a <a href="http://www.codeproject.com/">codeproject</a> article about using it with <a href="http://en.wikipedia.org/wiki/ASP.NET_MVC_Framework">MVC</a>. It is as simple as calling a single method for each of the services:</p>  <p><font color="#45a6cb">OAuthWebSecurity</font>.RegisterMicrosoftClient(     <br />&#160;&#160;&#160; clientId: &quot;<font color="#804000">123456789123456</font>&quot;,     <br />&#160;&#160;&#160; clientSecret: &quot;<font color="#800000">1234567892188a3456bq44535pk34rg1</font>&quot;);</p>  <p><font color="#45a6cb">OAuthWebSecurity</font>.RegisterTwitterClient(     <br />&#160;&#160;&#160; consumerKey: &quot;<font color="#804000">123456789123456</font>&quot;,     <br />&#160;&#160;&#160; consumerSecret: &quot;<font color="#800000">1234567892188a3456bq44535pk34rg1</font>&quot;);</p>  <p><font color="#45a6cb">OAuthWebSecurity</font>.RegisterFacebookClient(     <br />&#160;&#160;&#160; appId: &quot;<font color="#804000">123456789123456</font>&quot;,     <br />&#160;&#160;&#160; appSecret: &quot;<font color="#800000">1234567892188a3456bq44535pk34rg1</font>&quot;);</p>  <p><font color="#45a6cb">OAuthWebSecurity</font>.RegisterGoogleClient();</p>  <p>What if you want to authenticate your web application with a provider other than facebook, twitter etc? You can still do it but it needs a little more work. <a href="http://www.codeguru.com/columns/experts/implementing-oauth-features-in-asp.net-mvc-4.htm">Here</a> is an article that shows how to do this.     </p>
