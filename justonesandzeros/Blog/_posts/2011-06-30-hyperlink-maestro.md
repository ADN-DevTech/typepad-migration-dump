<p><strong>Update:&#0160; Hyperling Maestro has been updated for Vault 2014 by a third-party and posted on <a href="http://apps.exchange.autodesk.com/VLTC/en/Detail/Index?id=appstore.exchange.autodesk.com%3admhyperlinkmaestro_windows64%3aen" target="_blank">the Vault app store</a>.</strong></p>
<p>Link Maestro had too much coffee.&#0160; Now he&#39;s all hyper.   <br />Sorry, the jokes aren&#39;t going to get much better than that.&#0160; This is a corporate blog about API programming on a data management product.&#0160; <em>I don&#39;t have much to work with here</em>.</p>
<p>Anyway, I&#39;ve updated Link Maestro to work with Vault 2012.&#0160; The &quot;Copy Link&quot; command in Vault 2011 has been renamed to &quot;Copy Hyperlink&quot; in Vault 2012, so my app is now called Hyperlink Maestro.</p>
<p>This version contains all the cool features from <a href="http://justonesandzeros.typepad.com/blog/2011/01/link-maestro.html">version 1</a> (web client hyperlinks, multisite hyperlinks, etc).&#0160; Plus, I added a new feature:&#0160; <strong>HTML formatted hyperlinks</strong>.</p>
<p>If you paste into something that support HTML formatting, such as Microsoft Outlook or Word, it will show up much nicer.&#0160; Here are some examples of how the HTML formatted hyperlinks look better than the text formatted hyperlinks.</p>
