<p><strong>Features:</strong></p>
<ul>
<li><strong>Backup user settings</strong> - A quick way to zip up all your Vault user-settings. </li>
<li><strong>Create admin deployments</strong> - Select the components you want to deploy and they will be pushed to all other users with Thunderdome.       <br />Components include: User settings, saved searches, plug-in, <a href="http://justonesandzeros.typepad.com/blog/2012/07/deco-custom-ui-without-custom-code.html" target="_blank">DECO</a> files, and <a href="http://justonesandzeros.typepad.com/blog/2012/03/vlogic-2013.html" target="_blank">vLogic</a> scripts. </li>
</ul>
