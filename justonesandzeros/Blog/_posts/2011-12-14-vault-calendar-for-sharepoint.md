<p>For the millions and millions of people using the Vault integration for SharePoint 2010, here is a utility that you might find useful.&#0160; It’s a calendar web part that shows you when your Vault ECOs are due.</p>
<p><a href="http://justonesandzeros.typepad.com/images/2011/VaultCalendar/calendar_full.png" target="_blank"><img alt="" src="/assets/calendar_scaled.png" /></a></p>
<p><span style="color: #a5a5a5;">(click for full image)</span></p>
<p>The data can be driven from any Vault Change Order list in SharePoint.&#0160; Clicking on the calendar entry brings up the detail view for that ECO.</p>
<p><img alt="" src="/assets/detailView.png" /></p>
<hr noshade="noshade" style="color: #013181;" />
<p><strong>Requirements:</strong></p>
<ul>
<li>Vault Professional 2012 </li>
<li>SharePoint 2010 Standard or Enterprise </li>
<li><a href="http://underthehood-autodesk.typepad.com/blog/2011/06/new-videos-autodesk-vault-and-microsoft-sharepoint-2010-integration.html">The Vault integration for SharePoint 2010</a></li>
</ul>
<p>NOTE:&#0160; The SharePoint administrator needs to deploy and enable the web part before it can be used.&#0160; See readme file for instructions.</p>
