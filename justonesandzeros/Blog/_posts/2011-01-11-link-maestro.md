<p><strong>Update:</strong>&#0160; There is a newer version for <a href="http://justonesandzeros.typepad.com/blog/2011/06/hyperlink-maestro.html" target="_self">Vault 2012</a>.</p>
<p>Link Maestro is a rewrite of the Copy Link command.&#0160; In addition to the default functionality, Link Maestro adds the ability to link to a page in the Vault web client.&#0160; It can also do multi-site links, which is where each Vault site gets it&#39;s own link.&#0160; For Vault Professional users, you can now create Item and Change Order links.</p>
<p><img alt="" src="/assets/Menu.png" /></p>
<p><strong>Requirements:      <br /></strong>- Vault Workgroup/Collaboration/Professional 2011     <br />Some features may not work depending on the Vault product.</p>
<p><a href="http://justonesandzeros.typepad.com/Apps/LinkMaestro/LinkMaestro-1.0.1.0-bin.zip" target="_blank">Click here to download the application</a> <br /><a href="http://justonesandzeros.typepad.com/Apps/LinkMaestro/LinkMaestro-1.0.1.0-src.zip" target="_blank">Click here to download the source code</a></p>
<p><strong>To use:      <br /></strong>Run the install and start Vault Explorer.&#0160;</p>
<p>Under the Edit menu, select Configure Links.&#0160; A dialog will pop up allowing you to select what link format you want for each object type.<br /><img alt="" src="/assets/Settings.png" /></p>
<p>Multi-site links require that an administrator first set up the list of Vault sites.&#0160; <br /> <img alt="" src="/assets/multisite-settings_scaled.png" /></p>
<p>&#0160;</p>
<p><strong>Using Web Browser links:      <br /></strong>I do not recommend storing links to web client pages.&#0160; These links are not designed for long term use.&#0160; First, they link to a specific version.&#0160; So as time goes on, the link will go out of date.&#0160; Or the version could get purged, which will break the link.&#0160; Next, the link syntax may change in future releases.    <br /> <br />Web browser links work best for short term use, such as sending an email to a colleague with a download link.    <br /><img alt="" src="/assets/LinkExample_scaled.png" /></p>
