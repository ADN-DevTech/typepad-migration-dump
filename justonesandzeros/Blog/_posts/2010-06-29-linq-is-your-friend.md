<p><img src="/assets/TipsAndTricks2.png" /> </p> <p>When programming with Vault, you end up dealing with arrays a lot.&#0160; I find myself constantly looping through arrays trying to find the 1 object I am looking for.&#0160; Or I have an array of objects that I need to convert to an array of IDs.</p> <p>.NET 3.0 has a nice feature, LINQ, that makes these operations much easier than creating foreach loops all the time.&#0160; It&#39;s basically a query syntax for .NET collections.&#0160; If you are a fan of SQL, then you should have no trouble picking up LINQ.&#0160; Even if you are unfamiliar with SQL, the syntax is pretty easy to work with.&#0160;&#0160;
