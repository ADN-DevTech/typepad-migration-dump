<p>Revit の新バージョンとなる、Revit 2024 がリリースされました。まずは、概要をご紹介したいと思います。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b7517a4339200b-pi" style="display: inline;"><img alt="Revit2024-01-01" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b7517a4339200b image-full img-responsive" src="/assets/image_327901.jpg" title="Revit2024-01-01" /></a></p>
<hr />
<p><strong>Autodesk Revit 2024 の動作環境</strong></p>
<p>動作要件として、Revit 2023 に引き続き 64 ビット版 Windows 10 および 64 ビット版 Windows 11 をサポートしています。また、対応する.Net Framework は、.NET Framework Version 4.8 以降となります。</p>
<p>詳細なシステム要件については、オンラインドキュメントの以下のページをご参照ください。</p>
<p><a href="https://www.autodesk.com/support/technical/article/caas/sfdcarticles/sfdcarticles/System-requirements-for-Revit-2024-products.html" title="System requirements for Revit 2024 products">System requirements for Revit 2024 products</a></p>
<ul>
<li>推奨メモリ
<ul>
<li>&#0160;ドキュメント内に記載がありますが、推奨されるメモリに変更点がありますので、ご確認ください。
<ul>
<li>
<p>最小:エントリレベルの構成</p>
<p>Revit 2023: 8 GB の RAM <br /><strong>Revit 2024: 16 GB の RAM</strong></p>
</li>
<li>
<p>バリュー:価格と性能の両立</p>
<p>Revit 2023: 16 GB の RAM <br /><strong>Revit 2024: 32 GB の RAM</strong></p>
</li>
<li>
<p>パフォーマンス:大規模で複雑なモデル</p>
<p>Revit 2023: 32 GB の RAM <br /><strong>Revit 2024: 64 GB の RAM</strong></p>
</li>
</ul>
</li>
</ul>
</li>
</ul>
<hr />
<p><strong>Family テンプレート</strong></p>
<p>Revit 2024 をインストール後に日本語のファミリテンプレートが欠落している問題についてのご報告をいただいております。</p>
<p>Revit 2023 と比較するとテンプレート数が大幅に減少しており、詳細が分かり次第、こちらのブログ記事でもご案内させていただきます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b7519e9988200c-pi" style="display: inline;"><img alt="Revit2024-01-02" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b7519e9988200c image-full img-responsive" src="/assets/image_184662.jpg" title="Revit2024-01-02" /></a></p>

<p><strong>Autodesk Access</strong></p>
<p><a href="https://adndevblog.typepad.com/technology_perspective/2023/03/new-features-on-autocad-2024-part1.html">AutoCAD 2024 の記事</a>をご参照いただいた方は、既にご存知かと思いますが、Revit 2024 をインストールすると、これまでデスクトップに登録されていた Autodesk デスクトップ アプリではなく、Autodesk Access というアプリケーションがインストールされます。</p>
<p>Autodesk Access は、これまでの Autodesk デスクトップ アプリと同様に、インストールされている各製品のアップデートを知らせし、アプリから素早く簡単にインストールできます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b7517a434c200b-pi" style="display: inline;"><img alt="Revit2024-01-03" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b7517a434c200b image-full img-responsive" src="/assets/image_583935.jpg" title="Revit2024-01-03" /></a></p>
<p>今後、製品のインストールとアップデートをより速く、より簡単に、より信頼できるものにするために、時間をかけて新機能を追加していく予定です。</p>
<p>詳細につきまして、下記のページをご参照ください。</p>
<p><a href="https://www.autodesk.com/products/autodesk-access/overview">Autodesk Access: Easily keep your desktop products up to date</a></p>
