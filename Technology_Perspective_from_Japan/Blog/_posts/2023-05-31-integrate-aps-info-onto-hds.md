<p>ご存じのとおり、オートデスクのクラウド サービスの稼働状態は、<a href="https://health.autodesk.com/" rel="noopener" target="_blank">ヘルス ダッシュボード サイト（https://health.autodesk.com/）</a> で確認することが出来ます。昨年、ブランディングの変更でサイト デザインが一新した際には、<a href="https://adndevblog.typepad.com/technology_perspective/2022/02/new-health-dashboard.html" rel="noopener" target="_blank"></a>インシデント発生時に Subscribe すると、その後の状況を追跡することも出来ます。（<a href="https://adndevblog.typepad.com/technology_perspective/2022/02/new-health-dashboard.html" rel="noopener" target="_blank"> 新しくなったヘルス ダッシュボード</a> の記事参照）</p>
<p>このヘルス ダッシュボードですが、少し前から、クラウドサービスだけでなく、Autodesk Platform Services の API の稼働状況も表示するように改良が続けられています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b685386f5f200d-pi" style="display: inline;"><img alt="Aps_on_hds" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b685386f5f200d image-full img-responsive" src="/assets/image_937210.jpg" title="Aps_on_hds" /></a></p>
<p>すべての API が記載対象になっているわけではありませんが、徐々にですが、今後も主要 API の記載追加が計画されています。</p>
<p>また、<a href="https://aps.autodesk.com/" rel="noopener" target="_blank">APS ポータル（https://aps.autodesk.com/）</a>で「API Status」を選択すると、<a href="https://health.autodesk.com/#/APS" rel="noopener" target="_blank">https://health.autodesk.com/#/APS</a> が参照されて、APS の稼働状態だけを表示するヘルス ダッシュボードを表示させることが出来ます。</p>
