<p><a href="https://aps.autodesk.com/" rel="noopener" target="_blank">Autodesk Platform Services ポータル</a> の <a href="https://aps.autodesk.com/developer/documentation" rel="noopener" target="_blank">API ドキュメント&#0160; ページ</a>が刷新されて、よりユーザ フレンドリなエクスペリエンスを享受できるようになりました。</p>
<p>新しいページでは、すべてのオートデスク クラウドとデスクトップ製品に関する API ドキュメントを見つけることが出来るようになっています。ドキュメント自体は従来通り英語になってしまいますが、すべてのデスクトップ製品開発者、およびクラウド開発者リソースを単一のポータルに集約することで、開発者エコシステムも統合していくという狙いがあります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d39e2762200b-pi" style="display: inline;"><img alt="New_landing_page" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d39e2762200b image-full img-responsive" src="/assets/image_731749.jpg" title="New_landing_page" /></a></p>
<p>新しいドキュメント ページ左手には、デスクトップとクラウド API をフィルタするチェックボックスの他、業界（Industry）や機能（Capability）、開発言語（Language）を指定するトグルボタンが用意されているので、適切にフィルタすることで、目的にあったドキュメント、API や SDK、情報にたどり着くことが出来るはずです。&#0160;</p>
<p>By Toshiaki Isezaki</p>
