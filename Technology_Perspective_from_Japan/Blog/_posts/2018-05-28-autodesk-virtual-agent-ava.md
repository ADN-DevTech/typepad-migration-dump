<p>以前、<strong><a href="http://adndevblog.typepad.com/technology_perspective/2017/05/connected-world-with-web-service-api.html" rel="noopener noreferrer" target="_blank">Web サービス API で繋がる世界 </a></strong>でご紹介したことがある&#0160;<strong>Autodesk Virtual Agent</strong>（オートデスク バーチャル エージェント、<strong><a href="https://ava.autodesk.com/" rel="noopener noreferrer" target="_blank">https://ava.autodesk.com/</a></strong>）、通称、<strong>AVA </strong>が着々と進化しつつあります。</p>
<p>まだ、ベータ版の扱いではありますが、今回、従来の静的なテキストベースの&#0160;<strong><a href="https://ja.wikipedia.org/wiki/%E4%BA%BA%E5%B7%A5%E7%84%A1%E8%84%B3" rel="noopener noreferrer" target="_blank">チャットボット</a>&#0160;</strong>形態から、デジタル ヒューマンと呼ばれる形態になり、テキスト、音声、ビデオの任意の組み合わせでオートデスク製品に関連する疑問に回答するようになっています。</p>
<p>もちろん、また英語だけの対応ですが、一部、日本語の Autodesk Knowledge Network 記事と連動する内容もあります。また、<strong><a href="http://adndevblog.typepad.com/technology_perspective/2018/03/only-one-autocad.html" rel="noopener noreferrer" target="_blank">Only One AutoCAD - 業種別 AutoCAD 製品の提供方法変更</a></strong> でご案内した内容も含め、インストールやライセンスの問題、旧製品に対する質問など、以前より対象領域が増えています。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b0224df35e6cd200b-pi" style="display: inline;"><img alt="Ava" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0224df35e6cd200b image-full img-responsive" src="/assets/image_453845.jpg" title="Ava" /></a></p>
<p>ベータ版 AVA をお試しいただくには、最初に Autodesk ID で <strong><a href="https://ava.autodesk.com/" rel="noopener noreferrer" target="_blank">https://ava.autodesk.com/</a></strong> にサインインして利用規約に同意する必要がありますが、お試しいただくと製品サポートの将来像を垣間見ることが出来るかと思います。</p>
<p>By Toshiaki Isezaki</p>
