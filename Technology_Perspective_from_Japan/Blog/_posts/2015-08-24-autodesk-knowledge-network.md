<p>オートデスク製品上の操作方法や技術的な問題を自己解決する日本語 &#0160;FAQ サイトとして、<strong>Autodesk Technical Q&amp;A</strong> と称して専用ページが用意されています。もちろん、現在も、<strong><a href="http://www.autodesk.co.jp/tech-faq" target="_blank">http://www.autodesk.co.jp/tech-faq</a></strong> から &#0160;Autodesk Technical Q&amp;A を利用することが可能です。</p>
<p>今回、同様の目的で用意されていた英語版 <strong>Autodesk knowledge Network</strong>&#0160;に日本語コンテンツが登場しました。日本語の Autodesk Knowledge Base には、<a href="http://knowledge.autodesk.com/ja/support" target="_blank"><strong>http://knowledge.autodesk.com/ja/support</strong></a> から直接アクセスすることが出来ます。</p>
<p>また、米国オートデスクの Web サイトから、英語の &#0160;Autodesk Knowledge Base ページにアクセスした場合には、画面上に日本語サイトへの表示を尋ねるバナーが表示されます。例えば、AutoCAD の製品ページ&#0160;<a href="http://www.autodesk.com/autocad" target="_blank">http://www.autodesk.com/autocad</a>&#0160;から、画面上部の「Support &amp; Learning」リンクをクリックした場合、AutoCAD 用の&#0160;Autodesk Knowledge Base ページが英語で表示されます。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c7c36ff7970b-pi" style="display: inline;"><img alt="Akn_on_autodesk_com1" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c7c36ff7970b image-full img-responsive" src="/assets/image_755245.jpg" title="Akn_on_autodesk_com1" /></a></p>
<p>表示された英語版の&#0160;Autodesk Knowledge Base ページには、次のような表示を確認することが出来るはずです。ここで「<strong>はい、日本語のサイトに移動します</strong>」をクリックすることで、日本語版の&#0160;Autodesk Knowledge Base ページを表示させることが出来るはずです。&#0160;</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c7c36ffc970b-pi" style="display: inline;"><img alt="Akn_on_autodesk_com2" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c7c36ffc970b image-full img-responsive" src="/assets/image_561321.jpg" title="Akn_on_autodesk_com2" /></a></p>
<p>まだ、自動翻訳で公開されるコンテンツが多い状態ですが、始めから日本語で作成されたコンテンツも徐々に増えていくはずです。 今後、API カスタマイズに関する FAQ も、Autodesk Knowledge Base ページに移行していく予定です。なお、当面の間、<strong>Autodesk Technical Q&amp;A</strong>&#0160;との併用をしていく予定です。</p>
<p>Autodesk Technical Q&amp;A も&#0160;Autodesk knowledge Network 上のコンテンツは、オートデスクのオンライン ヘルプ内からだけではなく、通常の&#0160;Web ブラウザによるキーワード検索でコンテンツをヒット出来るようになっています。ぜひ、ご利用ください。</p>
<p>By Toshiaki Isezaki</p>
