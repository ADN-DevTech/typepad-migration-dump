<p>Autodesk Inventor 2017 の日本語バージョンの発表に伴い、Autodesk Inventor 2017 APIトレーニング教材一式「APIトレーニングテキスト」「APIトレーニング実習ガイドと演習＆回答」及び 「演習と実習用ファイル」を公開いたします。</p>

<p>公開されるコンテンツは、Autodesk Inventor 2017 APIトレーニングテキスト（.ppt P.368）を pdf ファイルに変換した "Inventor2017API .Net Training_J.pdf"、テキスト内の演習問題と各開発ガイド（実習）を説明した「演習と実習ヘルプファイル」の"Inventor2017API Exercise.chm"、「演習と各開発ガイド（実習）の回答」( 14の演習回答 と 80の実習の回答) をそれぞれVBAマクロファイルとVisual Studio 2015(VB.Net)のプロジェクトとしたファイル群、また「演習と各開発ガイド（実習）で使用するInventorファイル群」の公開となります。<br />

<p>任意のホルダーに　Inventor2017API_TrainigDoc.zip , Inventor2017API_LabDoc.zip , Inventor2017API_Labs.zip ,Inventor2017API_LabDrawings.zip　をダウンロード後にそれぞれ解凍をしていただき、以下のホルダー構造のように、Cドライブ直下に"Inventor API Training"ホルダーを移動配置いただきご利用下さい。<br />
（解凍していただくと、解凍ホルダー直下に\Inventor API Trainingホルダーが作成され中に\Documents,\Drawings ,\Lab Instructions,\Training Labs ホルダーが含まれます）<br />

<p>\Drawingsホルダー<br />
　演習と実習（Inventor2017API Exercise.chm）で使用するInventorファイル群</p>

<p>\Lab Instructionsホルダー<br />
　演習と実習解説chm（Inventor2017API Exercise.chm）</p>

<p>\Training Labsホルダー<br />
　演習の回答 ( 演習1～14 )　実習の回答( 80 )<br />
  (回答並びに解説は、全て上記の Cドライブのホルダー構造でハードコートされています)</p>

尚、アドイン作成には Visual Studio 2015 有償版(Professionalなど)の開発環境が必要となります。<br />
