<p>日本時間の先週金曜日、米国時間の 8月27日に、オートデスクは<a href="http://www.seecontrol.com/" target="_blank"><strong>&#0160;SeeControl 社</strong></a>の買収合意について<a href="http://news.autodesk.com/press-release/corporate-sustainability/autodesk-signs-agreement-acquire-seecontrol" target="_blank">発表</a>しました。&#0160;SeeControl 社は、サンフランシスコを拠点に持ち、クラウド ベースの IoT プラットフォームを提供する企業です。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d1502813970c-pi" style="display: inline;"><img alt="Seecontrol_acquisition" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d1502813970c img-responsive" src="/assets/image_515631.jpg" title="Seecontrol_acquisition" /></a></p>
<p>SeeControl プラットフォームの特徴は、IoT 機器として接続されたセンサーなどを監視、管理して、一括表示や解析、管理をおこなうダッシュボードを、プログラミングなしで構築できる点です。既定値で<a href="http://www.seecontrol.com/applications/" target="_blank">テンプレート</a>が用意されているので、比較的短い時間でダッシュボードの構築を完了することが可能なようです。</p>
<p style="text-align: center;"><iframe allowfullscreen="" frameborder="0" height="281" src="https://www.youtube.com/embed/NbLiMIX-bCU?feature=oembed" width="500"></iframe>&#0160;</p>
<p>開発者向けにも、REST API と SOAP API が用意されているようなので、今後の展開次第では、SeeControl プラットフォームを利用したカスタマイズなども考えられるようになるかも知れません。</p>
<p>買収完了後、今後のプランを含めた詳しい情報が公開されるものと思います。情報がアップデートされた時点で、またご案内したいと思います。</p>
<p>By Toshiaki Isezaki</p>
