<p>API の目的を反映する目的で、従来、Premium Reporting API と呼んでいた名称を <strong>Business Success Plan Reporting API</strong> に変更しました。</p>
<p><strong>Business Success Plan Reporting API</strong>（旧名 Premium Reporting API）は、従来通り、Premium プランのサブスクリプション契約下で使用される、オートデスク製品の使用状況分析とレポートを得るための API を提供します。</p>
<h4>開発者向けの情報</h4>
<p>エンドポイントに変更はありませんので、Premium Reporting API 時の作成したアプリは、以前と同じように機能します。</p>
