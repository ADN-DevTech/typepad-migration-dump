<div><strong>Design Automation API for Fusion</strong> のベータ プログラムが開始されました。定評のある&#0160;<a  _istranslated="1" href="https://help.autodesk.com/view/fusion360/ENU/?guid=GUID-A92A4B10-3781-4925-94C6-47DA85A4F65A" rel="noopener" target="_blank" title="Fusion Developer API">Fusion API </a>の活用領域をデスクトップからクラウドに移行することで、他の Design Automation API エンジンと同様の従量課金制演算サービスとして、7,000 を超える詳細な関数へのアクセスを提供します。</div>
<div>&#0160;</div>
<div>Fusion の強力な設計および製造機能を API を介して Web にもたらし、ユーザー、ハードウェアおよびソフトウェア パートナー、および開発者に可能性の世界を解き放ちます。スケッチ、モデリング、メッシュ編集から、積層造形や切削造形 CAM、2D および 3Dネ スティングまで、API の能力は Fusion が提供する設計および製造ワークフローのほぼ全体に及びます。</div>
<div class="clearfix text-formatted field field--name-field-body field--type-text-with-summary field--label-hidden field__item">
<h4>Design Automation API とは？</h4>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02e860e3da51200b-pi" style="display: inline;"><img alt="Da4f" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02e860e3da51200b img-responsive" src="/assets/image_438494.jpg" title="Da4f" /></a></p>
<p style="padding-left: 40px;">既に多くの建築家、エンジニア、デザイナーに使用されている <a  _istranslated="1" href="https://aps.autodesk.com/design-automation-apis" rel="noopener" target="_blank" title="Design Automation API">Design Automation API</a> は、AutoCAD、Revit、Inventor、3ds Max エンジンを従量課金制のクラウド サービスとして提供しています。基本的に、デスクトップ版の製品でスクリプト（アドイン、プラグイン）を使って作成出来るほぼすべてのタスクは、Design Automation API を使用して大規模にデプロイ出来るようになっています。</p>
<p style="padding-left: 40px;">Design Automation API 環境はクラウド ベースであるため、作業をより迅速かつ確実に完了でき、ビジネス システムとのシームレスな統合が可能です。これにより、プロセスを加速し、柔軟性を向上させ、新たな機会を解き放つエンドツーエンドのワークフローを実現出来ます。</p>
<h4>Fusion の Design Automation API とは？</h4>
<p style="padding-left: 40px;">Design Automation API for Fusion は、この強力な概念を Fusion に拡張し、Fusion のデザインと製作のプロセスのほとんどを自動化出来るようにします。Fusion スクリプトを Design Automation for Fusion に提出（登録）するだけで、Fusion Team に保存されているデータを使用して、クラウド上に用意された仮想環境で特定のタスクを実行します。</p>
<p style="padding-left: 40px;">時間のかかる反復的なタスクをデスクトップ版の Fusion からクラウドに移行したり、Fusion の機能を PLM や MES のワークフローにシームレスに統合したりですることが可能になるわけです。</p>
<h4>主な機能</h4>
<div class="video-embed-field-provider-youtube video-embed-field-responsive-video" style="text-align: center;"><iframe allowfullscreen="" frameborder="0" height="270" src="//www.youtube.com/embed/MxbrucvG9XU" width="480"></iframe></div>
<div class="video-embed-field-provider-youtube video-embed-field-responsive-video">&#0160;</div>
<div class="video-embed-field-provider-youtube video-embed-field-responsive-video" style="padding-left: 40px;">Design Automation API for Fusion は、Fusion インダストリー クラウドの基盤となるコンポーネントであり、サードパーティ アプリ統合のための強力なプラットフォームを提供します。主な機能は次のとおりです。</div>
<ul>
<li><strong>Fusion API の完全サポート:</strong><br />Design Automation API for Fusion は、デスクトップ版 Fusion の Fusion API を包括的にサポートします。現在、デスクトップ版 Fusion で自動化出来るほぼすべてのタスクを、クラウド上のエンジン（GUI、グラフィカル ユーザー インターフェイスのない Fusion）で実行出来ます。<br />&#0160;</li>
<li><strong>Fusion Team データとのシームレスな統合</strong>:<br />Design Automation API for Fusion を介して実行される自動化タスクでは、Fusion Team に保存されたデータに直接アクセスすることが出来ます。ネイティブな Fusion のデザイン データを読み取って解析、新しいデザイン、アセンブリ、製造セットアップを変更または作成後、結果を Fusion Team ハブに直接保存出来るため、プロジェクト データが最新に維持、適切なバージョン管理でアクセス可能なことを保証します。<br />&#0160;</li>
<li><strong>Autodesk Platform Services（APS）に接続</strong>:<br />Design Automation API for Fusion を、シームレスなファイル変換のための CAD トランスレータでもある Model Derivative API、自動化されたワークフローを実現する Webhooks API、高度な視覚化のための Viewer など、他の Autodesk Platform Services API と共に活用できます。<br />&#0160;</li>
<li><strong>スケーラブルな並列処理</strong>:<br />Design Automation API for Fusion では、複数の仮想環境を利用した自動化ジョブ（WorkItem）を同時に実行可能なため、通常、デスクトップ コンピュータ上で順に実行する必要があるタスクをクラウドで並列処理出来るようになり、効率が大幅に向上します。<br />&#0160;</li>
<li><strong>セキュアで信頼性の高い実行</strong>:<br />セキュリティは中核的な優先事項です。Design Automation API for Fusion は、スクリプト言語として TypeScript を使用し、開発中はローカル環境で、開発完了後にはクラウド環境で、すべてのスクリプトを安全なサンドボックス環境で実行します。</li>
</ul>
<h4>使用例の考察</h4>
<div class="video-embed-field-provider-youtube video-embed-field-responsive-video" style="text-align: center;"><iframe allowfullscreen="" frameborder="0" height="270" src="//www.youtube.com/embed/N-lDHDbBOw0" width="480"></iframe></div>
<div class="video-embed-field-provider-youtube video-embed-field-responsive-video" style="text-align: center;">&#0160;</div>
<div class="video-embed-field-provider-youtube video-embed-field-responsive-video" style="text-align: center;">Design Automation API for Fusion の特徴を考えると、同 API を利用するアプリの可能性は、次のような非常に多様です。</div>
<ol>
<li><strong>自動化主導のワークフローの強化</strong><br />Design Automation for Fusion を PLM、PDM、または MES ワークフローに統合して、デザインの編集プロセス調整を自動化し、運用効率を向上させます。<br />&#0160;</li>
<li><strong>CAM オペレーション</strong><br />&#0160;Design Automation for Fusionを使用して CAM プログラミングを自動化し、より迅速で正確な部品見積もりを可能にして販売プロセスを合理化します。<br />&#0160;</li>
<li><strong>AI と LLM の統合</strong>&#0160;<br />Web ベースの API として、Fusion インダストリー クラウドは、最先端の大規模言語モデル（LLM）を含む AI 統合に最適です。これにより、デジタル インテリジェンスと現実世界の製造オペレーションを橋渡しする AI 主導の自動化が可能になります。<br />&#0160;</li>
<li><strong>時間のかかるタスク</strong><br />ツールパス生成、ポスト プロセッシング、フラット パターン ネスティング、または積層 3D パッキングなどの計算負荷の高いタスクを移行することで、貴重な時間を節約できます。<br />&#0160;</li>
<li><strong>製品ファミリーのカスタマイズ</strong><br />製品ファミリーのデザイン ポートフォリオのカスタマイズを自動化し、一貫性を確保し、手作業を減らします。<br />&#0160;</li>
<li><strong>効率的なバッチ処理</strong><br />デザイン ファイルの調整、パラメータの更新、データ分析、製造計画のためのバッチ処理を効率化します。&#0160;</li>
</ol>
