<p>8月29日に開催する Autodesk University Japan では、昨年と同様にカスタマイズ トラックとして ADN が主催するセッションを4つ開催することになっています。今回は、オートデスクのエコシステムを構成する Autodesk Exchange Apps ストアの実例や、3D プリントの可能性やビジネスモデル、そして、クラウド サービスのカスタマイズを可能にする Web サービス API の紹介など、盛りだくさんです。</p>
<p>開催要項とお申込みは、<a href="http://au.autodesk.co.jp/2014/tc/adn01/">http://au.autodesk.co.jp/2014/tc/adn01/</a>&#0160;にてご確認ください。</p>
<p>&#0160;</p>
<p><span style="background-color: #40ffff;"><strong>C-1 13:00 - 13:50 (50</strong><strong>分</strong><strong>)</strong></span><strong><span style="background-color: #40ffff;">&#0160;<br /></span> Autodesk Exchange</strong><strong>ストアを用いたデータ変換プログラムの活用&#0160;</strong><span style="font-size: 8pt; color: #0000ff;">お客様事例</span></p>
<p style="text-align: right;">システムメトリックス株式会社R&amp;Dディレクター 前田 吉男 氏&#0160;</p>
<p style="padding-left: 30px;">Autodesk Exchange ストアを活用して AutoCAD 上に無償の PDF ファィル読みこみ、JWW ファィル読み書きコマンドを追加する方法を紹介します。Autodesk Exchange ストアでは、簡単な操作で AutoCAD 上に PDF 読みこみコマンドを追加することができます。</p>
<p style="padding-left: 30px;">CAD から書き出されたベクトルイメージを含む PDF ファィルは、線分、円、円弧、文字など AutoCAD で編集可能な図形に変換され、PDF ファィル内の画像イメージについては、PDF ファイルの中から画像ファイルが取り出され、イメージアタッチ図形として画像ファイルが参照された図形に変換されます。</p>
<p style="padding-left: 30px;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c6cf49e8970b-pi" style="display: inline;"><img alt="PDF_Import" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c6cf49e8970b image-full img-responsive" src="/assets/image_614066.jpg" title="PDF_Import" /></a></p>
<p style="padding-left: 30px;">同様に、Autodesk Exchange ストアから JWW 読み書きコマンドを追加すると、AutoCAD 上から、JWCAD のデータファィルである JWW/JWC ファィルを読み書きすることができる様になります。</p>

<p><span style="background-color: #40ffff;"><strong>C-2 14:10 - 15:00 (50</strong><strong>分</strong><strong>)<br /></strong></span>&#0160;<strong>Autodesk 360 </strong><strong>クラウド</strong><strong>プラットフォーム</strong><strong> Web </strong><strong>サービス</strong><strong> API </strong><strong>のご紹介</strong></p>
<p style="text-align: right;">オートデスク株式会社 伊勢崎 俊明&#0160;</p>
<p style="padding-left: 30px;">Autodesk 360 クラウド プラットフォームの Web サービス API 第一弾となる View &amp; Data API の概要と実例をご紹介します。従来の Design Review や DWG TrueView に変わるビューワー機能を、Web ブラウザとクラウドを利用して置き換え、さらに、独自アプリケーションやツール、サービスに組み込むことが出来ます。</p>

<p><span style="background-color: #40ffff;"><strong>C-3 15:30 - 16:20 (50</strong><strong>分</strong><strong>)<br /></strong></span><strong>3D プリンター関連技術を使って、新市場を創る !</strong><strong>&#0160;</strong><span style="color: #0000ff; font-size: 8pt;">お客様事例</span></p>
<p style="text-align: right;">株式会社アイジェット 久米原 勝 氏&#0160;</p>
<p style="padding-left: 30px;">3D プリンターとは、樹脂や石膏などの立体に印刷する機械です。金型ナシで精巧なモノをつくることができるため、製作時間もコストも大幅に縮小でき、試作品や模型作りなどに重宝されてきました。そんな 3D プリンターを使った出力サービスを国内でもいち早く事業化したアイジェットは出力造形に留まらず、3D データ入力・編集・仕上げ・梱包などお客様の手に届くまでを自社で行っています。</p>
<p style="padding-left: 30px;">これまで培った 3D プリンター関連技術で試作だけではなく製品化することができるのです。アイジェットは 3D プリンターと様々な生活場面を掛けあわせ、自分のためだけに生まれてくるモノとの出会いをお届けする新しい市場を創ります。</p>
<p style="padding-left: 30px; text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c6cc7ffe970b-popup" onclick="window.open( this.href, &#39;_blank&#39;, &#39;width=640,height=480,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0&#39; ); return false" style="display: inline;"><img alt="3dpipo" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c6cc7ffe970b img-responsive" src="/assets/image_944704.jpg" title="3dpipo" /></a></p>
<p style="padding-left: 30px;">CAD や CG など、一部だけで先行していた 3D の世界が一般化してきたことで、ユニークな新しいビジネスモデルが生まれつつあります。その一端をこのセッションでご確認ください。きっと新しいアイデアが生まれるはずです。&#0160;</p>
<p><span style="background-color: #40ffff;"><strong>C-4 16:40 - 17:30 (50</strong><strong>分</strong><strong>)<br /></strong></span><strong>AutoCAD 2015 </strong><strong>への移植と選択肢</strong></p>
<p style="text-align: right;">オートデスク株式会社 伊勢崎 俊明&#0160;</p>
<p style="padding-left: 30px;">最新バージョンである AutoCAD 2015 のカスタマイズ環境は、従来のバージョンと少し異なっています。このセッションでは、カスタマイズしたメニューやアドイン アプリケーションを AutoCAD 2015 に移植する際の注意点や具体的な方法をご案内します。また、移行に際して、新しい選択肢となる仮想環境でのAutoCAD運用にも言及します。お客様に移植の重要性をご理解いただく機会にもなりますので、AutoCAD 2015 への移行をご検討中のお客様にも有用と思いますので、本セッションをご紹介いただければと思います。</p>
