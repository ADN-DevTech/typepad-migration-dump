<p>BIM (Building Information Modeling) が提唱されてからずいぶん年月が経ちます。昨今では、世界的に見て、そのコンセプトがしっかりと根付いてきたと言えましょう。私の経験からも、１０年ほど前にRevitをサポートし始めた当時は、Revitを通してBIMとは何かといったようなお話させていただいたり、Revit API をできるだけ多くの人にマスターしていただこうと、クラスルームのトレーニングを用意したりといった活動を通して、どちらかというと「押し」のムードでRevit APIのお話をさせていただいたのを記憶しています。そのような押しムードも、ここ２，３年は、どちらかといえば、「押され」ムードに変わってきました。最近は、パートナーさんのほうから、「顧客からRevitを使うように頼まれている。API のトレーニングをしてくれないか」といった問い合わせに変わって来ているのです。</p>
<p>そのように普及してきたBIMですが、実は、BIMを取り巻く考え方や環境も、クラウド＆モバイル テクノロジーによって変化してきています。建築、設備、構造はもとより、設計、施工、管理、そして現場を抱えているといった、建築を取り巻く数多くの専門分野で携わる人たちのノーハウや、いろいろな場所で作業する人たちの間でのコラボレーションといったものを考えると、クラウドは、建築を取り巻く環境を飛躍させるものと考えられています。では、いったいどのように変わっていくのでしょう？</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017c380ac5df970b-pi" style="display: inline;"><img alt="2013 0322 BIM360 what mean2" border="0" class="asset  asset-image at-xid-6a0167607c2431970b017c380ac5df970b image-full" src="/assets/image_385619.jpg" title="2013 0322 BIM360 what mean2" /></a></p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017d4237c939970c-pi" style="display: inline;"></a></p>
<p>オートデスクでは、クラウドベースのビルディングインフォメーションモデリング（BIM）を BIM®360と総称して、次期世代のBIMと位置づけています。可動性、アクセシビリティ、および実質的に無限のコンピューティングパワーを提供するオートデスクの360クラウドベースのプラットフォームを通して、建築、インフラ、設計、施工の専門家は、インテリジェントなモデルベースの​​ワークフローにいつでも、どこでも、誰でもアクセスすることができます。</p>
<p>クラウドに計算集約型タスクを移動することによって、より迅速な可視化とシミュレーション、データが豊富なインテリジェントなモデルへのアクセスを最適化したコラボレーションを可能にし、オートデスクのBIM360は、複数のデザイン部門や施工チームが、プロジェクトの成果を向上させることができます。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017ee9ae0dcf970d-pi" style="display: inline;"><img alt="2013 0322 BIM360 infinite power2" border="0" class="asset  asset-image at-xid-6a0167607c2431970b017ee9ae0dcf970d image-full" src="/assets/image_38409.jpg" title="2013 0322 BIM360 infinite power2" /></a></p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017ee9abcbe5970d-pi" style="display: inline;"></a></p>
<ul>
<li>モバイルやデスクトップから、いつでも、どこでも、誰でもアクセスできる</li>
<li>一貫共有したデータにアクセスし、コラボレーションを可能にする</li>
<li>コンピューターの計算能力を多く必要とするタスクをクラウドに移行する</li>
</ul>
<p><a href="http://usa.autodesk.com/adsk/servlet/limage?siteID=123112&amp;imageID=20477672&amp;id=19676436" target="_blank" title="BIM 360">Autodesk BIM 360</a> のページにBIM 360 のアイデアを簡単に紹介するビデオ（英語）がありますので、参考にしてください。</p>
<p>具体的なサービスは、日本では今後徐々に提供されることになりますが、現時点では米国では以下のようなBIM 360 サービスが用意されています：</p>
<p>コラボレーションとアクセス</p>
<ul>
<li>干渉チェック、調整、およびコラボレーション (Autodesk® BIM 360 Glue) </li>
<li>概念設計と実現可能性の評価 (Autodesk® 360 Infrastructure Modeler ) </li>
<li>現場管理、委託、ハンドオーバ (Autodesk ® BIM 360 Field)</li>
</ul>
<p>シミュレーション</p>
<ul>
<li>メカニカル シミュレーション (Autodesk ® Simulation 360)</li>
<li>空気、流体の流れ、および熱的快適性 (Autodesk ® Simulation 360) </li>
<li>エネルギー分析 (Autodesk® 360 Energy Analysis for Autodesk ® Revit ® ) </li>
<li>建物全体の解析 (Autodesk ® Green Building Studio ®)</li>
<li>構造解析 (Autodesk ® 360 Structural Analysis for Autodesk ® Revit®) </li>
</ul>
<p>可視化</p>
<ul>
<li>レンダリング (Autodesk ® 360 Rendering) </li>
</ul>
<p>現時点での個々のサービスの詳細は、<a href="http://usa.autodesk.com/adsk/servlet/pc/index?siteID=123112&amp;id=19729180" target="_self" title="BIM 360 Services">BIM 360 サービスのページ（英語）</a>（<a href="http://www.autodesk.co.jp/adsk/servlet/pc/index?siteID=1169823&amp;id=20009033" target="_self" title="BIM 360 Services Japan">日本語</a>）を参考にしてください。</p>
<p>このように、これまでBIMといえば一括したモデルに頂点が当てられていましたが、今後は、さらに、解析、シミュレーション、コラボレーション、そして居場所にとらわれない情報へのアクセスとBIMの概念は大きく変化しています。</p>
