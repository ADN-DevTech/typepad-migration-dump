<p>翔泳社が主催する開発者イベントである <a href="http://event.shoeisha.jp/devsumi/" target="_blank"><strong>Developer Summit 2015 Winter</strong></a>、通称、<strong>デブサミ 冬</strong>&#0160;への参加が決定しました。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c73d004e970b-pi" style="display: inline;"><img alt="Devsumi_logo" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c73d004e970b image-full img-responsive" src="/assets/image_626982.jpg" title="Devsumi_logo" /></a></p>
<p>Web 開発者を含む多くに方々に、オートデスクの Web サービス API を広める目的で、オートデスクもセッションを 1 つ持たせていただくことが出来ました。</p>
<p>開催場所は、東京の<a href="https://www.megurogajoen.co.jp/access/index.html/" target="_blank">目黒雅叙園</a>、開催日時は&#0160;2015年2月19日（木）～20日（金）の2日間で、様々なセッションが用意されています。開催概要は、<a href="http://event.shoeisha.jp/devsumi/20150219" target="_blank"><strong>デブサミ 冬</strong></a> の Web サイトをご参照ください。<strong>参加は無料</strong>で、<a href="https://event.shoeisha.jp/order/apply/62/?utm_source=devsumi&amp;utm_medium=sitefloatingbanner&amp;utm_campaign=20141225" target="_blank"><strong>こちら</strong></a> からお申込みいただくことが出来ます。</p>
