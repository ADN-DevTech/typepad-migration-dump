<p><strong><a href="http://adndevblog.typepad.com/technology_perspective/2016/01/autodesk-forge-platfotrm.html" target="_blank">Autodesk Forge プラットフォームとは</a></strong> でもご案内ししましたが、Forge を大規模にご紹介する &#0160;<strong>Forge Developer Conference 2016</strong>、通称 <strong>Forge DevCon</strong> が6月15日、16日の2日間にわたって San Francisco の<strong> <a href="http://fortmason.org/" target="_blank">Fort Mason Center </a></strong>で開催されます。</p>

<p>いままで、<a href="http://forge.autodesk.com/conference/" target="_blank">http://forge.autodesk.com/conference/</a>&#0160;からイベントの概要のみが告知されていましたが、本日から参加申し込みが正式に開始されました。有償イベントとなりますが、<strong>4月22日まで</strong>ディスカウント価格でお申し込みいただくことが出来ます。学生は無償です。</p>
<p style="padding-left: 30px;">※ 旅費、及び、宿泊費は、お申し込みいただく方のご負担となります。</p>
<p>Forge Developer Conference は、オートデスクが提唱する&#0160;<strong><a href="http://www.autodesk.co.jp/fomt" target="_blank">Future Of Making Things</a></strong>&#0160;を実現するテクノロジや API を紹介するイベントです。次のような内容に興味をお持ちの方に最適です。</p>
<ul>
<li>クラウド API を使って既存のビジネスの強化や拡張の機会を見出したい</li>
<li>オートデスクの次世代のクラウド サービステクノロジと API を学びたい</li>
<li>スタートアップ企業の方で、今後の投資を得る機会を見出したい</li>
</ul>
<p>具体的には、次のようなセッションが用意されます。&#0160;</p>
<ul>
<li>バーチャル リアリティ(VR)/拡張現実(AR)</li>
<li>Internet of Things(IoT)</li>
<li>BIM(Biulding Information Modeling) －建設分野</li>
<li>製作（3D プリンタ関連） － 製造分野</li>
<li>管理（PLM、ERP、その他） － 製造分野</li>
<li>デザイン － 製造分野</li>
<li>ビジネス開発</li>
</ul>
<p>詳細なセッション内容は、<strong><a href="http://forge.autodesk.com/tracks-and-speakers/" target="_blank">Forge DevCon サイト</a></strong>をご参照ください。</p>
<p>海外での開催ではありますが、オートデスクが提供する未来をさまざまな業種で見出していただくことが出来るはずです。&#0160;お申し込みは、<a href="http://forge.autodesk.com/conference/">http://forge.autodesk.com/conference/</a> ページの&#0160;<span style="background-color: #ee8822;"> &#0160;<strong><span style="color: #ffffff;">Register Now &#0160; &#0160;&gt;</span></strong>&#0160; <span style="background-color: #ffffff;">&#0160;</span></span>リンクからお願いいたします。</p>
