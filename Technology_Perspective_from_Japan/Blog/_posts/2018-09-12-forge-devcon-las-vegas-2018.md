<p>例年通り、米国 Las Vegas で Autodesk University（11月13日～15日）と Forge DevCon（11月12日～13日）を開催いたします。ご参加いただくためのお申込みも既に開始されていますので、概要とともにお知らせいたします。今年も Forge DevCon（11月12日～13日）は Forge の最新情報やテクノロジを習得する場として設定され、基調講演、各種ワークショップ、事例、ラボなどで構成されるイベントとしての開催です。</p>
<p>一昨年（初回）、昨年と、従来1000名以上のエンジニアや開発者、また、各種業界の専門家の皆様にご参加いただいています。<strong><span style="background-color: #ffff00;">日本からの参加者も年々増加していて、初めての DevCon 開催となった一昨年の DevCon 2016 では 11 名のご参加、昨年の DevCon 2017 では 48 名となっていて、2017 年は米国以外の国別で最多になってます。</span></strong></p>
<p>今年の Forge DevCon Las Vegasでは、次の内容を調整中です。：</p>
<ul>
<li>Revit と Inventor 用に拡張された Design Automation API の詳細を披露。</li>
<li>現在の Data Management API のサブセットとして今後登場予定の Forge Data Platform の発表。</li>
<li>クラウドだけのアプリケーション開発プラットフォームとしてではなく、デスクトップ環境とクラウド環境を結びつける位置付けで、その方法論に言及するクラスの開催。</li>
<li>Issue、RFI、提出物、チェックリストなどを含む次世代の BIM 360 API の詳細。</li>
<li>大規模モデルを劇的に高速化させる Forge Viewer の紹介。</li>
</ul>
<p>現時点で予定しているセッション、クラスは次のとおりです。：</p>
<ul>
<li><strong>基調講演</strong> - オートデスクの新しいクラウド担当副社長 Sam Ramji と Forge 製品管理のシニアディレクター、Brian Roepke から、Forge プラットフォームの将来についての計画についてお話しします。より詳細な&#0160;<a href="https://autodeskuniversity.smarteventscloud.com/connect/sessionDetail.ww?SESSION_ID=227961">Forge Roadmap クラス&#0160;</a>もチェックしてください。</li>
<li><strong>Forge&#0160;</strong><strong>を知る</strong> - Forge Viewer、BIM 360 API、および Forge API で作成された業界固有のソリューションを深く理解いただけます。</li>
<li><strong>設計と製造のためのソリューション</strong> - 真にコラボレーティブなクラウドベースの設計と製作の可能性を探ります。Forg eのアップデート、統合、エンドツーエンドのワークフロー機能などもプレビューします。</li>
<li><strong>Forge&#0160;</strong><strong>プラットフォームへ追加された新機能</strong> – 現在のForgeで可能なことを理解したうえで、将来、Forgeがどこへ向かって行くかを把握することが出来ます。</li>
<li><strong>関連テクノロジ</strong> - AR、VR、機械学習、およびジェネレイティブ デザインをForgeを使用して利用できるようになります。その方法を把握することが可能です。</li>
<li><strong>Web&#0160;</strong><strong>プログラミング</strong> - Webアプリケーションのセキュリティからアーキテクチャ、ベストプラクティスのテスト、またはデスクトップから Web プログラミングへの移行に必要な基礎知識を習得出来ます。</li>
<li><strong>ユーザ事例</strong> - オートデスクの開発パートナーとエンドユーザが Forge を使用して実際のビジネス上の問題を解決した方法と、その過程で習得した点などをケーススタディとして取り上げます。</li>
<li><strong>Forge + BIM 360&#0160;</strong><strong>の統合テクノロジ</strong> - Forge と BIM 360 を使用して建設業界全体で利用可能な強力なアプリケーションを作成し、チームの革新、作業現場のデジタル化、データ損失の最小化、効率の向上を支援するテクニックを知ることが出来ます。<a href="https://autodeskuniversity.smarteventscloud.com/connect/search.ww#loadSearch-searchPhrase=&amp;searchType=session&amp;tc=0&amp;sortBy=relevance&amp;i(41005)=&amp;i(57245)=&amp;i(57211)=380130&amp;p=">もっと詳しく</a>。</li>
</ul>
<p><span style="background-color: #ffff00;"><strong>Forge DevCon は設計やエンジニアリング用途のソフトウェア技術者、プログラマだけをターゲットにしているわけではありません。</strong></span>企業や組織のリーダーや意思決定者は、自身でプログラムを書く方法を知る必要はありませんが、業界や技術の動向を常に把握して、適切に正しい判断を下す必要があります。そのような立場の方にも、次のような視点で事例や有用な情報をご提供します。<span style="color: #111111;"><strong><span style="background-color: #ffff00;"><a href="https://iot-jp.com/iotsummary/iottech/digital-twin%EF%BC%88%E3%83%87%E3%82%B8%E3%82%BF%E3%83%AB%E3%83%84%E3%82%A4%E3%83%B3%EF%BC%89/.html" style="background-color: #ffff00; color: #111111;">Autodesk University への参加をご検討中なら、ぜひ、Forge DevCon への参加をご検討ください。</a></span></strong></span></p>
<ul>
<li>Forge を使った洞察力を兼ね備えたダッシュボード例</li>
<li>プレハブ</li>
<li>メンテナンス用途のためのデジタルツイン</li>
<li>エンタープライズシステムと BIM 360 などの統合</li>
</ul>
<p>ご参加いただく際のコスト概要は下図のとおりです、詳細とお申込みは、<a href="https://forge.autodesk.com/devcon-2018#tickets">https://forge.autodesk.com/devcon-2018#tickets</a> をご確認ください。</p>

<p>なお、従来、米国では&#0160; Autodesk University（通称、AU）の前日に Autodesk University と同じ会場で ADN メンバー向けの開発者イベント Developer Days（通称、DevDays）を開催してきましたが、昨年から&#0160; DevDays&#0160; イベントを Forge DevCon に置き換えて開催しています。</p>
<p>Forge DevCon は Autodesk University と同じ会場で開催されるため、AU も併せて膨大な量の製品知識やネットワーキングの機会を得られるはずです。オートデスクは AU を旗艦イベントとして位置付けていて、このタイミングで新機軸のコンセプトやテクノロジを披露しています。イベントのご参加いただくことで、建築/建設、製造、エンジニアリング、デザイン、メディア業界向けに、Forge DevCon の約40クラスを含め、700以上のクラスからさまざまな情報を得ることが出来るはずです。海外での開催、英語での講義ということでご参加いただくハードルが高い点は否めませんが、百聞は一見に如かず、ぜひ、この機会に参加をご検討ください。</p>
