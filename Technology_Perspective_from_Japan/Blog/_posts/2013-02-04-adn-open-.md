<p>オートデスクでは、従来、Autodesk Developer Network の名のもとに、<strong>ADN Standard</strong> と <strong>ADN Professional</strong> という 2 種類の有償プログラムで、<strong>API</strong>(Application Programing Interface) を使ってオートデスク製品をカスタマイズする開発者の皆様をサポートしてきました。</p>
<p>これとは別に、米国では昨年より <strong>ADN Open</strong> と呼ばれる無償プログラムを先行して開始していて、ブログやディスカッション フォーラムを通しての開発サポートや情報公開をはじめています。今回、日本でも ADN Open の一貫として、このブログ「<strong>Technology Perspective from Japan</strong>」を開始することになりました。</p>
<p>すでに、一部製品の API について <strong><a href="http://www.autodesk.co.jp/tech-faq" target="_self">Autodesk Technical Q&amp;A</a></strong> ページで FAQ を公開し始めていて、今後は徐々に公開するコンテンツを増やしていく予定です。このブログ自体にも、最新の情報や API を使った Tips を記載していくつもりです。</p>
<p>このブログのもう 1 つの目的には、オートデスクが提供するクラウド サービスやモバイルを使ったテクノロジの紹介があります。クラウドとモバイルは、オートデスクが現在最も注力している分野でもあるのですが、具体的な内容がなかなか伝わらないジレンマがありました。今後は、こういったクラウドやモバイルの分野にも、API を使ったカスタマイズが導入されることが予想できますので、最新のクラウド サービスとともに、カスタマイズの可能性を探る手引きにしていただきたいと思っています。&#0160;</p>
<p>ADN Open には、ユーザ登録は不要です。いつでも、だれでもブログや Autodesk Q&amp;A などを使って情報取集してみてください。Google などの検索エンジンを使った検索でも、記事の内容にヒットするよう工夫されています。オートデスク製品をプログラムを使って開発する場面では、「はじめの一歩」から踏み出せない場合もあるかと思います。ぜひ、このブログ窓口としてご活用ください。</p>
<p>By Toshiaki Isezaki</p>
