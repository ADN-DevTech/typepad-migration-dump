<p>Inventor 2014 API クラストレーニング 向けに作成された教材一式「マテリアルと実習テキスト（chm）及び 演習と実習の回答 」を公開いたします。</p>

<p>公開されるコンテンツは、Inventor 2014 APIクラストレーニングテキスト（.ppt P.370）を pdf ファイルに変換した "Inventor2014API .Net Training_J.pdf"、テキスト内の演習問題と各開発ガイド（実習）を説明した「演習と実習ヘルプファイル」の"Inventor2014API Exercise.chm"、「演習と各開発ガイド（実習）の回答」( 14の演習回答 と 80の実習の回答) をそれぞれVBAマクロファイルとVisual Studio 2010(VB.Net)のプロジェクトとしたファイル群、また「演習と各開発ガイド（実習）で使用するInventorファイル群」の公開となります。</p>

関連ファイルのダウンロード先：<br />
１．	Inventor 2014 APIクラストレーニングテキストは、 <span class="asset  asset-generic at-xid-6a0167607c2431970b01901d5acfe5970b"><a href="http://adndevblog.typepad.com/files/inventor2014api_trainigdoc-1.zip">ココ</a></span> よりダウンロードできます。<br />
また「<a href="http://tech.autodesk.jp/faq/faq/adsk_result_dd.asp?QA_ID=8148">QA- Inventor 2014 APIの クラストレーニングテキスト</a>」 からもダウンロードできます<br />
２．	演習と実習のヘルプファイルと回答は、 <span class="asset  asset-generic at-xid-6a0167607c2431970b0192ab35e56d970d"><a href="http://adndevblog.typepad.com/files/inventor2014api_labdocs-2.zip">ココ</a></span> よりダウンロードできます。<br />
また「<a href="http://tech.autodesk.jp/faq/faq/adsk_result_dd.asp?QA_ID=8152">QA- Inventor 2014 APIの クラストレーニングテキスト内の演習問題と実習ガイドと回答</a>」 からもダウンロードできます。<br />
３．演習と実習に使用するファイル群は、 <span class="asset  asset-generic at-xid-6a0167607c2431970b0192ab192ce4970d"><a href="http://adndevblog.typepad.com/files/inventor2014api_labdrawings-1.zip">ココ</a></span> よりダウンロードできます。<br />
　　また「<a href="http://tech.autodesk.jp/faq/faq/adsk_result_dd.asp?QA_ID=8153">QA- Inventor 2014 APIの クラストレーニングテキスト内の演習と実習で使うファイル</a>」 からもダウンロードできます。</p>

<p>任意のホルダーに　Inventor2014API_TrainigDoc.zip , Inventor2014API_LabDocs.zip , Inventor2014API_LabDrawings.zip　をダウンロード後にそれぞれ解凍をしていただき、以下のホルダー構造のように、Cドライブ直下に"Inventor API Training"ホルダーを移動配置いただきご利用下さい。<br />
（解凍していただくと、解凍ホルダー直下に\Inventor API Trainingホルダーが作成され　\Documents,\Drawings ,\Lab Instructions,\Training Labs ホルダーが含まれます）</p>

<p><a class="asset-img-link"  style="display: inline;" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b0192ab1932a1970d-pi"><img class="asset  asset-image at-xid-6a0167607c2431970b0192ab1932a1970d" alt="Inventor2014APIHolder" title="Inventor2014APIHolder" src="/assets/image_933073.jpg" /></a><br /></p>

<p>\Documentsホルダー<br />
　Inventor 2014 APIのクラストレーニングpptを Inventor2014API .Net Training_J.pdfとして変換しています。</p>

<p>\Drawingsホルダー<br />
　演習と実習（Inventor2014API Exercise.chm）で使用するInventorファイル群</p>

<p>\Lab Instructionsホルダー<br />
　演習と実習解説chm（Inventor2014API Exercise.chm）</p>

<p>\Training Labsホルダー<br />
　演習の回答 ( 演習1～14 )　実習の回答( 80 )<br />
  (回答並びに解説は、全て上記の Cドライブのホルダー構造でハードコートされています)</p>

<p><br />
尚、アドイン作成には Visual Studio 2010 有償版(Professionalなど)の開発環境が必要となります。<br />
また、各演習の回答マクロの他に同梱されている Visual Studio向けのプロジェクトもVisual Studio 2010 有償版(Professionalなど)を対象として作成されています。</p>
