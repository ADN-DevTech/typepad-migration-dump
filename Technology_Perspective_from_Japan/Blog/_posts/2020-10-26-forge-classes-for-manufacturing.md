<p>クラウドの存在は製造業にも大きな影響を与えています。クラウドを利用することで、ユーザは接続されたプロセスやワークフローを実現することができます。Forge を使用すれば、ユーザはデスクトップ アプリケーションをクラウド上に拡張して、処理の自動化、リアルタイムデータの活用など、カスタマイズ ソリューションなどを実現することができます。</p>
<p>オンライン開催の AU 2020 にも製造業向けの Forge クラスが用意されています。クラウドを使用した製造プロセスに変革に触れることが出来る、それら製造業向け Forge クラスをご紹介したいと思います。</p>
<hr />
<ol>
<li><strong> Forge for Manufacturing Leadership Presentation [オンデマンド]</strong></li>
</ol>
<p style="padding-left: 40px;">Srinath Jonnalagadda と Brian Roepke が、クラウドが製造プロセスとワークフローをどのように破壊、再構築しているかを探ります。モイコン、アメリカン・エア・フィルター、東芝エレベータなどの企業から、Forge が今日のビジネスのあり方をご紹介します。ぜひ、ご参加ください。</p>
<ol start="2">
<li><strong> Using iLogic with Design Automation for Inventor to create a Configurator - MFG473705</strong></li>
</ol>
<p style="padding-left: 40px;">一般的に、Design Automation for Inventor を使用すると、デスクトップ API をクラウドで実行することができ、デスクトップ上での動作と同様の動作を実現しますが、インストールされたソフトウェアや依存するライセンスは必要ありません。</p>
<p style="padding-left: 40px;">見落とされがちな Design Automation for Inventor&#0160; のもう1つの利点は、クラウドで iLogic スクリプトを実行できる点です。これらのスクリプトは&#0160; CADファイル自体に埋め込まれているため、特定の条件下で自動実行するようにプログラムすることができます。</p>
<p style="padding-left: 40px;">このクラスでは、この機能を利用して Forge Design Automation と iLogic を組み合わせ、使いやすいユーザインタフェース上でユーザが提供した入力値に沿って、クラウド上の入力 Inventor ファイル（IPT）を修正してコンフィギュレータを作成する方法について説明します。作成された結果は、Forge Viewer を使って Web ブラウザ上ですぐに確認できるほか、IPT、IDW、PDF など複数の形式でダウンロードすることも可能です。</p>
<p style="padding-left: 40px;">本 Web アプリケーションサンプルは、最新の Web 技術を使用しており、OS に依存せず、モバイル端末にも対応しています。</p>
<ol start="3">
<li><strong> Using Autodesk Vault with Forge Design Automation for Inventor - MFG473706</strong></li>
</ol>
<p style="padding-left: 40px;">Autodesk University 2015 では、Vault API と Forge Viewer を組み合わせての使用が初めてデモされました。</p>
<p style="padding-left: 40px;">このクラスでは、その流れを踏襲し、Vault API と Design Automation for Inventor を使用して、Autodesk Vault 内に存在する Inventor ファイルをプログラムで修正する方法を紹介します。</p>
<p style="padding-left: 40px;">Vault からファイルをチェックアウトし、Design Automation for Inventor を使用してファイルを修正する方法を見ていきます。</p>
<p style="padding-left: 40px;">修正後、更新されたファイルを Vault にチェックインして新しいバージョンを作成し、ファイルの履歴を維持します。</p>
<ol start="4">
<li><strong> IoT Manufacturing Demo, Optimizing CNC toolpaths using Fusion 360 and Forge - PM469347</strong></li>
</ol>
<p style="padding-left: 40px;">このデモでは、ホビークラスの CNC ルーターからリアルタイムの製造データ（切削温度）を取得し、製造中の部品のビジュアルモデル上でそのデータを視覚化するという単純化された例を紹介します。&#0160;&#0160;切削工具とワークの温度は、切削効率と工具の健全性を直接示すものです。&#0160;&#0160;温度をマッピングすることで、公称切粉量を達成するための送り、速度、切り込み深さの調整を検証し、提案することができます。</p>
<p style="padding-left: 40px;">これは産業用ソリューションとは程遠いものですが、このデモでは簡略化された文脈の中で IoT の力を示し、Autodesk Forge で構築されたクラウドベースの製造ソリューションの有用性と将来性を探ります。</p>
<ol start="5">
<li><strong> Forging BIM Configurator for HVAC systems - FAB468859</strong></li>
</ol>
<p style="padding-left: 40px;">さまざまな業界で大量生産からカスタマイズソリューションへと移行しています。カスタム BIM ソフトウェアソリューションの需要は途方もなく増加しています。しかし、カスタムソリューションの開発にはコストがかかります。Autodesk Forge を使えば、最小限の投資と時間でカスタム BIM コンフィギュレータを構築することができます。&#0160;</p>
<p style="padding-left: 40px;">HVAC の専門家が持続可能な設計を作成するた めのカスタムシミュレーションアプリを作成しました。このアプリは、設計空間の性能と HVAC システムの構成をクラウド上でシミュレーションします。このクラスでは、Autodesk Forge を使用して、新しいユーザー定義のオフィスビルド空間をどのように作成できるかを実演します。複雑な 3D HVAC システム構成のアセンブリを作成する方法を詳しく説明します。Design Automation for Revit を使用して、サーバ上に正確な BIM モデルを構築する方法を学びます。参加者は、これらのテクニックを使用して、キッチン設計、家具設計、インフラストラクチャ設計などの膨大な種類のオンライン BIM システムコンフィギュレータを作成する方法を学ぶことができます。</p>
<ol start="6">
<li><strong> It&#39;s time to scale! Using Forge and cloud solutions to business growth - FAB468615</strong></li>
</ol>
<p style="padding-left: 40px;">目まぐるしく変化する今日の経済では、企業は新しい要求に迅速に適応しなければなりませんが、新しいテクノロジーを毎日導入しなければならない場合、労働者は追いつくことができません。人間の学習曲線は、テクノロジーの成長とビジネスへの期待とは一致しません。したがって、企業は成長とスケーラビリティを目指し、リードタイムを短縮するために、プロセスを自動化し、既存のソリューションに統合する必要があります。</p>
<p style="padding-left: 40px;">このクラスでは、Autodesk Forge と API がどのように生産性を高め、リスクとコストを削減し、複数のプラットフォームに接続して任意のワークフローを成形し、ビジネスと開発者に機会を開くことができるかを探ります。AU 2019 のクラス「計画、モデリング、ファブリケーションのためのForgeとジェネレーティブデザイン」に続くこのクラスでは、オートデスクのエコシステムによって、Shedmate が、どう Forgeを使用してスケーラブルなビジネスモデルと顧客開発を実現したかを紹介します。また、Forge によって営業チームがモデリングの知識がなくても数分で BIM データを作成することができ、関係者間でリアルタイムにデータをつなぐことができたことを紹介します。</p>
<ol start="7">
<li><strong> Sales Engineering Automation using Forge, Fusion Lifecycle, and Vault - MFG469225</strong></li>
</ol>
<p style="padding-left: 40px;">製造業種の企業は、お客様にユニークな製品を提供すると同時に、全体的な効率を向上させなければならないというプレッシャーにさらされています。&#0160;&#0160;このセッションでは、Fusion Lifecycle (FLC) と Vault と連携した Forge 上に構築されたコンフィギュレータ技術が、シームレスで統合されたソリューションを提供し、売上の増加、エンジニアリングサイクルタイムの短縮、そして最終的には効率性の向上を実現する方法をご紹介します。&#0160;&#0160;</p>
<p style="padding-left: 40px;">Web ブラウザを使用するだけで、営業スタッフや見込み客であっても、独自の要件に合わせて製品を設定することができ、視覚的なフィードバックと専門的な見積もりを即座に受け取ることができます。&#0160;&#0160;注文の準備ができたら、情報は Fusion Lifecycle と Vault の両方にシームレスに流れます。さらにVaultをカスタマイズすることで、DXF ファイルなどの下流のデータが追加で提供され、すべて Vault のライフサイクルエンジンを使って自動化されます。</p>
<p style="padding-left: 40px;">このセッションでは、最適な効率化、統合、自動化を実現するためのクラス最高のプロセスをご紹介します。</p>
