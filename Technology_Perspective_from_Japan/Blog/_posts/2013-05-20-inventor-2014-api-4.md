<p>「<a href="http://adndevblog.typepad.com/technology_perspective/2013/04/inventor-2014-api-%E6%96%B0%E6%A9%9F%E8%83%BD-%E6%9B%B4%E6%96%B0%E6%83%85%E5%A0%B1.html">Inventor 2014 API 新機能 その１</a>」「<a href="http://adndevblog.typepad.com/technology_perspective/2013/05/inventor-2014-api-%E6%96%B0%E6%A9%9F%E8%83%BD-%E3%81%9D%E3%81%AE%EF%BC%92.html">Inventor 2014 API 新機能 その２</a>」「<a href="http://adndevblog.typepad.com/technology_perspective/2013/05/inventor-2014-api-%E6%96%B0%E6%A9%9F%E8%83%BD-%E3%81%9D%E3%81%AE%EF%BC%93.html">Inventor 2014 API 新機能 その３</a>」に続き、Inventor 2014の新しい機能に対するAPIについてご紹介させていただきます。</p>

<p>「<a href="http://adndevblog.typepad.com/technology_perspective/2013/04/inventor-2014-api%E6%A9%9F%E8%83%BD-%E3%81%9D%E3%81%AE%EF%BC%91.html">Inventor 2014 機能 その１</a>」でもご案内させていただきましたが、大規模アセンブリファイルのファイルオープン時間の大幅な短縮機能を目的として、「簡易モード」によるファイルオープン機能が備わっています。<br />
<a class="asset-img-link"  style="display: inline;" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017eeb431423970d-pi"><img class="asset  asset-image at-xid-6a0167607c2431970b017eeb431423970d" alt="ProductFig12" title="ProductFig12" src="/assets/image_521058.jpg" /></a></p>

<p>APIにより、この「簡易モード」状態でアセンブリファイルを開く方法も用意されており、事前にNameValueMapオブジェクトにて、” ExpressModeBehavior”名で、値を” OpenExpress”の情報を用意した後に、Documents.OpenWithOptionsメソッドを使用する事で、「簡易モード」状態でアセンブリファイルを開く事が可能です。</p>

<p><strong>＜ 簡易モードによるアセンブリファイルオープン ＞</strong> <br />
<a class="asset-img-link"  style="display: inline;" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017eeb431585970d-pi"><img class="asset  asset-image at-xid-6a0167607c2431970b017eeb431585970d" alt="Fig1" title="Fig1" src="/assets/image_729544.jpg" /></a><br /><br />
<strong>Public Sub OpenWithExpressMode()</strong><br />
  Dim oAsmDoc As AssemblyDocument<br />
  Dim oDocOpenOptions As NameValueMap<br />
  Set oDocOpenOptions = _<br />
     ThisApplication.TransientObjects.CreateNameValueMap<br />
  Call oDocOpenOptions.Add("ExpressModeBehavior", "OpenExpress")<br />
  Set oAsmDoc = ThisApplication.Documents.OpenWithOptions _<br />
     ("C:\Temp\Bracket\Bracket.iam", oDocOpenOptions)<br />
<strong>End Sub</strong></p>

<p><br />
<strong>＜完全ロードモードによるアセンブリファイルオープン＞</strong><br />
<a class="asset-img-link"  style="display: inline;" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01901c45b2df970b-pi"><img class="asset  asset-image at-xid-6a0167607c2431970b01901c45b2df970b" alt="Fig2" title="Fig2" src="/assets/image_413299.jpg" /></a><br /> <br />
<strong>Public Sub OpenWithFullMode()</strong><br />
  Dim oAsmDoc As AssemblyDocument<br />
  Dim oDocOpenOptions As NameValueMap<br />
  Set oDocOpenOptions = _<br />
     ThisApplication.TransientObjects.CreateNameValueMap<br />
  Call oDocOpenOptions.Add("ExpressModeBehavior", "OpenFull")<br />
  Set oAsmDoc = ThisApplication.Documents.OpenWithOptions _<br />
     ("C:\Temp\Bracket\Bracket.iam", oDocOpenOptions)<br />
<strong>End Sub</strong></p>

<p><br />
尚、アプリケーションオプションのアセンブリタブで、簡易オプションを有効にした際の、任意のファイルのしきい値（設定したファイル数を超える場合に自動的に「簡易モードで開く」）を設定する事も出来ます。<br />
