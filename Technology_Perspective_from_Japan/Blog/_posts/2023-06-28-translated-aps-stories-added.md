<p>いままで英語のみで公開されていた 4 つの Autodesk Platform Services（旧 Forge）事例について、新たに日本語版を用意して公開しました。</p>
<p>ダム建設現場の課題を解決した <a href="https://www.capa.co.jp/" rel="noopener" target="_blank">CAPA</a>（株式会社キャパ）、Data Exchange API を用いたデータ交換例となる <a href="https://www.cctech.co.in/" rel="noopener" target="_blank">CCTech</a>、360 度カメラと Viewer を統合して施工管理を実現した <a href="https://www.earthcam.com/" rel="noopener" target="_blank">EarthCam</a>、ChatGPT を Viewer と融合させた <a href="https://sparkel.io/" rel="noopener" target="_blank">Sparkel</a>&#0160; の事例です。<a href="https://aps.autodesk.com/ja/customer-stories/capa" rel="noopener" target="_blank"></a></p>
