<p>Autodesk Inventor 2016 の日本語バージョンの発表に伴い、Autodesk Inventor 2016 APIトレーニング教材一式「APIトレーニングテキスト」「APIトレーニング実習ガイドと演習＆回答」及び 「演習と実習用ファイル」を公開いたします。</p>

<p>公開されるコンテンツは、Autodesk Inventor 2016 APIトレーニングテキスト（.ppt P.368）を pdf ファイルに変換した "Inventor2016API .Net Training_J.pdf"、テキスト内の演習問題と各開発ガイド（実習）を説明した「演習と実習ヘルプファイル」の"Inventor2016API Exercise.chm"、「演習と各開発ガイド（実習）の回答」( 14の演習回答 と 80の実習の回答) をそれぞれVBAマクロファイルとVisual Studio 2012(VB.Net)のプロジェクトとしたファイル群、また「演習と各開発ガイド（実習）で使用するInventorファイル群」の公開となります。<br />

<p>２．	演習と実習のヘルプファイルと回答は、 <span class="asset  asset-generic at-xid-6a0167607c2431970b01b7c76f18c7970b img-responsive"><a href="http://adndevblog.typepad.com/files/inventor2016api_labdocs.zip">Autodesk Inventor 2016 の APIトレーニングテキスト(実習ガイドと演習＆回答)</a></span> よりダウンロードいただけます。</p>

<p>３．演習と実習に使用するファイル群は、 <span class="asset  asset-generic at-xid-6a0167607c2431970b01b8d0f8b658970c img-responsive"><a href="http://adndevblog.typepad.com/files/inventor2016api_labdrawings.zip">Autodesk Inventor 2016 の APIトレーニングテキスト(演習と実習用ファイル)</a></span> よりダウンロードいただけます。</p>

<p><br />
任意のホルダーに　Inventor2016API_TrainigDoc.zip , Inventor2016API_LabDocs.zip、Inventor2016API_LabDrawings.zip　をダウンロード後にそれぞれ解凍をしていただき、以下のホルダー構造のように、Cドライブ直下に"Inventor API Training"ホルダーを移動配置いただきご利用下さい。<br />
（解凍していただくと、解凍ホルダー直下に\Inventor API Trainingホルダーが作成され中に\Documents,\Drawings ,\Lab Instructions,\Training Labs ホルダーが含まれます）</p>

<p><a class="asset-img-link"  style="display: inline;" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c76f1886970b-pi"><img class="asset  asset-image at-xid-6a0167607c2431970b01b7c76f1886970b img-responsive" alt="Inventor2016APIHolder" title="Inventor2016APIHolder" src="/assets/image_400207.jpg" /></a><br /><br />
\Documentsホルダー<br />
　Inventor 2016 APIのクラストレーニングpptを Inventor2016API .Net Training_J.pdfとして変換しています。</p>

<p>\Drawingsホルダー<br />
　演習と実習（Inventor2016API Exercise.chm）で使用するInventorファイル群</p>

<p>\Lab Instructionsホルダー<br />
　演習と実習解説chm（Inventor2016API Exercise.chm）</p>

<p>\Training Labsホルダー<br />
　演習の回答 ( 演習1～14 )　実習の回答( 80 )<br />
  (回答並びに解説は、全て上記の Cドライブのホルダー構造でハードコートされています)</p>

<p><br />
尚、アドイン作成には Visual Studio 2012 有償版(Professionalなど)の開発環境が必要となります。<br />
また、各演習の回答マクロの他に同梱されている Visual Studio向けのプロジェクトもVisual Studio 2012 有償版(Professionalなど)を対象として作成されています。</p>
