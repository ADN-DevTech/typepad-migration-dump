<div class="node__image">2019年10月の <strong><a href="https://adndevblog.typepad.com/technology_perspective/2019/10/design-automation-api-v3-release.html" rel="noopener" target="_blank">Design Automation API v3 の正式リリース</a></strong>にともない、2020年5月12日火曜日に Design Automation API v2 （v2 は Design Automation API for AutoCAD のみ）が廃止されることがアナウンスされています。 Design Automation API v2 を利用したアプリをお持ちで、まだ、Design Automation API v3 への移行がお済でない場合には、それまでに移行作業をお願いいたします。</div>
<div class="node__body">
<p>Design Automation API v3 では、WorkItem のキャンセル処理など、いくつかの機能強化が導入され、同時に、既存の AutoCAD コアエンジンに加えて、Inventor、Revit、3ds Max のコアエンジンのサポートが追加されています。 まだ、Design Automation API v2 をお使いのお客様には、Design Automation API v2 がまもなく廃止されることを直接メールでご連絡しています。</p>
