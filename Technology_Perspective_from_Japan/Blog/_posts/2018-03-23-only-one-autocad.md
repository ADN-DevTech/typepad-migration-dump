<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d2e1bbf5970c-pi" style="float: right;"><img alt="OneAutoCAD_logo-version2C" class="asset  asset-image at-xid-6a0167607c2431970b01b8d2e1bbf5970c img-responsive" src="/assets/image_779313.jpg" style="width: 200px; margin: 0px 0px 5px 5px;" title="OneAutoCAD_logo-version2C" /></a>AutoCAD 2019、および、AutoCAD LT 2019 がリリースされました。このバージョンは、2017、2018 バージョン製品同様、<strong><a href="https://www.autodesk.co.jp/subscription" rel="noopener noreferrer" target="_blank">サブスクリプション</a></strong>でのみ提供される製品となります。一方、AutoCAD Mechanical、AutoCAD Electrical、AutoCAD Map 3D などの AutoCAD ベースの業種別製品については、その提供方法に変更が加えられています。</p>
<p>2019 バージョンから AutoCAD サブスクライバ（サブスクリプション契約をお持ちの方）は、今迄、業種別製品で個別に提供してきた機能を<strong>ツールセット</strong>というかたちで導入して利用出来るようになります。すべての機能は AutoCAD のサブスクリプション契約に含まれることになるので、追加のお支払いは不要です。つまり、AutoCAD のサブスクリプション契約があれば、AutoCAD Mechanical、AutoCAD Electrical、AutoCAD Map3D などの業種別製品の機能をお使いいただけることになります。もちろん、それらの機能が不要な方は特に使用する必要はありません。</p>
<p class="asset-video"><iframe allow="autoplay; encrypted-media" allowfullscreen="" frameborder="0" height="281" src="https://www.youtube.com/embed/qHaQRUo0inA?feature=oembed" width="500"></iframe></p>
<p class="asset-video">新しいツールセットとしての業種別製品は、 「<strong>AutoCAD including specialized toolsets</strong>&#0160;（業種別ツールセット付き AutoCAD）」と呼ばれることになります。提供される業種別製品機能（従来製品名）と対応する個別のツールセット名、用途は次のとおりです。</p>
<ul>
<li>AutoCAD Architecture
<ul>
<li><strong>Architecture ツールセット</strong> ：建築設計&#0160;&#0160;&#0160;&#0160;</li>
</ul>
</li>
<li>AutoCAD Mechanical
<ul>
<li><strong>Mechanical ツールセット</strong>：機械設計</li>
</ul>
</li>
<li>AutoCAD Electrical
<ul>
<li><strong>Electrical ツールセット</strong>：電気制御設計</li>
</ul>
</li>
<li>AutoCAD Map 3D
<ul>
<li><strong>Map 3D ツールセット</strong>：地図情報</li>
</ul>
</li>
<li>AutoCAD MEP （AutoCAD MEP は従来日本で販売されていなかったので、英語版ツールセットで提供予定です）
<ul>
<li><strong>MEP ツールセット</strong>：設備設計</li>
</ul>
</li>
<li>AutoCAD Raster Design
<ul>
<li><strong>Raster Design ツールセット</strong>：ラスター 画像処理</li>
</ul>
</li>
<li>AutoCAD Plant 3D （AutoCAD P&amp;ID は Plant 3D に統合されています）
<ul>
<li><strong>Plant 3D ツールセット</strong>：3D プラント設計</li>
</ul>
</li>
</ul>
<p>紛らわしいのですが、AutoCAD Civil 3D はツールセットとしての提供はありません。従来通り、AutoCAD Civil 3D 製品としてサブスクリプションで提供されます。</p>
<p>AutoCAD including specialized toolsets は、Autodesk Desktop App や Autodesk Account ページを使って順次提供されます。必要に応じて各種ツールセットをダウンロードして、インストール済の AutoCAD 2019 にインストールすることが出来ます。</p>

<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb09fdac19970d-pi" style="display: inline;"><img alt="Autodesk_accounts" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb09fdac19970d image-full img-responsive" src="/assets/image_254795.jpg" title="Autodesk_accounts" /></a></p>
<p>業種別製品で既存のサブスクリプション契約をお持ちの方は、新しい AutoCAD including specialized toolsets に移行するか、現在契約している AutoCAD や AutoCAD 業種別製品のサブスクリプションを引き続き更新するかどうかを選択することが出来ます。</p>
<p>その他、AutoCAD including specialized toolsets に関しての FAQ が用意されていますので、詳細をお知りになりたい方は&#0160;<strong><a href="http://adndevblog.typepad.com/files/only-one-autocad-2019-customer-faq-ja.pdf">Only-One-AutoCAD-2019-よくある質問</a></strong> をご確認ください。</p>
<p><strong><a href="http://adndevblog.typepad.com/technology_perspective/2013/06/vertical-autocad-api-1.html" rel="noopener noreferrer" target="_blank">業種別 AutoCAD ベース製品の API カスタマイズ</a></strong> でご案内した業種別製品 API をお使いの開発者の方は、業種別製品 API ないし、OMF（Object Modeling Framework、AutoCAD Architecture 固有の ObjectARX ベース SDK） のような SDK の提供方法に変更はありません。適宜、2019 バージョンの開発環境を構築、当該 SDK を入手の上、お手持ちのアドインを純粋に AutoCAD 2019 へ移植していただく必要があります。&#0160;</p>
<p>なお、<strong><a href="https://apps.autodesk.com/ja" rel="noopener noreferrer" target="_blank">Autodesk App Store</a></strong> の製品カテゴリやアドインの<strong><a href="http://www.autodesk.co.jp/developapps" rel="noopener noreferrer" target="_blank"> 提出方法 </a></strong>は、従来通りで変更は予定されていません。</p>
