<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0263e96a857a200b-pi" style="display: inline;"></a>過去のブログ記事 <strong><a href="https://adndevblog.typepad.com/technology_perspective/2018/03/about-javascript-support-on-fusion-360-api.html" rel="noopener" target="_blank">Fusion 360 API での JavaScript サポートについて</a></strong> でご案内しましたが、数年前から Fusion 360 で JavaScript API によるスクリプトとアドインの作成が出来ない状態です。</p>
<p>ただし、現時点でも、JavaScript API を使用して作成済のスクリプトやアドインは引き続き実行出来るようになっています。</p>
<p>今回、新しく出されたアナウンスにより、JavaScript API 製のスクリプトとアドインの実行が、12 月にリリース予定の Fusion 360 から、この実行サポートを終了することが発表されていますので、お知らせいたします。</p>
<p>JavaScript API を使用するスクリプトとアドインは、Python または C ++ のいずれかに移行する必要があります。</p>
