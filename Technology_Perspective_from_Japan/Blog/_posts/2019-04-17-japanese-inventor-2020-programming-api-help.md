<p>Inventor 2020 API の日本語プログラミング用ヘルプ （admapi_24_0.chm）がリリースされました。 ZIP 圧縮したファイルをポストしていますので、次のリンクからダウンロードしてください。</p>
<p style="padding-left: 40px;"><span class="asset  asset-generic at-xid-6a0167607c2431970b0240a4a088f9200b img-responsive"><a href="https://adndevblog.typepad.com/files/admapi_24_0.zip"><strong>Inventor 2020 API の日本語プログラミング用ヘルプ </strong>をダウンロード</a></span></p>
<p>日本語版 Inventor 2020 に同梱されている API ヘルプは英語版オンライン ヘルプになっていて、<strong>C:\Users\Public\Documents\Autodesk\Inventor 2020\Local Help</strong>&#0160;にインストールされています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c8ea2c83970b-pi"> </a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a49cf383200b-pi" style="display: inline;"><img alt="Menu" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a49cf383200b image-full img-responsive" src="/assets/image_972012.jpg" title="Menu" /></a></p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c8ea2c83970b-pi"></a>ダウンロードした日本語版の Inventor 2020 API Help （admapi_24_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。&#0160;</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a47859df200d-pi" style="display: inline;"><img alt="Contents" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a47859df200d image-full img-responsive" src="/assets/image_823706.jpg" title="Contents" /></a></p>
<p>ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a453dab5200c-pi" style="display: inline;"><img alt="Blank" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a453dab5200c image-full img-responsive" src="/assets/image_172331.jpg" title="Blank" /></a></p>
<p>ページに何も表示されない場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、画面右下にある 「ブロックの解除」 (Windows 7)、「許可する」(Windows 10） にチェックを入れてください。</p>
<p style="text-align: center;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a44f1fc8200c-pi" style="display: inline;"><img alt="Block" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a44f1fc8200c img-responsive" src="/assets/image_295489.jpg" title="Block" /></a></p>
<p>なお、Inventor 2020 では従来の CHM ファイル形式とは別に、英語版 Inventor 2020 のみ、<strong><a href="https://help.autodesk.com/view/INVNTOR/2019/ENU" rel="noopener noreferrer" target="_blank">オンライン&#0160;ヘルプ</a></strong>&#0160;内に Programming Interface セクションが追加されています。残念ながら、日本語版 Inventor には反映されてはいませんが、記載されている内容は CHM 形式と同じものになっています。</p>
