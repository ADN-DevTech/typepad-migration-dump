<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c7e0d3b8970b-pi" style="float: right;"><img alt="Devweek_promo" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c7e0d3b8970b img-responsive" src="/assets/image_584189.jpg" style="margin: 0px 0px 5px 5px;" title="Devweek_promo" /></a>今年も、Developer Days を開催する時期になりました。Developer Days は、ADN（Autodesk Developer Network） プログラムにご加入いただいている開発者の皆様に、来るべきオートデスク テクノロジや API についてご案内するイベントです。ここ数年は、クラウド サービスとその API にフォーカスをあてたトピックを中心にご案内してきましたが、今年もの流れは変わりありません。</p>
<p>今回のイベントでは、大阪、東京での各1日の Developer Days とは別に、東京会場のみ Accelerator Workshop と題して、オートデスクの Web サービス API を使ったプロトタイプの集中開発の場もご提供します。</p>
<p>Developer Days、Accelerator Workshop ともに本来は ADN メンバー向けなのですが、新しい Web/クラウド開発者にも門戸を開く目的で ADN メンバー以外の参加も可能なようにしています。この場合には、事前に秘密保持契約書（NDA）への署名が必要となりますので、下記の概要/募集要項に沿って、その旨をお知らせください。お申込みは、Developer Days と Accelerator Workshop 個別にお願いします。なお、ADN メンバーの方には、別途、お申込みをご案内しています。</p>

<p><span style="font-size: 12pt;"><strong>Accelerator Workshop</strong></span></p>
<p><strong>概要と目的：</strong></p>
<p style="padding-left: 30px;">クラウド サービスや技術者にノートPC をご持参いただき、オートデスクにて、プロトタイプの開発作業に取り組んでいただきます。サポート対象となる Web サービス API に関するミニセミナーも順次開催致しますので、技術的な背景やテクニックを習得しながら、ゴールとなるプロトタイプを開発することが出来ます。このスタイルのイベントは、既に欧米で数回の実績があり、Web 開発の技術習得としてもご活用いただいています。Developer Days でご紹介する成功事例も、このイベントの成果の1 つです。<br />12 月17 日（木）夜には、来日する担当者も含め夕食会をご用意いたします。オートデスク社員との親睦やコミュニケーションにお役立てください。なお、最終日午後には、10 分程度で作成した成果物の概要発表をお願いいたします。&#0160;</p>
