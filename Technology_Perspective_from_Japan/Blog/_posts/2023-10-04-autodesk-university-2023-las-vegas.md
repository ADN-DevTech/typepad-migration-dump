<p>11月13日から15日まで、今年も米国 Las Vegas で <a href="https://conferences.autodesk.com/flow/autodesk/au2023/web/page/overview" rel="noopener" target="_blank">Autodesk University</a> が開催されます。オートデスクが推進している <a href="https://adndevblog.typepad.com/technology_perspective/2023/06/design-and-make-platforms.html" rel="noopener" target="_blank">Design &amp; Make プラットフォーム（デザインと創造 プラットフォーム）</a>と、その基盤になっている Autodesk Platform Services を多くご案内する予定です。主な内容は次の通りです。</p>
<ul>
<li>CEO の Andrew Anagnost と、CTO の Raji Arasu、ゲストによる基調講演（<a href="https://conferences.autodesk.com/flow/autodesk/au2023/sessioncatalog/page/inperson/session/1690395708359001BMNU" rel="noopener" target="_blank">General Session Day 1</a>）ぜひご覧ください。</li>
<li>11月14日（火）14:00-15:00に開催される <a href="https://conferences.autodesk.com/flow/autodesk/au2023/sessioncatalog/page/inperson/session/1690332400625001u9LM" rel="noopener" target="_blank">Autodesk Platform Services Forum</a> では、プラットフォームリーダー、データパイオニア、そして特別ゲストをお招きし、有意義な時間をお過ごしいただけます。Developer Enablement 部門 VP の Ben Cochran が、オートデスクの Design and Make Platform のビジョンと、開発者の重要性について語ります。続いて、Product Data 部門 VP の Shelly Mujtaba が、新しいデータ戦略と API サービスについてご紹介します。ステージでは、お客様の声を多数ご紹介し、データを最大限に活用する方法や、お客様との詳細なインタビューも予定しています！</li>
<li>APS のエキスパートに会い、直接、APS に関する質問をする機会です。<a href="https://aps.autodesk.com/customer-stories/moicon" rel="noopener" target="_blank">Moicon</a> は、Industry 4.0と Smart Manufacturing（分かり易いデータと KPI を 3D で可視化）に焦点を当てた素晴らしいデジタルツイン活用をおこなっています。 デジタルツインに反映される同社のセンサーで使って「実際の」ユースケースをご自身でお試しください。</li>
<li>APS をお使いのお客様が、示唆に富んだテーマでシアタートークに登壇します。11月13日（月）15:00-16:00 には、<a href="https://aps.autodesk.com/customer-stories/sweco" rel="noopener" target="_blank">Sweco</a> のチーフ・サステナビリティ・オフィサーである Elise Grosse 氏が「<a href="https://conferences.autodesk.com/flow/autodesk/au2023/sessioncatalog/page/inperson/session/1691427821327001hlie" rel="noopener" target="_blank">Sustainability by Design</a>」の講演をおこないます。Jacobs のリージョナル・テクノロジー・リードであるアンドリュー・デイビス氏は、11月13日（月）午後4時30分～5時30分に「<a href="https://conferences.autodesk.com/flow/autodesk/au2023/sessioncatalog/page/inperson/session/1691427453201001mBuo" rel="noopener" target="_blank">How Data Enables Better Outcomes Through Automation and Insights</a>」と題して講演します。</li>
<li>11月13日（月）午後3時30分から4時30分まで、「ソフトウェア開発者ミートアップ」を開催します： 11月13日（月）15:30～16:30に「<a href="https://conferences.autodesk.com/flow/autodesk/au2023/sessioncatalog/page/inperson/session/1689016852704001dLYz" rel="noopener" target="_blank">Coders Connect! Autodesk Platform Services</a>」では、APS で何が可能かを知るために実現した内容を共有し合うことが出来ます。</li>
</ul>
