<p><strong>Inventor2013 AddIn Wizard のインストール手順</strong></p>

<p>ウィザードのインストール手順を紹介したものです。<br />
VisualStudio2010などの VB C# などのテンプレートに用意され、「新しいプロジェクト」を選択した際に、採用される開発言語環境にてテンプレートを選択する事により、Inventorアドインを作成する際に、スケルトン環境がご利用できます。</p>

<p><iframe width="500" height="281" src="http://www.youtube.com/embed/as4dT6Hc4YM?feature=oembed" frameborder="0" allowfullscreen></iframe>&nbsp;<br />
<strong>Inventor2013 AddIn の作成とデバック</strong></p>

<p>Inventor2013 API  SDK内のアドインウィザード（アドイン作成を簡単に実装するためのスケルトン）を使い、Inventor Wizardアドインを作成する手順とデバックの仕方を説明したものです。<br />
アドイン実行時に、Inventor2013のソフトウェアーバージョンをメッセージボックス内に表現させます。</p>

<p><iframe width="500" height="281" src="http://www.youtube.com/embed/QYOyuH21vXw?feature=oembed" frameborder="0" allowfullscreen></iframe>&nbsp;<br />
次回は、外部EXEからInventor2013製品をCOM/ActiveXを使い制御する方法とデバックのビデオ公開を予定しています</p>
