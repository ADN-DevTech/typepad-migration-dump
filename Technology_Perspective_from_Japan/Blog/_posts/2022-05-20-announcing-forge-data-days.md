<p>お気づきかと思いますが、Forge ポータルの <a href="https://forge.autodesk.com/developer/documentation" rel="noopener" target="_blank"><strong>Documentation</strong> ページ</a>&#0160;に、最近新しい API が追加されています。<strong><a href="https://forge.autodesk.com/en/docs/fdx/v1/developers_guide/fd_overview/" rel="noopener" target="_blank">Data Exchange</a></strong> と <a href="https://forge.autodesk.com/en/docs/fusiondata/v1/developers_guide/overview/" rel="noopener" target="_blank"><strong>Fusion Data</strong></a> です。</p>
<p>まだ Beta 段階の Forge API ですが、これらはオートデスクが目指す<a href="https://adndevblog.typepad.com/technology_perspective/2021/10/forge-as-a-platform.html" rel="noopener" target="_blank"><strong>プラットフォーム</strong></a>戦略の基盤になる Forge Data をベースにした新しい API です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0282e15671e2200b-pi" style="display: inline;"><img alt="Forge_data_docs" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0282e15671e2200b image-full img-responsive" src="/assets/image_548362.jpg" title="Forge_data_docs" /></a></p>
<p>さまざまなフィードバックを経て、Forge の登場時から紹介されてきた HFDM のコンセプトを発展させたのが Forge Data です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0278807ded6b200d-pi" style="float: right;"><img alt="Tokyo_Japan_288x216" class="asset  asset-image at-xid-6a0167607c2431970b0278807ded6b200d img-responsive" src="/assets/image_400345.jpg" style="width: 150px; margin: 0px 0px 5px 5px;" title="Tokyo_Japan_288x216" /></a>この度、現状の Forge API の概要振り返りと可能な点、また、オートデスクのプラットフォーム戦略を含め、Forge Data の詳細を説明する無償イベントを&#0160;7 月 29 日にオートデスク東京オフィスで開催することになりました。</p>
<p>このイベントは、ワールドワイドで開催する対面形式のセミナーイベント <a href="https://forge.autodesk.com/forge-data-days" rel="noopener" target="_blank"><strong>Forge Data Days</strong></a> の 1 つです。東京会場では、日本人スタッフにより日本語での開催となります。</p>
<p>Forge Data Days は 1 日を費やすイベントとして予定されています。ご案内予定の内容は次のとおりです。</p>
<h4>Autodesk Forge プラットフォームの概要</h4>
<p style="padding-left: 40px;">&quot;現在の&quot; Forge がもたらすビジネス成長の機会をご紹介します。Forge エキスパートが、ライブデモや実際の使用例を交えて、何を、なぜ、いつ、どのように行うかを説明します。Forgeが提供する意思決定ダッシュボード、デジタルツイン、コンフィギュレーターを含むタスクの自動化、加工図を排除するダイレクトモデルからマシンデータへのフロー、その他多くの例をご覧いただけます。</p>
<p style="padding-left: 40px;">対象者：Forge 概要に興味をお持ちの方、Forge を初めて使う方</p>
<h4>Forge Data の概要</h4>
<p style="padding-left: 40px;">&quot;新しく登場する&quot; Forge Data は、個人とチームのためのツール（またはツール開発プラットフォーム）から、企業のデジタルトランスフォーメーションの実現、移行を強力に後押しする &quot;データ プラットフォーム&quot; を提供します。CAD によって異なるバイナリファイルの不透明なデータを、データアクセスが迅速、簡単、拡張可能、かつ連携可能なクラウドベースのデータベースへと導きます。その一端をご確認ください。</p>
<p style="padding-left: 40px;">対象者：Forge Data が提供する最新の計画についてお知りになりたい方</p>
<h4>Forge Data 開発の実際（複数セッションを予定）</h4>
<p style="padding-left: 40px;">設計・製造データのスピード、パワー、コラボレーション、セキュリティを低コストで実現する方法をご紹介します。このセッションでは、Model Derivative と Design Automation API、BIM 360/Autodesk Construction Cloud のモデル プロパティ、および Forge Data を基盤に構築された相互運用性を重視した Data Exchange API に焦点を当てます。最後に、AEC、製造、メディア＆エンターテインメントの 3 つの主要産業における、Forge Data に基づくクラウド情報モデル（Cloud Information Model）方向性についてご説明します。</p>
<p style="padding-left: 40px;">対象者：Forge API を利用してみたい開発者の方</p>
<p>チームやプロジェクトがより複雑になるにつれ、Forge はデータ エクスペリエンスの簡素化、相互運用性、直感的な操作性を保証します。Forge がどのようにデータをより細かく、標準化、透明化し、チーム、プロセス、ワークフローをつなぐ手助けをするのかをご覧いただけます。Forge の新しいデータ機能と API が生産性、自動化、収益性をどのように向上させるか、ぜひご覧ください。</p>
<p>お申し込み方法等のご案内は、6 月初旬を目途に本ブログでご案内する予定です。</p>
