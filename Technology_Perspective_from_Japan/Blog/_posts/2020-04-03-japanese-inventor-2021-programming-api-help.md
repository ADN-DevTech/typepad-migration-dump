<p>Inventor 2021 API の日本語プログラミング用ヘルプ （admapi_25_0.chm）がリリースされました。 ZIP 圧縮したファイルをポストしていますので、次のリンクからダウンロードしてください。</p>
<p>&#0160;<span class="asset  asset-generic at-xid-6a0167607c2431970b025d9b422c9f200c img-responsive"><a href="https://adndevblog.typepad.com/files/admapi_25_0.zip">Inventor 2021 API の日本語プログラミング用ヘルプ をダウンロード</a></span></p>
<p>日本語版 Inventor 2021 に同梱されている API ヘルプは英語版オンライン ヘルプになっていて、<strong>C:\Users\Public\Documents\Autodesk\Inventor 2021\Local Help</strong> にインストールされています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4f7909a200d-pi" style="display: inline;"><img alt="Inventor 2021 API Menu" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4f7909a200d image-full img-responsive" src="/assets/image_764593.jpg" title="Inventor 2021 API Menu" /></a></p>
<p>ダウンロードした日本語版の Inventor 2021 API Help （admapi_25_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a51c4541200b-pi" style="display: inline;"><img alt="Inventor 2021 API help" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a51c4541200b image-full img-responsive" src="/assets/image_898707.jpg" title="Inventor 2021 API help" /></a></p>
<p>&#0160;</p>
<p>ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。</p>
<p><img alt="Inventor 2021 online help empty" border="0" class="asset  asset-image at-xid-6a0167607c2431970b025d9b422fbf200c image-full img-responsive" src="/assets/image_295166.jpg" title="Inventor 2021 online help empty" /></p>
<p>ページに何も表示されない場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、右下にある 「許可する」 にチェックを入れてください。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a51c4776200b-pi" style="display: inline;"><img alt="Inventor 2021 online help property" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a51c4776200b img-responsive" src="/assets/image_142215.jpg" title="Inventor 2021 online help property" /></a></p>
<p>&#0160;</p>
<p>なお、従来の CHM ファイル形式とは別に、英語版 Inventor 2021 のみ、<a href="http://help.autodesk.com/view/INVNTOR/2021/ENU/" rel="noopener" target="_blank">オンライン ヘルプ</a> 内に Programming Interface セクションが追加されています。残念ながら、日本語版 Inventor には反映されてはいませんが、記載されている内容は CHM 形式と同じものになっています。</p>
