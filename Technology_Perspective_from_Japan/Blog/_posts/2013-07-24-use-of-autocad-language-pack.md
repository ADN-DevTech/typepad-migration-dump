<p>通常、日本のお客様は日本語版の AutoCAD をお使いいただいていると思います。ただ、海外業務の都合で英語版や中国語版、韓国語版の AutoCAD が必要になることもあるかと思います。また、AutoCAD API を使ったアプリケーション開発の場面では、<a href="http://apps.exchange.autodesk.com/" target="_self">Autodesk Exchange App ストア</a> で各国語に対応したアドオン アプリケーションの販売を計画されている方にも、テスト環境の入手の目的で日本語版以外の AutoCAD が必要になることも考えられます。</p>
<p>AutoCAD は、AutoCAD 2010 から1つの AutoCAD を利用した他言語対応を <strong>Language Pack（言語パック）</strong> の提供という形で進めてきました。当初、Language Pack の提供は、Subscription 保守契約にご加入いただいているお客様の特典として用意されたため、ご存じない方も多いのではないかと思います。</p>
<p>AutoCAD 2013 になって、どなたでも Language Pack を無償でお手持ちの AutoCAD 環境に適用することで、日本語以外の言語で AutoCAD を利用いただけるようになっています。もし、AutoCAD 2013 や AutoCAD 2014 をお持ちであれば、右上のメニューから Language Pack をダウンロードしていただくことが可能です。</p>

<p>Language Pack は自己解凍形式の EXE ファイルで提供されるので、ダウンロード後に任意フォルダーに解凍を済ませると、Language&#0160;Pack のインストーラが起動します。ボタン名の文字が各言語表記になってしまうので判読し難いのですが、あとは日本語版をインストールする際と同じ手順でインストールを完了することができます。もちろん、Language Pack には AutoCAD 本体は含まれませんので、日本語版の AutoCAD がインストールされている環境にしかインストールすることができません。次の画像は、簡体中国語の Language Pack インストーラ画面です。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b019101a2f165970c-popup" onclick="window.open( this.href, &#39;_blank&#39;, &#39;width=640,height=480,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0&#39; ); return false" style="display: inline;"><img alt="中国語インストーラ2" class="asset  asset-image at-xid-6a0167607c2431970b019101a2f165970c" src="/assets/image_641558.jpg" title="中国語インストーラ2" /></a>&#0160;</p>
<p>インストールが完了すると、デスクトップにその言語で起動するためのショートカットが作成されます。言語別のショートカットをダブルクリックすれば、その言語版の AutoCAD を起動することができます。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b019101a2e736970c-pi" style="display: inline;"><img alt="起動アイコン" border="0" class="asset  asset-image at-xid-6a0167607c2431970b019101a2e736970c" src="/assets/image_297143.jpg" title="起動アイコン" /></a></p>
<p style="text-align: left;">スタートメニューにも各国語のメニューが登録されています。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b019101a2f200970c-popup" onclick="window.open( this.href, &#39;_blank&#39;, &#39;width=640,height=480,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0&#39; ); return false" style="display: inline;"><img alt="スタートメニュー" class="asset  asset-image at-xid-6a0167607c2431970b019101a2f200970c" src="/assets/image_331621.jpg" title="スタートメニュー" /></a></p>
<p>インストールされた各言語のリソース ファイルは、日本語版 AutoCAD のインストール フォルダのサブフォルダにインストールされ、AutoCAD 本体を共有します。システム レジストリの内容も同様です。つまり、同じ環境で複数の言語の AutoCAD を同時に起動できる環境が整います。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b019101a2f241970c-popup" onclick="window.open( this.href, &#39;_blank&#39;, &#39;width=640,height=480,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0&#39; ); return false" style="display: inline;"><img alt="多言語UI" border="0" class="asset  asset-image at-xid-6a0167607c2431970b019101a2f241970c image-full" src="/assets/image_840244.jpg" title="多言語UI" /></a></p>
<p>Language Pack はコントロール パネルから個別に（言語別に）アンインストールすることもできますので、必要に応じて削除することもできます。</p>
<p>Language Pack を利用していただければ、別言語の AutoCAD をフルパッケージで購入する必要はもうありません。ぜひ、有効に活用してみてください。</p>
<p>1点注意してほしいのは、ビジネス的な判断から、日本語の Language Pack は提供されていない、という点です。例えば、英語版の AutoCAD 2014 を日本で購入いただいた場合でも、日本語 Language Pack を後からインストールして、日本語版 AutoCAD 2014 として利用することができません。</p>
