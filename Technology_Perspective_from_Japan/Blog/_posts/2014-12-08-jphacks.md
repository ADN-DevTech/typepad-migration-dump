<p>オートデスクは、東京大学が開催する学生を対象としたハッカソン イベント <a href="https://jphacks.com/" target="_blank"><strong>JPHACKS</strong></a> に協賛しています。本日、プレスリリースをおこないましたので、内容を転載したいと思います。</p>
<p>------------------------------------------------------------ ここから</p>
<p><span style="font-size: 13pt;"><strong>オートデスク、東京大学主催ハッカソン「JPHACK」にWeb サービス API を提供</strong></span></p>
<div id="section1">
<p>オートデスク株式会社は、12 月 13 日・14 日に国立大学法人東京大学本郷キャンパスで開催される国内最大規模の学生向けハッカソン「JPHACKS」に協賛して、参加者に当社の Web サービス API を提供します。</p>
<p>具 体的には、Web サービス上で 3D データを円滑に表示する「View &amp; Data API」と、複数の写真から 3D データを作成する「ReCap® Photo API」を提供して、デザインやエンジニアリングに関わる次世代の学生に自由な発想で活用してもらえるように支援します。当日は Google® Cardboard キットにスマートフォンを装着して、View &amp; Data API を応用したバーチャル・リアリティ技術のデモを披露します。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c71a317d970b-pi" style="display: inline;"><img alt="Gcb_lmv2" class="asset  asset-image at-xid-6a0167607c2431970b01b7c71a317d970b img-responsive" src="/assets/image_282167.jpg" style="width: 210px;" title="Gcb_lmv2" /></a> <a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d0a404e3970c-pi" style="display: inline;"><img alt="Gcb_lmv3" class="asset  asset-image at-xid-6a0167607c2431970b01b8d0a404e3970c img-responsive" src="/assets/image_984592.jpg" style="width: 210px;" title="Gcb_lmv3" /></a></p>
</div>
<div style="text-align: left;">当社は、豊かな発想でより良い世の中を創ろうとする次世代の設計者やデザイナーなどを育成する教育機関を支援しています。この一環で当社は、製造業、建設業 向け、CG・映像制作向けアプリケーションのライセンスを無償提供しています。JPHACK においても、新しい技術に興味を持つ技術力のある学生に当社の 3D 技術を体感していただき、継続的な支援につなげていきたいと考えています。</div>
<div id="section3">
<p>以上</p>
<p>------------------------------------------------------------ ここまで</p>
<p>出来るだけ多くの参加者にオートデスクの Web サービス API を使っていただき、<a href="http://adndevblog.typepad.com/technology_perspective/2014/10/autodesk-3d-hackathon-results.html" target="_blank"><strong>オートデスク 3D ハッカソン</strong></a>と同様、多様な結果が生まれることを期待しています。</p>
