<p><a href="http://adndevblog.typepad.com/technology_perspective/2013/03/new-features-on-autocad-2014-part1.html" target="_blank">AutoCAD 2014 の新機能</a> や <a href="http://adndevblog.typepad.com/technology_perspective/2014/04/new-features-on-autocad-2015-part3.html" target="_blank">AutoCAD 2015 の新機能</a> のブログ記事で紹介した&#0160;GEOGRAPHICLOCATION コマンドを使ったモデル空間背景に地図表示をする機能が、昨年の10月以降利用できなくなっている可能性があります。地理的位置の機能をお使いでない場合は特に対処する必要はありませんが、稀にしか利用されない方の中には、この点にお気づきでない方もいらっしゃるようです。</p>
<p>該当する方は、ぜひ、GEOGRAPHICLOCATION コマンドを実行して地図から位置を指定してみてください。次のようなエラーが表示される場合には、このコマンドを実装しているファイルを置き換える処置が必要です。対象製品は、AutoCAD 2014 と AutoCAD 2014 ベースの業種別製品、AutoCAD LT 2014、AutoCAD 2015 と AutoCAD 2015 ベースの業種別製品、AutoCAD LT 2015 です。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c74384eb970b-pi" style="display: inline;"><img alt="Geo_map_error" class="asset  asset-image at-xid-6a0167607c2431970b01b7c74384eb970b img-responsive" src="/assets/image_23298.jpg" style="width: 260px;" title="Geo_map_error" /></a></p>
<p style="text-align: left;">このダイアログで [閉じる] ボタンをクリックすると、更に下記のようなダイアログが表示されるはずです。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c74384d9970b-pi" style="display: inline;"><img alt="Geo_map_error2" class="asset  asset-image at-xid-6a0167607c2431970b01b7c74384d9970b img-responsive" src="/assets/image_131603.jpg" style="width: 400px;" title="Geo_map_error2" /></a></p>
<p>ここで「更新情報を見る」のリンクをクリックすることで、置き換え対象のファイルが公開されている <a href="http://knowledge.autodesk.com/support/autocad/downloads/caas/downloads/content/autodesk-C2-AE-autocad-C2-AE-and-autocad-lt-C2-AE-2014-and-2015-geolocation-online-map-hotfix.html" target="_blank">Autodesk Knowledge Network</a>（英語） が Web ブラウザ上に表示されるはずです。このページ下部のリンクから、お使いの製品に該当するリンクをクリックすると、更に別ページにジャンプします。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d0cd0d49970c-pi" style="display: inline;"><img alt="Hotfix" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d0cd0d49970c image-full img-responsive" src="/assets/image_798263.jpg" title="Hotfix" /></a></p>
<p>あとは、各ページから ZIP ファイルをダウンロードして入手した&#0160;AcGeoLocationUI.arx を、お使いの製品のインストール フォルダにあるものと置き換えるだけです。念のため、置き換え前に、インストールされている&#0160;AcGeoLocationUI.arx の拡張子を&#0160;AcGeoLocationUI.bak などに変更して保存しておきことをお勧めします。もちろん、作業中は AutoCAD を終了しておく必要があります。</p>
<p>詳細は、各ページの Readme ドキュメント（英語）か、次のドキュメントを参照してみてください。</p>
