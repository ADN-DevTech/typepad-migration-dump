<p>AU Las Vegasに先立って、Forge Platform API が拡張されました。といっても、いままで Beta 扱いだった&#0160;<strong><a href="http://adndevblog.typepad.com/technology_perspective/2016/08/reality-capture-api-and-related-products.html" rel="noopener noreferrer" target="_blank">Reality Capture API</a></strong> と BIM 360 API から Beta が外れ、正式にサポートされるようになっています。変更は <strong>Forge サイト</strong>（<strong><a href="https://forge.autodesk.com" rel="noopener noreferrer" target="_blank">https://forge.autodesk.com</a></strong>）、<strong>デベロッパ ポータル</strong>（<strong><a href="https://developer.autodesk.com">https://developer.autodesk.com</a></strong>）で確認することが出来ます。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d2bc45f8970c-pi" style="display: inline;"><img alt="Solutions" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d2bc45f8970c image-full img-responsive" src="/assets/image_270738.jpg" title="Solutions" /></a></p>
<p>特に、いままで特定のデベロッパの方にのみ提供していた Reality Capture API&#0160; を利用される場合には、デベロッパ キー（Client ID と Client Secret）の 作成や編集に当該 API の追加が必要になりますので注意してくだい。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c931e985970b-pi" style="display: inline;"><img alt="Enabled_api" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c931e985970b image-full img-responsive" src="/assets/image_441228.jpg" title="Enabled_api" /></a></p>
<p>クラウド クレジットの消費量については、Design Automation API と同様に使用する CPU 時間を基準として算出される予定です。同情報は、<strong><a href="https://forge.autodesk.com/pricing" rel="noopener noreferrer" target="_blank">https://forge.autodesk.com/pricing</a></strong> から参照いただくことが出来ます。その他 API の消費についての考え方は、以前と変わりありません。過去のブログ記事 <strong><a href="http://adndevblog.typepad.com/technology_perspective/2017/01/charge-scheme-for-forge-at-us-and-europe.html" rel="noopener noreferrer" target="_blank">欧米での Forge 課金の仕組み</a></strong> を参照してみてください。</p>
