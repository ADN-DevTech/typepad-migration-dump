<p>Autodesk Developer Networkの工藤　暁です。本日は先日最新バージョン1.1がリリースされました“Autodesk® FBX® Review”のご案内させて頂きます。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b019b00ccab55970d-pi" style="display: inline;"><img alt="Title" border="0" class="asset  asset-image at-xid-6a0167607c2431970b019b00ccab55970d image-full img-responsive" src="/assets/image_997910.jpg" title="Title" /></a></p>
<p><a href="http://area.autodesk.com/products/features/2013fbx">Autodesk® FBX® Review</a> は３Ｄオーサリングツールを使用せず３Ｄコンテンツを御確認頂けるWindows 7 と Windows 8 Store用のツールです。<a href="http://apps.microsoft.com/windows/en-us/app/fbx-review/6d5638e5-a427-4c34-9126-75ab2d587821">Windows Store</a> 及び <a href="http://apps.exchange.autodesk.com/3DSMAX/en/Detail/Index?id=appstore.exchange.autodesk.com%3aappautodesk00aefbx00aereview%3aen">Autodesk Exchange</a>より入手可能です。このツールを使用しアセットの共有や開発工程の確認を素早く行う事が可能となります。<a href="http://usa.autodesk.com/adsk/servlet/pc/item?siteID=123112&amp;id=10775847">FBX SDK</a>や<a href="http://usa.autodesk.com/adsk/servlet/pc/item?id=10775855&amp;siteID=123112">FBX Converter</a>にて作成されたＦＢＸファイル等の確認にも御利用頂けます。</p>
<ul type="disc">
<li>サポートファイルタイプ:.fbx, .3ds, .obj, .dxf, .dae, .bvh, .htr, .trc, .asf, .amc, .c3d, .aoa, .mcd.</li>
<li>サポートシェーディングモード: wireframe, faceted, shaded, textured, wireframe overlay.</li>
<li>scene light, shadow, &#0160;tessellationのON/OFF.</li>
<li>&#0160;DirectX 11 によるビジュアルエフェクト</li>
</ul>
<p><strong>iOS </strong><strong>サポートと</strong><strong> Almebic </strong><strong>サポートも予定しております。</strong></p>
<p>&#0160;</p>
<p>What’s new in FBX Review v1.1?</p>
<p>&#0160;</p>
<p>- Features:</p>
<p style="padding-left: 30px;">-Settings properly persisted across sessions</p>
<p style="padding-left: 30px;">-Enable/disable back-face culling(enabled by default)</p>
<p style="padding-left: 30px;">-Scene(file) name to the scene statistics heads-up display</p>
<p style="padding-left: 30px;">-Menu pane(ESC key) for easier access to info</p>
<p>&#0160;</p>
<p>- Improvements:</p>
<p style="padding-left: 30px;">-Loading screen look and resizing</p>
<p style="padding-left: 30px;">-Time display to include fractional frames</p>
<p style="padding-left: 30px;">-Texture search paths to try more locations</p>
<p style="padding-left: 30px;">-Loading/rendering performance</p>
<p style="padding-left: 30px;">-Memory use</p>
<p style="padding-left: 30px;">-Samples available as download package</p>
<p style="padding-left: 30px;">-Renamed &quot;SMPTE&quot; option in time display mode to &quot;Timecode&quot; for clarity</p>
<p>&#0160;</p>
<p>- Fixed issues with:</p>
<p style="padding-left: 30px;">-OBJ material rendering completely invisible</p>
<p style="padding-left: 30px;">-Crash on startup with some graphics h/w</p>
<p style="padding-left: 30px;">-Texture not loading when not embedded</p>
<p style="padding-left: 30px;">-Time bar slider sticking to random values using keyboard</p>
<p style="padding-left: 30px;">-Imported camera orbiting not behaving</p>
<p style="padding-left: 30px;">-Shadow maps when loading scenes with lights/shadows disabled and then turning them on</p>
<p style="padding-left: 30px;">-Main controls background not rescaling properly &gt; 1920x1080</p>
<p style="padding-left: 30px;">-Inverted normals loading DXF files</p>
<p style="padding-left: 30px;">-Crash loading OBJ files</p>
<p style="padding-left: 30px;">-Intense specular blow-out affecting some materials</p>
<p style="padding-left: 30px;">-Diffuse texture rendering as black material on some OBJ files</p>
<p style="padding-left: 30px;">-Cameras using an up target not always correctly oriented</p>
<p style="padding-left: 30px;">-Main controls hide with a single click/tap and not hide when using the mouse-wheel</p>
<p style="padding-left: 30px;">-Zooming using default Maya controls (Alt+RMB) horizontally inversed</p>
<p style="padding-left: 30px;">-Step forward/backward buttons/shortcuts not snapping back to closest frame</p>
<p style="padding-left: 30px;">-Dynamic tessellation break mesh silhouette</p>
<p style="padding-left: 30px;">-Cameras using aim/up target objects can be moved Anim</p>
