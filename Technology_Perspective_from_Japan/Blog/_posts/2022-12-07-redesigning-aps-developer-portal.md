<p>Autodesk Forge の Autodesk Platform Services への改名にともない、開発者ポータル サイトが再デザインされました。URL も <a href="https://forge.autodesk.com" rel="noopener" target="_blank">https://forge.autodesk.com</a> から <a href="https://aps.autodesk.com" rel="noopener" target="_blank">https://aps.autodesk.com</a> に変更されています。従来の URL <a href="https://forge.autodesk.com" rel="noopener" target="_blank">https://forge.autodesk.com</a> は、<a href="https://aps.autodesk.com" rel="noopener" target="_blank">https://aps.autodesk.com</a> にリダイレクトされるようになります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02af14a91126200b-pi" style="display: inline;"><img alt="Portal_renewal" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02af14a91126200b image-full img-responsive" src="/assets/image_855617.jpg" title="Portal_renewal" /></a></p>
<p><a href="https://aps.autodesk.com/code-samples" rel="noopener" target="_blank">https://aps.autodesk.com/code-samples</a> に記載されているサンプルは、サイトのリニューアルにともない、整理されています。ソースコードを記載する GitHub リポジトリは、<a href="https://github.com/autodesk-platform-services" rel="noopener" target="_blank">https://github.com/autodesk-platform-services</a> に集約されています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02af1c92c0d0200d-pi" style="display: inline;"><img alt="Github_repo" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02af1c92c0d0200d image-full img-responsive" src="/assets/image_937320.jpg" title="Github_repo" /></a></p>
<p>以前、サンプル ページに記載していたサンプルは、<a href="https://github.com/Autodesk-Forge" rel="noopener" target="_blank">https://github.com/Autodesk-Forge</a>、 <a href="https://github.com/Developer-Autodesk" rel="noopener" target="_blank">https://github.com/Developer-Autodesk</a> からご参照いただけます。</p>
