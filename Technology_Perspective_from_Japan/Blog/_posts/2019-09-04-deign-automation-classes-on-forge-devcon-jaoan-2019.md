<p>10 月 8 日～ 9 日に開催する Forge DevCon 2019 では、Design Automation API についてご案内するセッションを 2 クラスご用意しています。</p>
<p>8 日の <strong><a href="https://www.event-entry.net/auj/reg/modal.php?id=auj025" rel="noopener" target="_blank">1S-3 あなたの AutoCAD アドインアプリをクラウドで活用</a></strong> では Design Automation API for AutoCAD を、<strong><a href="https://www.event-entry.net/auj/reg/modal.php?id=auj027" rel="noopener" target="_blank">2S-23 Revit アドインのクラウド移行で繋げる BIM ワークフロー</a></strong> では Design Automation API for Revit を、それぞれご案内いたします。</p>
<p>いずれも、現在、Beta としてご提供していて、近々に正式なリリースが見込まれる <a href="https://forge.autodesk.com/en/docs/design-automation/v3/developers_guide/overview/" rel="noopener" target="_blank">Design Automation API <strong>v3</strong></a>（<strong>バージョン ３</strong>）をベースに、既にお持ちのアドイン アプリケーション資産をどのように活用していくか、を骨子として、シンプルな例を用いてご案内していく予定です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4a84474200d-pi" style="display: inline;"><img alt="Migration" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4a84474200d image-full img-responsive" src="/assets/image_869761.jpg" title="Migration" /></a></p>
<p><a href="https://forge.autodesk.com/en/docs/design-automation/v3/developers_guide/overview/" rel="noopener" target="_blank">Design Automation API <strong>v3</strong></a> では、AutoCAD、Revit のコアエンジンのほかにも、Inventor、3ds Max のコアエンジンも導入されます。残念ながら（大人の都合で）、それらの専用セッションを持つことが出来なかったのですが、コアエンジン、すなわち、アドイン（プラグイン）アプリケーションの実装が異なるだけで、考え方や実際に利用する RESTFul API 部分の endpoint は共通です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4a83576200d-pi" style="display: inline;"><img alt="Da_v3" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4a83576200d image-full img-responsive" src="/assets/image_683171.jpg" title="Da_v3" /></a></p>
<p>コアエンジンの違いにかかわらず、プロセスの自動化に貢献する Design Automation API について、関係セッションへの参加をご検討いただけますと幸いです。</p>
<p>Autodesk University Japan、Forge DevCon への参加費用は共通で、ご登録は<a href="http://www.autodesk.co.jp/auj" rel="noopener" target="_blank"><strong>http://www.autodesk.co.jp/auj&#0160;</strong></a>でお願いいたします。</p>
