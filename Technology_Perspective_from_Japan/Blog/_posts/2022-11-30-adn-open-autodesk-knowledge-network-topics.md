<p><a href="https://www.autodesk.co.jp/developer-network/membership-options" rel="noopener" target="_blank">Autodesk Developer Network（ADN）</a>は、主に、オートデスクのデスクトップ製品用 API をサポートする有償サポート プログラム（年間契約）です。詳しくは、<a href="https://adndevblog.typepad.com/technology_perspective/2017/12/adn-faq.html">ADN よくある質問</a> に記事に譲りますが、<a href="https://www.autodesk.co.jp/developer-network/open" rel="noopener" target="_blank">ADN オープン</a> と称して、ブログ記事などを介したデスクトップ製品用 API 情報の無償公開もおこなっています。</p>
<p>このブログでもデスクトップ製品 API を題材にすることがありますが、実際のプログラムを記載することは多くありません（除く、Autodesk Platform Services - 旧 Forge 記事）。プログラムの具体的な内容は、ブログ記事と区別して Autodesk Knowledge Network 記事として記載しているためです。</p>

<p>Autodesk Knowledge Network には、よくある質問を中心に、次のような内容を記載しています。</p>
