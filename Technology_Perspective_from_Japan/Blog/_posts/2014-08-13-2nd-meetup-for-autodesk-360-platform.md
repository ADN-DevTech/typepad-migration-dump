<p>以前、<a href="http://adndevblog.typepad.com/technology_perspective/2014/06/%E3%82%AF%E3%83%A9%E3%82%A6%E3%83%89%E3%82%B5%E3%83%BC%E3%83%93%E3%82%B9%E3%81%A8-api-%E3%81%AB%E3%81%A4%E3%81%84%E3%81%A6%E3%81%AE%E3%83%96%E3%83%AA%E3%83%BC%E3%83%95%E3%82%A3%E3%83%B3%E3%82%B0-meetup-.html" target="_blank">開催した meetup イベント</a>の2回目を、9月10日 19:00~ に開催します。内容は前回とほぼ同様です。</p>
<p>この Meetup は、Autodesk 360 クラウド プラットフォームと Web サービス API の勉強会です。3D モデルや 3D デザインをクラウドを介して Web や モバイルで活用したい Web デザイナー、プログラマ、その他、オートデスクが既存のビジネスで思いつかないようなアイデアをお持ちのクリエイティブな方など、大歓迎です。</p>
<p>ご賛同、ご参加いただける方は、<strong><a href="http://www.meetup.com/3D-design-technology-on-Cloud-Mobile-developer/events/199746952/" target="_blank">http://www.meetup.com/3D-design-technology-on-Cloud-Mobile-developer/events/199746952/</a>&#0160;</strong>から、<strong>Join and RSVP</strong>&#0160;をクリックしてご参加ください。</p>

<p>なお、従来、Viewing Service API の名前でご案内してきた Web サービス API 名は、<strong>View &amp; Data Service API</strong>&#0160;に変更されていますので、当日は、こちらを反映したご案内となります。&#0160;</p>
