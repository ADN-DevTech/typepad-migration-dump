<p>今回の記事では引き続き、Inventor 2021での「<strong>Inventor のエクスペリエンス</strong>」、「<strong>連携ワークフロー</strong>」、「<strong>お客様の要求に基づく改善</strong>」の機能拡張についてお伝えをしたいと思います。</p>
<p>&#0160;</p>
<p><strong><strong class="ph b">Inventorのエクスペリエンス</strong></strong></p>
<ul>
<li class="p">ダーク テーマのプレビュー</li>
</ul>
<p class="p">Inventor 2020 では、新しい明るい色調のインタフェースが提供されました。Inventor 2021 には、Inventor の新しいダーク テーマのプレビューが含まれています。</p>
<p class="p"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4febc44200d-pi" style="display: inline;"><img alt="DarkTheme" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4febc44200d image-full img-responsive" src="/assets/image_112976.jpg" title="DarkTheme" /></a></p>
<p class="p">&#0160;</p>
<ul>
<li class="p">コマンドの UI と生産性の機能強化</li>
</ul>
<p class="p">プロパティ パネルは、引き続き Inventor のより多くの領域で使用されるようになっています。[曲げ]、[コイル]、[結合]、[オブジェクトをコピー]、[デカール]、[面を削除]、[分割]、[厚み/オフセット]、およびフレーム ジェネレータのすべてのコマンドで、合理化されたプロパティ パネル ワークフローを使用できるようになりました。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4febc4f200d-pi" style="display: inline;"><img alt="CommandUI" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4febc4f200d image-full img-responsive" src="/assets/image_97134.jpg" title="CommandUI" /></a></p>
<p>&#0160;</p>
<p>Inventor 2021でのエクスペリエンスの機能拡張についてご紹介した、以下の動画もご確認ください。</p>
<p><iframe allowfullscreen="" frameborder="0" height="270" src="//www.youtube.com/embed/ZmbkgQ-F_VU" width="480"></iframe></p>
<p>&#0160;</p>
<p><strong><strong class="ph b">連携ワークフロー</strong></strong></p>
<ul>
<li>相互運用性</li>
</ul>
<p>Revit AnyCAD: カスタム加工業者は、ビルディング インフォメーション モデリングを導入することでお客様に多くの価値を提供したいと考えていることと思います。Autodesk Inventor は、Revit データとシームレスに統合することで、建設業界に関係する企業に提供されています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b025d9b4971c0200c-pi" style="display: inline;"><img alt="Revit" border="0" class="asset  asset-image at-xid-6a0167607c2431970b025d9b4971c0200c image-full img-responsive" src="/assets/image_946556.jpg" title="Revit" /></a></p>
<ul style="list-style-type: square;">
<li>作業時に、挿入された Revit データへの関連リンクを維持することで、Revit と Inventor との相互運用性が改善されました。Revit プロジェクトは、ローカル ファイルからアクセスしたり、Autodesk BIM 360 の共通データ環境を使用してクラウドを介して調整することができます。</li>
</ul>
<p>&#0160;</p>
<ul style="list-style-type: square;">
<li>設計プロセス中の変更を避けることはできません。Revit モデルが変更された場合は、Inventor モデルを更新して適合させる必要があります。Revit モデルはファイル変換を行うことなく Inventor アセンブリ内で参照されるため、Revit プロジェクトに加えた変更は Inventor アセンブリ内でも更新されます。</li>
</ul>
<p>&#0160;</p>
<p>Inventor 2021での Revit との AnyCAD 連携の機能拡張についてご紹介した、以下の動画もご確認ください。</p>
<p><iframe allowfullscreen="" frameborder="0" height="270" src="//www.youtube.com/embed/CNfebRrGe7Y" width="480"></iframe></p>
<p>&#0160;</p>
<ul>
<li><strong class="ph b">トランスレータ</strong></li>
</ul>
<p>JT エクスポートで、ソフトウェア解釈可能なセマンティック製品製造情報(PMI)のエクスポートがサポートされるようになりました。 以前は、[製品の製造情報]ボックスをオンにすると、グラフィック PMI データがエクスポートされていました。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a5237d1a200b-pi" style="display: inline;"><img alt="JT" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a5237d1a200b img-responsive" src="/assets/image_873561.jpg" title="JT" /></a></p>
<p>&#0160;</p>
<ul style="list-style-type: square;">
<li>Parasolid バージョン 32 のインポートに対するサポートが追加されました。</li>
</ul>
<p>&#0160;</p>
<ul style="list-style-type: square;">
<li>Solid Edge、SolidWorks、Unigraphics のインポート形式と AnyCAD 形式が更新されました。</li>
</ul>
<p>&#0160;</p>
<p><strong><strong class="ph b">お客様の要求に基づく改善</strong></strong></p>
<p>Inventor コミュニティでは、Inventor Ideas および Feedback Community からフィードバックやリクエストを継続して提供します。 お客様からのフィードバックは、パーツ、アセンブリ、図面の改善に役立っています。</p>
<p>&#0160;</p>
<ul>
<li>アセンブリ</li>
</ul>
<p>Autodesk Inventor 2021 では、チューブ＆パイプ、アセンブリ構造、およびファイル命名規則(LT では使用できません)に関してアセンブリ レベルの拡張が行われています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b025d9b49727d200c-pi" style="display: inline;"><img alt="Assembly" border="0" class="asset  asset-image at-xid-6a0167607c2431970b025d9b49727d200c image-full img-responsive" src="/assets/image_731733.jpg" title="Assembly" /></a></p>
<p>&#0160;</p>
<ul style="list-style-type: square;">
<li>[ファイルを命名]ダイアログ ボックスが[ミラー]および[コピー]コマンドに拡張されました。</li>
</ul>
<p>&#0160;</p>
<ul style="list-style-type: square;">
<li>ファイルを保存するときに、ファイルを保存するかどうかの既定の動作を設定する新しいオプションが有効になりました。保存ダイアログの新しい[状態を保存]列は、保存を促すプロンプトが表示されるファイルと保存を促す理由を理解するのに役立ちます。</li>
</ul>
<p>&#0160;</p>
<ul style="list-style-type: square;">
<li>アセンブリの構造を容易にするために、軸コンポーネント ジェネレータに使用できる機能をボルト締結に拡張しました。このオプションをオンにすると、ディスク上にフォルダまたはサブアセンブリは作成されず、個々のコンポーネントがアセンブリに直接挿入されます。この機能により、アセンブリを構造化し、ニーズに最も適した日付を構成するためのオプションが増えます。</li>
</ul>
<p>&#0160;</p>
<p>Inventor 2021でのアセンブリの機能拡張についてご紹介した以下の動画もご確認ください。</p>
<p><iframe allowfullscreen="" frameborder="0" height="270" src="//www.youtube.com/embed/9-HugqKkcUc" width="480"></iframe></p>
<p>&#0160;</p>
<ul>
<li>パーツ</li>
</ul>
<p>Autodesk Inventor 2021 では、シート メタル、マルチボディ パーツ、3D 注記のパーツ レベルの機能が強化されました。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4febd0f200d-pi" style="display: inline;"><img alt="Parts" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4febd0f200d img-responsive" src="/assets/image_50593.jpg" title="Parts" /></a></p>
<ul style="list-style-type: square;">
<li>シート メタル フランジを参照面または面で定義できるようになったため、正確な角度に簡単に一致させることができるようになりました(LT では使用できません)。</li>
</ul>
<p>&#0160;</p>
<ul style="list-style-type: square;">
<li>使い慣れたウィンドウ選択を使用してパーツ内の複数のボディまたは面を選択することにより、マウスのクリック操作を大幅に削減できます。</li>
</ul>
<p>&#0160;</p>
<ul style="list-style-type: square;">
<li>引出線注記や一般注釈に言及することで、3D 注記とカスタム iProperty の間の整合性を維持できます。</li>
</ul>
<p>&#0160;</p>
<p>Inventor 2021でのパーツの機能拡張についてご紹介した以下の動画もご確認ください。</p>
<p><iframe allowfullscreen="" frameborder="0" height="270" src="//www.youtube.com/embed/MJmh7trzwpo" width="480"></iframe></p>

<ul>
<li>チューブ＆パイプ</li>
</ul>
<p>チューブ＆パイプ設計のファイル命名の機能は引き続き改善されています(LT では使用できません)。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b025d9b4972e3200c-pi" style="display: inline;"><img alt="TubeAndPipe" border="0" class="asset  asset-image at-xid-6a0167607c2431970b025d9b4972e3200c image-full img-responsive" src="/assets/image_849830.jpg" title="TubeAndPipe" /></a></p>
<p>&#0160;</p>
<p>作成するルートが複数存在するということはよくあります。[自動ルート]オプションを使用する際に[適用]ボタンを使用すると、コマンドを実行したままの状態で次のルートを作成できます。</p>
