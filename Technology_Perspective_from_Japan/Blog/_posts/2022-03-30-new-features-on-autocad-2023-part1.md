<p>誕生から 40 年を迎える AutoCAD、今年も新バージョンとなる AutoCAD 2023 と AutoCAD LT 2023 がリリースされました。まずは、概要のご紹介です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02788074066d200d-pi" style="display: inline;"><img alt="40th_aniversary" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02788074066d200d image-full img-responsive" src="/assets/image_988088.jpg" title="40th_aniversary" /></a></p>
<p>昨年と同様、Windows 版と Mac 版の同時リリースとなります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02788073d26e200d-pi" style="display: inline;"><img alt="2023s" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02788073d26e200d image-full img-responsive" src="/assets/image_241112.jpg" title="2023s" /></a></p>
<p>また、AutoCAD Web アプリ、AutoCAD モバイルアプリを含めた AutoCAD ファミリーとした包括運用による柔軟なワークフローに対応しています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02942f9f7420200c-pi" style="display: inline;"><img alt="Autocad_family" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02942f9f7420200c image-full img-responsive" src="/assets/image_532994.jpg" title="Autocad_family" /></a></p>
<p>Windows 版では、業種別ツールキットの利用も可能なので、業務に合わせてお使いいただくことで、AutoCAD 単体の利用に比較して、さらに生産性を向上させることが出来るようになります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02942fa204f9200c-pi" style="display: inline;"><img alt="Ist" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02942fa204f9200c image-full img-responsive" src="/assets/image_11760.jpg" title="Ist" /></a></p>
<ul style="list-style-type: circle;">
<li>なお、昨年の&#0160;<a href="https://adndevblog.typepad.com/technology_perspective/2021/05/reborned-autocad.html" rel="noopener" target="_blank"><strong>生まれ変わった AutoCAD</strong></a> の内容は、このバージョンでも同じなので、実質的な新規販売は <span style="text-decoration: underline;">単体の AutoCAD</span> と<span style="text-decoration: underline;">業種別ツールセットを含む AutoCAD Plus（別名 AutoCAD including specialized toolsets）</span>となります。Mac 版には業種別ツールセットを含む AutoCAD Plus の提供はありません。</li>
</ul>
<p>デスクトップ版 AutoCAD 2023 がサポートする OS 環境は次のとおりです。</p>
<p><strong>Windows 10</strong></p>
<ul>
<li>Microsoft がサポート中の Windows 10、Windows 11
<ul>
<li>注：<strong><a href="https://knowledge.autodesk.com/ja/customer-service/account-management/users-software/previous-versions/previous-version-support-policy">製品サポートのライフサイクル</a></strong></li>
</ul>
</li>
<li>64 ビット版のみの提供（32 ビット版の提供はなし）</li>
<li>.NET Framework Version 4.8 以降</li>
</ul>
<p><strong>Mac</strong></p>
<ul>
<li>Apple<sup>®</sup> macOS<sup>®</sup> Monterey v12 (Update 2022.2 が必要)</li>
<li>Apple macOS Big Sur v11</li>
<li>Apple macOS Catalina v10.15</li>
<li>Apple macOS Mojave v10.14</li>
</ul>
<p><a id="open_save"></a>AutoCAD 2023 は、引き続き 2018 図面ファイル形式を採用します。DWG と DXF の入出力の能力は従来と同様です。</p>
<p><strong>図面読み込み：</strong></p>
<ul>
<li>すべての AutoCAD バージョンで作成した DWG ファイル
<ul>
<li>TrustedDWG を扱うことが出来ます</li>
</ul>
</li>
<li>すべての AutoCAD バージョンで作成した DXF ファイル</li>
</ul>
<p><strong>図面の保存：</strong></p>
<ul>
<li>R14, 2000, 2004, 2007, 2010, 2013, 2018 形式の DWG ファイル</li>
<li>R12, 2000, 2004, 2007, 2010, 2013, 2018 形式の DXF ファイル</li>
</ul>
