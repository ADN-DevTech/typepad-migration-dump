<div id="section1">
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d133d54c970c-pi" style="display: inline;"><img alt="Cloud_API_Seminar_Title" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d133d54c970c image-full img-responsive" src="/assets/image_2015.jpg" title="Cloud_API_Seminar_Title" /></a></p>
<p>Autodesk A360 をはじめ、オートデスクは既に数多くのクラウド サービスを提供しています。加えて、オートデスク クラウド サービスで提供される API を利用することで、プラットフォームとして既存の Web ページやサービス、モバイル アプリケーションに新しい機能を組み込んだり、クラウド サービスの機能を自動化するようなカスタマイズが可能です。そこで、今回、API を持つ クラウドサービスを一挙にご紹介するセミナーを開催することになりました。設計やその環境がクラウド化していくにあたって、何が出来るようになるのか、どのように実装していくのかなど、今後の対応のために概要を把握していただくことが出来ます。開催概要と、ご紹介する内容、お申込み方法は次のとおりです。&#0160;</p>
</div>

</div>
<div>&#0160;</div>
<div style="text-align: left;"><strong><span style="font-size: 1.17em;">内容 － アジェンダ</span></strong></div>
<div>&#0160;</div>
<div style="padding-left: 30px;"><strong>13:00～13:20<span style="font-size: 12pt;"> クラウド サービスの基本</span></strong></div>
<div>
<p style="padding-left: 60px;">オートデスクのクラウドサービスの構成と仕組み、ライセンスの考え方について簡単にご説明します。</p>
<p style="padding-left: 30px;"><strong>13:20～14:20 <span style="font-size: 12pt;">View and Data API</span></strong></p>
<p style="padding-left: 60px;">Autodesk A360 上で CAD データの表示、検索を受け持つ基本機能を抜き出して、独自にカスタマイズできる API です。その利用方法や事例をご紹介します。</p>
<p style="padding-left: 30px;"><strong>14:20～14:30 &#0160; &#0160; &lt;休憩&gt;</strong></p>
<p style="padding-left: 30px;"><strong>14:30～15:20 <span style="font-size: 12pt;">AutoCAD I/O</span></strong></p>
<p style="padding-left: 60px;">AutoCAD が持つコア エンジンをクラウドに搭載してカスタマイズ利用できる環境です。その成り立ちとカスタマイズ方法とサンプルとデモをご案内します。</p>
<p style="padding-left: 30px;"><strong>15:20～16:20<span style="font-size: 12pt;"> BIM 360 Glue API</span></strong></p>
<p style="padding-left: 60px;">BIM とクラウドを組み合わせたコラボレーション サービスである BIM 360 Glue API で何が出来るか、概要と事例をご紹介します。</p>
<p style="padding-left: 30px;"><strong>16:20～16:30 &#0160; &#0160; &lt;休憩&gt;</strong></p>
<p style="padding-left: 30px;"><strong>16:30～17:20 <span style="font-size: 12pt;">Fusion 360 API</span></strong></p>
<p style="padding-left: 60px;">多彩な API を持つ Fusion 360 API について、開発環境や言語、実装差に加え、Windows 版、Mac 版との違いについて、コードサンプルやデモでご案内します。</p>
