<p>Autodesk Platform Services をはじめ、オートデスクのクラウド サービスの稼働状況を示すヘルスダッシュボードには、数多くの製品や API が記載されています。</p>
<p>最近では、クラウドを利用する製品や機能の増加にともない、調べたい製品を見出すのに少し探さなければならない状態になっていました。（2025年1月15日現在 <strong>131 個</strong>）</p>
<p>そこで、今回、既定で表示する製品数が従来より抑止されるようになっています。といっても、設定を変更すればすべての製品/API を表示出来るのは言うまでもありません。</p>
<p>ご注意いただきたいのは、<a href="https://health.autodesk.com/" rel="noopener" target="_blank">https://health.autodesk.com/</a> を表示して既定で表示される内容から、Autodesk Platform Services の API が除外されている点です。この場合、ページ上部の「Products」フィルターを使って、表示させたい製品/API を指定することが出来ます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3c9404e200c-pi" style="display: inline;"><img alt="Products_combobox" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3c9404e200c image-full img-responsive" src="/assets/image_627868.jpg" title="Products_combobox" /></a></p>
<p>「Products」フィルターでは、Autodesk Platform Services 全体の他、個別 API 毎に表示を指定することが出来ます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3c94059200c-pi" style="display: inline;"><img alt="Aps_filter" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3c94059200c img-responsive" src="/assets/image_341721.jpg" title="Aps_filter" /></a></p>
<p>フィルターの指定が面倒な場合には、直接、<a href="https://health.autodesk.com/?search=APS" rel="noopener" target="_blank"><strong>https://health.autodesk.com/?search=APS</strong></a> の URL を参照することも出来ます。</p>
