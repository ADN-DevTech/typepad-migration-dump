<p><span style="background-color: #ffff00;">Autodesk Platform Services（旧 Autodesk Forge）の課金制度は、2022年11月7日に Autodesl Flex に移行しています。詳しくは、<strong><a href="https://adndevblog.typepad.com/technology_perspective/2022/11/flex-token-adoption-into-aps-on-11-7.html" rel="noopener" style="background-color: #ffff00;" target="_blank">11月7日 APS へ Flex トークンを導入</a></strong>&#0160;の記事をご確認ください。</span></p>
<p><span style="text-decoration: line-through;"><span style="background-color: #ffffff;"><em><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02942f98cf82200c-pi" style="display: inline; background-color: #ffffff;"><img alt="Brand-Portrait-SF-Gallery-1989" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02942f98cf82200c image-full img-responsive" src="/assets/image_604421.jpg" title="Brand-Portrait-SF-Gallery-1989" /></a><br /></em></span></span></p>
<p><span style="text-decoration: line-through;"><span style="background-color: #ffffff;"><strong><em>この情報は、Token Flex Enterprise Business Agreement (EBA) を介して Forge にアクセスするお客様を対象としたものではありません。EBA のお客様には、契約管理者の方へ、別途、情報が提供される予定です。&#0160;</em></strong></span></span></p>
<p><span style="text-decoration: line-through;">この記事は、3月29日（GMT） からの実施を予定している Forge API の価格変更について、Forge をお使いの皆様にお知らせするものです。&#0160;</span></p>
<p><span style="text-decoration: line-through;">オートデスクは、<a href="http://www.autodesk.co.jp/flex" rel="noopener" target="_blank">Autodesk Flex</a> と呼ばれる新しい従量制モデルを導入しています。 今後 1 年間で、Forge&#0160; のサインアップ、トライアル、年間サブスクリプションのワークフローに一連の変更を加え、Forge&#0160; の消費を&#0160; Flex トークン（現在使用されている クラウドクレジットに代替）に移行する予定です。 この結果、Forge&#0160; へのサインアップと追加消費トークンの購入がよりシンプルになります。</span></p>
<p><span style="text-decoration: line-through;">しかしながら、移行作業が複雑であるため、段階的にこれらの変更を行う必要があり、移行プロセス中の皆様のご理解をお願いいたします。これらの移行手順について不都合や混乱が生じてしまった場合には、お手数ですが&#0160;<a href="mailto:forge.sales@autodesk.com" rel="noreferrer noopener" target="_blank">forge.sales@autodesk.com</a> までご連絡ください。</span></p>
<p><span style="text-decoration: line-through;">2022年3月29日に行われる移行の第一段階として、クラウドクレジットの価格と購入モデルが変更される予定です。（<span style="background-color: #ffff00;">下記は2022年3月29日以降に適用</span>）</span></p>
<ol role="list" start="1">
<li aria-setsize="-1" data-aria-level="1" data-aria-posinset="1" data-font="" data-leveltext="%1." data-listid="28">
<p><span style="text-decoration: line-through;">クラウドクレジットは、1 クレジットあたり 1 米国ドル（または現地通貨換算額）から 3 米国ドル（または現地通貨換算額）に価格改定されます。 （<span style="background-color: #ffff00;">日本では現在 1 クラウドクレジット税抜 160 円です。</span>）</span></p>
</li>
</ol>
<ol role="list" start="2">
<li aria-setsize="-1" data-aria-level="1" data-aria-posinset="2" data-font="" data-leveltext="%1." data-listid="28">
<p><span style="text-decoration: line-through;">（Autodesk Flex の完全移行までForge で使用するクラウドクレジット追加購入分は、）Autodesk から直接購入のみとなり、Autodesk eStore や Autodesk パートナーを通じた購入はできなくなります。</span></p>
</li>
</ol>
<p><span style="text-decoration: line-through;">クラウドクレジットの価格は Autodesk Flex 価格に合わせた再設定に対応するため、Forge API の消費クラウドクレジット数も改定します。 以下は、2022年3月29日に自動的に適用される新しい消費クラウドクレジット数です。</span></p>
<p><span style="text-decoration: line-through;">クラウドクレジット単価は、1クラウドクレジットあたり 3 米国ドル（480円-税抜）です。 &#0160;</span></p>
<p><span style="text-decoration: line-through;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02a30d47f489200b-pi" style="display: inline;"><img alt="Table_revised" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02a30d47f489200b image-full img-responsive" src="/assets/image_991171.jpg" title="Table_revised" /></a></span></p>
<p><span style="text-decoration: line-through;">*注1）Model Derivative API を使用した IFC ファイルの変換処理は、コンプレックス ジョブとして分類されるようになりました。以前、シンプル ジョブ に分類されていました。ンプレックス ジョブとシンプル ジョブ については、<a href="https://forge.autodesk.com/ja/pricing" rel="noopener" target="_blank">https://forge.autodesk.com/ja/pricing</a> をご参照ください。</span></p>
<p><span style="text-decoration: line-through;"><sup>†</sup>注2：3 月 29 日以降、Reality Capture API は新価格、ギガピクセルあたり 1 クラウドクレジットという暫定的な料金が適用されます。&#0160;</span></p>
<p><span style="text-decoration: line-through;">Forge クラウドクレジットから Flex トークンへの移行の進捗は、本ブログと電子メールの両方で、変更点を逐次にお知らせしていきます。&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>よくある質問：</strong></span></p>
<p><span style="text-decoration: line-through;"><strong>Q.</strong> 今回の変更で、Forge の消費料金の総額に与える影響を把握することができますか？&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>A.</strong>これらの変更がお客様にどのように影響するかを理解するためにさらにサポートが必要な場合は、<a href="mailto:forge.sales@autodesk.com" rel="noreferrer noopener" target="_blank">forge.sales@autodesk.com</a> にメールをお送りください。&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>Q.&#0160;</strong>予期せずにクラウドクレジットを使い果たしてしまった場合はどうなりますか？&#0160;&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>A.&#0160;</strong> 予想外にクラウドクレジットを使い果たしてしまった場合、Forge アカウントの所有者に電子メールでその旨を通知し、少なくとも 2 週間の応答とコミュニケーションの猶予期間を提供します。突然Forge への API アクセスを停止することはありません。</span></p>
<p><span style="text-decoration: line-through;"><strong>Q.&#0160;</strong>3月29日以前に購入したクラウドクレジットはどうなりますか？&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>A.</strong> 3月29日までに購入したクラウドクレジット（Forge の年間サブスクリプションに付帯する 100 クラウドクレジットを含む）は、残高を使い切るか、購入から 1 年の有効期限が切れるまで、引き続き使用することができます。&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>Q.</strong> Forge サブスクリプションの価格はどうなりますか？&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>A.</strong> Forge サブスクリプションには引き続き 100 クラウドクレジットが含まれますが、クラウドクレジットの更新価格を反映し、年間 Forge サブスクリプションは費用が変更されます。 100 米国ドル（または現地通貨での相当額）だった Forge サブスクリプションは、今後 300 米国ドル（または現地通貨での相当額）になります。</span></p>
<p><span style="text-decoration: line-through;"><strong>Q. </strong>クラウドクレジット消費対象外の Forge API のみを使用する場合はどうなりますか？&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>A.</strong> 現在、クラウドクレジット消費の Forge API のみが今回の価格改定の影響を受けます。 現時点では、新しい API にクラウドクレジット消費を追加する計画はありません。&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>Q.</strong> さらに質問がある場合はどうしたらよいですか？&#0160;</span></p>
<p><span style="text-decoration: line-through;"><strong>A.</strong> 我々は、今回の変化が決して簡単ではないことを理解しています。ご質問は、<a href="mailto:forge.sales@autodesk.com" rel="noreferrer noopener" target="_blank">forge.sales@autodesk.com</a> までお気軽にお寄せください。&#0160;</span></p>
