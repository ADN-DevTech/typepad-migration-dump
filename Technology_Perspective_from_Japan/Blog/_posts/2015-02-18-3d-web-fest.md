<p>Web の世界では、静止画像などの 2D 主体のコンテンツから、3D モデルや 3D 表現を多用するリッチでインタラクティブなコンテンツが急速に広まりつつあります。WebGL、HTML5、CSS3 などのテクノロジを用いることで、いままで実現が難しかった、驚くようなコンテンツがどんどん登場してきています。</p>
<p>今回、クリエイティブなアーティストやデベロッパーを対象にした <strong>3D Web Festival</strong> が開催されます。3D Web Festival では、ライブで参照可能な&#0160;3D を題材にした 1 ～ 3&#0160;分程度の Web コンテンツを競いあうコンテストです。3D と音楽、3D とゲーム、3D と XX、3D の Web コンテンツを無限に表現する「何か」を驚きを与えてください。応募と詳細は、次のURL から参照いただけます。</p>
<p style="text-align: center;"><span style="font-size: 24pt;"><a href="http://www.3dwebfest.com" target="_blank"><strong>http://www.3dwebfest.com</strong></a></span></p>
