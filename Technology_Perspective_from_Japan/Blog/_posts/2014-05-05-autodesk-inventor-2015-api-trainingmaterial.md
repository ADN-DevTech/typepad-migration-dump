<p>今回は、Autodesk Inventor 2015 製品の新機能のご紹介をいったん中断し、Autodesk Inventor 2015 の日本語による APIトレーニング教材一式「APIトレーニングテキスト」「APIトレーニング実習ガイドと演習＆回答」及び 「演習と実習用ファイル」を公開いたします。</p>

<p>公開されるコンテンツは、Autodesk Inventor 2015 APIトレーニングテキスト（.ppt P.368）を pdf ファイルに変換した "Inventor2015API .Net Training_J.pdf"、テキスト内の演習問題と各開発ガイド（実習）を説明した「演習と実習ヘルプファイル」の"Inventor2015API Exercise.chm"、「演習と各開発ガイド（実習）の回答」( 14の演習回答 と 80の実習の回答) をそれぞれVBAマクロファイルとVisual Studio 2012(VB.Net)のプロジェクトとしたファイル群、また「演習と各開発ガイド（実習）で使用するInventorファイル群」の公開となります。</p>

１．	<strong>Autodesk Inventor 2015 の APIトレーニングテキスト</strong>は、<span class="asset  asset-generic at-xid-6a0167607c2431970b01a3fcffd438970b img-responsive"><a href="http://adndevblog.typepad.com/files/inventor2015api_trainigdoc-1.zip">Autodesk Inventor 2015 の APIトレーニングテキスト</a></span> よりダウンロードできます。<br />

３．<strong>演習と実習に使用するファイル群</strong>は、<span class="asset  asset-generic at-xid-6a0167607c2431970b01a73dbaaf6f970d img-responsive"><a href="http://adndevblog.typepad.com/files/inventor2015api_labdrawings-1.zip">Autodesk Inventor 2015 の APIトレーニングテキスト(演習と実習用ファイル)</a></span> よりダウンロードできます。</p>

<p>任意のホルダーに　Inventor2015API_TrainigDoc.zip , Inventor2015API_LabDocs.zip、Inventor2015API_LabDrawings.zip　をダウンロード後にそれぞれ解凍をしていただき、以下のホルダー構造のように、Cドライブ直下に"Inventor API Training"ホルダーを移動配置いただきご利用下さい。<br />
（解凍していただくと、解凍ホルダー直下に\Inventor API Trainingホルダーが作成され中に\Documents,\Drawings ,\Lab Instructions,\Training Labs ホルダーが含まれます）</p>

<p>\Documentsホルダー<br />
　Inventor 2015 APIのクラストレーニングpptを Inventor2015API .Net Training_J.pdfとして変換しています。</p>

<p>\Drawingsホルダー<br />
　演習と実習（Inventor2015API Exercise.chm）で使用するInventorファイル群</p>

<p>\Lab Instructionsホルダー<br />
　演習と実習解説chm（Inventor2015API Exercise.chm）</p>

<p>\Training Labsホルダー<br />
　演習の回答 ( 演習1～14 )　実習の回答( 80 )<br />
  (回答並びに解説は、全て上記の Cドライブのホルダー構造でハードコートされています)</p>

<p><br />
尚、アドイン作成には Visual Studio 2012 有償版(Professionalなど)の開発環境が必要となります。<br />
また、各演習の回答マクロの他に同梱されている Visual Studio向けのプロジェクトもVisual Studio 2012 有償版(Professionalなど)を対象として作成されています。</p>

<p>次回は、引き続き図面の新機能についてご紹介する予定です。</p>
