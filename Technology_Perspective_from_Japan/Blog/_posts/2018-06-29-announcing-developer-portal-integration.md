<p>Autodesk Forge の情報を公開するオフィシャルサイトには、実際の開発に利用することになる <strong><a href="https://developer.autodesk.com/" rel="noopener noreferrer" target="_blank">デベロッパ ポータル（https://developer.autodesk.com）</a></strong>&#0160;と、主に Forge 全般を紹介するマーケテイング要素の&#0160;<strong><a href="https://forge.autodesk.com/" rel="noopener noreferrer" target="_blank">Forge サイト（https://forge.autodesk.com/）</a></strong> が存在しています。前者は Forge が正式に登場する前の <a href="http://adndevblog.typepad.com/technology_perspective/2014/10/a360-view-data-service-api-startup-guide.html" rel="noopener noreferrer" target="_blank">View and Data API 時代</a>から用意されていたサイトで、後者は <a href="http://adndevblog.typepad.com/technology_perspective/2015/12/autodesk-forge-announcement.html" rel="noopener noreferrer" target="_blank">Forge アナウンス時</a>に用意されたサイトです。</p>
<p>オートデスクは長らく2つの Forge サイトを並行運用してきましたが、今後の Forge Platform API の増加にともなって、情報を <strong><a href="https://forge.autodesk.com/" rel="noopener noreferrer" target="_blank">Forge サイト（https://forge.autodesk.com/）</a></strong> に集約することが決定されました。</p>
<p>段階的に予定されている変更とマイルストーンは次のとおりです。</p>
<ul>
<li><strong><a href="https://developer.autodesk.com/" rel="noopener noreferrer" target="_blank">デベロッパ ポータル（https://developer.autodesk.com）</a></strong>&#0160;に記載・提供されている内容や機能は、7 月上旬を目途に&#0160;<strong><a href="https://forge.autodesk.com/" rel="noopener noreferrer" target="_blank">Forge サイト（https://forge.autodesk.com/）</a></strong>&#0160;に移行・統合されます。</li>
<li>developer.autodesk.com 配下の URL は、移行先となる新しい forge.autodesk.com 配下の URL にリダイレクトされます。</li>
</ul>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b022ad39cc931200b-pi" style="display: inline;"><img alt="Forge_unification" border="0" class="asset  asset-image at-xid-6a0167607c2431970b022ad39cc931200b image-full img-responsive" src="/assets/image_20875.jpg" title="Forge_unification" /></a></p>
<p>デベロッパ ポータルの URL をブックマークされている場合には、サイト統合後に新しい URL でブックマークし直すことが推奨されています。両サイトの今後の変化にご注意ください。</p>
<p>By Toshiaki Isezaki</p>
