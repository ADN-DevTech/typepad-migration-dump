<p>Inventorの新バージョンとなる、Inventor 2024がリリースされました。</p>
<p>Inventor 2024の API の日本語プログラミング用ヘルプ （admapi_28_0.chm）を ZIP 圧縮したファイルをポストしていますので、次のリンクからダウンロードしてください。<a href="https://adndevblog.typepad.com/files/admapi_26_0.zip"></a></p>
<p><span class="asset  asset-generic at-xid-6a0167607c2431970b02b7519d7391200c img-responsive"><a href="https://adndevblog.typepad.com/files/admapi_28_0-1.zip">Inventor 2024 API の日本語プログラミング 用ヘルプ をダウンロード</a></span></p>
<p>日本語版 Inventor 2024 に同梱されている API ヘルプは英語版オンライン ヘルプになっていて、</p>
<p><strong>C:\Users\Public\Documents\Autodesk\Inventor 2024\Local Help</strong> にインストールされており、Inventorからは[ヘルプ]- [プログラミング/ API ヘルプ]より起動が可能です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02942fa2dbe9200c-pi" style="display: inline;"><img alt="API Help" class="asset  asset-image at-xid-6a0167607c2431970b02942fa2dbe9200c img-responsive" src="/assets/image_311888.jpg" title="API Help" /></a></p>
<p>ダウンロードした日本語版の Inventor 2024 API Help （admapi_28_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b7519d73d1200c-pi" style="display: inline;"><img alt="1" class="asset  asset-image at-xid-6a0167607c2431970b02b7519d73d1200c img-responsive" src="/assets/image_248088.jpg" title="1" /></a></p>
<p>ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b027880751956200d-pi" style="display: inline;"> </a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68530425b200d-pi" style="display: inline;"><img alt="2" class="asset  asset-image at-xid-6a0167607c2431970b02b68530425b200d img-responsive" src="/assets/image_855449.jpg" title="2" /></a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b027880751956200d-pi" style="display: inline;"><br /></a></p>
<p>ページに何も表示されない場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、右下にある 「許可する」 にチェックを入れてください。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b75179214f200b-pi" style="display: inline;"><img alt="3" class="asset  asset-image at-xid-6a0167607c2431970b02b75179214f200b img-responsive" src="/assets/image_185313.jpg" title="3" /></a></p>
<p>&#0160;</p>
<p>なお、従来の CHM ファイル形式とは別に、英語版 Inventor 2024 オンラインヘルプ内に Programming Interface セクションが追加されています。記載されている内容は 、CHM 形式の内容とと同じです。なお、残念ながら日本語版のInventor 2024 オンラインヘルプにはこのヘルプは掲載されてはいません。</p>
