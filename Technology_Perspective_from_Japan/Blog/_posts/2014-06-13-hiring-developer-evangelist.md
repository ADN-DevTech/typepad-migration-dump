<p><span style="color: #0000ff; background-color: #ffff00;"><strong>おかげさまで、本求人は終了いたしました。多くのお申込み、大変ありがとうございました。</strong></span></p>
<p>Autodesk Developer Network(ADN) では、まもなく、オートデスク 株式会社 東京オフィスでデベロッパー エバンジェリストの求人を開始します。ADN チームメンバーは、世界中のオフィスに点在しながら、US 本社を中心に他国のメンバと共同で活動する、社内でも少しユニークな存在です。</p>
<p>もし、次の項目に該当する方、あるいは、該当しそうな方をご存じであれば、ぜひ、その方にこの記事をご紹介ください。求人を開始し次第、このブログでお知らせします。</p>
<p>-----------------</p>
<p>現在、オートデスクは、数多くの新しいクラウド サービスを市場に送り出そうとしています。すべてのクラウド サービスは、API を使ってカスタマイズできるように、開発者用のプラットフォームとしてデザインされています。そして、それら新しい API を世界中に広める必要があります。ADN の&#0160;Developer Technical Services チームに参加して、一緒にそんな活動してみませんか !!</p>
<p>業務内容：</p>
<ul>
<li>サードパーティ デベロッパーと共に作業して、彼らが持つ問題をオートデスク API を使って解決に導くサポートをする。</li>
<li>社内のソフトウェア開発チームへ社外からの要望を伝えるなど、サードパーティ デベロッパー コミュニティが成功するために必要とする API を理解させる。</li>
<li>カンファレンス、ハッカソンやミートアップ、ブログや Web キャストなどのソーシャルメディアを通じて、オートデスクの新しいテクノロジを喧伝、啓蒙する。</li>
</ul>
<p>求められる特性（もちろん全部でなくても大丈夫です）：</p>
<ul>
<li>探究心が旺盛で、テクノロジや環境の変化に柔軟に適応して、楽しみながら問題解決することが出来る、スマート エンジニア。</li>
<li>新しいテクノロジの習得と、他の人へのテクノロジの啓蒙に興味を持つプログラマー。</li>
<li>JavaScript、REST API、HTML5 など、Web プログラミング テクノロジの基礎知識がある。または、それらを習得したいと考えている。</li>
<li>Java やObjective-C などのモバイル プラットフォーム プログラミングの基礎知識がある。または、それらを習得したいと考えている。</li>
<li>カンファレンスで大衆へプレゼンテーションしたり、1 対 1 で会話したり、人との対話を楽しめる。</li>
<li>英語の読み書きが出来る。自身がなくても、社内のコミュニケーションを通じて英語スキルを習得できます。そんなところにも興味を持てる方。.</li>
<li>ブログやツイッターなどのソーシャル メディアへの投稿を楽しめる。</li>
<li>建設業やビルディング インフォメーション モデリング(BIM) の知識がある（あるといいかも）。</li>
</ul>
<p>-----------------</p>
<p>The ADN team is hiring an API evangelist based in our Tokyo office. Please spread the word. Experience with AutoCAD APIs - or any other Autodesk technology - is not required (but would be a bonus).</p>
<p><a href="https://www.linkedin.com/jobs2/view/13975557" title="https://www.linkedin.com/jobs2/view/13975557"><em style="color: #000000;">Autodesk is hiring an API evangelist!&#0160; If you’re a software engineer who’s enthusiastic about new technologies, enjoys interacting with others and sharing this enthusiasm, and loves to evangelize code –then this is the job for you!</em></a></p>
<p>-----------------</p>
<p>Autodesk is in the process of launching a large number of new cloud services – all of which are designed as platforms for developers to build upon.&#0160; We need to tell the world about these new APIs.&#0160; As the successful candidate, you will join the Developer Technical Services team (our developer evangelist team at Autodesk, and part of our Cloud Platforms Division).</p>
<p>Responsibilities:</p>
<ul>
<li>Working directly with third party developers to help them solve their problems using our APIs.&#0160;</li>
<li>Acting as developer advocate to our internal software development teams – helping them understand the APIs our developer community needs to be successful.</li>
<li>Evangelizing our new technology through conferences, hackathons, meetups, blogging, webcasts, social media, and whatever other communication channels you dream up.</li>
</ul>
<p>To be successful you will have the following attributes:</p>
<ul>
<li>You’re a gifted programmer who is excited about learning new technology and teaching others. (You already know deep in your heart if you’re a natural programmer – if you don’t then you’re not).</li>
<li>You have a good grounding in web programming technologies, including JavaScript, REST and HTML5.</li>
<li>You have a good grounding in, or are willing to learn, mobile platform programming technologies, such as Java and Objective-C.</li>
<li>You’re intelligent and inquisitive, and enjoy solving problems.</li>
<li>You enjoy interacting with people – including one-on-one conversations and presenting to large audiences at conferences.</li>
<li>You have basic writing/reading English skills. If you are not, you can learn English by having communication with colleagues in ADN and our engineering team.</li>
<li>You enjoy writing – including blogging, tweeting, and other social media outlets.</li>
<li>You have background for AEC industry or Building Information Modeling(BIM), that will be bonus.</li>
</ul>
