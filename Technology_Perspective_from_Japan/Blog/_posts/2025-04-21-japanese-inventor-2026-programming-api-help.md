<p>Inventorの新バージョンとなる、Inventor 2026がリリースされました。</p>
<p>Inventor 2026の API の日本語プログラミング用ヘルプ （admapi_30_0.chm）を ZIP 圧縮したファイルをポストしていますので、次のリンクからダウンロードしてください。<a href="https://adndevblog.typepad.com/files/admapi_26_0.zip"></a></p>
<p><span class="asset  asset-generic at-xid-6a0167607c2431970b02c8d3d1408b200c img-responsive"><a href="https://adndevblog.typepad.com/files/admapi_30_0.zip">Inventor 2026の API の日本語プログラミング用ヘルプ</a></span></p>
<p>&#0160;</p>
<p>日本語版 Inventor 2026 に同梱されている API ヘルプは英語版オンライン ヘルプになっていて、</p>
<p><strong>C:\Users\Public\Documents\Autodesk\Inventor 2026\Local Help</strong> にインストールされており、Inventorからは[ヘルプ]- [プログラミング/ API ヘルプ]より起動が可能です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02942fa2dbe9200c-pi" style="display: inline;"><img alt="API Help" class="asset  asset-image at-xid-6a0167607c2431970b02942fa2dbe9200c img-responsive" src="/assets/image_311888.jpg" title="API Help" /></a></p>
<p>ダウンロードした日本語版の Inventor 2026 API Help （admapi_30_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02dad0c054bb200d-pi" style="display: inline;"><img alt="APIHelp" class="asset  asset-image at-xid-6a0167607c2431970b02dad0c054bb200d img-responsive" src="/assets/image_891082.jpg" title="APIHelp" /></a></p>
<p>ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3d142ce200c-pi" style="display: inline;"><img alt="Empty" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3d142ce200c img-responsive" src="/assets/image_94213.jpg" title="Empty" /></a></p>
<p>ページに何も表示されない場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、右下にある 「許可する」 にチェックを入れてください。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02dad0c054de200d-pi" style="display: inline;"> </a> <a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3d141a4200c-pi" style="display: inline;"><img alt="Fileproperty" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3d141a4200c img-responsive" src="/assets/image_151193.jpg" title="Fileproperty" /></a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02dad0c054de200d-pi" style="display: inline;"><br /></a></p>
<p>なお、従来の CHM ファイル形式とは別に、英語版 Inventor 2026 オンラインヘルプ内に Programming Interface セクションが追加されています。記載されている内容は 、CHM 形式の内容と同じです。なお、残念ながら日本語版のInventor 2026 オンラインヘルプにおいてもこのヘルプは英語の状態となり日本語への翻訳版は掲載されてはいません。</p>
