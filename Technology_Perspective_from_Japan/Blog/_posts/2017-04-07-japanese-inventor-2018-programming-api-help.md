<p>Inventor 2018 API の日本語プログラミング用ヘルプ&#0160;（admapi_22_0.chm）がリリースされました。 ZIP 圧縮したファイルをポストしていますので、次のリンクからダウンロードしてください。</p>
<p style="padding-left: 30px;"><span class="asset  asset-generic at-xid-6a0167607c2431970b01b8d273a09e970c img-responsive"><a href="http://adndevblog.typepad.com/files/admapi_22_0.zip"><strong>日本語版 Inventor 2018 API Help</strong> をダウンロード</a></span></p>
<p>日本語版 Inventor 2018 に同梱されている API ヘルプは英語版オンライン ヘルプになっていて、C:\Program Files\Autodesk\Inventor 2018\Local Help にインストールされています。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c8ea2c83970b-pi" style="display: inline;"><img alt="Access_api_help" class="asset  asset-image at-xid-6a0167607c2431970b01b7c8ea2c83970b img-responsive" src="/assets/image_755776.jpg" style="width: 600px;" title="Access_api_help" /></a></p>
<p>ダウンロードした日本語版の Inventor 2018 API Help （admapi_22_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。&#0160;</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d273a152970c-pi" style="display: inline;"><img alt="Appear_contents" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d273a152970c image-full img-responsive" src="/assets/image_409455.jpg" title="Appear_contents" /></a></p>
<p>ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。その場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、画面右下にある 「ブロックの解除」 にチェックを入れてください。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d273a194970c-pi" style="display: inline;"><img alt="Remove_block" class="asset  asset-image at-xid-6a0167607c2431970b01b8d273a194970c img-responsive" src="/assets/image_943146.jpg" title="Remove_block" /></a></p>
<p>なお、今回の Inventor 2018 では従来の CHM ファイル形式とは別に、英語版 Inventor 2018 のみ、<a href="http://help.autodesk.com/view/INVNTOR/2018/ENU/" rel="noopener noreferrer" target="_blank"><strong>オンライン</strong> ヘルプ</a>&#0160;内に Programming Interface セクションが追加されています。残念ながら、日本語版 Inventor には反映されてはいませんが、記載されている内容は CHM 形式と同じものになっています。</p>
