<p>今回は、Autodesk Revit 2024 の新しいコア機能をご紹介いたします。</p>
<p><strong>Revit の[暗]テーマ</strong><br />Revit 2024 では、最上位レベルのユーザ インタフェースの[暗]（ダーク）テーマがサポートされました。作図領域(キャンバス)のテーマも、リボンと[オプション]ダイアログの両方で[暗]または[明]に設定できます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68532e7b6200d-pi" style="display: inline;"> </a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68532e7bf200d-pi" style="display: inline;"><img alt="Revit2024-02-01" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b68532e7bf200d image-full img-responsive" src="/assets/image_628105.jpg" title="Revit2024-02-01" /></a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68532e7b6200d-pi" style="display: inline;"><br /></a></p>
<p>テーマの設定を変更するには、[ファイル]タブ-&gt;[オプション]ダイアログ-&gt;[色]タブ-&gt;[UI のアクティブなテーマ]をクリックします。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68532e7ca200d-pi" style="display: inline;"><img alt="Revit2024-02-02" class="asset  asset-image at-xid-6a0167607c2431970b02b68532e7ca200d img-responsive" src="/assets/image_611273.jpg" title="Revit2024-02-02" /></a></p>
<p>作図領域のテーマを[明]または[暗]に設定するには、[ファイル]タブ-&gt;[オプション]ダイアログ-&gt;[色]タブ-&gt;[キャンバスのカラースキーム]をクリックします。</p>
<p>[表示]タブの[キャンバス テーマ]ボタンを使用してリボンから現在の作図領域テーマを、目的のテーマ([明]または[暗])に変更できます。</p>
<hr />
<p><strong>新しい[カラー ブック]ダイアログ</strong><br />[オプション]ダイアログの[色]タブでは、サードパーティのカラーブック、またはユーザ定義のカラーブックを使用して色を指定できるようになりました。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b7517bcd81200b-pi" style="display: inline;"><img alt="Revit2024-02-03" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b7517bcd81200b image-full img-responsive" src="/assets/image_550436.jpg" title="Revit2024-02-03" /></a></p>
<hr />
<p><strong>新しいサンプル モデル</strong><br />Revit 2024 で新しいサンプル モデルが追加されました。また、日本仕様のサンプルモデルも引き続きインストールされます。</p>
<p>サンプル モデルは、Revit を使用して作成された多目的プロジェクトの例です。<br />モデルには、プロジェクトの作成に使用される多くの要素の例が示されています。完全なモデルを提供するため、設計図書の例や、構造、MEP、外構、点群をまとめたファイルが含まれています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b7517bcd96200b-pi" style="display: inline;"><img alt="Revit2024-01-01" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b7517bcd96200b image-full img-responsive" src="/assets/image_751053.jpg" title="Revit2024-01-01" /></a></p>
<p>モデル内にある[ヘルプ]アイコンを使用して、そのモデルで説明されている特定の要素とワークフローについて学習します。モデルに含まれている「手順」集計表を使用して、サンプル モデルについての学習の進捗を確認します。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68532e7dc200d-pi" style="display: inline;"><img alt="Revit2024-02-04" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b68532e7dc200d image-full img-responsive" src="/assets/image_596230.jpg" title="Revit2024-02-04" /></a></p>
<p>プロパティの「Learning Link」をクリックすると、Revit ヘルプページを開くことができます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68532e80f200d-pi" style="display: inline;"><img alt="Revit2024-02-05" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b68532e80f200d img-responsive" src="/assets/image_721134.jpg" title="Revit2024-02-05" /></a></p>
<hr />
<p><strong>プロジェクト ブラウザで検索する</strong><br />プロジェクト ブラウザの上部に検索機能が追加されました。</p>
<p>入力すると、検索が自動的に開始され、関連する結果が更新されます。一致するキーワードは、プロジェクト ブラウザのすべての階層レベルで、フィルタされたリストの中でハイライト表示されます。</p>
<p>ファミリを検索したい場合などにご活用いただけます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b751a02626200c-pi" style="display: inline;"><img alt="Revit2024-02-06" class="asset  asset-image at-xid-6a0167607c2431970b02b751a02626200c img-responsive" src="/assets/image_986290.jpg" title="Revit2024-02-06" /></a></p>
<hr />
<p><strong>コーディネーション モデルをリンクする</strong><br />Navisworks® のファイルまたは Autodesk Docs のコーディネーション モデル ビューをリンクし、それらを Revit モデルの参照として使用して、設計の周辺環境を提供します。<br />ファイルをローカル リソースにダウンロードせずに、Autodesk Docs からビューに直接リンクすることができます。<br />Autodesk Docs からリンクされたビューでは、Autodesk Docs アカウントの権限を使用してアクセスを管理できるため、IP 保護が重要な場合でもコラボレーション ワークフローをより簡単に使用できます​。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b751a02634200c-pi" style="display: inline;"><img alt="Revit2024-02-07" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b751a02634200c image-full img-responsive" src="/assets/image_455957.jpg" title="Revit2024-02-07" /></a></p>
<hr />
<p><strong>コーディネーション モデルのスナップ点</strong><br />スナップ点を使用して、状況に合わせて簡単にモデリングしたり、コーディネーション モデルの点に基づいて 2 点間を計測したり、コーディネーション モデルをより適切に配置します。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b7517bcdc3200b-pi" style="display: inline;"><img alt="Revit2024-02-08" class="asset  asset-image at-xid-6a0167607c2431970b02b7517bcdc3200b img-responsive" src="/assets/image_501878.jpg" title="Revit2024-02-08" /></a></p>
<hr />
<p><strong>クラウド モデル コラボレーションのキャッシュ管理</strong><br />クラウド モデルのキャッシュを管理しやすくするために、[オプション]ダイアログ ボックスでフォルダ パスを指定できます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68532e7f8200d-pi" style="display: inline;"><img alt="Revit2024-02-09" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b68532e7f8200d image-full img-responsive" src="/assets/image_53798.jpg" title="Revit2024-02-09" /></a></p>
