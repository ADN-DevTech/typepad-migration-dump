<p><strong><a href="https://developer.autodesk.com/" rel="noopener noreferrer" target="_blank">デベロッパ &#0160;ポータル（https://developer.autodesk.com/）</a></strong>の Resouces メニューに <strong><a href="https://developer.autodesk.com/en/docs/quickstarts/v1/overview/" rel="noopener noreferrer" target="_blank">Quickstarts</a></strong>（クイック スタート ガイド）が追加されました。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb097dfd00970d-pi" style="display: inline;"><img alt="Quickstarts" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb097dfd00970d image-full img-responsive" src="/assets/image_511877.jpg" title="Quickstarts" /></a></p>
<p>いままでも、Forge Platform API の各 API では、API Basic &#0160;や Field Guide で当該 API の概要を、Step-by-Step Tutorials でシナリオに応じた実装内容や手順を、それぞれ解説していました。例えば、Model Derivative API の場合、次のようなページ内リンクでそれぞれの内容を表示することが可能です。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb097dfd4e970d-pi" style="display: inline;"><img alt="Api_basic_example" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb097dfd4e970d image-full img-responsive" src="/assets/image_744516.jpg" title="Api_basic_example" /></a></p>
<p>ただ、これらの内容は、選択した API の endpoint を中心に記述されているので、いくつかの API を横断的に使用する必要がある場合には、全体の把握がし難い場合がありました。また、Step-by-Step Tutorials では、特定のフレームワークや開発言語/環境には触れていませんでした。</p>
<p>今回追加された Quickstarts は、主な開発環境/フレームワーク別に GitHub 上のリポジトリから <strong><a href="http://adndevblog.typepad.com/technology_perspective/2017/01/forge-sdk.html" rel="noopener noreferrer" target="_blank">Forge SDK</a></strong> とサンプルソース一式をクライアントにクローンして、ローカル コンピュータに開発環境を構築する手助けとなります。クローンした実装は、Authentication API、Data Management API、Model Derivative API を使用して、最終的に Forge Viewer モデルを表示するまでを把握することが出来る、いわば、Tutorial とも言えるものです。&#0160;</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb097dfdd8970d-pi" style="display: inline;"><img alt="Quickstarts_main" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb097dfdd8970d image-full img-responsive" src="/assets/image_297993.jpg" title="Quickstarts_main" /></a></p>
<p>既にお持ちの開発知識にマッチした内容を選択することが出来ますので、API Basic &#0160;や Field Guide Step-by-Step Tutorials、また、既存の&#0160;<a href="http://adndevblog.typepad.com/technology_perspective/2017/01/forge-viewer-tutorial-part1-implement-nodejs-server.html" rel="noopener noreferrer" target="_blank">チュートリアル</a>&#0160;等と併用して参照することで、すばやい習得が可能になるはずです。</p>
<p>By Toshiaki Isezaki</p>
