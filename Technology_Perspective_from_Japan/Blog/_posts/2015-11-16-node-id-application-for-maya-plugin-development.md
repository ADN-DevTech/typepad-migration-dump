<p>Autodesk Maya でプラグインを開発する際、Maya 上でプラグイン アプリケーションのノードが他のアプリケーションと競合しないよう、一意な Node ID を持つカスタムノードの利用が推奨されています。従前から <strong>Autodesk&#0160;Maya Developer Center</strong>（<a href="http://www.autodesk.co.jp/developmaya" target="_blank"><strong>http://www.autodesk.co.jp/developmaya</strong></a>） で、Node ID の申請ページをご用意していましたが、サーバー不良で申請ページが無効になっていました。</p>
<p>今回、新たに Node ID の申請ページをご用意しましたので、新規に Node ID を申請いただく場合には、<strong><a href="http://mayaid.autodesk.io/" target="_blank">http://mayaid.autodesk.io/</a></strong>&#0160;から直接アクセスしていただくか、Developer Center ページの次のリンクから、アクセスしていただきますようお願いいたします。</p>
