<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c8002d83970b-pi" style="float: right;"><img alt="Devsumi_logo" class="asset  asset-image at-xid-6a0167607c2431970b01b7c8002d83970b img-responsive" src="/assets/image_599275.jpg" style="margin: 0px 0px 5px 5px;" title="Devsumi_logo" /></a>昨年に続いて、今年も翔泳社主催のデベロッパー サミット、通称、デブサミ に参加することが決まりました。前回は、View and Data API の概要をセッションでご紹介しましたが、今年は、先月発表された Autodesk Forge 全般について、いくつかの具体例とともにご紹介したいと思っています。</p>
<p>6 月には、初めての<strong> <a href="http://forge.autodesk.com/conference/" target="_blank">Forge Developer Conference</a></strong>&#0160;が San Francisco で開催される予定です。3D プリンタ プラットフォームである Spark や View and Data API や IoT も含め、Autodesk Forge には、クラウドを中心とした「接続された」エコシステムの開発を手助けする多くの API や SDK、テクノロジが含まれています。Autodesk Forge をご紹介する初めての場ともなりますので、ぜひ、ご参加いただければと思います。</p>
<p>デブサミは無償で参加いただけるイベントです。参加登録も含め、<strong><a href="http://event.shoeisha.jp/devsumi/20160218/" target="_blank">http://event.shoeisha.jp/devsumi/20160218/</a></strong>&#0160;で詳細をご確認ください。</p>
<p>By Toshiaki Isezaki</p>
