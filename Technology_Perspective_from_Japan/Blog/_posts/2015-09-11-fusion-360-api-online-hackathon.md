<p>Autodesk Developer Network (ADN) が主催する、 開発者向けの Fusion 360 ハッカソンが初めて開催されます。</p>
<p>通常のハッカソンとは異なり、オンライン イベント（英語）で Fusion 360 API を習得していただき、約 5 週間の間にカスタム アプリを開発後、<strong><a href="https://apps.exchange.autodesk.com/en" target="_blank">Autodesk Exchange Apps ストア</a>&#0160;</strong>で開発したアプリを公開することをゴールとしています。</p>
<p>もし、米国太平洋標準時の&#0160;10月30日までにアプリを提出いただければ、$500 を得ることが出来ます。参加は無償で事前登録が必要です。詳細は&#0160;<a href="http://www.fusion360hackathon.com" target="_blank"><strong>www.fusion360hackathon.com</strong>&#0160;</a>をご確認ください。&#0160;</p>
<p>日本語リソースはまだ多くありませんが、<a href="http://adndevblog.typepad.com/technology_perspective/2015/07/summary-for-cloud-api-seminar.html" target="_blank"><strong>クラウド API セミナーのサマリー</strong></a> &#0160;ページに記載されている Fusion 360 API のセッション動画とプレゼンテーション資料で概要を把握していただけます。英語でもOK という方がいらっしゃいましたら、ぜひチャレンジしてみてください。</p>
<p>Fusion 360 API アプリを含む Autodesk Exchange Apps ストアへのアプリ公開方法等は、<a href="http://www.autodesk.co.jp/developapps" target="_blank"><strong>www.autodesk.co.jp/developapps</strong></a> でご案内しています。</p>
<p>By Toshiaki Isezaki</p>
