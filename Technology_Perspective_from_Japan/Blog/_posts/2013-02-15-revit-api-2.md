<p><a href="http://adndevblog.typepad.com/technology_perspective/2013/02/revit-api%E5%85%A5%E9%96%80.html" target="_self" title="Revit API getting started">前回</a>は、これからRevit APIを使用してカスタマイズを開始しようとなさっている方のために、まずどういった環境設定が必要なのかをご紹介しました。では、実際にプログラムを作成するにはどこからはじめればよいのでしょう？うれしいことにSDKには１００以上のサンプルもあって、Revitに関しては資料は実に豊富にあるのですが、逆にありすぎて戸惑ってしまわれる方も多いようです。ということで、ここではいくつかの学習ようの教材をご紹介します。　</p>
<p>2）どこから始めればよいか</p>
<p>（1）トレーニングビデオを見る</p>
<p>開始する最善の方法としては、Developer Technical Service (DevTech) チームの作成したWebcastのレコーディングを見ていただくとよいと思います。ただ、ドキュメント同様、日本語となると数も限られてくるので、英語版も可能であれば見ていただければと思います。</p>
<p>日本語版のレコーディングはAutodesk Developer Network OpenサイトのAPIトレーニングセクション、Webcast アーカイブのページよりダウンロードできます。</p>
<ul>
<li>Webcast レコーディング アーカイブ<br /><a href="http://www.adskconsulting.com/adn/cs/api_course_webcast_archive.php">http://www.adskconsulting.com/adn/cs/api_course_webcast_archive.php</a></li>
</ul>
<p>言語を日本語でフィルターしていただくと日本語のレコーディングのみが表示されます。 Revit API の基本はここ数年さほど変わっていないので、Revit 2011で紹介版を録音して、それ以降はアップデート版として、追加機能に重点をのみを紹介しています。ですから、以下の順序で見ていただくとよいと思います。ただし、必要に応じてバージョン番号をお使いのRevitのバージョンにあわせていただく必要があります。</p>
<ul>
<li>Revit API 2011 入門編 Webcast <a href="http://download.autodesk.com/media/adn/Revit2011APIWebcast_Japanese_07082010.zip">http://download.autodesk.com/media/adn/Revit2011APIWebcast_Japanese_07082010.zip</a></li>
<li>Revit API 2012 新機能紹介 Webcast<br /><a href="http://download.autodesk.com/media/adn/Revit2011APIWebcast_Japanese_07082010.zip">http://download.autodesk.com/media/adn/Revit 2012 API webcast Japanese July 14.zip.zip</a></li>
<li>Revit API 2013 新機能紹介 Webcast<br /><a href="http://download.autodesk.com/media/adn/Revit2011APIWebcast_Japanese_07082010.zip">http://download.autodesk.com/media/adn/Revit2013APIWebcastJapanese_July26_2012.zip</a></li>
</ul>
<p>これらのレコーディングには、Webcastで使用したスライドも含まれています。</p>
<p>あとは、英語になりますが、英語だとさらに以下のようなトレーニングマテリアルが用意されています。</p>
<ul>
<li>DevTV <br /><br /><a href="http://www.autodesk.com/developrevit">http://www.autodesk.com/developrevit</a><br /><br />ADN Open のRevit のページにDevTVと呼ばれるDevTechチームが作成したレコーディングがいくつか用意されています。&#0160;（ページの中ごろにありますので、スクロールダウンしてください。）自分のチームの作成したのをほめるようでなんですが、特にAdam Nagy の作成した Revit Intro Part 2 は部屋の番号をマウスをクリックする順に付け替えるプログラムを例にフィルターの説明をおこなっており、小さなプログラムでありながら、即実践で使えるなかなかのすぐれものです。ぜひプログラムもダウンロードして、実際に使いながら学習してしてみてください。<br /><br /><a href="http://download.autodesk.com/media/adn/Revit_2011_2012_DevTV_EN/Revit_2011_2012_DevTV_EN.html" target="_self" title="DevTV Revit Intro Part1"><strong>DevTV: Introduction to Revit 2011/2012/2013 Programming - Part 1</strong><br /></a><br /><strong><a href="http://download.autodesk.com/media/adn/DevTV_Revit_2012_NET_English-Part2/Revit%20DevTV%20-%20part%202.html" target="_self" title="DevTV Revit Intro Part2">DevTV: Introduction to Revit 2011/2012/2013 Programming – Part 2</a></strong><br /><br /></li>
<li>My First Revit Plug-in<br /><br />プログラミング そのものに慣れていないかたには、&quot;MyFirst Revit Plug-in&quot; トレーニング教材を見ていただくとよいかと思います。<br /><a href="http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=16777469">http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=16777469</a><br />もしも、.NET でのプログラミングの経験をお持ちのかたは、適宜プログラミングの基礎の部分をスキップしてみていただくとよいかと思います。</li>
<li>Webcast レコーディング アーカイブ（英語版）<br /><a href="http://www.adskconsulting.com/adn/cs/api_course_webcast_archive.php">http://www.adskconsulting.com/adn/cs/api_course_webcast_archive.php</a><br /><br />さきに日本語版の&#0160;レコーディングの話をしましたが、同じページからそれらの英語版が入手できます。<br /><br />Revit API 2011 Introduction&#0160;Webcast<br /><a href="http://download.autodesk.com/media/adn/RevitAPI_20May2010.zip">http://download.autodesk.com/media/adn/RevitAPI_20May2010.zip</a><br /><br />これはバージョン2011以降のRevit APIの基本をカバーしています。 （注：RevitのAPIは2011バージョンで大きな改良をしました。）このWebcastの教材には、レコーディング、pptおよびトレーニングラボのコードが含まれています。 今年のWebキャストでは、主要な拡張機能をカバーしています。<br /><br />Revit API 2013&#0160;New Feature&#0160;Webcasts<br /><a href="http://download.autodesk.com/media/adn/DevDaysOnline-Revit2013API.zip">http://download.autodesk.com/media/adn/DevDaysOnline-Revit2013API.zip</a><br /><a href="http://download.autodesk.com/media/adn/Revit2013APIWebcast_May17_2012.zip">http://download.autodesk.com/media/adn/Revit2013APIWebcast_May17_2012.zip</a><br /><br />ファミリーのAPIに興味がある場合は、ファミリーAPIに関するWebcastの録音もあります：<br /><a href="http://download.autodesk.com/media/adn/Revit_2010_Family_API_Webcast-July2009.zip">http://download.autodesk.com/media/adn/Revit_2010_Family_API_Webcast-July2009.zip</a></li>
</ul>
<p>（2）トレーニングラボ</p>
<p>ADN Open のサイトから入手できる最新のトレーニングラボを試してください。<br /><a href="http://www.autodesk.com/developrevit">http://www.autodesk.com/developrevit</a><br />（ &quot;Learning&quot; &gt;&gt; &quot;Samples and Documentation&quot;の下にあります。スクロールダウンしてください。）</p>
<p>このラボは私たちDevTechメンバーがクラスルームのトレーニングを行い際に用いる使用しているトレーニングマテリアルです。インストラクター付で2日から3日のトレーニングに使用しています。教材はC#とVB.NETの両方で用意されていますので、個人の好で選択してください。</p>
<p>（3）SDKのサンプルをみる。</p>
<p>ラボのコードをみていただくとともに、ご自分でHelloRevitサンプルを試してみて、さらにSDKのサンプルのいくつかを見ていただくとよいでしょう。特に、次のサンプルは良い出発点になります。</p>
<ul>
<li>RevitCommands</li>
<li>Fire Rating</li>
<li>Viewer/ ElementViewer</li>
<li>CreateBeamsColumnsBraces</li>
</ul>
