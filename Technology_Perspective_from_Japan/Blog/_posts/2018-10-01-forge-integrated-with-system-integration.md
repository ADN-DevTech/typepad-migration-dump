<p><strong><a href="https://forge.autodesk.com/customers" rel="noopener noreferrer" target="_blank">Success Stories</a></strong>&#0160; を見ていくと、多様な業種・業態で Forge が利用されていることがわかりますが、最近では、安井建築設計事務所の事例のように <strong><a href="http://adndevblog.typepad.com/technology_perspective/2015/08/about-internet-of-things.html" rel="noopener noreferrer" target="_blank">IoT</a></strong> センサーとのマッシュアップで、今迄、簡単には実現できなかったような事例も見受けられるようになりました。例えば、<strong><a href="https://forge.autodesk.com/customer-stories/moicon" rel="noopener noreferrer" target="_blank">Moicon（https://forge.autodesk.com/customer-stories/moicon）</a></strong>や <strong><a href="https://forge.autodesk.com/customer-stories/eutech" rel="noopener noreferrer" target="_blank">Eutech（https://forge.autodesk.com/customer-stories/eutech）</a></strong>です。</p>
<p><a href="https://www.iviva.com/smartbim/?wvideo=c4qr25uyrm"><img height="425" src="/assets/e182f742382c8642bc35fbedb0cbd4d9096fb5ff.jpg" style="width: 700px; height: 425px;" width="700" /></a></p>
<p><a href="https://www.iviva.com/smartbim/?wvideo=c4qr25uyrm">Smart BIM</a></p>
<p>これらの事例は Forge ポータルに記載されているので Forge を利用していることが自明ですが、Forge を明示していない事例も見られます。例えば、</p>
<p><strong><a href="https://willowinc.com/" rel="noopener noreferrer" target="_blank">willow Inc. https://willowinc.com）</a></strong>の例では<strong><a href="https://willowinc.com/products/" rel="noopener noreferrer" target="_blank"> https://willowinc.com/products/</a></strong> ページで、Microsoft のブログ記事<strong><a href="https://blogs.microsoft.com/iot/2018/09/24/announcing-azure-digital-twins-create-digital-replicas-of-spaces-and-infrastructure-using-cloud-ai-and-iot/" rel="noopener noreferrer" target="_blank">（https://blogs.microsoft.com/iot/2018/09/24/announcing-azure-digital-twins-create-digital-replicas-of-spaces-and-infrastructure-using-cloud-ai-and-iot/）</a></strong>の例では、それぞれページ内に記載されたキャプチャ画像から Forge Viewer を見て取ることが出来ます。</p>
<p>また、YouTube で公開されている IBM Reaserch の&#0160;Cognitive Building の例では IBM Watson IoT プラットフォームとのマッシュアップを見ることが出来ます（音声で Watson と会話していますので、デバイスのサウンドを有効にしてご確認ください）。&#0160;</p>
<p class="asset-video"><iframe allowfullscreen="" frameborder="0" height="450" src="https://www.youtube.com/embed/38BJkQPhj_o" width="680"></iframe></p>
<p class="asset-video">このように、Forge は既存のデザイン データを活用する目的で広く利用されるようになっています。Forge をはじめとした Webサービス（Web API）は、他の Web サービスと組み合わせて利用する<strong><a href="https://ja.wikipedia.org/wiki/%E3%83%9E%E3%83%83%E3%82%B7%E3%83%A5%E3%82%A2%E3%83%83%E3%83%97_(Web%E3%83%97%E3%83%AD%E3%82%B0%E3%83%A9%E3%83%9F%E3%83%B3%E3%82%B0)" rel="noopener noreferrer" target="_blank">マッシュアップ</a></strong>で新しい用途を作り出すことが可能なのです。故に、各社が用意するさまざまな機能を持つ Web API を、よく色々な色や形を持つブロックに例えて説明しています。IoT センサーも、その 1 つです。</p>
<p>なお、オートデスクは 3 年前にクラウド ベースの IoT プラットフォームを提供していた SeeControl を<strong><a href="http://adndevblog.typepad.com/technology_perspective/2015/08/about-see-control.html" rel="noopener noreferrer" target="_blank">買収</a></strong>していて、Fusion Connect に流用しています。昨年の DevCon では、Forge IoT として API 化のロードマップを<strong><a href="http://adndevblog.typepad.com/technology_perspective/2017/11/devcon-2017-forge-roadmap.html" rel="noopener noreferrer" target="_blank">発表</a></strong>していましたが、残念ながら、現在、このプランは白紙撤回されています。このため、前述の IoT 事例では、独自に Socket IO 等でデバイスとの接続を確立し、Forge Viewer のベースになっている Three.js &#0160;<a href="https://threejs.org/">https://threejs.org/</a> を使ってカンバスのオーバーレイするなどしていることになります。Forge 側で一貫したインターフェースを用意しているわけではありませんのでご注意ください。</p>
