<p>今日は、新しくリリースされた AutoCAD 2014 の機能をご紹介していきましょう。この最新バージョンにも開発コンセプトがありますが、まずは、クラウドと繋がることで利用可能な <strong>接続性</strong> について見ていきます。</p>
<p>AutoCAD 2014 は、クラウドやソーシャル メディア、モバイルへのアクセスで「繋がる」利便性を享受できるデザイン アプリケーションです。Autodesk ID で AutoCAD 2014 の中から Autodesk 360 にサインインすることで、プロジェクト関係者とオンランでコラボレーションしたり、どこにいても AutoCAD の機能を利用することができます。</p>
<p><strong>設計フィード</strong></p>
<p style="padding-left: 30px;">先行して AutoCAD WS に導入された設計フィードが、AutoCAD 2014、AutoCAD LT 2014 で利用できるようになりました。従来の [オンライン] リボンタブが [Autodesk 360] リボン タブに変更されていて、このタブから [設計フィード] パレットを表示させることができます。設計フィードを使った投稿には、テキスト メッセージを入力したり、モバイル端末で撮った写真イメージをアタッチできるので、Autodesk 360 を使用している同僚、顧客、コンサルタントなどと明快なオンライン コラボレーションが可能になります。また、投稿は、図面の特定の領域や点を指定して投稿できるで、図面のどの部分の問題について議論すべきかが明確になる利点があります。</p>
<p style="text-align: center; padding-left: 30px;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017c380f4c68970b-popup" onclick="window.open( this.href, &#39;_blank&#39;, &#39;width=640,height=480,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0&#39; ); return false" style="display: inline;"><img alt="Design Feed" class="asset  asset-image at-xid-6a0167607c2431970b017c380f4c68970b" src="/assets/image_432557.jpg" title="Design Feed" /></a></p>
<p style="padding-left: 30px;">もちろん、設計フィードの機能は AutoCAD WS ユーザと一緒に利用することができるので、AutoCAD WS Mobile を使って、現場から即座にフィードバックを得ることができるようになります。&#0160;&#0160;</p>
<p style="text-align: center; padding-left: 30px;"><iframe frameborder="0" height="344" src="http://www.youtube.com/embed/SUO2sYPe8ac?feature=oembed" width="459"></iframe>&#0160;</p>
<p><strong>地理的位置</strong></p>
<p style="padding-left: 30px;">従来からあった地理的位置の指定方法が大きく強化されています。住所からの検索や、ライブラリから選択可能な任意の測地系座標を使って緯度と経度を指定して、図面上で対応する位置を指定することで、AutoCAD のモデル空間に投影することができるようになります。投影する地図は、Microsoft 社から提供される Bing マップを使用しています。また、道路地図や航空写真、航空写真に道路情報を表記したハイブリッドなどから表示内容を選択することも可能です。</p>
<p style="text-align: center; padding-left: 30px;"><iframe frameborder="0" height="344" src="http://www.youtube.com/embed/9IV94nXsi1Q?feature=oembed" width="459"></iframe>&#0160;&#0160;</p>
<p style="padding-left: 30px;">AutoCAD 上で 3D モデルを使ったコンセプトモデルを作成すれば、把握しにくい都市計画プロジェクトを容易に視覚化して近隣住民の説明に役立てることができます。</p>
<p style="text-align: center; padding-left: 30px;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017d423e6759970c-popup" onclick="window.open( this.href, &#39;_blank&#39;, &#39;width=640,height=480,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0&#39; ); return false" style="display: inline;"><img alt="Live Map" class="asset  asset-image at-xid-6a0167607c2431970b017d423e6759970c" src="/assets/image_442899.jpg" title="Live Map" /></a></p>
<p style="padding-left: 30px;">また、<strong>Autodesk 360 Rendering サービスと併用</strong>することで、フォトリアルなレンダリング画像を短時間で入手したり、日照シミュレーションを活用できます。</p>

<p><strong>点群サポートと Autodesk ReCap</strong></p>
<p style="padding-left: 30px;">AutoCAD 2014 には、主要なレーザースキャナーから計測した点群データを読み込んで、マークアップを加えたり、必要な部分だけを切り出して、別の点群データとして書き出せる点群エディタ Autodesk ReCap が同梱されています（Autodesk ReCap は Reality Capture の略です）。既定では、Autodesk ReCap は AutoCAD 2014 のインストールと同時インストールされます。点群データの読み込み時には、ゴミの除去を指定することも可能です。</p>
<p style="text-align: center; padding-left: 30px;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017ee9b279ae970d-popup" onclick="window.open( this.href, &#39;_blank&#39;, &#39;width=640,height=480,scrollbars=no,resizable=no,toolbar=no,directories=no,location=no,menubar=no,status=no,left=0,top=0&#39; ); return false" style="display: inline;"><img alt="ReCap" class="asset  asset-image at-xid-6a0167607c2431970b017ee9b279ae970d" src="/assets/image_188800.jpg" title="ReCap" /></a></p>
<p style="padding-left: 30px;">Autodesk ReCap は、直観的に理解しやすいインタフェースを備えています。一見、Windows 8 のモダンスタイル アプリケーションのように見えますが、デスクトップ アプリケーションとして動作します。</p>
<p style="text-align: center;">&#0160;<iframe frameborder="0" height="344" src="http://www.youtube.com/embed/8vsX84bTpiM?feature=oembed" width="459"></iframe>&#0160;</p>
<p style="padding-left: 30px;">Autodesk ReCap から出力したファイルを AutoCAD に読み込むことで、3D モデルが存在しない現況をCAD 内で再現させて、追加の 3D モデリングなどに役立てることができます。</p>
