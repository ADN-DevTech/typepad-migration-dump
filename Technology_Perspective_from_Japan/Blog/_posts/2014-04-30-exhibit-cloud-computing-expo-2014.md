<p>オートデスクのクラウド サービス群を新しいカスタマイズ プラットフォームとする考え方が、<strong>&#0160;Autodesk 360 クラウド プラットフォーム</strong>です。そして、Autodesk 360 クラウド プラットフォームを構成するサービスを、徐々に Web サービス API として公開します。第一弾のお披露目の場として <a href="http://www.cloud-japan.jp/" target="_blank"><strong>クラウド コンピューティング EXPO</strong></a> に出展することになりました。&#0160;</p>

<p>ADN として出展するので小さな小間ですが、東京ビッグサイトでの開催です。会場にお越しの場合には、ぜひ <a href="http://www.cloud-japan.jp/To-Visit_Haru/Floor-Plan/" target="_blank"><strong>東ホール</strong> <strong>東21-30</strong></a>&#0160;の小間にお立ち寄りください。</p>
