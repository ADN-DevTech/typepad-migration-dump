<p>Inventor 2016 API Help&#0160;ドキュメントが日本語化されました。残念ながら Inventor 2016 に同梱されている Inventor 2016 API Help は英語版ですが、次の URL から日本語版 Inventor 2016 API Help をダウンロードすることが出来ます。</p>
<p style="padding-left: 30px;"><strong><span class="asset  asset-generic at-xid-6a0167607c2431970b01b7c80f5d90970b img-responsive"><a href="http://adndevblog.typepad.com/files/admapi_20_0.zip">日本語版 Inventor 2016 API Help をダウンロード</a></span></strong></p>
<p style="padding-left: 30px;"><span style="background-color: #ffff00;">※ ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。その場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、画面右下にある 「ブロックの解除」 にチェックを入れてください。</span></p>
<p style="padding-left: 30px;"><span style="background-color: #ffff00;"><span style="background-color: #ffffff;"> <a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb08b4213b970d-pi" style="display: inline; background-color: #ffffff;"><img alt="Inventor_help_blank_contents" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb08b4213b970d image-full img-responsive" src="/assets/image_408555.jpg" title="Inventor_help_blank_contents" /></a></span><br /></span></p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d1164a82970c-pi" style="display: inline;"><img alt="Inventor_2016_api_help" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d1164a82970c image-full img-responsive" src="/assets/image_683894.jpg" title="Inventor_2016_api_help" /></a></p>
<p style="text-align: left;">もともとの英語版 Inventor 2016 API Help （admapi_20_0.chm）は、C:\Program Files\Autodesk\Inventor 2016\Local Help にインストールされているので、ダウンロードした日本語版&#0160;Inventor 2016 API Help （admapi_20_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。</p>
