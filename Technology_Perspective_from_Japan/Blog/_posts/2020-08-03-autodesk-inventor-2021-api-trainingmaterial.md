<p>Autodesk Inventor 2021 の日本語バージョンの発表に伴い、Autodesk Inventor 2021 APIトレーニング教材一式「APIトレーニングテキスト」「APIトレーニング実習ガイドと演習＆回答」及び「演習と実習用ファイル」を公開いたします。</p>
<p>公開されるコンテンツは、以下の2つとなります。</p>
<ul>
<li>Autodesk Inventor 2021 APIトレーニングテキスト（397 page）を pdf ファイルに変換した &quot;Inventor2021API .Net Training_J.pdf&quot;</li>
<li>「演習の解答」( 10の演習の解答) をそれぞれVBAマクロファイルとVisual Studio 2019(VB.Net)のプロジェクトとしたファイル群</li>
<li>「演習で使用するInventorファイル群」</li>
</ul>
<p>&#0160;</p>
<p>以下のリンクからファイルをダウンロードしてください</p>
<p>１．Autodesk Inventor 2021 の APIトレーニング テキスト<br /><span class="asset  asset-generic at-xid-6a0167607c2431970b0263e957b402200b img-responsive"><a href="https://adndevblog.typepad.com/files/inventor2021api-.net-training.pdf">Inventor2021API .Net Trainingをダウンロード</a></span></p>
<p>&#0160;</p>
<p>２ ．演習の解答と演習で使用するInventorファイル群</p>
<p><span class="asset  asset-generic at-xid-6a0167607c2431970b0263ec29502a200c img-responsive"><a href="https://adndevblog.typepad.com/files/autodesk-inventor-2021-%E3%81%AE-api%E3%83%88%E3%83%AC%E3%83%BC%E3%83%8B%E3%83%B3%E3%82%B0-%E6%BC%94%E7%BF%92%E8%A7%A3%E7%AD%94.zip">Autodesk Inventor 2021 の APIトレーニング 演習＆解答をダウンロード</a></span></p>
<p><br />尚、アドイン作成には Visual Studio 2019 有償版(Professionalなど)の開発環境が必要となります。<br />また、各演習の回答マクロの他に同梱されている Visual Studio向けのプロジェクトもVisual Studio 2017 有償版(Professionalなど)を対象として作成されています</p>
<p>なお、Inventor 2019以前のAPIトレーニングマテリアルで演習の解答と合わせてご提供しておりましたデベロッパーガイドについては、今回公開するAPIトレーニングには含まれておりませんので、ご注意ください。</p>
