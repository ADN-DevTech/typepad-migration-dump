<p><span style="background-color: #ffff00;">ご注意：View and Data API は2016年6月に Viewer と &#0160;Model Derivative API に分離、及び、名称変更されました。</span></p>
<p>Autodesk Forge のプラットフォーム API の 1 つである View and Data API の詳細ページへは、<strong><a href="https://developer.autodesk.com/" target="_blank">https://developer.autodesk.com/</a></strong>&#0160;サイト上部の API メニューからジャンプすることが出来ます。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb08c5afd2970d-pi" style="display: inline;"><img alt="Forge_platform_apis" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb08c5afd2970d image-full img-responsive" src="/assets/image_359800.jpg" title="Forge_platform_apis" /></a></p>
<p>View and Data API 詳細ページには、開発者用リポジトリ サービスである <strong><a href="https://ja.wikipedia.org/wiki/GitHub" target="_blank">GitHub</a></strong>&#0160;に記載されているリポジトリへのリンクも含まれます。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d1ab7824970c-pi" style="display: inline;"><img alt="Forge_platform_apis" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d1ab7824970c image-full img-responsive" src="/assets/image_597819.jpg" title="Forge_platform_apis" /></a></p>
<p>GitHub に記載されているサンプル &#0160;プログラムの中には、View and Data API をステップ毎に実装していくことが可能な、自己学習に最適なチュートリアルも含まれています。</p>
<p>まず、上図にある<strong> <a href="http://developer-autodesk.github.io/" target="_blank">View more samples in GitHub </a></strong>リンクをクリックして、<strong><a href="http://developer-autodesk.github.io/" target="_blank">http://developer-autodesk.github.io/</a>&#0160;</strong>へジャンプしてください。新しく表示されるページ内に、<strong>Template Projects</strong> に <strong>JavaScript Node.js - Web</strong>&#0160;と書かれたサンプルを見出すことが出来るはずです。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c8212eb3970b-pi" style="display: inline;"><img alt="Basic_tutorial" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c8212eb3970b image-full img-responsive" src="/assets/image_743574.jpg" title="Basic_tutorial" /></a></p>
<p>新しいページが開くと、&#0160;<strong>Autodesk View and Data API Node.js Basic Sample</strong>&#0160; リポジトリ ページが表示されるはずです。ページ中ほどに、チュートリアル リポジトリへのリンク <strong><a href="https://github.com/Developer-Autodesk/tutorial-getting.started-view.and.data" target="_blank">tutorial material </a></strong>が埋め込まれています。このリンクをクリックすることで、実際のチュートリアル ページである&#0160;<strong><a href="https://github.com/Developer-Autodesk/tutorial-getting.started-view.and.data" target="_blank">Autodesk View &amp; Data API – Getting Started Tutorial リポジトリ</a></strong>&#0160;表示させることが出来ます。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb08c5b77a970d-pi" style="display: inline;"><img alt="Tutorial_materrial" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb08c5b77a970d image-full img-responsive" src="/assets/image_392616.jpg" title="Tutorial_materrial" /></a></p>
<p><strong><a href="https://github.com/Developer-Autodesk/tutorial-getting.started-view.and.data" target="_blank">Autodesk View &amp; Data API – Getting Started Tutorial</a>&#0160;</strong>リポジトリは、最初の部分で <strong>Autodesk View and Data API Node.js Basic Sample </strong>リポジトリの内容を流用していますが、徐々に実装を増やしていくことで、View and Data API &#0160;の基礎的な機能や実装方法を把握できるようになっています。</p>
<p><strong><a href="http://adndevblog.typepad.com/technology_perspective/2016/03/view-and-data-api-tutorial-part1-nodejs-basic.html" target="_blank">次回</a></strong>から数回に分けて、このチュートリアルについて、その内容をご紹介していく予定です。</p>
<p>もし、開発環境や View and Data API の利用手順を把握したい場合には、過去のブログ記事 <strong><a href="http://adndevblog.typepad.com/technology_perspective/2015/09/view-and-data-api-workflow.html" target="_blank">View and Data API ワークフロー</a></strong>&#0160;をご一読いただくことをお勧めします。</p>
