<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a28d0b200d-pi" style="display: inline;"><img alt="Adn-aps-migration" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a28d0b200d image-full img-responsive" src="/assets/image_537977.jpg" title="Adn-aps-migration" /></a></p>
<p>開発者エクスペリエンスの向上を目的として、いままで「ADN オープン」の名のもとに公開されていたページが APS ポータルに統合されつつあります。</p>
<p>11 月 9 日に新たにページ統合が実施されています。すでに統合が完了したページも含め、対象は次の「メンバーシップ オプション」、「プラットフォーム オプション」「APP STORE」のカテゴリです。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a2baa8200d-pi" style="display: inline;"><img alt="Dev_center" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a2baa8200d image-full img-responsive" src="/assets/image_261487.jpg" title="Dev_center" /></a></p>
<p>APS ポータルへの移行先は、それぞれ「メンバーシップ オプション」が「Documentation」、「プラットフォーム オプション」が「Community」配下の「Developer Programs」、「APP STORE」が「App Store」配下の「Publisher Center」です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a2bac6200d-pi" style="display: inline;"><img alt="Destination" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a2bac6200d image-full img-responsive" src="/assets/image_103688.jpg" title="Destination" /></a></p>
<p>「メンバーシップ オプション」は &#0160;<a href="https://adndevblog.typepad.com/technology_perspective/2023/10/integrate_developer-center_pages-to_aps_portal.html">ADN 加入ページの APS ポータル統合</a> の記事でご紹介しましたので割愛しますが、今回、<a href="https://adndevblog.typepad.com/technology_perspective/2023/09/new-aps-document-page.html" rel="noopener" target="_blank">新しい APS ドキュメント ページ&#0160;</a> でご案内した API 概要ページの一部が日本語化されています。</p>
<p><a href="https://aps.autodesk.com/developer/documentation" rel="noopener" target="_blank">https://aps.autodesk.com/developer/documentation</a> ページで「Desktop」を選択すると、「プラットフォーム オプション」に記載されていたデスクトップ製品 API について概説するページへのリンクがリストされます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a26168200b-pi" style="display: inline;"><img alt="Api_docs_landing" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a26168200b image-full img-responsive" src="/assets/image_916865.jpg" title="Api_docs_landing" /></a></p>
<p>ほぼ英語ページへのリンクですが、AutoCAD、Revi、Inventor については、お使いのブラウザの言語設定が日本語になっていると、旧サイトのように日本語ページが表示されます。</p>
<p>デスクトップ製品に同梱されるオンライン ヘルプの各種 API ページへのリンクや、SDK や Visual Studio 上で使用する Wizard ツール のダウンロード リンクが含まれています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a261d4200b-pi" style="display: inline;"><img alt="Document_autocad" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a261d4200b image-full img-responsive" src="/assets/image_489063.jpg" title="Document_autocad" /></a></p>
<p><a href="https://aps.autodesk.com/app-store/publisher-center" rel="noopener" target="_blank">https://aps.autodesk.com/app-store/publisher-center</a> では、<a href="https://apps.autodesk.com/ja" rel="noopener" target="_blank">Autodesk App Store</a> へアプリを公開するための情報が記載されています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a261c6200b-pi" style="display: inline;"><img alt="Store_specifics" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a261c6200b image-full img-responsive" src="/assets/image_334391.jpg" title="Store_specifics" /></a></p>
<p>アドイン（プラグイン）の記載等、対応するデスクトップ製品やクラウド サービスによって異なる規定が設けられていますが、お使いのブラウザの言語設定が日本語になっていると、それら情報ページも日本語ページで表示されるはずです。</p>
