<p><strong>内容：</strong></p>
<h4 style="padding-left: 40px;"><strong>9:30 am - 11:00 am: Autodesk Forge プラットフォームの概要</strong></h4>
<p style="padding-left: 80px;">&quot;現在の&quot; Forge がもたらすビジネス成長の機会をご紹介します。Forge エキスパートが、ライブデモや実際の使用例を交えて、何を、なぜ、いつ、どのように行うかを説明します。Forgeが提供する意思決定ダッシュボード、デジタルツイン、コンフィギュレーターを含むタスクの自動化、加工図を排除するダイレクトモデルからマシンデータへのフロー、その他多くの例をご覧いただけます。</p>
<p style="padding-left: 80px;"><strong>対象者：</strong>Forge 概要に興味をお持ちの方、Forge を初めて使う方</p>
<h4 style="padding-left: 40px;"><strong>11:15 am - 12:45 pm: Forge Data の概要</strong></h4>
<p style="padding-left: 80px;">&quot;新しく登場する&quot; Forge Data は、個人とチームのためのツール（またはツール開発プラットフォーム）から、企業のデジタルトランスフォーメーションの実現、移行を強力に後押しする &quot;データ プラットフォーム&quot; を提供します。CAD によって異なるバイナリファイルの不透明なデータを、データアクセスが迅速、簡単、拡張可能、かつ連携可能なクラウドベースのデータベースへと導きます。その一端をご確認ください。</p>
<p style="padding-left: 80px;"><strong>対象者：</strong>Forge Data が提供する最新の計画についてお知りになりたい方</p>
<h4 style="padding-left: 40px;"><strong>1:45 pm - 3:30 pm: Forge Data 開発の実際 - Part 1</strong></h4>
<p style="padding-left: 80px;">設計・製造データのスピード、パワー、コラボレーション、セキュリティを低コストで実現する方法をご紹介します。このセッションでは、Model Derivative API および Design Automation API、BIM 360/Autodesk Construction Cloud のモデル プロパティを紹介します。</p>
<p style="padding-left: 80px;"><strong>対象者：</strong>Forge API を利用してみたい開発者の方</p>
<h4 style="padding-left: 40px;"><strong>3:45 pm - 5:15 pm: Forge Data 開発の実際 - Part 2</strong></h4>
<p style="padding-left: 80px;">まもなくリリースされる Data Exchange API に焦点を当て、Forge Data をベースに構築された相互運用性のすべてを紹介します。最後に、3つの主要産業（AEC、製造業、メディア＆エンターテインメント）向けの Forge Data を活用する Cloud Information Modeling（クラウド情報モデリング）の方向性について掘り下げます。</p>
<p style="padding-left: 80px;"><strong>対象者：</strong>Forge API を利用してみたい開発者の方</p>
