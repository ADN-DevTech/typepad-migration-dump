<p>Autodesk Platform Services（旧 Forge）を学習する場合、<a href="https://adndevblog.typepad.com/technology_perspective/2022/12/aps-learning-site.html" rel="noopener" target="_blank">APS 学習サイト：APS チュートリアル</a> でご紹介した、英語表記の&#0160;<a href="https://tutorials.autodesk.io/" rel="noopener" target="_blank">Learn APS Tutorial</a>&#0160;を使って、既存 API の役割や動作を把握いただくことが可能です。Learn APS Tutorial&#0160; の日本語化を検討・調査していたのですが、複数ページで構成される Learn APS Tutorial が APS ポータル内部に組み込まれているため、日本語化で問題が発生することがわかりました。</p>
<p>そこで、ご参考まで、暫定策として、以前、独立したサイトで日本語化を進めて公開した <a href="https://adndevblog.typepad.com/technology_perspective/2021/10/localized-learn-forge.html" rel="noopener" target="_blank">Learn Forge</a> について、ローカル環境で実行・内容を参照する方法をご紹介しておきたいと思います。なお、Learn Forge は、当時の URL（https://learnforge.autodesk.io/）を入力しても Learn APS Tutorial にリダイレクトされてしまい、Live 版を参照出来なくなっています。</p>
<p>Learn Forge は Node.js を使用します。Node.js の入手先とインストールに関しては、必要に応じて <a href="https://adndevblog.typepad.com/technology_perspective/2017/01/development-environment-for-forge.html" rel="noopener" target="_blank">APS の開発環境</a>&#0160;の内容をご確認ください。</p>
<hr />
<ol>
<li>Learn Forge は <a href="https://github.com/Developer-Autodesk/forge.learning" rel="noopener" target="_blank">https://github.com/Developer-Autodesk/forge.learning</a> GitHub リポジトリで公開されています。（Learn Forge をローカル環境で実行する方法は、<a href="https://nam11.safelinks.protection.outlook.com/?url=https%3A%2F%2Fgithub.com%2FDeveloper-Autodesk%2Fforge.learning%23running-locally&amp;data=05%7C01%7Ctoshiaki.isezaki%40autodesk.com%7Caa198ed3d6d04308ee6a08db455e15bf%7C67bff79e7f914433a8e5c9252d2ddc1d%7C0%7C0%7C638180042631321153%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C3000%7C%7C%7C&amp;sdata=MEFLjmIk%2BvBCGaxQx1AxAXTleYc1XjLTrCR%2FZyoOzjA%3D&amp;reserved=0">https://github.com/Developer-Autodesk/forge.learning#running-locally</a> で概説されています。）</li>
<li>コマンドプロンプトを起動後、Git コマンド <strong>npm i docsify-cli -g</strong> と入力して、Learn Forge がドキュメント管理で使用している <a href="https://docsify.js.org/#/?id=docsify" rel="noopener" target="_blank">docsify</a> パッケージを Node.js 環境にインストールします。<br /><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b751a73502200c-pi" style="display: inline;"><img alt="Install_docsify" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b751a73502200c image-full img-responsive" src="/assets/image_64517.jpg" title="Install_docsify" /></a></li>
<li>CD コマンドでリポジトリをクローンしたいフォルダに移動したら、<strong>git clone https://github.com/Developer-Autodesk/forge.learning</strong> と入力して、リポジトリをローカル環境にクローンします。<br /><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b75182d06f200b-pi" style="display: inline;"><img alt="Clone_learnforge" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b75182d06f200b image-full img-responsive" src="/assets/image_363719.jpg" title="Clone_learnforge" /></a></li>
<li>forge.learning フォルダが作成されてリポジトリがクローンされているので、CD コマンドで forge.learning フォルダに移動します。</li>
<li>forge.learning フォルダ直下の index,html をエディタで開いて、8 行目のリダイレクト処理の定義を削除、または、コメント化します。<br /><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68539d449200d-pi" style="display: inline;"><img alt="Comment_out" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b68539d449200d image-full img-responsive" src="/assets/image_257436.jpg" title="Comment_out" /></a></li>
<li>続いて、<strong>docsify serve ./docs</strong> と入力して、docsify に docs サブフォルダにクローンされているドキュメントを認識、ローカルに配信させます。<br /><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b75182d07a200b-pi" style="display: inline;"><img alt="Run_docsify" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b75182d07a200b image-full img-responsive" src="/assets/image_250123.jpg" title="Run_docsify" /></a></li>
<li>Web ブラウザを起動して、ローカルで実行されている Learn Forge を表示します。<br /><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b75182e70b200b-pi" style="display: inline;"><img alt="Learn_forge_en" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b75182e70b200b image-full img-responsive" src="/assets/image_276181.jpg" title="Learn_forge_en" /></a></li>
<li>Learn APS Tutorial にリダイレクトせずに英語の Learn Forge が表示されるので、右上の JA をクリックして、日本語の Learn Forge を表示させます。<br /><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b68539d464200d-pi" style="display: inline;"><img alt="Learn_forge_ja" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b68539d464200d image-full img-responsive" src="/assets/image_825859.jpg" title="Learn_forge_ja" /></a></li>
</ol>
<hr />
<p>ローカル環境でLearn Forgeを参照出来るようになったら、昨年開催した Forge Online Training の内容をご確認いただけるはずです。</p>
