<p>Inventor 2019 API の日本語プログラミング用ヘルプ&#0160;（admapi_23_0.chm）がリリースされました。 ZIP 圧縮したファイルをポストしていますので、次のリンクからダウンロードしてください。</p>
<p style="padding-left: 30px;"><span class="asset  asset-generic at-xid-6a0167607c2431970b01b8d2da2715970c img-responsive"><a href="http://adndevblog.typepad.com/files/admapi_23_0.zip"><strong>Inventor 2019 API の日本語プログラミング用ヘルプをダウンロード</strong></a></span></p>
<p>日本語版 Inventor 2019 に同梱されている API ヘルプは英語版オンライン ヘルプになっていて、<strong>C:\Users\Public\Documents\Autodesk\Inventor 2019\Local Help</strong>&#0160;にインストールされています。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c8ea2c83970b-pi"><img alt="Access_api_help" class="asset asset-image at-xid-6a0167607c2431970b01b7c8ea2c83970b img-responsive" src="/assets/image_755776.jpg" title="Access_api_help" /></a></p>
<p>ダウンロードした日本語版の Inventor 2019 API Help （admapi_23_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。&#0160;</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c94fbbe7970b-pi" style="display: inline;"><img alt="Api_help" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c94fbbe7970b image-full img-responsive" src="/assets/image_645992.jpg" title="Api_help" /></a></p>
<p>ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。その場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、画面右下にある 「ブロックの解除」 (Windows 7)、「許可する」(Windows 10） にチェックを入れてください。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb09f36a0d970d-pi" style="display: inline;"><img alt="Permision_settings" class="asset  asset-image at-xid-6a0167607c2431970b01bb09f36a0d970d img-responsive" src="/assets/image_131770.jpg" title="Permision_settings" /></a></p>
<p>なお、今回の Inventor 2019 では従来の CHM ファイル形式とは別に、英語版 Inventor 2019 のみ、<strong><a href="https://help.autodesk.com/view/INVNTOR/2019/ENU" rel="noopener noreferrer" target="_blank">オンライン&#0160;ヘルプ</a></strong>&#0160;内に Programming Interface セクションが追加されています。残念ながら、日本語版 Inventor には反映されてはいませんが、記載されている内容は CHM 形式と同じものになっています。</p>
