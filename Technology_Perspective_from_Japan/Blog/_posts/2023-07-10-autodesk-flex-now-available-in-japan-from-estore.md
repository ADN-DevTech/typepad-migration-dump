<p>Autodesk Platform Services（旧 Forge）の課金制度は、<a href="https://adndevblog.typepad.com/technology_perspective/2022/11/flex-token-adoption-into-aps-on-11-7.html" rel="noopener" target="_blank">2022年11月</a>に Autodesk Flex を利用する従量課金に移行しています。</p>
<p>Autodesk Flexの ご購入に際して、当初、eStore（オンライン ストア）では販売をサポート出来ていなかったため、お見積書の作成も含め、Flex 特約を締結済の Platinum/Gold 認定リセラーのいずれかに直接コンタクトして入手いただく必要がありました。</p>
<p>この度、2023年7月7日から eStore（オンライン ストア）で Autodesk Flex の販売が出来るようになりましたので、クレジットカードを使ったご購入方法についてご案内しておきたいと思います。</p>
<p>eStore をご利用いただく場合には、<a href="https://aps.autodesk.com/pricing" rel="noopener" target="_blank">https://aps.autodesk.com/pricing</a> へアクセスをしていただき、トライアル期間の終了した Autodesk ID でサインイン後、ページ中段の <strong><span style="color: #ffffff; background-color: #111111;">[ FLEX を購入 &gt; フレックスページ ]</span> </strong>をクリックしてください。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c1a6cceb53200b-pi" style="display: inline;"><img alt="Pricing_page" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c1a6cceb53200b image-full img-responsive" src="/assets/image_294006.jpg" title="Pricing_page" /></a></p>
<p><a href="https://www.autodesk.co.jp/buying/flex" rel="noopener" target="_blank"><strong>Autodesk Flex ページ</strong>（https://www.autodesk.co.jp/buying/flex）</a>にジャンプしますので、「購入するトークン」セクションで購入するトークン数を選択後、<strong>[トークンを購入]</strong> をクリックします。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c1b258cc82200d-pi" style="display: inline;"><img alt="Purches_token" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c1b258cc82200d image-full img-responsive" src="/assets/image_769594.jpg" title="Purches_token" /></a></p>
<p>eStore ページに遷移して、最初に Autodesk ID でサインインを求められます。画面の指示に従ってユーザ名とパスワードを入力し、サインインを完了します。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b751aa56ba200c-pi" style="display: inline;"><img alt="Signin" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b751aa56ba200c image-full img-responsive" src="/assets/image_515693.jpg" title="Signin" /></a></p>
<p>次に、購入したい Autodesk Flex のトークン数を確認して、必要に応じて<a href="#amount_of_flex">入力</a>し直します。</p>
<ul>
<li>ここでは画面上に「新規サブスクリプション」と表示されますが、Autodesk Platform Services はサブスクリプション制を廃止していますので、この表現は無視していただいて結構です。</li>
<li>ただし、ご購入いただくトークンは購入から 1 年間の有効期限を持ちますので、消費出来ないような大量のトークン購入には注意が必要です。</li>
<li>なお、アプリがどの程度トークンを消費しているかは、<a href="https://adndevblog.typepad.com/technology_perspective/2022/11/updated-usage-analytics-page-for-flex.html">Usage Analytics ページ：Flex トークン残高と消費量の確認</a> の記事でご案内した Usage Analytics ページでご確認いただくことが出来ます。（90 日間の無償トライアル期間中は、Usage Analytics ページにトークン残高や消費量は表示されません。）</li>
</ul>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c1b258cb86200d-pi" style="display: inline;"><img alt="Amount_of_token" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c1b258cb86200d image-full img-responsive" src="/assets/image_883205.jpg" title="Amount_of_token" /></a></p>
<p>続いて、お客様情報を入力します、サインインした Autodesk ID の住所等の情報が登録されていれば、それらが表示されてきます。変更が必要な場合は、「お客様情報」セクション右上の [編集] ボタンをクリックして内容を編集することが出来ます</p>
<p><span style="background-color: #111111;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c1b258cb99200d-pi" style="display: inline; background-color: #111111;"><img alt="Account_info" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c1b258cb99200d image-full img-responsive" src="/assets/image_865031.jpg" title="Account_info" /></a></span></p>
<p>支払い方法に「クレジットカード」を選択、適宜、画面の指示に従って請求先情報を入力したら、<span style="background-color: #111111; color: #ffffff;"><strong>[注文の確認]</strong> </span>をクリックしてください。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c1b258cbf9200d-pi" style="display: inline;"><img alt="Payment_info" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c1b258cbf9200d image-full img-responsive" src="/assets/image_807364.jpg" title="Payment_info" /></a></p>
<p>ページ右側に入力した注文内容が表示されるので、内容に間違いがなければ、<span style="color: #ffffff; background-color: #111111;"><strong>[注文の送信 &gt; ]</strong></span> をクリックして注文を確定します。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c1a6cceb6f200b-pi" style="display: inline;"><img alt="Place_order" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c1a6cceb6f200b image-full img-responsive" src="/assets/image_120927.jpg" title="Place_order" /></a></p>
<p>次のページが表示されたら、ご購入のお手続きは完了です。</p>
<ul>
<li>ページ右の青矢印部分に、購入されたトークンの有効期限が表示されるていることにご注意ください。ご購入から 1 年間を経過する時点で、購入されたトークンに残高があった場合には、有効期限切れとともにトークン残高が失効することになります。（Usage Analytics ページに表示される「Available」値から失効したトークン数が減算されます。）</li>
</ul>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b751aa5794200c-pi" style="display: inline;"><img alt="Thank_you" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b751aa5794200c image-full img-responsive" src="/assets/image_85359.jpg" title="Thank_you" /></a></p>
<ul>
<li>なお、Autodesk Flex を購入した方と実際に Autodesk Platform Services を使った開発者の方が同一（Autodesk Flex 購入者の Aitodesk ID と&#0160;<a href="https://adndevblog.typepad.com/technology_perspective/2016/07/app-registration-and-getting-keys-for-forge-api.html" rel="noopener" target="_blank">APS API を利用するアプリの登録とキーの取得</a> の手順で開発に必要なデベロッパーキーを取得した方の Autodesk ID が同じ）の場合には、追加の作業は不要です。</li>
<li>もし、Autodesk Flex 購入者と Autodesk Flex を消費する開発者が異なる場合には、開発者がトークン消費出来るよう、Autodesk Platform Services を割り当てる必要があります。その場合には、<span style="background-color: #111111; color: #ffffff;"><strong>[ユーザーを割り当てる]</strong></span> をクリックして、割り当て処理をおこなってください。</li>
</ul>
<p>注文が処理されると、「請求書/納品書/領収書【オートデスク株式会社】」のタイトルを持つメールが送られます。メールには「請求書/納品書/領収書」と題した PDF ファイルが添付されているので、必要に応じて購買処理にお使いいただけます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b751aa5b2e200c-pi" style="display: inline;"><img alt="Invoice" class="asset  asset-image at-xid-6a0167607c2431970b02b751aa5b2e200c img-responsive" src="/assets/image_407375.jpg" title="Invoice" /></a></p>
<p>その他の補足事項は次のとおりです。</p>
<ul>
<li>購入した Autodesk Flex トークンが Usage Analytics ページに反映されるには、最大で 24 時間かかる場合があります。</li>
<li>既に APS を使用していて、お手持ちのトークン（「Available」値）がマイナスになっている場合、猶予期間中に購入したトークン数からマイナス消費分が相殺されて表示されます。</li>
<li>あいにく、現在、見積書の自動作成機能がありません。見積書をご入用の場合には、<a href="https://customersuccess.autodesk.com/partners/search?search=&amp;locations%5B%5D=Japan&amp;languages%5B%5D=Japanese&amp;services%5B%5D=Flex" rel="noopener" target="_blank">Autodesk Customer Success Hub</a>&#0160;から検索可能な認定リセラーにご用命の上、ご購入をお願いいたします。</li>
<li><a id="amount_of_flex"></a>購入トークン数は、最小 100 から 1 トークン刻みで購入可能です。</li>
</ul>
