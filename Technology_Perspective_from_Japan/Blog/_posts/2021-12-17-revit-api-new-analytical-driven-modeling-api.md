<p><a href="https://trello.com/b/ldRXK9Gw/revit-public-roadmap"><strong>Revit Public Roadmap</strong></a> では、Revit の最新のロードマップを公開しておりますが、このほど新しい項目が追加され、Revit で解析モデリングを行うための新しい API 、<strong>Analytical Driven Modeling API</strong> が発表されました。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0282e138e8cb200b-pi" style="display: inline;"><img alt="RevitRoadmap" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0282e138e8cb200b image-full img-responsive" src="/assets/image_140888.jpg" title="RevitRoadmap" /></a></p>
<blockquote>
<p><em>構造解析モデルを直接制御できる新しい API です。物理的な要素を抽象化して動作するシンプルな実装方法を提供します。サードパーティのデベロッパーは、解析要素の作成と修正を独立して操作できるようになります。</em></p>
</blockquote>
<p><strong><a href="https://help.autodesk.com/view/RVT/2022/JPN/?guid=Revit_API_Revit_API_Developers_Guide_Discipline_Specific_Functionality_Structural_Engineering_Analytical_Model_html">既存の解析モデル API</a> </strong>を利用してアドインを開発されている場合は、今後、<strong>既存の API の一部は廃止予定</strong>となり、新しい API にマイグレーションいただく必要がございますので、ご注意ください。</p>
<p>By Ryuji Ogasawara</p>
