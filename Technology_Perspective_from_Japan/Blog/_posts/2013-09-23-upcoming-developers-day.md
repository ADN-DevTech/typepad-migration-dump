<p>クラウドとモバイル アプリのアイデアを得る機会をお探しですか?<br />オートデスクがデスクトップ コンピューティングやクラウド テクノロジをどこに収束させようとしているか把握したいとお考えですか?</p>
<p>オートデスクのプラットフォーム テクノロジ上でアプリケーション開発をされていたり、あるいは、まだ Autodesk Developer Network カンファレンスにご参加いただけていない場合には、オートデスクの最新テクノロジと、来るべき次世代の製品群を深く紹介する Developer Days への参加をご検討ください。Autodesk Developer Network では、本年度も 2013年11月から2014年2月にかけて、世界中で Developer Days の開催を予定しています。</p>
<p>DevDays 2013 と呼ぶ今年のテーマは、rEvolution – &quot;デスクトップがクラウドに遭遇するところ&quot; です。 ぜひ、DevDays 2013 にご参加いただいて、オートデスクが検討している新しい Web サービス API の概要を把握してください。これら新しい Web サービス API が既存のデスクトップのパワーをどう拡張し得るか、容易なコラボレーションの提供、制限のない演算パワーやビッグデータの扱いなど、開発者とお客様に提供することができるプラットフォームを見ていただくことができるはずです。製造、建設、インフラストラクチャなどの数多くの業種に向けた、クラウド べースのプラットフォームを提供する &quot;360 ベース&quot; の Web サービス API を習得いただく機会となります。もちろん、デスクトップ製品（AutoCAD®, Autodesk® Revit®, Autodesk® Inventor®, AutoCAD® Civil 3D® 等）の次期リリースについてもご案内する予定です。</p>
<p>なお、高い秘密保持情報を扱う&#0160;Developer Days へご参加いただくいためには、ADN メンバとなっていただく必要があります。 <a href="http://www.autodesk.co.jp/adnopen" target="_blank">ADN オープン</a> やブログ記事、フォーラムをお使いで、DevDays への参加をご希望の場合には、まずは&#0160;<a href="http://www.autodesk.co.jp/adn" target="_blank">www.autodesk.co.jp/adn</a>&#0160;をご参照ください。</p>
<p>既に ADN にご加入いただいているメンバの方には、来週以降、ご担当者宛てに Developer Days のご案内メールが届きます。お申込みまで、もうしばらくお待ちください。</p>
