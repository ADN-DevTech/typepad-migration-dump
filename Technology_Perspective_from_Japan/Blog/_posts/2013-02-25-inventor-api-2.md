<p><strong>Autodesk Inventor API 日本語デベロッパガイド ( admapi_17_0.chm )</strong></p>

<p>API 開発に必要不可欠なデベロッパガイドが日本語に翻訳されましたので、お知らせします。<br />
<a class="asset-img-link"  style="display: inline;" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017ee8b791bc970d-pi"><img class="asset  asset-image at-xid-6a0167607c2431970b017ee8b791bc970d image-full" alt="Developer Guide" title="Developer Guide" src="/assets/image_124270.jpg" border="0" /></a><br />
これらは、Autodesk Inventor API リファレンスの一部としてリリースされており、Autodesk Technical Q&A <a href="http://tech.autodesk.jp/faq/faq/adsk_result_dd.asp?QA_ID=7857">Q&A-7857 「日本語に翻訳した Inventor2013 APIの ユーザーマニュアル は有りますか？」</a> にも同時掲載しています。</p>

<p><em><strong>注意と制限：</strong></em></p>

<p>Local Help ホルダー内の オリジナルの api_17_0.chm ファイルと置き換えての VBAIDE環境内の "F1" キーによる 連動表示 は、日本語化によって chmファイル内のマッピングテーブルが異なるために ご利用いただけません。<br />
そのために、Local Help ホルダー以外の、任意のホルダー内に解凍いただいた状態 で、単独でご使用下さい。</p>

<p>デベロッパガイドは、<span class="asset  asset-generic at-xid-6a0167607c2431970b017d41439d8c970c"><a href="http://adndevblog.typepad.com/files/admapi_17_0_j.zip">こちら</a></span> からもダウンロードしてご利用いただけます。</p>
