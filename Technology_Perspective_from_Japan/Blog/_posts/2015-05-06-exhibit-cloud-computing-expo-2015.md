<p>5.月13日から開催されるクラウドコンピューティングEXPO に於いて、日本でAmazon Web Services を運営するアマゾンデータ サービスジャパン株式会社のパートナーとして、同社ブース内に出展することが決まりました。</p>

<p>建築・建設業、製造業向けに最適化されたクラウドソリューション A360 と、API カスタマイズが可能な Web サービス、View and Data API をご紹介します。</p>
