<div class="clearfix text-formatted">
<h2 class="h3">Forge プラットフォームについて</h2>
</div>
</div>
<div class="paragraph--type--faqs  col-xs-12 js-faq-accordion" data-collapse="open">
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>Forge とは何ですか?</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Autodesk Forge は Web サービス、テクニカル リソース、コミュニティを含む 、“繋げる” 開発者向けのクラウド プラットフォームです。Forge は一般的なデータ環境で動作し、オートデスクが 35 年間培ってきた業界の専門知識、テクノロジ、グローバル ネットワークを利用して、デザインやエンジニアリングの課題の解決、製造業のチャレンジにフォーカスしたスケーラブルでカスタマイズされたソリューション開発を、多くの企業が迅速に開始できるプラットフォームを提供します。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__text">
<div><strong>Forge で何が出来ますか?</strong></div>
</div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<ul>
<li>既存のオンライン ワークフローの利用をさらに加速させたり、新しいソリューションを構築することが出来ます。</li>
<li>関係者間のコンポーネント、プロセスを接続することで、製造業、エンジニアリング業、建設業の複雑さを合理化します。</li>
<li>データを一元化し、業界のサプライチェーンのすべての拠点を接続し、製造プロセスを変革させる。</li>
<li>いつでもどこでも、あらゆる種類のデザインやエンジニアリング データにアクセスできるデジタル体験を作成します。</li>
<li>ユーザのニーズや革新度に応じて拡張性を変化させる柔軟性を提供します。</li>
<li>業界標準の Web テクノロジを使用して、既存のソフトウェアを改善したり、新しいカスタムソリューションを構築出来ます。</li>
</ul>
<p>未来を切り開くためにオートデスクのユーザが、どのように Forge を利用しているか <strong><a href="https://forge.autodesk.com/customers" rel="noopener noreferrer" target="_blank">ユーザ事例</a></strong>&#0160;をご確認ください。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>Forge は誰のためのものですか?</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Forge は、将来のデジタルニーズを満たすカスタマイズされたスケーラブルなソリューションの開発を可能にすることで、企業がエンジニアリング、建設、製造の課題に取り組むのに役立ちます。 新しいアプリケーションを構築、または、既存の製品をアップグレードするのに役立つ API（Application Programming Interface）とサービスを提供することでに、 Forge は、デザイン データを活用して革新とコラボレーションの機会を創出します。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>Forge は他の PaaS（Platform as a Service）とどう異なるのですか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Forge は、BIM（Building Information Modeling）、製造、CAD（Computer Aided Design）、PLM（Product lifecycle Management）などの SaaS（Software as a service）を提供している、3D 設計、エンジニアリング、建設、エンターテイメント ソフトウェアにおける 35 年以上の業界専門知識を持つリーダーでもあるオートデスクから提供させるの唯一の PaaS &#0160;です。また、オートデスクのパートナーが持つクラウド サービスとユーザのためのソリューションを結ぶ基盤となるクラウド サービスと IT インフラストラクチャを実行環境でもあります。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>プラットフォームには何が含まれていますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p style="padding-left: 30px;"><strong>Viewer <a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c925ee76970b-pi" style="float: right;"><img alt="Icon - viewer" class="asset  asset-image at-xid-6a0167607c2431970b01b7c925ee76970b img-responsive" src="/assets/image_307588.jpg" style="width: 80px; margin: 0px 0px 5px 5px;" title="Icon - viewer" /></a><br /></strong></p>
<p style="padding-left: 30px;">基 本的な VR（Virtual Reality）と 50 種類以上のファイル形式の関連データを含む 2D と 3D デザイン ファイルを Web ブラウザにストリーミング表示します。 ユーザは Web ブラウザを介して設計情報を共有し、プロジェクト ダッシュボード、原価見積、BOM（Bill of Materials）やスケジュール ビューなど、カスタマイズされたさまざまなユーザ エクスペリエンスを作成できます。 ビューアは、他のアプリケーションからの情報も活用出来ます。</p>
<p style="padding-left: 30px;"><strong>Model Derivative API <a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c925ee6e970b-pi" style="float: right;"><img alt="Icon - model derivative api" class="asset  asset-image at-xid-6a0167607c2431970b01b7c925ee6e970b img-responsive" src="/assets/image_485179.jpg" style="width: 80px; margin: 0px 0px 5px 5px;" title="Icon - model derivative api" /></a></strong></p>
<p style="padding-left: 30px;">さまざまな形式をデザイン ファイルをオンラインで表示するために変換、共有し、ジオメトリやオブジェクトのプロパティ データを抽出します。 設計データは、ERP（Enterprise Resource Planning）、PLM（Product Lifecycle Management）、コスト算出、スケジュール管理など、他のシステムと迅速かつ容易に統合することができます。</p>
<p style="padding-left: 30px;"><strong>Design Automation API <a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb09c903bb970d-pi" style="float: right;"><img alt="Icon - design automation api" class="asset  asset-image at-xid-6a0167607c2431970b01bb09c903bb970d img-responsive" src="/assets/image_406886.jpg" style="width: 80px; margin: 0px 0px 5px 5px;" title="Icon - design automation api" /></a></strong></p>
<p style="padding-left: 30px;">AutoCAD、Revit、Inventor、3ds Max のコア エンジンを使って、クラウド上でアドイン/プラグインを実行することで、デザイン ファイルの生成エンジンとして使用したり、何千もの デザインファイルから属性/メタデータを抽出したりする Web アプリケーションを開発することが出来ます。</p>
<p style="padding-left: 30px;"><strong>Data Management API <a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c925ee8d970b-pi" style="float: right;"><img alt="Icon - data management" class="asset  asset-image at-xid-6a0167607c2431970b01b7c925ee8d970b img-responsive" src="/assets/image_481592.jpg" style="width: 80px; margin: 0px 0px 5px 5px;" title="Icon - data management" /></a></strong></p>
<p style="padding-left: 30px;">A360、Fusion 360、BIM 360 Docs が利用するユーザ ストレージ領域や、Forge ネイティブの OSS（ Object Storage Service）に保存されているファイルを管理します。 統合されたインタフェースを使用して、異なるオートデスク製品や Web サービスからデータファイルをアップロード/ダウンロード出来ます。 他の企業内システムやプロジェクト システムと統合するために、プロジェクト レベル、または、ファイル レベルのデータにアクセスすることも可能です。</p>
<p>詳細な情報は&#0160;<a href="https://forge.autodesk.com/developer/documentation" rel="noopener noreferrer" target="_blank">API ドキュメント</a> をご参照ください。<strong>&#0160;</strong></p>
</div>
</div>
</div>

<div class="clearfix text-formatted">
<h2 class="h3">価格</h2>
</div>
</div>
<div class="paragraph--type--faqs  col-xs-12 js-faq-accordion" data-collapse="open">
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__text">
<div><strong>Forge の利用コストはいくらなのでしょうか?</strong></div>
</div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>利用コストは期間契約（サブスクリプション）によって提供されるクラウド クレジットによってまかなわれます。Forge を使用したアプリケーションの開発を始めるためには、<strong><a href="https://adndevblog.typepad.com/technology_perspective/2016/07/app-registration-and-getting-keys-for-forge-api.html" rel="noopener" target="_blank">Forge ポータルでアプリ作成を実施してデベロッパキー</a></strong>を入手する必要があります。無償トライアルもご利用いただくことは出来ます。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>クラウド クレジットとは何ですか?</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>クラウド クレジットは、オートデスクのクラウド サービスで消費されるもので、ファイルを変換したり、レンダリングしたりするなど、クラウド サービスの特定のタスクを実行するために必要な重量課金（計測）単位です。</p>
<p>追加のクラウド クレジットは、推定消費量に基づいて購入することができます。 詳細については、当社の&#0160;<a href="https://forge.autodesk.com/en/pricing" rel="noopener noreferrer" target="_blank">価格ページ</a> <a href="https://developer.autodesk.com/en/pricing" rel="noopener noreferrer" target="_blank"></a>をご覧いただくか、<a href="mailto:forge.sales@autodesk.com">forge.sales@autodesk.com</a>までお問い合わせください。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>Forge をビジネス モデル（独自に課金中のクラウド サービスなど）に統合するにはどうすればいいですか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Forge ビジネス モデルを独自のアプリケーションに統合するには、そのサービスの使用量に応じてコストをユーザに課金するなど、まざまな方法が考えられます。オートデスクは Forge を利用する開発者にのみクラウド クレジットでコストとして要求するためり、3rd Party の開発者が独自に提供しているサービスの課金内容には関知しません。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>無償トライアル（体験版）はありますか?</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>はい。 100 クラウド クレジット（ 1 年間有効）を無償提供するトライアルをお使いいただけます。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>Forge のクラウド クレジットを購入して使用できますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>いいえ、Forge に登録して、プラットフォームを使用するサブスクリプションを購入し、付帯するクラウド クレジットを使用する必要があります。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__text"><strong>現在、Forge プラットフォーム内の 1 つの API のみを使用しています。 個々の API の価格設定パッケージはありますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Forge サブスクリプションでは、クラウド クレジットを任意の API に適用することができます。</p>
</div>
</div>
</div>

<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>未使用のクラウド レジットは翌月に繰り越しすることは出来ますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>翌月にのみ繰り越すことが出来ます。例えば、1 か月目に余ったクラウドクレジットは翌月に繰り越せますが、そのクラウド クレジットは&#0160; 2 か月めに余ったとしても、3 か月めに繰り越すことは出来ません。なお、すべてのクラウド クレジットは、受領後 12 ヶ月以内に失効します。</p>
</div>

<div class="clearfix text-formatted">
<h2 class="h3">使用方法と利用状況</h2>
</div>
</div>
<div class="paragraph--type--faqs  col-xs-12 js-faq-accordion" data-collapse="open">
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__text"><strong>サブスクリプションからすべてのクラウド クレジットを消費してしまった場合どうなりますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>サブスクリプションの利用限度に達すると（手持ちのクラウド クレジットがなくなると）、追加のクラウド クレジットを購入するためのオプションが提供されます。<span style="text-decoration: underline;">通知なしに</span> API へのアクセスを無効にするようなことはありません。 ただし、オートデスクは、クラウド クレジットなしの状態で過剰な API の使用を制限する権利を留保し、API の使用を制限する前に Forge を利用中の開発者、または、ユーザとコミュニケーションを開始します。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>Forge プラットフォームはどのようなプログラミング言語をサポートしていますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Forge プラットフォームの各種 API は、クライアントである Web ブラウザで動作する Viewer の JavaScript ライブラリを除いて、一連の RESTful API を提供しています。 このため、開発言語とツールを自由に選択、使用することができます。&#0160;</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>このプラットフォームはどの地域に対応していますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Forge 無償トライアルとサブスクリプションは全世界で利用することが出来ます。 米国、または、ヨーロッパに在住の場合には、Forge サブスクリプションをオンラインで購入することができます。 米国、ないし、ヨーロッパ以外で Forge の購入にご興味をお持ちの場合には、<a href="mailto:forge.sales@autodesk.com">forge.sales@autodesk.com</a> までメールでご連絡ください。</p>

<div class="clearfix text-formatted">
<h2 class="h3">セキュリティ</h2>
</div>
</div>
<div class="paragraph--type--faqs  col-xs-12 js-faq-accordion" data-collapse="open">
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__text"><strong>セキュリティ/データプライバシーはどのように確保されていますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<ol>
<li>オートデスクは、製品ライフサイクル全体にわたって安全な開発作業を実行することで、製品にセキュリティを構築しています。 これによって、脅威に対してより柔軟性のある製品を開発し、攻撃者の悪用を困難にし、ユーザとその資産を保護しています。</li>
<li>オートデスクは、アクティブな監視と脅威インテリジェンスを使用して、オートデスク クラウドに対する脅威を検出し、その脅威を事前に防止します。</li>
<li>ユーザのリスクを効果的に低減するよう対応していきます。 オートデデスクが防御策を構築する際にどれほど厳格であったとしても、攻撃者は引き続きそれらを突破する方法を見いだすかもしれません。 オートデスクは積極的にセキュリティの脆弱性の優先付けと対処をおこない、迅速に脅威を無効化することができます。 さらに、ユーザと透過的にコミュニケーションしながら、ユーザ自身を保護するための措置を講じることで、製品セキュリティの重要箇所を構成していきます。</li>
<li>オートデスクは業界の規制と基準を遵守しています。環境を確保するための適切な要素と活動に焦点を当てていることを確認するため、第 3 者認証を使用しています。 コンプライアンスは良好なセキュリティの副産物でなければならないということです。</li>
<li>オートデスクのセキュリティプラクティスの詳細については、<a href="https://www.autodesk.co.jp/trust/security" rel="noopener noreferrer" target="_blank">Autodesk Trust Center</a> をご覧ください。 安全なアプリケーションの開発についてもっと学びたい場合は、RSA カンファレンスの<a href="https://www.rsaconference.com/videos/rethinking-product-security-cloud-demands-a-new-way" rel="noopener noreferrer" target="_blank">プレゼンテーション</a>をご覧ください。</li>
</ol>
</div>

<div class="clearfix text-formatted">
<h2 class="h3">他のオートデスク製品との統合</h2>
</div>
</div>
<div class="paragraph--type--faqs  col-xs-12 js-faq-accordion" data-collapse="open">
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__text"><strong>Fusion 360 / A360 / BIM 360 をサブスクリプション契約を持っています。 この状態で Forge を利用する場合、別途 Forge サブスクリプションが必要ですか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>既存の製品サブスクリプションを通じて、Forge の関連 API にアクセスできます（例：BIM 360 サブスクリプションで BIM 360 API を利用）。 Forge プラットフォームでしか利用できない API にアクセスするには、Forge サブスクリプションに登録する必要があります。</p>
</div>
</div>
</div>
</div>
<div>
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>すでに BIM 360 サブスクリプション契約があり、ファイルを変換したいと考えています。 この用途ではクラウド クレジットが請求されますか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Model Derivative API を直接呼び出してファイル変換を実行すると、ファイルがどこにあるかに関係なくクラウド クレジットが請求されます。 ただし、ファイルを BIM 360 Docs にユーザ操作（手動）でアップロードした場合、BIM 360 Docs 自身がファイルを変換出来るため、クラウド クレジットは請求されません。さらに、Forge Data Management API を利用すれば、変換された情報にもアクセスすることができます。</p>
<p>クラウド クレジットの消費例につていては、過去のブログ記事 <strong><a href="https://adndevblog.typepad.com/technology_perspective/2018/01/about-charging-to-forge.html" rel="noopener noreferrer" target="_blank">Forge 課金について</a></strong> を参照することが出来ます。</p>
</div>

<h2 class="h3">販売パートナーシップ</h2>
</div>
</div>
<div class="paragraph--type--faqs  col-xs-12 js-faq-accordion" data-collapse="open">
<div class="paragraph--type--faq-item">
<div class="paragraphs__faq__question js-faq-accordion--question paragraphs__faq__question--accordion active">
<div class="paragraphs__faq__question__icon"><strong>弊社はシステム インテグレータ、独立系ソフトウェアベンダー、または、Autodesk Developer Network メンバで、Forge API を再販したいと考えています。 どうすればいいですか？</strong></div>
</div>
<div class="paragraphs__faq__answer js-faq-accordion--answer paragraphs__faq__answer--accordion">
<div class="clearfix text-formatted">
<p>Forge は販売パートナーによる再販は認められていません。しかし、販売パートナーが認定システムインテグレータに参加し、Forge プラットフォームを利用する独自のソリューションとアプリケーションを販売することは可能です。持続した成長を続けるための大きなチャンスでもあります。</p>
