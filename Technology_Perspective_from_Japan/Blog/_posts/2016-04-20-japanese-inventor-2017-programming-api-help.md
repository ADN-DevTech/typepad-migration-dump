<p>Inventor 2017 API の日本語プログラミング用ヘルプ&#0160;（admapi_21_0.chm）がリリースされました。 ZIP 圧縮したファイルをポストしていますので、次のリンクからダウンロードしてください。</p>
<p style="padding-left: 30px;"><span class="asset  asset-generic at-xid-6a0167607c2431970b01bb08d50553970d img-responsive"><a href="http://adndevblog.typepad.com/files/admapi_21_0.zip"><strong>日本語版 Inventor 2017 API プログラミング用ヘルプ</strong>をダウンロード</a></span></p>
<p>製品に同梱されている API ヘルプは英語版オンライン ヘルプになっていて、C:\Program Files\Autodesk\Inventor 2017\Local Help にインストールされています。ダウンロードした日本語版の Inventor 2017 API Help （admapi_21_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。&#0160;</p>
