<p>オートデスクのデスクトップ製品 API や Autodesk Platform Services を使った開発作業では、Web で参照可能な情報が多くあります。どうしても英語のページが多くなりがちですが、翻訳サービスとの併用も出来るような時代になってきているので、日本語情報も含め、改めて、ご紹介しておきたいと思います。</p>
<hr />
<p><strong>開発者ドキュメント</strong></p>
<p style="padding-left: 40px;">開発者ポータル（<a href="https://aps.autodesk.com/" rel="noopener" target="_blank">https://aps.autodesk.com</a>）配下の <a href="https://aps.autodesk.com/developer/documentation" rel="noopener" target="_blank">Documentation</a> には、各種 API の導入初期や検討時に役立つ概要情報や開発者ガイドへのリンクが記載されています。</p>
<p style="padding-left: 40px;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a9a5aa200b-pi" style="display: inline;"><img alt="Developer_portal" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a9a5aa200b image-full img-responsive" src="/assets/image_790512.jpg" title="Developer_portal" /></a></p>
<p style="padding-left: 40px;"><a href="https://aps.autodesk.com/developer/documentation" rel="noopener" target="_blank">Documentation</a>&#0160;ページではクラウド（Cloud）＝ APS とデスクトップ（Desktop）のチェックボックスで、表示させる API タイルをフィルタリングすることが出来ます。&#0160;</p>
<p style="padding-left: 40px;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a9a5db200b-pi" style="display: inline;"> </a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a9a5fa200b-pi" style="display: inline;"><img alt="Documentations" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a9a5fa200b image-full img-responsive" src="/assets/image_966717.jpg" title="Documentations" /></a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a9a5db200b-pi" style="display: inline;"><br /></a></p>
<p style="padding-left: 40px;">すべてではありませんが、お使いの Web ブラウザの言語設定が日本語に設定されていると、<a href="https://aps.autodesk.com/developer/overview/autocad" rel="noopener" target="_blank">AutoCAD</a>、<a href="https://aps.autodesk.com/developer/overview/revit" rel="noopener" target="_blank">Revit</a>、<a href="https://aps.autodesk.com/developer/overview/inventor" rel="noopener" target="_blank">Inventor</a> ページのみ日本語で表示されます。</p>
<p style="padding-left: 40px;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a9a609200b-pi" style="display: inline;"><img alt="Revit" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a9a609200b image-full img-responsive" src="/assets/image_155933.jpg" title="Revit" /></a></p>
<hr style="padding-left: 40px;" />
<p><strong>ブログ</strong></p>
<p style="padding-left: 40px;">オートデスクの公式ブログは数多く存在します。各ブログの一覧は、<strong><a href="https://www.autodesk.com/blogs/" rel="noopener" target="_blank">https://www.autodesk.com/blogs</a></strong> で把握することが出来ますが、開発者向けのブログも複数公開されています。</p>
<p style="padding-left: 40px;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3aa0b2b200d-pi" style="display: inline;"><img alt="Blogs" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3aa0b2b200d image-full img-responsive" src="/assets/image_840221.jpg" title="Blogs" /></a></p>
<hr />
<p><strong>フォーラム</strong></p>
<p style="padding-left: 40px;">無償で問い合わせが可能なフォーラムは、いわゆるコミュニティーサポート&#0160; リソースでもあります。世界中の開発者が投稿した質問に対して、解となる投稿を見出すことが出来ます。英語フォーラムは、<a href="https://forums.autodesk.com/" rel="noopener" target="_blank">https://forums.autodesk.com</a> からアクセスすることが出来ます。</p>
<p style="padding-left: 40px;">対象製品によって異なりますが、開発者の多い AutoCAD では、「Other Forums」タブにカスタマイズ用のフォーラムが独立して設定されています。「Product Forums」の製品フォーラム ページ配下に 「*** Customization」が用意されている場合もあります。</p>
<p style="padding-left: 40px;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a9a66a200b-pi" style="display: inline;"><img alt="Forums" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a9a66a200b image-full img-responsive" src="/assets/image_18012.jpg" title="Forums" /></a></p>
<p style="padding-left: 40px;">AutoCAD Customization フォーラムでは、API 毎の投稿内容を閲覧、開発作業に役立てることが出来ます。</p>
<p style="padding-left: 40px;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a9a682200b-pi" style="display: inline;"><img alt="Autocad_customization" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a9a682200b image-full img-responsive" src="/assets/image_87503.jpg" title="Autocad_customization" /></a></p>
<hr />
<p><strong>ナレッジベース</strong></p>
<p style="padding-left: 40px;"><a href="https://forums.autodesk.com/t5/ri-ben-yu/ct-p/511" rel="noopener" target="_blank">日本語フォーラム ページ</a> には、デスクトップ製品 AutoCAD、Inventor、Vault、Revit の API について、過去のお問い合わせ実績を<a class="lia-link-navigation lia-message-unread" data-wat-link="true" data-wat-loc="board title" data-wat-val="autodesk community tips- adnオープン" href="https://forums.autodesk.com/t5/autodesk-community-tips-adnopun/tkb-p/tkbJP" id="link_66" rel="noopener" target="_blank">Autodesk Community Tips- ADNオープン</a> の名前でナレッジベース化しています。このナレッジベースには、本ブログのヘッダーからアクセスすることも出来ます。</p>
<p style="padding-left: 40px;"><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02c8d3a5afe1200c-pi" style="display: inline;"><img alt="Access" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02c8d3a5afe1200c image-full img-responsive" src="/assets/image_326340.jpg" title="Access" /></a></p>
<p style="padding-left: 40px;">過去のナレッジベース記事は、<a href="https://forums.autodesk.com/t5/autodesk-community-tips-adnopun/guo-quno-api-guan-lian-autodesk-knowledge-network-ji-shi/ta-p/11558155">過去の API 関連 Autodesk Knowledge Network 記事 - Autodesk Community</a>&#0160;でまとめたページも記載しています。</p>
<hr />
<p><strong>Stackoverflow</strong></p>
<p style="padding-left: 40px;">Web サービス API である Autodesk Platform Services（APS、旧 Forge）は、<a href="https://aps.autodesk.com/get-help" rel="noopener" target="_blank">https://aps.autodesk.com/get-help</a> から投稿された内容を Stackoverflow でコミュニティーサポートされています。デスクトップ製品 API のフォーラムと同様に、世界中の開発者が投稿した質問に対して、解となる投稿を見出すことが出来ます。</p>
<p style="padding-left: 40px;">英語 Stackoverflow でおサポートになっているため、投稿も英語が要求されていますが、過去の投稿とその解は、<a href="https://aps.autodesk.com/get-help" rel="noopener" target="_blank">https://aps.autodesk.com/get-help</a> からキーワード検索でヒットさせることが可能です。</p>
