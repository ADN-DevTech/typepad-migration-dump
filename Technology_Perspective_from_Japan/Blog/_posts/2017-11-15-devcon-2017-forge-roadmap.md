<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c93bb918970b-pi" style="float: right;"><img alt="Under_construction" class="asset  asset-image at-xid-6a0167607c2431970b01b7c93bb918970b img-responsive" src="/assets/image_942355.jpg" style="width: 230px; margin: 0px 0px 5px 5px;" title="Under_construction" /></a>Forge DevCon 2017 では、「Forge ロードマップ」と題して、先にご案内したブログ記事&#0160;<strong><a href="http://adndevblog.typepad.com/technology_perspective/2017/11/forge-devcon-2017-expand-forge-platform-api-detail.html" rel="noopener noreferrer" target="_blank">DevCon 2017 : Forge Platform API の拡張詳細</a></strong>、<strong><a href="http://adndevblog.typepad.com/technology_perspective/2017/11/announcing-forge-vrar-toolkit.html" rel="noopener noreferrer" target="_blank">DevCon 2017 : Forge VR/AR Toolkit 発表</a></strong> で触れた API 以外にも、現在計画中の API の公開が紹介されています。</p>
<p>いずれもリリース時期も未定ではありますが、大変意欲的なものもあり、将来の Forge と異なる方向性を持たせて新しい可能性をもたらします。詳細未定のため、ここでは、非常に簡単な記載にとどまりますが、その概要をお知らせしておきたいと思います。</p>
<hr />
<p><strong>Design Automation API - 3ds Max</strong></p>
<p>すでに AutoCAD エンジンをクラウドで実行、カスタムアドインをロードさせてバッチ処理を実現させている Design Automation API。同じ仕組みを利用する Inventor エンジンと Revit エンジンのサポートが計画されていますが、この他に 3ds Max エンジンのサポートもアナウンスされました。</p>
<p><strong>Forge App Framework </strong></p>
<p>別名 Forge Integrated Developer Experience、Forge IDX と呼ばれているフレームワークです。ブログ記事 <strong><a href="http://adndevblog.typepad.com/technology_perspective/2017/11/consider-about-forge-hfdm.html" rel="noopener noreferrer" target="_blank">Forge HFDM の考え方</a></strong> でご紹介した High Frequency Data Management、HFDM を利用して CAD 的なアプリの作成を可能にします。</p>
<p><strong>Forge IoT</strong></p>
<p>Fusion Connect のように IoT デバイス差を吸収する Device Adapter Layer を提供して、IoT ワークフローで利用するアプリ開発を可能にします。</p>
<p><strong>Design Graph</strong></p>
<p>オートデスクが初めて運用している機械学習メカニズム <strong><a href="http://adndevblog.typepad.com/technology_perspective/2016/08/design-graph.html" rel="noopener noreferrer" target="_blank">Design Graph</a></strong> を利用出来る API です。</p>
