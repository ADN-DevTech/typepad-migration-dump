<p>新バージョンとなる AutoCAD 2024 と AutoCAD LT 2024 がリリースされました。まずは、概要のご紹介です。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b6852e3266200d-pi" style="display: inline;"><img alt="History" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b6852e3266200d image-full img-responsive" src="/assets/image_499779.jpg" title="History" /></a></p>
<p>今回も、Windows 版と Mac 版の同時リリースとなります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b685306547200d-pi" style="display: inline;"><img alt="Win_mac" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b685306547200d image-full img-responsive" src="/assets/image_987947.jpg" title="Win_mac" /></a></p>
<p>なお、2021 年の <a href="https://adndevblog.typepad.com/technology_perspective/2021/05/reborned-autocad.html" rel="noopener" target="_blank"><strong>生まれ変わった AutoCAD</strong></a> による決定を継承して、実質的な新規販売は 単体の AutoCAD と、業種別ツールセットを含む AutoCAD Plus（別名 AutoCAD including specialized toolsets）となります。Mac 版には業種別ツールセットを含む AutoCAD Plus の提供はありません。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b7519c0e1c200c-pi" style="display: inline;"><img alt="Autocad_portfolio" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b7519c0e1c200c image-full img-responsive" src="/assets/image_699187.jpg" title="Autocad_portfolio" /></a></p>
<p>業種別ツールキットを利用出来る Windows 版では、AutoCAD 単体の利用に比較して、さらに生産性の向上を実現することが可能になっています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02942fa204f9200c-pi"><img alt="Ist" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02942fa204f9200c image-full img-responsive" src="/assets/image_11760.jpg" title="Ist" /></a></p>
<p><strong>動作環境</strong></p>
<p>AutoCAD 2024 がサポートする OS 環境は次のとおりです。</p>
<p style="padding-left: 40px;"><strong>Windows</strong></p>
<ul>
<li>Microsoft がサポート中の Windows 10、Windows 11、Windows Server 2019
<ul>
<li>注：<strong><a href="https://knowledge.autodesk.com/ja/customer-service/account-management/users-software/previous-versions/previous-version-support-policy">製品サポートのライフサイクル</a></strong></li>
</ul>
</li>
<li>64 ビット版のみの提供（32 ビット版の提供はなし）</li>
<li>.NET Framework Version 4.8 以降</li>
</ul>
<p style="padding-left: 40px;"><strong>Mac</strong></p>
<ul>
<li>Apple®&#0160;macOS® Ventura v13</li>
<li>Apple macOS Monterey v12</li>
<li>Apple macOS Big Sur v11</li>
</ul>
<p><strong>互換性</strong></p>
<p><a id="open_save"></a>AutoCAD 2024 は、引き続き 2018 図面ファイル形式を採用します。DWG と DXF の入出力形式は従来と同様です。</p>
<p style="padding-left: 40px;"><strong>図面読み込み：</strong></p>
<ul>
<li>
<ul>
<li>すべての AutoCAD バージョンで作成した DWG ファイル
<ul>
<li>TrustedDWG を扱うことが出来ます</li>
</ul>
</li>
<li>すべての AutoCAD バージョンで作成した DXF ファイル</li>
</ul>
</li>
</ul>
<p style="padding-left: 40px;"><strong>図面の保存：</strong></p>
<ul>
<li>
<ul>
<li>R14, 2000, 2004, 2007, 2010, 2013, 2018 形式の DWG ファイル</li>
<li>R12, 2000, 2004, 2007, 2010, 2013, 2018 形式の DXF ファイル</li>
</ul>
</li>
</ul>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b6852e3283200d-pi" style="display: inline;"><img alt="Intercompatibility" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b6852e3283200d image-full img-responsive" src="/assets/image_501675.jpg" title="Intercompatibility" /></a></p>
<p><strong>Autodesk Access</strong></p>
<p>AutoCAD 2024 をインストールすると、いままでと違い、デスクトップに登録されていた Autodesk デスクトップ アプリが表示されない点に気付かれるはずです。&#0160;</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b02b751770d76200b-pi" style="display: inline;"><img alt="Desktop_app" border="0" class="asset  asset-image at-xid-6a0167607c2431970b02b751770d76200b image-full img-responsive" src="/assets/image_542494.jpg" title="Desktop_app" /></a></p>
<p>Autodesk デスクトップ アプリはオートデスク製品と一緒にインストールされるコンパニオン アプリで、製品の Update モジュールや新バージョンを表示・インストールするメインテナンスを目的としたアプリケーションでした。</p>
<p>2024 バージョンからは、この Autodesk デスクトップ アプリに代わって、新しく Autodesk Access がインストールされるようになります。Autodesk デスクトップ アプリ同様、製品の Update モジュールをリストアップする機能を持っています。</p>
<p>Autodesk Access には、システム トレイからアクセスすることが出来ます。</p>
