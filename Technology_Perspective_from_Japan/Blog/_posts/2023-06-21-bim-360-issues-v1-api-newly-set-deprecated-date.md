<p>BIM 360 Issues (v1) API エンドポイントの廃止日が 1 か月少々延長されました。新しい日付は <strong>2023 年 8 月 15 日</strong>です。</p>
<p>以前にご案内しましたように、BIM 360 Issues (v2) API エンドポイントの導入にともない、オートデスクは 11 個の BIM 360 Issues (v1) API エンドポイントを廃止する予定です。アプリケーションでまだ BIM 360 Issues (v1) API をお使いの場合には、2023 年 8 月 15 日までに BIM 360 Issues (v1) API に移行をお願いいたします。</p>
<p>既に BIM 360 Issues (v2) API に移行されている場合には、本記事をスキップしていただいて結構です。早々にご対応いただきまして、大変ありがとうございました。</p>
<p>BIM 360 Issue API (v1) から (v2) への移行の詳細については、<a  _istranslated="1" href="https://adndevblog.typepad.com/technology_perspective/2023/01/retiring-bim-360-issues-v1-endpoints.html" rel="noopener" target="_blank">こちらの投稿 </a> をご参照ください。</p>
<p>この移行について問題が発生した場合、または、ご質問をお持ちの場合には、aps.help@autodesk.com にお問い合わせいただければと思います。</p>
