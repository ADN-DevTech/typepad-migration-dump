<p><a href="https://adndevblog.typepad.com/technology_perspective/2024/07/partial-packages-on-aps-net-sdk-release.html" rel="noopener" target="_blank">APS .NET SDK 一部パッケージ公開</a>&#0160;でご案内した Authentication API（OAuth API）v2 と Model Derivative API v2 をラップする NuGet パッケージに続いて、Data Management API の Object Storage Service（OSS）機能をラップするパッケージが APS .NET SDK に加わりました。</p>
<p><a href="https://www.nuget.org/packages/Autodesk.Oss" rel="noopener" target="_blank">OSS SDK</a> と命名された本 SDK を使用すると、S3 Bucket へのアップロードで必要とされていたアップロード完了のリクエストを省略するオートコンプリートを利用するこうとが出来ます。</p>
<p>OSS SDK の詳細に関するドキュメントは、<a href="https://aps.autodesk.com/en/docs/data/v2/reference/dot-net-sdk-oss/" rel="noopener" target="_blank">.NET SDK Reference (OSS)</a>&#0160;をご確認ください。</p>
