<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb07c64c51970d-pi" style="float: right;"><img alt="Jphacks" class="asset  asset-image at-xid-6a0167607c2431970b01bb07c64c51970d img-responsive" src="/assets/image_739573.jpg" style="width: 150px; margin: 0px 0px 5px 5px;" title="Jphacks" /></a>12月13、14日 の両日、東京大学本郷キャンパスに於いて、オートデスクもスポンサーとなった学生向けハッカソン <a href="https://jphacks.com/" target="_blank"><strong>JPHACKS</strong></a> が開催されました。当初、100 名の参加を目標としていたそうですが、最終的に110 名、全32 チームがこの開発イベントに参加してくれたようです。</p>
<p>オートデスクもテクノロジー セッションで、提供した View and Data API と ReCap Photo API の紹介をさせていただきました。いくつかのチームが、これらを使ったアプリケーション開発に取り組んでくれたようなので、結果が楽しみです。</p>
<p>当日の模様は、<a href="https://www.facebook.com/jphacks" target="_blank"><strong>JPHacks の Facebook ページ</strong></a> に写真が多数記載されていますので、ぜひ参照してみてください。</p>
<p>さて、競技イベント自体は終了していますが、上位 8 チームの発表と最優秀チームの発表を含むアワードイベントは、別途、12月20日に開催される予定です。こちらの模様は、ニコニコ生放送で中継されることを決まっているそうです。中継 URL と内容は、次のとおりです。</p>
