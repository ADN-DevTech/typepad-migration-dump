<p>APS サービスの新しい SDK のベータ版に関するフィードバックを募集しています。</p>
<h3>参加方法は？</h3>
<p>はじめに&#0160;<a href="https://feedback.autodesk.com/key/apssdkbeta" rel="noopener" target="_blank">https://feedback.autodesk.com/key/apssdkbeta</a> にご参加ください。</p>
<h3>どのような API や言語に対応していますか？</h3>
<p>今回の第 1 弾では、次の API をカバーする .NET 6（LTS）用の .NET Standard 2.0 ビルドを提供する予定です。</p>
<ul>
<li>Data Management API（Hub、Project、Folder、Item、Version 関連の Data Management API エンドポイント）</li>
<li>OSS（OSS Bucket 関連の Data Management API エンドポイント）</li>
<li>Model Derivative API</li>
<li>Webhooks API</li>
</ul>
<h3>他の API や言語の対応は？</h3>
<p>優先順位をつけるために、皆さまからのフィードバックは重要です。フィードバック・プログラムにご参加いただくか、<a href="mailto:sdk.beta.feedback@autodesk.com" rel="nofollow noopener" target="_blank">sdk.beta.feedback@autodesk.com</a> までメッセージをお寄せください。</p>
