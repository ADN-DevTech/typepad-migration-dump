<p lang="EN-US" xml:lang="EN-US">従来の US、EMEA データセンターに加え、新たにオーストラリア データセンターが開設されて、一部の Autodesk Construction Cloud 製品でプロジェクト データを同データセンターでプライマリ ストレージするオプションが追加されました。たとえば、オーストラリアで提供されている Autodesk Build 地域向け製品では、オーストラリアを Autodesk Build プロジェクト データのプライマリ ストレージの場所として指定できる Autodesk Build アカウントを作成できます。</p>
<p lang="EN-US" xml:lang="EN-US">オーストラリアを新しいストレージ地域とする利点は、次のような点が挙げられます：</p>

<p lang="EN-US" xml:lang="EN-US">今回の発表にともない、オーストラリア向けの Autodesk Docs および Autodesk Build 地域向けサービスが公開されました。オーストラリア地域向けの一部の Autodesk Platform Services API は、将来的にコードの変更が必要になる可能性があるため、ベータ版としてご利用いただけます。ACC 製品で利用可能なサービスやツールの詳細については、<span style="text-decoration: underline;"><a href="https://aps.autodesk.com/en/docs/acc/v1/overview/acc-regions/" rel="noopener" target="_blank">こちら</a></span> のドキュメントをご覧ください。</p>

<p lang="EN-US" xml:lang="EN-US">新しいサービスを利用するためには、いくつかのアプリをアップデートする必要があるかもしれません。プログラマー向けの詳細な技術ブログは近日公開予定です。また、オーストラリア地域をサポートするためにアプリを拡張するのに役立つ Webinar（英語）も開催予定です。</p>
