<p>12月1日から米国 Las Vegas で Autodesk University 2015 が開催されています。今年で通算 23 回目の開催だそうで、IT 業界で 1 つのイベントがここまで長く続くのは珍しいそうです。今年も多くのセッションが開催されていて、盛りだくさんの内容です。また、Autodesk University 2015 に先立って、ADN メンバ向けの Developer Days も開催されました。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb08998af2970d-pi" style="display: inline;"><img alt="AU" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb08998af2970d image-full img-responsive" src="/assets/image_611378.jpg" title="AU" /></a></p>
<p>まだ公表できない内容が多くあるので、Developer Days の詳細をでご紹介することは出来ないのですが、この Developer Days でも言及のなかったアナウンスが、今日、12月1日にありました。それが、<strong>Autodesk Forge</strong> の発表です。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb0898243b970d-pi" style="display: inline;"><img alt="Forge" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01bb0898243b970d image-full img-responsive" src="/assets/image_399915.jpg" title="Forge" /></a></p>
<p>Autodesk Forge の概要は、<a href="http://www.autodesk.com/temp/pdf/Autodesk_Forge_JP_final.pdf" rel="noopener" target="_blank">プレス リリース</a>に譲りますが、一言で言うと開発者向けのプログラムとなります。プログラムというと Autodesk Developer Network に見られる年間契約の有償プログラムであるような印象を持ちますが、そうではありません。Autodesk Forge は、オートデスクが掲げる The Future Of Making Things － 創造の未来 － を支える新しいテクノロジの素材とる API と SDK などのプラットフォーム テクノロジ（<strong>Forge Platform</strong>）、その活動を啓蒙してエコシステムを構築するためのファンド（<strong>Forge Fund</strong>）、それを支えるコミュニティ プログラム（<strong>Forge Program</strong>）、という 3 つを柱として持つプログラムです。別の言い方をするなら、オートデスクが提供するクラウドベースの Web サービス API のブランド名と捉えることも出来るかと思います。</p>
<p>Autodesk Forge の詳細は、専用ポータル サイト <a href="http://forge.autodesk.com/" rel="noopener" target="_blank">http://forge.autodesk.com/</a>&#0160;から知ることが可能です。現在のところ、Autodesk Forge として公開される Web サービス API、および、SDK には、それぞれ 5 つにカテゴリ化されています。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c7f407ba970b-pi" style="display: inline;"><img alt="Five_categories" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c7f407ba970b image-full img-responsive" src="/assets/image_828305.jpg" title="Five_categories" /></a></p>
<p>各カテゴリでサポートされる API や SDK は、今のところ、次のとおりです。</p>
<p>Design - デザイン</p>
<ul>
<li>Fusion 360</li>
<li>AutoCAD I/O</li>
</ul>
<p>Make - 製作</p>
<ul>
<li>Spark</li>
</ul>
<p>Use - 利用</p>
<ul>
<li>SeeControl</li>
</ul>
<p>Visualize - ビジュアライズ</p>
<ul>
<li>View and Data</li>
<li>ReCap Photo</li>
<li>Rendering</li>
</ul>
<p>Collaborate - コラボレーション</p>
<ul>
<li>BIM 360 Glue</li>
<li>Comments</li>
<li>Marup</li>
</ul>
<p>まだ未公開の API も含まれますが、まもなく正式に登場する予定です。特に、Rendering API は Rendering in A360 機能を実装できる API と考えていただくと分かりやすいかと思います。また、ReCap Photo API は、従来、独立して運用していたポータル サイトを、デベロッパ ポータル（<a href="https://developer.autodesk.com/" rel="noopener" target="_blank">https://developer.autodesk.com/</a>）へ統合されています。まもなく、API 利用時に必要なキーの申請や管理が出来るようになるはずです。</p>
<p><iframe allowfullscreen="" frameborder="0" height="281" src="https://www.youtube.com/embed/bL5mdT4hlHg?feature=oembed" width="500"></iframe>&#0160;</p>
<p>さて、今回、Autodesk Forge の発表に続いて、Forge Program の活動として&#0160;<strong>Forge Developer Conference</strong> の開催も同時に発表されています。こちらは、開発者向けのカンファレンスで、来年、2016年6月15日と16日の 2 日間にわたって、San Francisco で開催の予定です。概要は、<a href="http://forge.autodesk.com/conference/" rel="noopener" target="_blank">http://forge.autodesk.com/conference/</a>&#0160;を参照してみてください。計画では、Google や Apple、Microsoft などの年次開発者会議のようなイベントになるようです。徐々に詳細が決定していくはずなので、わかり次第ご案内します。</p>
