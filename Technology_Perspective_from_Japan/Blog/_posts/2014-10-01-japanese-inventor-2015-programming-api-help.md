<p>従来、製品に同梱される API ドキュメントは英語のままでした。一昨年の Inventor 2013 時には、Inventor<strong> 2013</strong> API Help のうち、ユーザ マニュアルの部分のみを翻訳して提供した経緯があります。この時には、リファレンスやサンプルコードの部分が英語のままでした。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d07423bb970c-pi" style="display: inline;"><img alt="Inventor_2013_api_help" class="asset  asset-image at-xid-6a0167607c2431970b01b8d07423bb970c img-responsive" src="/assets/image_764882.jpg" style="width: 400px;" title="Inventor_2013_api_help" /></a></p>
<p>最新バージョンの Inventor 2015 でも、製品に同梱される API ドキュメントは英語版のままですが、今回、完全なかたちで Inventor <strong>2015</strong> API Help が日本語化されました。翻訳対象は、ユーザ マニュアルを含むすべてのトピックです。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb078f4ae7970d-pi" style="display: inline;"><img alt="Inventor_2015_api_help" class="asset  asset-image at-xid-6a0167607c2431970b01bb078f4ae7970d img-responsive" src="/assets/image_799970.jpg" style="width: 380px;" title="Inventor_2015_api_help" /></a></p>
<p>日本語版&#0160;&#0160;Inventor 2015 API Help は、次の Autodesk Technical Q&amp;A からダウンロードしていただくことが出来ます。</p>
<p style="padding-left: 30px;"><strong><span class="asset  asset-generic at-xid-6a0167607c2431970b01b8d199810a970c img-responsive"><a href="http://adndevblog.typepad.com/files/admapi_19_0.zip">日本語版 Inventor 2015 API Help をダウンロード</a></span></strong></p>
<p style="padding-left: 30px;"><span style="background-color: #ffff00;">※ ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。その場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、画面右下にある 「ブロックの解除」 にチェックを入れてください。</span></p>
<p>もともとの英語版 Inventor 2015 API Help （admapi_19_0.chm）は、C:\Program Files\Autodesk\Inventor 2015\Local Help にインストールされているので、ダウンロードした日本語版&#0160;Inventor 2015 API Help （admapi_19_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。</p>
<p style="text-align: center;"><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b8d0741fa8970c-pi" style="display: inline;"><img alt="Inventor_api_help_access" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b8d0741fa8970c image-full img-responsive" src="/assets/image_927674.jpg" title="Inventor_api_help_access" /></a></p>
<p>完全なかたちで Inventor API ドキュメントを日本語化されたのは、Inventor 5.3 以来です。Windows OS をはじめ、Inventor API を取り巻く環境も大分変ってしまいましたが、Inventor API も大きな進化と遂げています。&#0160;Inventor 2015 API Help 内のサンプルも書き換えられたものが存在しますので、この機会に内容をチェックしてみてはいかがでしょうか。</p>
<p>By Toshiaki Isezaki</p>
