<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b026bdec98112200c-pi" style="display: inline;"><img alt="API reference" border="0" class="asset  asset-image at-xid-6a0167607c2431970b026bdec98112200c image-full img-responsive" src="/assets/image_911771.jpg" style="width: 1093.19px;" title="API reference" /></a></p>
<p>&#0160;</p>
<p>Inventorの新バージョンとなる、Inventor 2022がリリースされました。</p>
<p>新バージョンでの機能の概要については後程別の記事でご紹介をしたいと思いますが、取り急ぎInventor 2022の API の日本語プログラミング用ヘルプ （admapi_26_0.chm）を ZIP 圧縮したファイルをポストしていますので、次のリンクからダウンロードしてください。</p>
<p><span class="asset  asset-generic at-xid-6a0167607c2431970b025d9b422c9f200c img-responsive"><a href="https://adndevblog.typepad.com/files/admapi_26_0.zip"></a><a href="https://adndevblog.typepad.com/files/admapi_26_0.zip">Inventor 2022 API の日本語プログラミング 用ヘルプ をダウンロード</a></span></p>
<p>&#0160;</p>
<p>日本語版 Inventor 2022 に同梱されている API ヘルプは英語版オン&#0160;<br />ライン ヘルプになっていて、<strong>C:\Users\Public\Documents\Autodesk\Inventor 2022\Local Help</strong> にインストールされています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4f7909a200d-pi" style="display: inline;"><img alt="Inventor 2021 API Menu" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4f7909a200d image-full img-responsive" src="/assets/image_764593.jpg" title="Inventor 2021 API Menu" /></a></p>
<p>ダウンロードした日本語版の Inventor 2022 API Help （admapi_26_0.chm）で上書き保存していただければ、Inventor のメニューから日本語版ヘルプを起動することが出来るようになります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0263e996b2fa200b-pi" style="display: inline;"><img alt="API" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0263e996b2fa200b image-full img-responsive" src="/assets/image_502939.jpg" style="width: 1093.19px;" title="API" /></a></p>
<p>ダウンロードして展開した .chm ファイルは、セキュリティ機能によってコンテンツ表示がブロックされる場合があります。コンテンツがブロックされている場合には、各ページが白く表示されて何もない状態となります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0278801bec48200d-pi" style="display: inline;"><img alt="API_empty" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0278801bec48200d image-full img-responsive" src="/assets/image_94492.jpg" title="API_empty" /> </a><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0278801bec48200d-pi" style="display: inline;"><br /></a></p>
<p>&#0160;</p>
<p>ページに何も表示されない場合には、エクスプローラから .chm ファイルを右クリックして [プロパティ] ダイアログの [全般] タブを開き、右下にある 「許可する」 にチェックを入れてください。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0278801bec73200d-pi" style="display: inline;"><img alt="FileProperty" class="asset  asset-image at-xid-6a0167607c2431970b0278801bec73200d img-responsive" src="/assets/image_403488.jpg" title="FileProperty" /></a></p>
<p>&#0160;</p>
<p>なお、従来の CHM ファイル形式とは別に、英語版 Inventor 2022 のみ、オンラインヘルプ内に Programming Interface セクションが追加されています。残念ながら、日本語版 Inventor には反映されてはいませんが、記載されている内容は CHM 形式と同じものになっています。</p>
