<p>米国サンフランシスコのオートデスク オフィスで開催されている <strong><a href="http://autodeskcloudaccelerator.com/" rel="noopener noreferrer" target="_blank">Autodesk Clould Accelerator</a></strong> に来ています。オートデスクは、世界各地で定期的に Cloud Accelerator イベントを開催していますが、今回は、まだリリースされていない HFDM を利用した次世代の&#0160; App Framework API Beta の評価をしていただく特殊なものです。オートデスク社外のデベロッパの方に実際に利用していただき、将来の正式リリースに向けて、個々のアイデアの基づいた実装が可能か、実現に何が必要か、などを洗い出すことが目的です。</p>
<p>このイベントで得られた情報は、<strong><a href="http://au.autodesk.com/japan" rel="noopener noreferrer" target="_blank">AU Japan 2018/Forge DevCon Japan</a></strong> の&#0160; <strong>[H4] Forge が描くデザインとつながる未来</strong> セッションでご紹介する予定です。</p>
