<p>今回のイベントの最後に、Autodesk University の展示会場、通称、<strong>Expo</strong> にも触れておきたいと思います。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4c908ce200d-pi" style="display: inline;"><img alt="IMG_0716" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4c908ce200d image-full img-responsive" src="/assets/image_620380.jpg" title="IMG_0716" /></a></p>
<p>Expo は、Autodesk University の会期３日間を通してオープンしている非常に大きな展示会場で、Forge DevCon 開催日（Autodesk University 前日）のみの１日間だけオープンしていた the Village とは、まったくの別もの/別会場です。毎年、スポンサー企業のブース展示をはじめ、オートデスク自身の展示など多様性にあふれています。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a49fda12200c-pi" style="display: inline;"><img alt="Expo" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a49fda12200c image-full img-responsive" src="/assets/image_127907.jpg" title="Expo" /></a></p>
<p>今年もオートデスクの展示エリアがもっとも面積を占有していたのは言うまでもありませんが、意外だったのは Forge の扱いです。</p>
<p>従来は、AutoCAD や Revit、Inventor、Fusion 360、BIM 360 など、主要製品やクラウドサービスのブースやサインが目立つ位置に配置されていましたが、今年は <strong>Forge Developer Zone</strong> として、Forge が中央に配置されていました。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4edc43a200b-pi" style="display: inline;"><img alt="Developer_zone" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4edc43a200b image-full img-responsive" src="/assets/image_299827.jpg" title="Developer_zone" /></a></p>
<p>Forge は API ですが、デスクトップの製品カスタマイズ用 API と同じく、一部の開発者が利用する特殊なもの、と認識されている場合がほとんどです。そんな状況で 20 年以上 API に携わってきた自身にとって、製品やクラウドサービスなどの物理的になユーザインタフェースを持つものでなく、API という抽象的なサービスがここまでフォーカスされていることに驚きを禁じえません。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4c90a3f200d-pi" style="display: inline;"><img alt="Gui_api" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4c90a3f200d image-full img-responsive" src="/assets/image_286970.jpg" title="Gui_api" /></a></p>
<p>デスクトップ製品からクラウドサービスへの移行、売り切りの永久ライセンス販売からサブスクリプションへの移行を経て、オートデスク自身も「<strong>デジタル フォーメーション</strong>」してきているのを強く感じた次第です。</p>
<p>Facebook の <strong><a href="https://www.facebook.com/pg/adn.open.japan/" rel="noopener" target="_blank">ADN オープン ページ</a></strong>で会場の模様を写した<a href="https://www.facebook.com/pg/adn.open.japan/photos/?tab=album&amp;album_id=2483732291900706" rel="noopener" target="_blank"><strong>アルバム</strong></a>を公開していますので、そちらもご覧ください。</p>
