<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a51a6ede200b-pi" style="display: inline;"></a>AutoCAD 2021、AutoCAD LT 2021 がリリースされました。AutoCAD の誕生から 38 年を経て、更なる生産性の向上とワークフローの進化を目指したバージョンとなります。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a51a2df5200b-pi" style="display: inline;"><img alt="Autocad_history" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a51a2df5200b image-full img-responsive" src="/assets/image_254237.jpg" title="Autocad_history" /></a></p>
<p>このバージョンでも、引き続き、業種別ツールキット（AutoCAD のみ）と AutoCAD Web アプリ、AutoCAD モバイル アプリ をお使いいただくことが出来ます。</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b025d9b400ed4200c-pi" style="display: inline;"><img alt="Industry_toolset" border="0" class="asset  asset-image at-xid-6a0167607c2431970b025d9b400ed4200c image-full img-responsive" src="/assets/image_703848.jpg" title="Industry_toolset" /></a></p>
<p>新機能をご紹介していく前に、まずは、本バージョンのコンセプトを紹介する動画をご確認ください。</p>
<p class="asset-video"><iframe allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen="" frameborder="0" height="354" src="https://www.youtube.com/embed/ooDEgk2xmWo?feature=oembed" width="630"></iframe></p>
<p>AutoCAD 2021 と業種別ツールセット、また、AutoCAD LT 2021 は、Microsoft が現在サポートしている Windows の 64 ビットバージョンをサポートしています。Microsoft が Windows バージョンのサポートを終了した場合、その Windows バージョンで実行しているオートデスク製品のサポートも終了することになりますのでご注意ください。</p>
<p>現段階では、次の環境をサポートすることになります。インストール環境には、Microsoft .NET Framework 4.8 が必要になります。（Microsoft .NET Framework 4.8 がまだインストールされていない場合は、AutoCAD 2021 と業種別ツールセットおよび AutoCAD LT 2021 と一緒にインストールされます。）</p>
<p><a class="asset-img-link" href="https://adndevblog.typepad.com/.a/6a0167607c2431970b0240a4f58625200d-pi" style="display: inline;"><img alt="Support_os" border="0" class="asset  asset-image at-xid-6a0167607c2431970b0240a4f58625200d image-full img-responsive" src="/assets/image_73375.jpg" title="Support_os" /></a></p>
