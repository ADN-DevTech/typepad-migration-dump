<p>オートデスクのクラウドWebサービスを活用した新しいプロダクトの開発にご興味のある方、本社のクラウドエンジニアと直接コミュニケーションしながらアイデアをブラッシュアップしたいという方がいらっしゃいましたら、ぜひAutodesk Cloud Accelerator 2015への参加をご検討ください。</p>
<p>Autodesk Cloud Accelerator 2015は、米国サンフランシスコのオフィスにて、2015年3月9日から3月20日の2週間にわたって開催されるワークショッププログラムです。ソフトウェアデベロッパーを対象としております。</p>
<p><a class="asset-img-link" href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c73231cf970b-pi" style="display: inline;"><img alt="CloudAccelerator" border="0" class="asset  asset-image at-xid-6a0167607c2431970b01b7c73231cf970b image-full img-responsive" src="/assets/image_614453.jpg" title="CloudAccelerator" /></a></p>
<p>参加者の方々には、事前に申請して頂いたアイデアの提案書をもとに、クラウドエンジニアチームのサポートやトレーニングを直接受けながら、プロジェクトに取り組んで頂きます。<br />期間中はエキスパートによる講習会が実施され、1オン1のヘルプを受けることができます。<br />アクセラレータープログラムの最後には、みなさまのプロトタイプを弊社のシニアエグゼクティブにプレゼンテーションする機会もあります。</p>
<p>お申込みには1000字程度の提案書の概要（英文）を提出して頂く必要がございます。<br />また提出期限は1月17日(土)までとなっております。</p>
<p>提案書には以下のいずれかのオートデスクのクラウドサービスをご利用いただくことを明記してください。<br />アクセラレータプログラムの最終日にデモンストレーション可能なプロトタイプとなるような提案をお願い致します。<br />・View &amp; Data<br />・AutoCAD® I/O<br />・Autodesk® Fusion 360™<br />・Autodesk® BIM 360™<br />・Autodesk® ReCap 360™<br />・Autodesk® InfraWorks™</p>
<p>ご参加にあたり、オートデスクは指定のホテルでの14日間の宿泊費用（1社あたり2名様まで）を負担させて頂きますが、移動に伴う費用や航空代金、食費につきましては参加者の方々にご負担いただくこととなります。ただし講習会での昼食と夕食につきましては提供させていただく予定です。<br />また英語でのコミュニケーションとなりますことをご理解頂きますようお願い申し上げます。</p>
<p>より詳細な情報につきましては、<a href="http://autodeskcloudaccelerator.com/" target="_self" title="Autodesk Cloud Accelerator 2015のWebサイト">Autodesk Cloud Accelerator 2015のWebサイト</a>をご参照ください。</p>
<p>エキスパートのサポートを受けながらプロトタイプ開発を行える機会をぜひご活用いただければと思います。</p>
<p>By Ryuji Ogasawara</p>
