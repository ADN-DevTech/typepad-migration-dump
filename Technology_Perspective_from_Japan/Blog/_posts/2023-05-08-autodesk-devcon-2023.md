<h3>対象</h3>
<p>Autodesk DevCon は、Autodesk Platform Services (APS) を使い始めたばかりの開発者から、APS を長年使っているベテランまで、幅広い層に向けたイベントです。建設業、製造業を問わず、Autodesk API の新規ユーザー、ベテランユーザー、シニアリーダーを対象としたセッションが用意されています。</p>
<h3>参加の利点</h3>
<p>習得や人的ネットワークの構築、そして洞察を得る目的で、2日間にわたり、オートデスクのエンジニアやテクノロジーの意思決定者、その他の業界プロフェッショナルから、企業の成長とデジタル トランスフォーメーションを加速する方法を学ぶことができます。オートデスクのテクノロジー ロードマップ セッションや API テクニカル ディープダイブを体験してください。最新の Autodesk Construction Cloud API と、アプリ マーケットプレイス が企業の成長を加速させるためにできることを学びます。このイベントに参加して、設計と製造のデータを解き放つのに役立つ最新のツールについて学びましょう。直感的でリアルタイムな次世代のソリューションを実現するツールは、デジタル トランスフォーメーションをビジョンから現実のものにします。</p>
<h3>セッション内容（予定）</h3>
<p>建設業、および製造業向けのロードマップ セッション、テクニカル ディープダイブ、パネル ディスカッションなどをご期待ください。</p>
<strong>建築、建設（含む Forma）</strong>
<p>建設業の参加者は、Autodesk Construction Cloud の初級および上級 API の技術的なヒント、Revit をクラウドに接続する方法、デジタルツインとダッシュボードの作成などのクラスを受講することができます。</p>
<strong>製造（含む Fusion）</strong>
<p>製造業の参加者は、コンフィギュレータの構築、Viewer の使用、Inventor と Fusion のクラウド接続など、様々なクラスに受講することができます。</p>
<strong>Design and Make Data</strong>
<p>新しいデータ機能とデータモデルが、生産性、自動化、収益性をどのように向上させるかをご確認ください。</p>
<strong>エグゼクティブ - ビジネスのデジタル化</strong>
<p>アプリ マーケットプレイスでのビジネス機会、新しいビジネス モデル、セキュリティと信頼性、そして認定パートナー企業がどのように企業デジタル化の旅路をサポートするかご確認ください。</p>
<strong>サステナビリティ - Hack the planet</strong>
<p>APS の活用で、未来に向けてより持続可能な成果を実現するための洞察を提供する方法をご紹介します。</p>
