<p>ここ数年、春になると各種デスクトップ製品の新バージョンが発表されてきます。そして、これに合わせて、新しいバージョン用の SDK がどこからダウンロードできるのか、よくご質問いただきます。もちろん、新バージョンの発売開始時期はまちまちなので、ひとくくりに回答することは難しいのですが、ここを見ておけば、近いうちに SDK や開発ツールが公開されます、という Web ページがありますのでご紹介しておきましょう。</p>
<p>最も早く SDK が公開されるのは、有償の ADN 会員向けの専用サイト <a href="http://adn.autodesk.com">http://adn.autodesk.com</a> です。ただ、このサイトを参照できるのは、ADN Professional や ADN Standard にご加入いただいている開発者の方のみです。</p>
<p>原則、無償で情報提供している ADN Oepn のページでは、若干のタイムラグがありますが、間をおかずに新バージョン用の SDK が公開されてきます。ADN Oepn ページは、いままで Developer Center と呼ばれていたページで昨年になって改名された英語のサイトを指しています。日本語の Developer Center ページも順次 ADN Open 化していく予定ですが、英語ページを見ていただくのが一番早く SDK を入手していただく方法です。</p>
<p>自動生成されるオートデスクの Web ページは、<a href="http://www.autodesk.co.jp/adsk/servlet/pc/index?id=16541932&amp;siteID=1169823">http://www.autodesk.co.jp/adsk/servlet/pc/index?id=16541932&amp;siteID=1169823</a>&#0160;のようにアルファベットや数字で URL が構成されていて記憶しにくいので、分かり易くする<strong> MURL</strong> （Marketing URL）というのが用意されています。次にご紹介するのは、各種デスクトップ製品の API 情報を記載している MURL の一覧です。このページの中から、お使いの製品を見つけて、ご希望の SDK をダウンロードしていただけるはずです。&#0160;</p>
