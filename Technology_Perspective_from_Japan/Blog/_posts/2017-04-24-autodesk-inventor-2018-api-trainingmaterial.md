<p>Autodesk Inventor 2018 の日本語バージョンの発表に伴い、Autodesk Inventor 2018 APIトレーニング教材一式「APIトレーニングテキスト」「APIトレーニング実習ガイドと演習＆回答」及び 「演習と実習用ファイル」を公開いたします。</p>

<p>公開されるコンテンツは、Autodesk Inventor 2018 APIトレーニングテキスト（.ppt P.368）を pdf ファイルに変換した "Inventor2018API .Net Training_J.pdf"、テキスト内の演習問題と各開発ガイド（実習）を説明した「演習と実習ヘルプファイル」の"Inventor2018API Exercise.chm"、「演習と各開発ガイド（実習）の回答」( 14の演習回答 と 80の実習の回答) をそれぞれVBAマクロファイルとVisual Studio 2015(VB.Net)のプロジェクトとしたファイル群、また「演習と各開発ガイド（実習）で使用するInventorファイル群」の公開となります。</p>
