<p>by <a href="http://adndevblog.typepad.com/manufacturing/vladimir-ananyev.html">Vladimir Ananyev</a></p>  <p><b>Issue</b></p>  <p>Is there a way to determine if a given part occurrence is a tube? </p>  <p><a name="section2"></a><b>Solution</b>     <br />For tube part occurrence, there is an attribute set called “IFC_ATTRIBUTER_SET” on the occurrence object and there is an attribute called PATHCOMPTYPE. If the value of the attribute is 9, it means this is an occurrence of a tube part.</p>  <p>Otherwise you can check the PropertySets of the document. There is a PropertySet called &quot;Content Library Component Properties&quot; in which &quot;Member&quot; Property contains data in XML format that also contains whether the Document is a TubeAndPipeComponent.</p>  <p>Below is a code that checks a Document's PropertySets and lists them in the Debug (Immediate) window in the VBA environment, and prints the property value mentioned above.</p>  <p>&#160;</p>  <p><strong>VB .NET</strong></p>  <p style="line-height: normal;
