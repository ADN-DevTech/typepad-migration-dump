<p>by <a href="http://adndevblog.typepad.com/manufacturing/vladimir-ananyev.html">Vladimir Ananyev</a></p>  <p>Question: </p>  <p>How to insert Standard and Custom iParts in assembly using Inventor API?</p>  <p>Solution</p>  <p>To add an iPart member, you need to use <font face="Consolas">Occurrences.AddiPartMember()</font><font face="Arial"> method.</font></p>  <p><font face="Consolas">Public Function AddiPartMember( _      <br /></font><font face="Consolas">&#160;&#160; ByVal FactoryFileName As String, _      <br /></font><font face="Consolas">&#160;&#160; ByVal Position As Matrix, _      <br /></font><font face="Consolas">&#160;&#160; Optional ByVal Row As Variant _      <br /></font><font face="Consolas">) As ComponentOccurrence</font></p>  <p>To add a custom member, you need to use <font face="Consolas">Occurrences.AddCustomiPartMember()</font><font face="Arial"> method.</font></p>  <p><font face="Consolas">Public Function AddCustomiPartMember( _      <br /></font><font face="Consolas">&#160;&#160; ByVal FactoryFileName As String, _      <br /></font><font face="Consolas">&#160;&#160; ByVal Position As Matrix, _      <br /></font><font face="Consolas">&#160;&#160; ByVal FullFileName As String, _      <br /></font><font face="Consolas">&#160;&#160; Optional ByVal Row As Variant, _      <br /></font><font face="Consolas">&#160;&#160; Optional ByVal CustomInput As Variant _      <br /></font><font face="Consolas">) As ComponentOccurrence</font></p>  <p>&#160;</p>  <p>The optional argument Row specifies the row for the member within the factory and can be:</p>  <ol>   <li>a Row index (Long) </li>    <li>an iPartTableRow object (doesn’t work for Custom iParts). </li>    <li>a String (part identifier, i.e. ''[Height=1.000 in][Length=2.000 in][Radius=0.250 in]'') </li> </ol>  <p>Following is a VB.NET code.
