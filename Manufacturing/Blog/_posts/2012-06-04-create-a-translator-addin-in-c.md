<p>By <a href="http://adndevblog.typepad.com/manufacturing/barbara-han.html" target="_self">Barbara Han</a></p>  <p><b>Issue</b></p>  <p>We need to create a translator addin in C#. Using VC 2005 we can create a project of translator addin using Inventor AddIn wizard, but for C#, those options are not available when using Inventor AddIn wizard.   <br />Could you please guide us how to create translator addin project in C# using Visual Studio 2010?</p>  <p><a name="section2"></a></p>  <p><b>Solution</b></p>  <p>The following is the steps for creating translator AddIn in C# from Visual Studio 2010:   <br />1.Install Inventor AddIn wizard first.
