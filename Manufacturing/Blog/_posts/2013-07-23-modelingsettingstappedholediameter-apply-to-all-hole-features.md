<p>By <a href="http://adndevblog.typepad.com/manufacturing/wayne-brill.html" target="_self">Wayne Brill</a></p>  <p>The HoleTapInfo.DiameterInModel property used to work in older releases of Inventor but this does not work in the current version. Is there anyway we can access this property?</p>  <p>The DiameterInModel property is not supported on a per hole basis in Inventor. Hence the API property on HoleTapInfo has been removed. This property is now a document setting (ModelingSettings.TappedHoleDiameter ) and applies to all the hole features in the document.
