Actually, this applies to other types of feature which are supported in assembly document.</p>  <p>For <strong>choice 1</strong>, we need to input the proxy geometries (say EdgeProxy, FaceProxy) to the methods of chamfer feature. e.g. the code below assumes a face is selected in the assembly document.
