<p><strong>Issue</strong></p>
<p>Question: How do I change text items in the TitleBlockDefinition in Inventor?</p>
<p><a name="section2"></a><strong>Solution</strong></p>
<p>Answer: Assuming you have an active drawing document open, first you need to access the TitleBlockDefinitions collection. Step through the collection to find the Title Block of interest, as shown in the following code fragment, the &quot;ANSI - Large&quot; title block definition. </p>
<p>When the TitleBlockDefinition of interest is found, we select the TextBoxes collection. Step through the collection until the TextBox of interest is found. For a TextBox the &#39;Text&#39; property is a read/write property. The most important code in the following code fragment is placing the TitleBlockDefinition into edit mode with a call to &#39;Edit&#39; and exiting the edit mode with a call to &#39;ExitEdit&#39;.</p>
<p>Two code samples and a sample drawing document (&quot;Drawing.zip&quot;) demonstrate this.</p>
