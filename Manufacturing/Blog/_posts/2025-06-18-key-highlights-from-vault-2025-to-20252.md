<ul>
<li><strong>Copy Design Improvements</strong>:<br />Vault 2025.1 introduced support for reusing library files, making design reuse more efficient.</li>
<li><strong>ECO Enhancements</strong>:<br />ECO property sync became bidirectional in 2025.1, improving integration between item properties and change orders.</li>
<li><strong>Thin Client Upgrades</strong>:<br />Password reset and enhanced BOM property views were added in 2025.1 for easier web access.</li>
<li><strong>Admin &amp; Deployment Tools</strong>:<br />Vault 2025.1 allowed admins to reset user dialogs and client settings; 2025.2 expanded this with centralized deployment tools for add-ins and settings.</li>
<li><strong>Hyperlink and Property Enhancements</strong>:<br />2025.2 introduced hyperlink-type user-defined properties (UDPs) for more dynamic metadata usage.</li>
<li><strong>Search &amp; Navigation Improvements</strong>:<br />Vault 2025.1 added support for @me filters and saved searches, streamlining user workflows.</li>
<li><strong>Cloud API Layer (Vault Data APIs)</strong>:<br />Vault 2025.2 debuted secure APIs enabling integrations with platforms like Power BI, Microsoft Teams, and SAP.</li>
<li><strong>UI/UX Enhancements</strong>:<br />Folder pickers and login identity feedback were improved in 2025.2 for better usability.</li>
</ul>
