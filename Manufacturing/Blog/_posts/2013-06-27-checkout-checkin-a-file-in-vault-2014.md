<p>There have been some questions about using the API to checkout, download and Check in a file. This example is an update to this Vault SDK sample:</p>
<p>C:\Program Files (x86)\Autodesk\Autodesk Vault 2014 SDK\VS10\VB\VaultList</p>
<p>It has two new buttons. One does the checkout and download and the other does the CheckIn. They both use a function that does a search for a file named &quot;12345 Testing XXXX.csv&quot;. That function returns a FileIteration object. </p>
