<p>By <a href="http://adndevblog.typepad.com/manufacturing/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>  <p>The question is to to hide the first element of a circular pattern feature of a boundary patch using the API.&#160; The first element is the original boundary patch. There are two ways to hide it:</p>  <p>way1:</p>  <p>BoundaryPatchFeature.SurfaceBody.Visible = False</p>  <p>way2:</p>  <p>BoundaryPatchFeature.SurfaceBody.Parent.Visible = False</p>  <p>with way1, before Inventor 2012, the first element is hidden   <br />but if the file is closed and re-opened the first element is visible again.
