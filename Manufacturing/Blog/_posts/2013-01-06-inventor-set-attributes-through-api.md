<p>by <a href="http://adndevblog.typepad.com/manufacturing/vladimir-ananyev.html">Vladimir Ananyev</a></p>  <p><b>Issue</b></p>  <p>How can I add an attribute to features to tag them through API or UI, in order to perform further processing or identify them?</p>  <p><a name="section2"></a><b>Solution</b></p>  <p>Attributes are only available through the API and it is not possible to access Attributes through the default User Interface. The following VB.NET code and VBA example allows the user to set the attribute value to one or more PartFeatures. Before running these examples you need to select the features. A MsgBox is used to get the value of the attribute. The code will set or modify the attribute value of the selected features. </p>  <p>You could enhance this example to also get the name to use for the AttributeSet as well. It is fast and easy because there is no need to develop an Add-In. Also it could be easily modified to handle other types of objects other than features. </p>  <p>This VB.NET example could be called from a button on a form in a standalone exe that connects to Inventor from out of process. A VBA Example is below this.</p>  <p>&#160;</p>  <p>Related posts:</p>  <p><a href="http://modthemachine.typepad.com/my_weblog/2009/07/introduction-to-attributes.html">Introduction to Attributes</a></p>  <p><a href="http://modthemachine.typepad.com/my_weblog/2009/09/attribute-utility.html">Attribute Utility AttributeHelper</a></p>  <p>&#160;</p>  <p>VB.
