<p>By <a href="http://adndevblog.typepad.com/manufacturing/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>  <p>Progressive Tooltip provides the expanded help tip with more various information. </p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b019b03ac98a5970c-pi"><img style="background-image: none; border-bottom: 0px; border-left: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top: 0px; border-right: 0px; padding-top: 0px" title="image" border="0" alt="image" src="/assets/image_04b814.jpg" width="308" height="352" /></a></p>  <p>&#160;</p>  <p>Inventor 2011 has exposed the relevant API. The ControlDefinition.ProgressiveToolTip allows you to customize your button. Although we normally create button in an add-in, the following VBA code is a demo how the workflow could be. I have answered a forum question last year with such code. It adds a button in Part&gt;&gt;3D Model &gt;&gt; Work Features with the progressive tooltip.</p>  <p><em><font size="1">Public Sub ProgressiveToolTips()&#160;&#160;&#160; <br />&#160;&#160; Dim g_FilePath As String        <br />&#160;&#160; g_FilePath = &quot;C:\temp\&quot;        <br />&#160; <br />&#160;&#160;&#160; ' Create the button definition.        <br />&#160;&#160;&#160; Dim smallIcon As IPictureDisp        <br />&#160;&#160;&#160; Set smallIcon = LoadPicture(g_FilePath &amp; &quot;SmallProgTooltip.bmp&quot;)        <br />&#160;&#160;&#160; <br />&#160;&#160;&#160; Dim largeIcon As IPictureDisp        <br />&#160;&#160;&#160; Set largeIcon = LoadPicture(g_FilePath &amp; &quot;LargeProgTooltip.bmp&quot;)        <br />&#160;&#160;&#160; <br />&#160;&#160;&#160; Dim buttonDef As ButtonDefinition        <br />&#160;&#160;&#160; Set buttonDef = ThisApplication.CommandManager.ControlDefinitions.AddButtonDefinition(&quot;Sample1&quot;, &quot;SampleCommand&quot;, kQueryOnlyCmdType, &quot;&quot;, &quot;Sample command to show progressive tool tips.&quot;, &quot;Sample&quot;, smallIcon, largeIcon)        <br />&#160;&#160;&#160; <br />&#160;&#160;&#160; ' Add a control to the Work Feature panel of the Model tab of the Part ribbon.        <br />&#160;&#160;&#160; Call ThisApplication.UserInterfaceManager.Ribbons.Item(&quot;Part&quot;).RibbonTabs.Item(&quot;id_TabModel&quot;).RibbonPanels.Item(&quot;id_PanelA_ModelWorkFeatures&quot;).CommandControls.AddButton(buttonDef, True)        <br />&#160;&#160;&#160; <br />&#160;&#160;&#160; ' Define the progressive tooltip.&#160; This would typically be done in the        <br />&#160;&#160;&#160; ' same section of code where the button definition is created but there's        <br />&#160;&#160;&#160; ' a problem with the beta version where the progressive tooltip can only        <br />&#160;&#160;&#160; ' be defined on a control has been created.        <br />&#160;&#160;&#160; With buttonDef.ProgressiveToolTip        <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; .Description = &quot;The short description.&quot;        <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; .ExpandedDescription = &quot;This is the long expaned version of the description that could have a more complete description to accompany the picture.&quot;        <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; Dim progImage As IPictureDisp        <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; Set progImage = LoadPicture(g_FilePath &amp; &quot;koala.jpg&quot;)        <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; .Image = progImage        <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; .IsProgressive = True        <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; .Title = &quot;Sample1&quot;        <br />&#160;&#160;&#160;&#160;&#160;&#160; <br />&#160;&#160;&#160; End With        <br />End Sub</font></em></p>  <p>&#160;</p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b019b03ac995e970c-pi"><img style="background-image: none;
