<p>by <a href="http://adndevblog.typepad.com/manufacturing/vladimir-ananyev.html">Vladimir Ananyev</a></p>  <p><b>Issue</b></p>  <p>I need to find the base view of all projected views in the drawing. I use the DrawingView.ParentView however the ParentView is nothing if the view is an isometric view. Is there a way to get the baseview?</p>  <p><b>Solution</b></p>  <p>Isometric projected views are not associated with the base views. You can see this in the browser. When you create an isometric projected view, it is not nested under the base view. Also deleting the base view does not delete the isometric view. There is no a property that could return the projected view from the base view.
