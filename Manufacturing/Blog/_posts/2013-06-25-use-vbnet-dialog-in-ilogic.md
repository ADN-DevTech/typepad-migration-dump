<p>By <a href="http://adndevblog.typepad.com/manufacturing/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>
<p>The Form ability of iLogic is helpful in most cases. However, it is not so flexible to design the layout. And textbox does not support multi-lines (iProperties&gt;&gt;Comments cannot be shown) etc.</p>
<p>iLogic bases on VB.NET. Actually it can call native form of VB.NET. Following is the steps of a demo:</p>
<p><span class="asset  asset-generic at-xid-6a0167607c2431970b01b8d1511ffe970c img-responsive"><a href="http://adndevblog.typepad.com/files/vbnetformforilogic-1.zip">Download VBNetFormforiLogic</a></span></p>
<p>1) Create a VB.NET class library project. Add one form with single-line textbox to communicate with the parameter which is single value; a combo box for a parameter which is multi values; a multi-line textbox for iProperties&gt;&gt;Comments.</p>
<p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b019103d4c6bf970c-pi"><img alt="image" border="0" height="274" src="/assets/image_d747e1.jpg" style="display: inline; border-width: 0px;" title="image" width="433" /></a></p>
<p>2) In Form_Load, assign the parameters values to the controls. In OK button, update the parameters values with the controls values.</p>
