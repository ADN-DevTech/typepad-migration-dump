<p>By <a href="http://adndevblog.typepad.com/manufacturing/sajith-subramanian.html" target="_self">Sajith Subramanian</a></p><p>The end of support for the JavaScript API was announced a couple of years ago.</p><p>However, so far any scripts or add-ins that were written using the JavaScript API have continued to run. <p>Please note that this is set to change, and as of the <strong>December release of Fusion 360 they will no longer run at all</strong>. <p>Any script or add-in using the JavaScript API will need to be migrated to either <strong>Python </strong>or <strong>C++</strong>.<p>Please find the official announcement here:<p><a href="https://forums.autodesk.com/t5/fusion-360-api-and-scripts/notice-javascript-api-is-being-fully-retired/td-p/9748484">Notice: JavaScript API is being fully retired</a>
