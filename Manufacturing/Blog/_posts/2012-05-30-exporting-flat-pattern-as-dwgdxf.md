<p>By <a href="http://adndevblog.typepad.com/manufacturing/philippe-leefsma.html" target="_self">Philippe Leefsma</a></p>  <p><b>Q:</b></p>  <p>How to create a DWG/DXF file flat pattern from an Inventor sheet metal part?</p>  <p><a name="section2"></a></p>  <p><b>A:</b></p>  <p>The WriteDataToFile method of the DataIO object supports the option to write the flat pattern out as a DWG file. You can also supply arguments in a method similar to http URL variables (starting with '?', 'key=value', multiple keys and values separated by '&amp;'). </p>  <p>   <br />The valid arguments are:     <br />TangentLayer = [name]     <br />BendLayer = [name]     <br />OuterProfileLayer = [name]     <br />AcadVersion = {&quot;2005&quot;,&quot;2004&quot;,&quot;2002&quot;, &quot;2000&quot;, &quot;R14&quot;, &quot;R13&quot;, &quot;R12&quot;,} // R12 is only supported for the dxf format type.</p>  <p>For example:</p>  <p>&#160; <br />&quot;FLAT PATTERN DWG?AcadVersion=2000&quot;     <br />&quot;FLAT PATTERN DWG?AcadVersion=R14&quot;     <br />&quot;FLAT PATTERN DWG?AcadVersion=R13&amp;TangentLayer=myTangLayer&quot;     <br />&quot;FLAT PATTERN DXF?AcadVersion=R14&amp;BendLayer=myBendLayer&quot;
