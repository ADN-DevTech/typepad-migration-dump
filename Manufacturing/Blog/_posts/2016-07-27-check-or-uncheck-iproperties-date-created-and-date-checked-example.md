<p>By <a href="http://adndevblog.typepad.com/manufacturing/wayne-brill.html" target="_self">Wayne Brill</a></p> <p>It is not obvious how to add or remove a check to an iProperty such as Creation Date: </p> <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb0923d1da970d-pi"><img title="image" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; padding-top: 0px; padding-left: 0px; display: inline; padding-right: 0px; border-top-width: 0px" border="0" alt="image" src="/assets/image_692dd5.jpg" width="660" height="255"></a></p> <p>&nbsp;</p> <p>In the VBA watch window you will see that when the "Date Created" iProperty is not checked the Expression property of the iProperty is not a valid date. When it is checked it has a valid date. If you give the iProperty Expression a valid date value it will be checked. Using a date value of "1601, 1, 1" removes the checkmark.&nbsp;&nbsp; <br></p> <p>Here is a VBA example:</p> <p style="font-size: 10pt; font-family: arial; background: #eeeeee; color: black; line-height: 140%">Public Sub iProperties_Test()<br>&nbsp;&nbsp;&nbsp; ' Get the active document.<br>&nbsp;&nbsp;&nbsp; Dim invDoc As Document<br>&nbsp;&nbsp;&nbsp; Set invDoc = ThisApplication.ActiveDocument<br>&nbsp;&nbsp;&nbsp; <br>&nbsp;&nbsp;&nbsp; ' Get the design tracking property set.<br>&nbsp;&nbsp;&nbsp; Dim invDesignInfo As PropertySet<br>&nbsp;&nbsp;&nbsp; Set invDesignInfo = invDoc.PropertySets.Item("Design Tracking Properties")<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>&nbsp;&nbsp;&nbsp; Dim iProp As Property<br>&nbsp;&nbsp;&nbsp; For Each iProp In invDesignInfo<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 'If iProp.Name = "Date Checked" Then<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; If iProp.DisplayName = "Date Created" Then<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 'Property is unchecked with this<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ' iProp.Expression = "1601, 1, 1"<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 'Property is checked with this<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; iProp.Expression = #7/22/2016#<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Exit For<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; End If<br>&nbsp;&nbsp;&nbsp; Next iProp<br>&nbsp;&nbsp;&nbsp; <br>End Sub</p>
