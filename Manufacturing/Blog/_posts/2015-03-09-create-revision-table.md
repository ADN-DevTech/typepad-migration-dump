<p>By <a href="http://adndevblog.typepad.com/manufacturing/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>  <p>API provides the corresponding methods like UI to manipulate revision table. </p>  <ul>   <li>RevisionTables: the collection of revision table in a sheet. Its Add or Add2 method can create a new revision table.</li>    <li>RevisionTable: the specific revision table. You can set its title, position and the styles of header of column or row, or even delete it.</li>    <li>RevisionTableColumns: the collection of the columns. It supports to add a new column. </li> </ul>  <blockquote>   <p><em><font size="1">RevisionTableColumns.<b>Add</b>(           <br /><b>PropertyType</b> As </font></em><em><font size="1">RevisionTablePropertyEnum</font></em><em><font size="1">,         <br /> [<b>PropertySetId</b>] As String,          <br /> [<b>PropertyIdentifier</b>] As Variant,          <br /> [<b>TargetIndex</b>] As Long,           <br />[<b>InsertBefore</b>] As Boolean ) As </font></em><em><font size="1">RevisionTableColumn</font></em></p> </blockquote>  <blockquote>   <p><em>RevisionTablePropertyEnum</em> specifies the property type to associate with the column. </p> </blockquote>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c75e8a5d970b-pi"><img style="background-image: none; border-bottom: 0px; border-left: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top: 0px; border-right: 0px; padding-top: 0px" title="image" border="0" alt="image" src="/assets/image_2d3019.jpg" width="185" height="344" /></a></p>  <ul>   <li>RevisionTableColumn: the specific column. You can control the justification, title, even delete the column.</li>    <li>RevisionTableRows: the collection of the rows. It supports to add a new row or custom row.</li>    <li>RevisionTableRow: the specific row that can access the cell by RevisionTableRow.Item. </li>    <li>RevisionTableCell : the specific cell of the row. it tells the value of the cell.</li> </ul>  <p>The following is a small code that demos how to create a new revision table and some kinds of columns.</p>  <p>   <p style="line-height: normal;
