<p>By <a href="http://adndevblog.typepad.com/manufacturing/wayne-brill.html" target="_self">Wayne Brill</a> (Information provided by Brian Ekins)</p> <p>Compared to Inventor 2016 there are actually six removed objects and 45 removed functions.&nbsp; (The data in the slide one this <a href="http://adndevblog.typepad.com/manufacturing/2016/04/inventor-2017.html">post</a> was generated long before the release). The complete list is below.&nbsp; The reason for so many removed functions is that some features implemented the “Definition” way of encapsulating the input used to create and edit those features. The “removed” functions weren’t actually removed but have been hidden and can continue to be used, although they’re not officially supported and developers should begin using the new way of creating and editing those features.</p> <p><strong>6 Removed Objects:</strong></p> <p>&nbsp;&nbsp;
