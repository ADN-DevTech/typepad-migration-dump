<p>By <a href="http://adndevblog.typepad.com/manufacturing/barbara-han.html" target="_self">Barbara Han</a></p>  <p>Some times back, Gary posted a blog talking about “<a href="http://adndevblog.typepad.com/manufacturing/2012/06/how-to-set-the-shortcut-key-to-a-customized-command-in-inventor.html">how to set the shortcut key to a customized command in Inventor</a>”, however the code in that sample is quite out-of-date, so I want to give a update and also added one more point that you need to notice:</p>  <p>There are two types of shortcut key - Shortcut and Alias. Inventor API help document mentioned that the accelerator shortcut key can't be duplicate, but the alias shortcut key can. The alias shortcut key is interpreted as &quot;Alt+&lt;key&gt;&quot; (&lt;key&gt; is the letter showing in the Customize –&gt; keyboard window), for instance, &quot;I&quot; alias shortcut key means that you can press Alt+I to bring out that command.</p>  <p>The following is a VBA code sample that assigns the alias shortcut key &quot;I&quot; to a custom command along with the code of deleting the command and custom ribbon. With VB.NET, the code is similar.</p>  <p>Below code is saved VBA Editor’s DocumentProject –&gt;ThisDocument module:</p>  <p><font size="1" face="Courier New">Private WithEvents oBtnDef As shortcut key to the button command</font></p>  <p><font size="1" face="Courier New">oBtnDef.OverrideShortcut = &quot;I&quot;</font></p>  <p><font size="1" face="Courier New">' Find the drawing
