<p>By <a href="http://adndevblog.typepad.com/manufacturing/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>  <p>If the file is in sketch edit mode, Inventor will pop out a message even if you do nothing with a new document. To have a silent saving, the first we will think of is the API SilentOperation.&#160; But with SilentOperation =true, saving will fail.</p>  <p>This is the current design. Both UI and API do not support to save a part document in sketch editing mode. User can check if the document is in part environment, if no exit the activated object mode. The similar known situations are:    <br />1. Sketch mode in part, assembly, drawing.    <br />2. Custom positional representation mode in assembly.    <br />3. Occurrence in process edit mode in assembly.    <br />4. Flat pattern edit mode in sheet metal. </p>  <p>Following is a workaround for situation 1. But as commenting in the code, there is still one case(though not so popular) for a newly created drawing definition edit mode that the code could not cover.
