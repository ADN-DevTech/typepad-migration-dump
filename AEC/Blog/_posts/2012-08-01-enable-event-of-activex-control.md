<p>By <a href="http://adndevblog.typepad.com/aec/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>  <p>The ActveX control derives from Windows control. It impelements some override events. You can find the events list in API help reference: _InwControlEvents </p>  <p>In default, some events are not enabled such as OnLButtonDown, OnLButtonUp, OnKeyDown, OnKeyUp. InwControl4.ForceEvents is a flag that can toogle the events to fire.&#160; </p>  <p>Following HTML code demo uses&#160; <em> Navisworks Integrated Control 10 SDI</em> .&#160; The button 'Force Events' will set ForceEvents as true.
