<p>More than often,&#0160;I receive inquiries about how to get started with Revit API.&#0160;Attached&#0160;here is the document&#0160;that I have been keeping and updating since Revit 2009:&#0160;<span class="asset  asset-generic at-xid-6a0167607c2431970b016768295ac8970b"><a href="http://adndevblog.typepad.com/files/getting-started-with-revit-2013-api-2.pdf">Download Getting Started with Revit 2013 API</a></span><span class="asset  asset-generic at-xid-6a0167607c2431970b017743006b0e970d">&#0160; This gets quite handy when I get an inquiry.&#0160;I&#39;m also copy and pasting the contents here so that from now on, I can point to our blog, too: </span></p>
<p><span class="asset  asset-generic at-xid-6a0167607c2431970b017743006b0e970d">1. Setting up the Environment</span></p>
<ul>
<li><strong>Revit Product</strong> – you can download the latest version of released products from Autodesk product page.&#0160; Go to <a href="http://www.autodesk.com">http://www.autodesk.com</a>, look under “Products” menu, “Alphabetically.”&#0160; From each Revit product page, look under Product Download. Following are the examples of the links as of the time when this documentation was written. (Note: exact link may change):&#0160;</li>
</ul>
<p style="padding-left: 30px;">Revit (OneBox available with Design Suites): <br /><a href="http://usa.autodesk.com/revit/trial/">http://usa.autodesk.com/revit/trial/</a></p>
<p style="padding-left: 30px;">Revit Architecture:<br /><a href="http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=9262388">http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=9262388</a></p>
<p style="padding-left: 30px;">Revit Structure:<br /><a href="http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=9339705">http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=9339705</a></p>
<p style="padding-left: 30px;">Revit MEP: <br /><a href="http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=9262907">http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=9262907</a></p>
<p style="padding-left: 30px;">Without an activation code, you can use Revit as viewing-only mode.</p>
<ul>
<li><strong>Revit SDK</strong> – you can find Revit SDK in every Revit install.&#0160;</li>
</ul>
<p style="padding-left: 30px;">The SDK install is located under “Install Tools and Utilities” menu on the main page of Revit installer. (Note: there is another menu called “Utilities” you see right after the product installation, which contains only the link Content Batch utilities. Click on “Back to First Page” button to move back to the main page of the installer to install SDK.)</p>
<p style="padding-left: 30px;">Alternatively, you can also find the SDK in the extraction folder, under:<br />&lt;extraction folder&gt;\support\SDK\RevitSDK.exe<br />If you have accepted the default location, which typically looks like: <br />C:\Program Files\Autodesk\Revit XXX 201x\support\SDK\RevitSDK.exe</p>
<p style="padding-left: 30px;"><strong>Note</strong>:&#0160; SDK may be updated.&#0160; Make sure you check the latest at either ADN site (member only) or public Autodesk Developer Network site for Revit (available to public): <br /><a href="http://www.autodesk.com/developrevit">http://www.autodesk.com/developrevit</a></p>
<p style="padding-left: 30px;"><strong>RevitAPI.dll</strong> and <strong>RevitAPIUI.dll</strong> is in every Revit installation under &lt;Revit install&gt;\Program\</p>
<ul>
<li>Development Environment – Microsoft Visual Studio (MSVS) 2010 for Revit 2013. (In previous releases, 2011 and 2010, you may also use VS 2008 + SP1, but debugging is limited. All the samples are on VS 2010).</li>
<li>Revit Documentations – Documentation about Revit and Revit API are all on the Autodesk Wiki now.&#0160; Revit SDK includes Developer Guide and many samples.&#0160;</li>
</ul>
<p>2. Where to Start</p>
<p style="padding-left: 30px;">(1) Watch the training video - The best way to start will be to watch the recoding of our webcast. <br />There is recordings called DevTV, which is posted on the public ADN site for Revit: <a href="http://www.autodesk.com/developrevit">http://www.autodesk.com/developrevit</a> <br />&#0160;(Scroll down toward the bottom.) <br />If you are new to the programming as well, take a look at “My First Revit Plug-in” training materials: <br /><a href="http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=16777469">http://usa.autodesk.com/adsk/servlet/index?siteID=123112&amp;id=16777469</a><br />You can also download previous year’s webcast recording from here: <br /><a href="http://download.autodesk.com/media/adn/RevitAPI_20May2010.zip">http://download.autodesk.com/media/adn/RevitAPI_20May2010.zip</a><br />This covers the basic of Revit API since 2011. (Note: Revit API had a major renovation in 2011). The materials from this webcast include recordings, ppt and training labs code.&#0160; (You can find the same from ADN site, under Revit knowledgebase section, or public ADN site&#0160;Training schedule page.)</p>
<p style="padding-left: 30px;">This year’s webcasts cover major enhancements: <br /><a href="http://download.autodesk.com/media/adn/DevDaysOnline-Revit2013API.zip">http://download.autodesk.com/media/adn/DevDaysOnline-Revit2013API.zip</a>&#0160; <br /><a href="http://download.autodesk.com/media/adn/Revit2013APIWebcast_May17_2012.zip">http://download.autodesk.com/media/adn/Revit2013APIWebcast_May17_2012.zip</a>&#0160;</p>
<p style="padding-left: 30px;">If you are interested in Family API, there is a webcast recording about Family API, too. You can find it from the location below:&#0160; <br /><a href="http://download.autodesk.com/media/adn/Revit_2010_Family_API_Webcast-July2009.zip">http://download.autodesk.com/media/adn/Revit_2010_Family_API_Webcast-July2009.zip</a><br /><br />(2) Try the latest training labs, which is available from the public ADN site: <br /><a href="http://www.autodesk.com/developrevit">http://www.autodesk.com/developrevit</a> <br />(Scroll down toward the bottom, under &quot;Learning&quot; &gt;&gt; &quot;Samples and Documentation&quot;). <br /><br />(3) Try HelloRevit sample by yourself, go through Labs code, and look at some for samples in the SDK. <br />The following samples will be a good starting point: <br />•&#0160;RevitCommands<br />•&#0160;Fire Rating<br />•&#0160;Viewers/ElementViewer<br />•&#0160;CreateBeamsColumnsBraces</p>
