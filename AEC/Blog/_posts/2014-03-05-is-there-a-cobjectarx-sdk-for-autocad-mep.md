<p>By <a href="http://adndevblog.typepad.com/aec/gopinath-.html">Gopinath Taget</a></p>  <p>There is no C++ based ARX SDK specifically for AutoCAD MEP. AutoCAD MEP only has a .NET API. You will find more information about the MEP .NET SDK in <a href="http://adndevblog.typepad.com/aec/2012/07/where-is-autocad-mep-sdk.html">this blog post</a>.</p>  <p>If you are looking for generic vanilla AutoCAD ARX SDK (this <strong><font color="#000000">does not</font></strong> contain APIs for MEP entities or functionality), you will find it here: <a href="http://www.objectarx.com">http://www.objectarx.com</a>. The ARX SDK will work with Vanilla AutoCAD and AutoCAD verticals based on it.     </p>
