<p>By <a href="http://adndevblog.typepad.com/aec/joe-ye.html">Joe Ye</a></p>  <p>&#160;</p>  <p>I want to set the crop box of a 3d view to correspond to a single room in plan.&#160; I have already been successful in setting the crop box in plan but want to find a similar view in 3d.&#160; It won't be exactly the same since the view is angled but I'd like to get to something close.</p>  <p><b>Solution</b></p>  <p>In the purpose of setting the crop box for 3D view, we need to get bottom-left and up-right points of a box that exactly encloses the room.    <br />First we get all the vertices of the room in WCS, and project all the vertices to the 3D view. The projection can be done by means of the inverse transform returned by View3D.CropBox.Transform.     <br />Then find the bottom-left point and the up-right point in these converted points.     <br />Finally assign the two points to the CropBox property.     <br />Please see the code of the command.
