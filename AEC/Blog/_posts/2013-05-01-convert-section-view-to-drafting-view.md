<p>By <a href="http://adndevblog.typepad.com/aec/joe-ye.html">Joe Ye</a></p>  <p>Revit didn’t expose API to change the the property of a section view to make it a drafting view.&#160; Here introduce a workaround how to put the content in a section view to a drafting view.</p>  <p>The solution is to using the view export and dwg file import. You can export any view to dwg file via Document.Export() method. After export the dwg file, import this dwg to the target drafting view via Document.Import() method.
