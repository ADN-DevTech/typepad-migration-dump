<p>By <a href="http://adndevblog.typepad.com/aec/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>
<p>We have introduced how to add hyperlink in <a href="http://adndevblog.typepad.com/aec/2012/05/create-hyperlinks-for-model-objects-using-net-api.html">this blog</a>. A further requirement is to open the hyperlink when the user double clicks the object. This will need a couple of APIs:</p>
<p>- InputPlugin to interact with UI for mouse clicking</p>
<p>- A common plugin (say AddInPlugin) to provide a button to invoke InputPlugin</p>
<p>- Call state.GetOverrideURL of COM to get the hyperlinks of the object</p>
<p>The following are a demo code. The plugin <em>EnableToolPluginExample</em> will create a button under Tool Add-ins tab. When the button is clicked, the custom interactive mode is activated. When the user double clicks an object, the first URL will be opened in the default browser.</p>
<p>The complete project is available <span class="asset  asset-generic at-xid-6a0167607c2431970b01b7c7a8b494970b img-responsive"><a href="http://adndevblog.typepad.com/files/doubleclick_hyperlink.zip">Download DoubleClick_Hyperlink</a></span>.</p>
