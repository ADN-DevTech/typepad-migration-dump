<p>By <a href="http://adndevblog.typepad.com/aec/katsuaki-takamizawa.html" target="_self">Katsuaki Takamizawa</a></p>  <p>This post illustrates commonly available ways to override the display of an element in order. It is good to know.</p>  <p><a href="http://revitclinic.typepad.com/my_weblog/2012/02/revit-visibility-hierarchy.html">http://revitclinic.typepad.com/my_weblog/2012/02/revit-visibility-hierarchy.html</a></p>  <p>Currently , these are only available view specific overrides for an element. </p>  <p>CutColorOverrideByElement -    <br />&#160; Override the cut lines' color by Element in this view. </p>  <p>CutLinePatternOverrideByElement -    <br />&#160; Override the cut lines' pattern by Element in this view. </p>  <p>CutLineWeightOverrideByElement -    <br />&#160; Override the cut lines' weight by Element in this view. </p>  <p>ProjColorOverrideByElement -    <br />&#160; Override the projection lines' color by Element in this view. </p>  <p>ProjLinePatternOverrideByElement -    <br />&#160; Override the projection lines' pattern by Element in this view. </p>  <p>ProjLineWeightOverrideByElement -    <br />&#160; Override the projection lines' weight by Element in this view.     </p>
