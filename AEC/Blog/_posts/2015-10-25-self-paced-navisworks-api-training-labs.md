<p>By <a href="http://adndevblog.typepad.com/aec/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>  <p>We have produced a training lab material. The first version covers 11 topics that we think are basic to the developers who are getting started with Navisworks API. It could be also a reference for those who have some practices because the contents and samples might include the tricks of API.</p>  <p>The material is available at <a href="http://www.autodesk.com/developnavisworks">ADN Open</a>. It contains: PPTs, samples, Readme and necessary files that for some samples. The following is an overview of the material. </p>  <p>&#160;</p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01b7c7e301b1970b-pi"><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="clip_image001" border="0" alt="clip_image001" src="/assets/image_571382.jpg" width="365" height="63" /></a></p>  <p>&#160;</p>  <p><strong>Lab1: Overview</strong></p>  <p>In this lab, we will quickly go through Navisworks product family, basics of Navisworks API, common use case scenarios of APIs, and available learning resources.</p>  <p><strong>Lab2: Plugins</strong></p>  <p>This lab introduces the plugin system, and demonstrate how to create a basic AddInPlugin and DockPanePlugin.</p>  <p><strong>Lab3: Document</strong></p>  <p>This lab introduces API structure in .NET, how to manipulate Document, what document parts, document info and grids &amp; levels are.</p>  <p><strong>Lab4: Model</strong></p>  <p>This lab introduces model tree, model item, and how to access geometry attributes of model item.</p>  <p><strong>Lab5: Properties</strong></p>  <p>This lab introduces the properties of objects and how to effectively find the specific properties.</p>  <p><strong>Lab6: Search</strong></p>  <p>This lab introduces the ways to find objects, the most efficient way and various scenarios to work with Search API.</p>  <p><strong>Lab7: Selection</strong></p>  <p>This lab introduces current selection and selection sets, and how to manipulate them.</p>  <p><strong>Lab8: COM Interop</strong></p>  <p>This lab introduces how to work with COM API from .NET with a couple of examples, e.g. add custom properties, read geometry primitives etc.</p>  <p><strong>Lab9: Events</strong></p>  <p>This lab introduces .NET events API.</p>  <p><strong>Lab10: Viewpoint</strong></p>  <p>This lab introduces viewpoint and saved viewpoint. How to manipulate camera position is also discussed.</p>  <p><strong>Lab11: .NET Control</strong></p>  <p>This lab introduces how to use Navisworks .NET controls, i.e. ApplicationControl, DocumentControl and ViewControl.</p>  <p>&#160;</p>  <p>Enjoy!</p>
