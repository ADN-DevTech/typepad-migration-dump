<p>By <a href="http://adndevblog.typepad.com/aec/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>
<p>Have you ever wished to have a viewer to check Navisworks file on iPad? It came true today!&#0160; </p>
<p>The Autodesk BIM 360 Glue mobile app enables users to access Building Information Modeling (BIM) projects on a mobile device. Building, infrastructure, design, and construction professionals can access and intuitively explore the multidisciplinary models online or offline, access all saved views, and review intelligent object properties.&#0160; You can now easily&#0160; access to Navisworks models. It provides the similar experience when you access model on web Glue.</p>
<p>&#0160;</p>
<p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017c387c7dff970b-pi"><img alt="image" border="0" height="284" src="/assets/image_665804.jpg" style="display: inline; border-width: 0px;" title="image" width="365" /></a> </p>
<p>&#0160;</p>
<p>And it has the following key abilities: </p>
<p>•Publish Navisworks models to BIM 360 Glue’s mobile app for access in the field</p>
<p>•Navigate through saved views and access intelligent object properties</p>
<p>•Intuitive exploration of multi-discipline BIM projects in the field</p>
<p>•Navigate using intuitive gesture-based pan, zoom and orbit </p>
<p>•Walk through the model with gravity-assisted navigation optimized for BIM</p>
<p>•View project model details with immersive, gyroscope-enabled look-around </p>
