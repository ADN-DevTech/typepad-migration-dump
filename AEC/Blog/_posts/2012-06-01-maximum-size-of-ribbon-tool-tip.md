<p><strong>Issue</strong></p>
<p>I&#39;m trying to define the guidelines for people creating tool tip images&#0160;using the Revit API. What is the maximum size that we can use for Ribbon tool tip images?&#0160;</p>
<p><strong>Solution</strong></p>
<p>The maximum size of the Ribbon tool tip is 355 pixels in width, and no limitation is set for height.&#0160;</p>
