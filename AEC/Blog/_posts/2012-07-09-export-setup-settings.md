<p>By <a href="http://adndevblog.typepad.com/aec/katsuaki-takamizawa.html" target="_self">Katsuaki Takamizawa</a></p>  <p>You can access to DWG/DXF and DGN Export Setups in a dialog box from R&gt;Export&gt;Options menu in UI. DWG/DXF and DGN Export Setups are now available in Revit 2013 API as ExportDWGSettings and ExportDGNSettings element respectively. The most settings of Export Setups are accessible in these elements expect settings of individual tables appear in each tab. The tables would become accessible using API near future. </p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01676857204e970b-pi"><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="ExportSetup" border="0" alt="ExportSetup" src="/assets/image_982578.jpg" width="244" height="142" /></a></p>
