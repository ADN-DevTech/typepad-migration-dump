<p><a href="http://adndevblog.typepad.com/aec/joe-ye.html">Joe Ye</a></p>  <p>&#160;</p>  <p>Through Revit UI command, we can hide group by picking it and click the hide command like other elements. Howerver I cannot hide group via API.</p>  <p>Here is the DevelopSharp code I used.    <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; public void hidegroup()     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; {     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; Document doc = this.ActiveUIDocument.Document;     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; Selection sel = this.ActiveUIDocument.Selection;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; Reference ref1 = sel.PickObject(ObjectType.Element,&quot;Please pick a group&quot;);     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; Element elem = doc.GetElement(ref1);     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; IList&lt;ElementId&gt; lists = new List&lt;ElementId&gt;();     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; lists.Add(elem.Id);     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; Transaction trans = new Transaction(doc);     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; trans.Start(&quot;hidegroup&quot;);     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; doc.ActiveView.HideElements(lists);     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; trans.Commit ();     <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160; }</p>  <p>Launch the command, pick any group,&#160; error occurs. The error message is :    <br />Revit failed to execute hidegroup.     <br />A problem has been detected. </p>  <p>Then how can we hide a group? Any tricks?</p>  <p>&#160;</p>  <p><strong>Solution</strong></p>  <p>Groups cannot be hidden or overridde like ordinary elements. In order to hide a group, we need to hide the element set in that group by HideElements() method. The good aspect is that we can hide part of a group.</p>  <p>Here is the code to hide the whole group.</p>  <blockquote>   <p>public void hidegroup()      <br />{       <br />&#160;&#160;&#160; Document doc = this.ActiveUIDocument.Document;</p>    <p>&#160;&#160;&#160; Selection sel = this.ActiveUIDocument.Selection;&#160;&#160;&#160; </p>    <p>&#160;&#160;&#160; Reference ref1 = sel.PickObject(ObjectType.Element,&quot;Please pick a group&quot;);</p>    <p>&#160;&#160;&#160; Element elem = doc.GetElement(ref1);</p>    <p>&#160;&#160;&#160; IList&lt;ElementId&gt; lists = new List&lt;ElementId&gt;();</p>    <p>&#160;&#160;&#160; Group g = elem as Group;</p>    <p>&#160;&#160;&#160; IList&lt;ElementId&gt; ids = g.GetMemberIds();</p>    <p>&#160;&#160;&#160; Transaction trans = new Transaction(doc);</p>    <p>&#160;&#160;&#160; trans.Start(&quot;hidegroup&quot;);</p>    <p>&#160;&#160;&#160; this.ActiveUIDocument.ActiveView.HideElements(ids);</p>    <p>&#160;&#160;&#160; trans.Commit ();</p>    <p>}</p></blockquote>
