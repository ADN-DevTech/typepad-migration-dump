Type duplicate methods are not under the Autodesk.Revit.Creation.Document class. Then how to create new type then?</p>  <p>Normally Revit document resident elements are created via the methods in Autodes.Revit.Creation.Document class. However there is no method in this class can be used to create new types, for example, DimensionType, WallType, FloorType...    <br />The way to create new Revit types is different from other elements. Each type class provides Duplicate(string name) method . It is this method that we can use to duplicate a new type, which has the same parameter values and property values but different name.
