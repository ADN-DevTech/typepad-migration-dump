<p>By <a href="http://adndevblog.typepad.com/aec/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>  <p><strong>Issue</strong>     <br />In the SDK sample <strong>CustomRibbon</strong> I saw a comment about the attribute <strong>ShortcutWindowTypes.</strong> But I did not know how it works. </p>  <p><strong>Solution</strong></p>  <p>As the SDK code explains: The ability of the attribute ShortcutWindowTypes of CommandHandlerPlugin is:</p>  <p><em>If you have defined a DockPanePlugin, you can specify that the Shortcut only applies when that window is active. Use the Id for the&#160; DockPanePlugin.</em></p>  <p>This attribute is just to specify if the shortcut can work or not.</p>  <p>Assume we have had a DockPanePlugin e.g. the SDK sample BasicDockPanePlugin. And in CommandHandlerPlugin, we define a command e.g.</p>  <div style="font-family: courier new; background: white; color: black; font-size: 8pt">   <p style="margin: 0px"><span style="line-height: 140%"> [</span><span style="line-height: 140%; color: #2b91af">Command</span><span style="line-height: 140%">(</span><span style="line-height: 140%; color: #a31515">&quot;ID_Button_3&quot;</span><span style="line-height: 140%">,</span></p>    <p style="margin: 0px"><span style="line-height: 140%">&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; CanToggle = </span><span style="line-height: 140%; color: blue">true</span><span style="line-height: 140%">,</span></p>    <p style="margin: 0px"><span style="line-height: 140%">&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; Shortcut = </span><span style="line-height: 140%; color: #a31515">&quot;Shift+Z&quot;</span><span style="line-height: 140%">)]</span></p> </div>  <p>In default, the shortcut can work anytime when we press “Shift+Z”. But if we add the attribute&#160; ShortcutWindowTypes</p>  <div style="font-family: courier new; background: white; color: black; font-size: 8pt">   <p style="margin: 0px"><span style="line-height: 140%">[</span><span style="line-height: 140%; color: #2b91af">Command</span><span style="line-height: 140%">(</span><span style="line-height: 140%; color: #a31515">&quot;ID_Button_3&quot;</span><span style="line-height: 140%">,        <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; CanToggle = </span><span style="line-height: 140%; color: blue">true</span><span style="line-height: 140%">,&#160; <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; Shortcut = </span><span style="line-height: 140%; color: #a31515">&quot;Shift+Z&quot;</span><span style="line-height: 140%">,&#160; <br />&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; ShortcutWindowTypes = </span><span style="line-height: 140%; color: #a31515">&quot;BasicDockPanePlugin.BasicDockPanePlugin.ADSK&quot;</span><span style="line-height: 140%">)]</span></p>    <p style="margin: 0px"><span style="line-height: 140%"></span></p>    <p style="margin: 0px">&#160;</p> </div>  <p>Where: BasicDockPanePlugin.BasicDockPanePlugin.ADSK is the ID of the BasicDockPanePlugin.</p>  <p>What will happen?</p>  <p>-&#160;&#160;&#160;&#160;&#160;&#160;&#160; If the window of BasicDockPanePlugin is not displayed and not activate(Activate means it is being focused), the shortcut will NOT work anymore.</p>  <p>-&#160;&#160;&#160;&#160;&#160;&#160;&#160; If the window of BasicDockPanePlugin is displayed and activate, the shortcut will work as normal.</p>  <p>In a word, this attribute is to specify if the shortcut can work or not. In addition, when you set ShortcutWindowTypes, please make sure to set the <strong>correct</strong> ID (name) of the DockPanePlugin. Otherwise, the shortcut will still work.</p>
