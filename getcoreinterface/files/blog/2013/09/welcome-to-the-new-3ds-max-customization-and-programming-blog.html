Welcome to GetCOREInterface!

Welcome to GetCOREInterface! This is a new blog dedicated to 3ds Max customization and programming. It will also have some sprinkling of FBX SDK along with other products relating to 3ds Max and 3ds Max Design. This blog will be maintained by Kevin Vandecar who is part of the Media & Entertainment worldwide team of API gurus providing technical services through the Autodesk Developer Network (ADN).

Why did I choose GetCOREInterface for the name? Well if you know any basic 3ds Max SDK programming, you know that GetCOREInterface is the entry point API to programming 3ds Max. So why not use it as the entry point for the blog itself? It's easy to remember and search once you become acquainted with programming in 3ds Max. It is also used in the .NET API environment, and it is the underlying access point in MAXScript, too (you just don't need to know that from a scripting standpoint). That being said, you can expect all forms of programming topics to be covered here including MAXScript, .NET API, and of course the C++ SDK. I will categorize each topic with the primary concepts (.NET API, C++, or MAXScript). But also remember that many techniques can be "ported" to the other environments with minimal effort.

You can expect mostly technical (how-to) solutions and content derived from answering questions from our ADN partners and forum users including sample code. I will also provide pointers to white papers, screencasts, conference presentation material, and other information derived from our work in ADN. This blog is meant to provide up-to-date information, in one place, so you can easily find what you need to solve your programming problems.

For more about me, see here: Kevin Vandecar
