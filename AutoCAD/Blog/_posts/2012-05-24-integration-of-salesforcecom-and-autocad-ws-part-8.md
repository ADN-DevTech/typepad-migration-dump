<p>By <a href="http://adndevblog.typepad.com/autocad/daniel-du.html">Daniel Du</a></p>  <p><img alt="" src="/assets/image_845925.jpg" width="259" height="81" /></p>  <p>Until <a href="http://adndevblog.typepad.com/autocad/2012/05/integration-of-salesforcecom-and-autocad-ws-part-7.html" target="_blank">last post</a>, we have already implemented of salesforce.com and AutoCAD WS. We need to pass the case Id to the visual force page, the URL is similar like : <a title="https://c.na12.visual.force.com/apex/OpeninAutocadWS?id=500U0000001yRc4" href="https://c.na12.visual.force.com/apex/OpeninAutocadWS?id=500U0000001yRc4">https://c.na12.visual.force.com/apex/OpeninAutocadWS?id=500U0000001yRc4</a></p>  <p>In this post, I would introduce how to integrate this visual force page to our case layout, so that we do not need to input the case ID manually. </p>  <p>Refer to Your Name &gt; Setup &gt; App Setup &gt; Customize &gt; Cases &gt; Page Layouts,&#160; Edit “Case layout”.</p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b016766bdb636970b-pi"><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="image" border="0" alt="image" src="/assets/image_914511.jpg" width="469" height="134" /></a></p>  <p>select “VisualForce Pages ”, all visual force pages applying to case will show up, drag the one we created “ OpenInAutocadWS” to somewhere below, you can get a WYSIWYG interface.&#160; Click “Quick Save”&#160; or “Save” when you are done. </p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b016305c9a840970d-pi"><img style="background-image: none; border-bottom: 0px; border-left: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top: 0px; border-right: 0px; padding-top: 0px" title="image" border="0" alt="image" src="/assets/image_689759.jpg" width="463" height="170" /></a></p>  <p>Here is what I did, When you open one case from salesforce.com, you will see the visual force page is embedded in case layout.</p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b016305c9a87c970d-pi"><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="image" border="0" alt="image" src="/assets/image_690587.jpg" width="475" height="243" /></a></p>  <p>So far we are finishing this serials of post. It is just a prototype and many stuffs need to be done to make it more practical. Anyway, it is really a good experiences to investigate those two cloud based applications. If you have any suggestions to make it better, please do let me know. </p>  <p>Have fun !</p>
