    </p>
    <p>Dear Developers,</p>
    <p>Thank you for being part of the Autodesk AutoCAD developer ecosystem.</p>
    <p>
      As part of Autodesk Opensource Initiative, we are glad to announce a new
      member to the family, the “<a href="https://marketplace.visualstudio.com/items?itemName=Autodesk.autolispext">VSCode extension for AutoLISP</a>” which is now Opensource in
      <a href="https://github.com/Autodesk-AutoCAD/AutoLispExt">Github</a>.
    </p>
    <p>
      This extension adds support for AutoCAD AutoLISP source (LSP) files to
      Visual Studio Code. It allows you to edit and debug your lisp programs
      with AutoCAD 2021. For information on creating lisp programs for AutoCAD
      with the AutoLISP programming language, see the:
    </p>
    <ul>
      <li>
        AutoCAD AutoLISP:
        <a href="https://www.autodesk.com/autolisp-developers-guide">Developer’s Guide</a>
      </li>
      <li>
        AutoCAD AutoLISP:
        <a href="https://www.autodesk.com/autolisp-reference">Reference</a>
      </li>
      <li>
        AutoCAD AutoLISP:
        <a href="https://www.autodesk.com/autolisp-tutorials">Tutorials</a>
      </li>
    </ul>
    <p>
      The extension is open source and distributed under
      <a href="https://github.com/Autodesk-AutoCAD/AutoLispExt/blob/main/LICENSE.md">Apache License v2.0</a>. A friendly summary can be found at
      <a href="https://tldrlegal.com/license/apache-license-2.0-(apache-2.0)">tldrlegal</a>.
    </p>
    <p>
      Your feedback is highly appreciated! Should you have any suggestions,
      please create an issue
      <a href="https://github.com/Autodesk-AutoCAD/AutoLispExt/issues">here</a>.
    </p>
    <p>
      We welcome contributions through
      <a href="https://docs.github.com/en/github/collaborating-with-issues-and-pull-requests/about-pull-requests">PR</a>’s for more details please read through the
      <a href="https://github.com/Autodesk-AutoCAD/AutoLispExt/blob/main/CONTRIBUTING.md">CONTRIBUTION</a>
      guide.
