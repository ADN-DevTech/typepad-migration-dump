<p>By <a href="http://adndevblog.typepad.com/autocad/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>
<p><strong>Issue      <br /></strong>Although there are many different variant data types available in ActiveX, Visual LISP does not use all of them. What variant data types are supported by Visual LISP?</p>
<p><strong>Solution      <br /></strong>The following code demonstrates the different data types and constants available to Visual LISP. Please note that to convert between actual values and variants, the following functions are used:</p>
<p>(vlax-make-variant [value] [type])    <br />(vlax-variant-change-type var type)     <br />(vlax-variant-type var)     <br />(vlax-variant-value var)</p>
For more information on converting the data types, see DevNote #53397 and the product documentation.
