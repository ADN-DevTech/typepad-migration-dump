<p>By <a href="http://adndevblog.typepad.com/autocad/augusto-goncalves.html">Augusto Goncalves</a></p>  <p>There is no specific isClosed() method defined for AcDbSpline as there is for    <br />AcDbPolyline. However, in many cases, an AcDbSpline can be closed. Here is a     <br />simple solution to implement a setClosed function for splines. It uses the     <br />getNurbsData() and setNurbsData() AcDbSpline methods. getNurbsData will gather     <br />the information about the spline, including a Boolean closed that indicates if     <br />the spline is closed or not. setNurbsData() takes the same arguments as     <br />getNurbsData(), so only the closed argument will be changed to Adesk::kTrue to     <br />obtain the desired result.</p>  <p>And the same idea can be applied to the .NET Spline.NurbData property.</p>  <div style="font-family: ;
