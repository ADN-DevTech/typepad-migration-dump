<p>By <a href="http://adndevblog.typepad.com/autocad/xiaodong-liang.html" target="_self">Xiaodong Liang</a></p>  <p><b>Issue</b></p>  <p>How do I get all block references on a particular layout, such as &quot;Layout1&quot;?</p>  <p><a name="section2"></a></p>  <p><b>Solution</b></p>  <p>These steps show how this can be done:</p>  <p>1. Iterate through AcDbBlockTables and get the AcDbBlockTableRecords.    <br />2. Check to see if AcDbBlockTableRecord is a layout using AcDbBlockTableRecord::isLayout(). If the AcDbBlockTableRecord is of type layout,then get its Id, open the layout using the Id and get the name of the layout.     <br />3. When you have retrieved the required name, create an iterator for that AcDbBlockTableRecord.     <br />4. Cycle through all entities for the layout. If the entity type is AcDbBlockReference, add it to the selection set.
