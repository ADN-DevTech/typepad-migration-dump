<p>By <a href="http://adndevblog.typepad.com/autocad/philippe-leefsma.html" target="_self">Philippe Leefsma</a></p>  <p><b>Q:</b></p>  <p>I would like to add some menu items to the Help menu on the AutoCAD window title bar. Is there a way to do this?</p>  <p><a name="section2"></a></p>  <p><b>A:</b></p>  <p>There are two ways to do this:</p>  <p>1) Through the registry on a per profile basis:</p>  <p>   <br />You can add a registry key for each of your menu item under the &quot;Help Button&quot; AutoCAD registry key and then populate them with information similar to the example below. The &quot;Help Button&quot; Key will not exist by default so you may need to add it. All of the entries are strings. (With regedit use New &gt; &quot;String Value&quot;) </p>  <p>   <br /><strong>[HKEY_CURRENT_USER\Software\Autodesk\AutoCAD\R19.0\ACAD-8001:409\Profiles\&lt;&lt;Unnamed Profile&gt;&gt;\Help Button\MyHelp]</strong></p>  <p><strong>     <br />&quot;Text&quot;=&quot;Test Help&quot;      <br />&quot;InsertBefore&quot;=&quot;AcHelpAbout&quot;      <br />&quot;Command&quot;=&quot;_LINE&quot;      <br />&quot;Separator&quot;=&quot;After&quot;</strong>    <br /></p>  <p>Note: &quot;Command&quot; is interpreted as a Help Topic, not an actual AutoCAD &quot;command&quot;, so &quot;_LINE&quot; will not start the line command, but will display the LINE help topic instead. You can use acedSetFunHelp() to associate the topic with a specific help file if you wish.</p>  <p>&quot;InsertBefore&quot; requires the id of the menu item that should follow your item. E.g. it is &quot;AcHelpAbout&quot; for &quot;ABOUT&quot;.</p>  <p>&#160;</p>  <p>2) Programmatically, using the Ribbon Runtime API</p>  <p>&#160;</p>  <p style="line-height: normal; margin: 0in 0in 0pt;
