<p>By <a href="http://adndevblog.typepad.com/autocad/gopinath-taget.html">Gopinath Taget</a></p>  <p>To copy entities from Model Space to Paper Space in ObjectARX is actually quite straightforward. You can utilize the services of deepCloneObject() function to copy your select entities from Model Space into Paper Space. This is useful in situations where for example an older drawing contains a title block and you want to copy that title block into Paper Space. The following code defines command called &quot;MS2PS&quot;.
