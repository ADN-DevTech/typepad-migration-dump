This post shows how to navigate in the other direction – finding all the references to a dynamic block. </p>  <p>As a reminder – when you manipulate a dynamic block, the changes are stored behind the scenes as anonymous blocks. An anonymous block is created for each different state of the dynamic block in the drawing.
