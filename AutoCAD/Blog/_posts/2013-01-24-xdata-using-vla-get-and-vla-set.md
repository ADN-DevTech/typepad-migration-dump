<p>By <a href="http://adndevblog.typepad.com/autocad/augusto-goncalves.html">Augusto Goncalves</a></p>  <p>You can use the (vla-GetXData) and (vla-SetXData) functions. These functions are not documented in the Visual LISP online help.</p>  <p>To get more information about these functions, look at the function description of GetXData and SetXData in the ACADAUTO.CHM help file (the AutoCAD ActiveX and VBA Reference).</p>  <p>Here is a short description of the functions:</p>  <p><span style="line-height: 120%; font-family: &#39;courier new&#39;, courier; font-size: 8pt"><span style="color: #ff0000">(</span><span style="color: #0000ff">vla-GetXData</span> entity appName 'codes 'datas<span style="color: #ff0000">)</span></span></p>  <p>entity:   <br />This is the entity name of an AutoCAD entity.</p>  <p>appName:   <br />This is a string with the application name. The functions returns all XDatas belonging to the specified application. If appName is &quot;&quot;, the functions returns all XDatas attached to 'entity'.</p>  <p>codes:   <br />This list receives the XData group codes.</p>  <p>datas:   <br />This list receives the data of the group codes specified in 'codes'</p>  <p><span style="line-height: 120%; font-family: &#39;courier new&#39;, courier; font-size: 8pt"><span style="color: #ff0000">(</span><span style="color: #0000ff">vla-SetXData</span> entity codes datas<span style="color: #ff0000">)</span></span></p>  <p>entity:   <br />This is the entity name of an AutoCAD entity.</p>  <p>codes:   <br />This list defines the XData group codes.</p>  <p>datas:   <br />This list contains the data for the specified group codes</p>  <p>The following is a sample code which attaches some XData to every entity in model space.After this, it extracts the XData of every entity from every application:</p>  <p><span style="line-height: 120%;
