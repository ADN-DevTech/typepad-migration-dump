<p>By <a href="http://adndevblog.typepad.com/autocad/fenton-webb.html">Fenton Webb</a></p>  <p><b>Issue</b></p>  <p>How do I configure the 3D Graphics System in AutoCAD OEM? </p>  <p><a name="section2"></a></p>  <p><b>Solution</b></p>  <p>In AutoCAD we can configure 3D Graphics System by using the Current 3D Graphics Display sub-option, which is found on Options Dialog.</p>  <p>However, AutoCAD OEM does not support the Current 3D Graphics Display sub-option, which is found on Options Dialog, System tab for standard AutoCAD. This functionality is available to you, the developer, however. </p>  <p>There are two ways to handle setting the values that are available on the 3D Graphics System Configuration dialog under the Properties button. </p>  <ol>   <li>You can enable the &quot;3DConfig&quot; command to gain direct access to the 3D Configuration screens for the graphics. </li>    <li>Alternately you can change the parameters in the registry to set certain configuration options by setting the following flags. These are found in the registry for your product under:</li> </ol>  <p>&#160;</p>  <p><strong>HKEY_CURRENT_USER\Software\&lt;company name&gt;\&lt;stamped oem appname&gt;\&lt;Your version&gt;\&lt;prod-key&gt;\3DGS Configuration\GSHEIDI10</strong></p>  <p>These values should be set before your product’s OEM engine initializes. If the values required are fixed for the requirements of you applications you could set these during install one time. Or if you only need a couple to be configurable, you could provide your own UI and set them as necessary, however, note that the OEM engine will need to be restarted in order for them to take affect.</p>  <p><strong>Flags =&#160; 0x00240000 - This the default, no options set value.</strong></p>  <p>Add the following flags to enable specific options:</p>  <p><strong><font size="1">0x00004000 = Render Option checked no others</font></strong></p>  <p><strong><font size="1">0x00008000 = Lights</font></strong></p>  <p><strong><font size="1">0x00000010 = Materials</font></strong></p>  <p><strong><font size="1">0x00020000 = Textures (High quality/Slower)</font></strong></p>  <p><strong><font size="1">0x00040000 = Textures (Medium Quality)</font></strong></p>  <p><strong><font size="1">0x00080000 = Textures (Low quality/Faster)</font></strong></p>  <p><strong><font size="1">0x00002000 = Adaptive Degradation checked no others</font></strong></p>  <p><strong><font size="1">0x00000001 = Flat Shaded</font></strong></p>  <p><strong><font size="1">0x00000002 = Wireframe</font></strong></p>  <p><strong><font size="1">0x00000004 = Bounding Box</font></strong></p>  <p><strong><font size="1">0x00000040 = Iso Lines on top</font></strong></p>  <p><strong><font size="1">0x00000008 = Discard Back Faces</font></strong></p>  <p><strong><font size="1">0x00000200 = Dynamic tessellations</font></strong></p>  <p><strong><font size="1">Surface Tessellation = SurfaceTessellationTol value = 0 – 0x64 (0 – 100 decimal)</font></strong></p>  <p><strong><font size="1">Curve Tessellation = CurveTessellationTol value = 0 – 0x64 (0 – 100 decimal)</font></strong></p>  <p><strong><font size="1">Number of Tessellations to Cache = MaxTessellations</font></strong></p>  <p><strong><font size="1">Maintain Speed at n FPS = FramesPerSecond</font></strong></p>  <p><strong><font size="1"></font></strong></p>  <p><font size="1">Some examples:</font></p>  <p><strong><font size="1">0x00244000 = Render Option checked no others</font></strong></p>  <p><strong><font size="1">0x0024c000 = Render Option + Lights</font></strong></p>  <p><strong><font size="1">0x0024c010 = Render Option + Lights + Materials</font></strong></p>  <p><strong><font size="1">0x00244010 = Render Option + Materials</font></strong></p>  <p><strong><font size="1">0x0026c010 = Render Option + Lights + Materials + Textures (High quality/Slower)</font></strong></p>  <p><strong><font size="1">0x002ac010 = Render Option + Lights + Materials + Textures (Medium quality)</font></strong></p>  <p><strong><font size="1">0x0032c010 = Render Option + Lights + Materials + Textures (Low quality/Faster)</font></strong></p>  <p><strong><font size="1">0x0032e010 = Adaptive Degradation + above</font></strong></p>  <p><strong><font size="1">0x00242000 = Adaptive Degradation checked no others</font></strong></p>  <p><strong><font size="1">0x00246000 = Adaptive Degradation + Render Option</font></strong></p>  <p><strong><font size="1">0x00242001 = Adaptive Degradation + Flat Shaded</font></strong></p>  <p><strong><font size="1">0x00242003 = Adaptive Degradation + Flat Shaded + Wire Frame</font></strong></p>  <p><strong><font size="1">0x00242007 = Adaptive Degradation + Flat Shaded + Wire Frame + Bounding Box</font></strong></p>  <p><strong><font size="1">0x00242040 = IsoLines on top</font></strong></p>  <p><strong><font size="1">0x00242008 = Discard Back Faces</font></strong></p>  <p><strong><font size="1">0x00242048 = IsoLines on top + Discard Back faces</font></strong></p>  <p><strong><font size="1">0x00240200 = Dynamic
