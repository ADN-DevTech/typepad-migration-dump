<p>By <a href="http://adndevblog.typepad.com/autocad/gopinath-taget.html">Gopinath Taget</a></p>  <p>Consider this: You would like to stop the user from selection ALL when using acedSSGet()? This because acedSSGet selects all of the entities in all of the spaces and this is not your intent. You only want it selects ALL the entities from the current space. So how can you go about doing this?</p>  <p>In AutoCAD there is no way to stop the acedSSGet() from allowing the user to use the &quot;All&quot; option; this is as designed. The best way to do what you want is to simply check the selection set after the acedSSGet() and make sure that each entity is owned by the current space...
