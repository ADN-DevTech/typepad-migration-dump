<p>By <a href="http://adndevblog.typepad.com/autocad/adam-nagy.html" target="_self">Adam Nagy</a></p>
<p>I have a variable which contains the name of a LISP function, but I do not know how to use that in order to call the function.</p>
<p><strong>Solution</strong></p>
<p>You can use <strong>(read)</strong> to get the function from the name then <strong>(mapcar)</strong> or <strong>(apply)</strong> to invoke it. Here are two functions that we will be calling knowing their names. One has an input parameter, the other one does not.</p>
