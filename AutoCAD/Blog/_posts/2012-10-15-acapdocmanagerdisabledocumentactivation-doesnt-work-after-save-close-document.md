<p>By <a href="http://adndevblog.typepad.com/autocad/philippe-leefsma.html" target="_self">Philippe Leefsma</a></p>  <p><b>Q:</b></p>  <p>AcApDocManager::disableDocumentActivation() should disable switching documents, opening new documents and closing documents. Usually, it does all this. However, if when saving the current document (eg. SAVE from command line), it is possible to directly close (fe. CLOSE from command line) the current document after that. Also opening a new document immediately after a SAVE is doable.</p>  <p>How can to make it work after SAVE?</p>  <p><a name="section2"></a></p>  <p><b>A:</b></p>  <p>A workaround is to use AcEditorReactor and override commandEnded().
