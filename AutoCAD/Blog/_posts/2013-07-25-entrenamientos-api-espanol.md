<h1><strong>AutoCAD API</strong></h1>
<p><strong>Descripción</strong></p>
<p>Esta capacitación está dirigido a desarrolladores y usuarios avanzados de AutoCAD, con ganas de aprender o mejorar sus habilidades para crear aplicaciones usando AutoCAD NET. Además de una descripción detallada de estas tecnologías, la capacitación incluye ejemplos y ejercicios con código.</p>
<p><strong>Requerimientos</strong></p>
<p>Conocimiento del desarrollo de software orientado a objetos, el conocimiento de lenguajes de programación VB.NET. Deseable conocimientos básicos de AutoCAD a nivel de usuario.</p>
<p><strong>Objetivo</strong></p>
<p>Al finalizar la capacitación, los participantes podrán:</p>
<ul>
<li>comprender los principios y el uso de la AutoCAD .NET.;</li>
<li>distinguir entre las categorías existentes de API y entender las posibles sinergias entre ellos;</li>
<li>para poder crear aplicaciones usando AutoCAD .NET.</li>
</ul>
<p><strong>Tabla de Contenidos</strong></p>
<p>La capacitación abarcará los siguientes temas (pero no limitados a):</p>
<ul>
<li>Conceptos básicos. NET</li>
<li>Configuración de proyecto Visual Studio</li>
<li>Función de entrada</li>
<li>Base de datos de AutoCAD .NET</li>
<li>Los controladores de eventos (controladores de eventos) en AutoCAD .NET</li>
<li>Desarrollar una interfaz de usuario: Diálogos WinForms y paletas de AutoCAD</li>
<li>Los materiales adicionales, el tiempo lo permite</li>
</ul>
<p><strong>Materiales</strong></p>
<p>Los ejemplos de código de la formación se presentarán en la lenguaje Visual Basic .NET.<br /><br /><strong>Duración</strong></p>
<p>2 días, desde las 9:00 hasta las 17:00 horas</p>
<p><strong>Inscripción</strong></p>
<p><a href="https://www.surveymonkey.com/s/entrena_API" target="_self">Llenar el formulario</a>.</p>
<h1>Revit API</h1>
<p><strong>Descripción</strong></p>
<p>Este curso está diseñado para enseñar los principios básicos de la programación con la API de Revit.</p>
<p><strong>Requerimientos</strong></p>
<p>Usted debe tener una comprensión básica de Autodesk Revit (Architecture, Structure y/o MEP), y también la lenguaje de programación C#.</p>
<p><strong>Objetivos</strong></p>
<p>Los participantes del curso aprenderán:</p>
<ul>
<li>Conceptos y funciones básicas asociadas con la API de Autodesk Revit.</li>
<li>En la última versión de la API de Revit.</li>
</ul>
<p><strong>Tabla de Contenido</strong><br /><br /></p>
<ul>
<li>Visión general de la API de Revit.</li>
<li>Extracción / Añadiendo propiedades de los elementos.</li>
<li>Obtención de información acerca de la geometría de los elementos.</li>
<li>Crear objetos básicos.</li>
<li>Acceso a la base de datos de Revit.</li>
<li>Acceso a la información sobre el proyecto.</li>
<li>Descripción de las familias estándar.</li>
<li>Adición de eventos a nivel de documento.</li>
</ul>
<p><strong>Materiales<br /></strong></p>
<p>Las presentaciones, ejemplos de proyectos que contienen el código, cursos y ejercicios de laboratorio de origen.</p>
<p><strong>Materiales</strong></p>
<p>Los ejemplos de código de la formación se presentarán en la lenguaje C#.<br /><br /><strong>Duración</strong></p>
<p>1 dia, desde las 9:00 hasta las 17:00 horas</p>
