<p>By <a href="http://adndevblog.typepad.com/autocad/stephen-preston.html" target="_self">Stephen Preston</a></p>  <p>Another feature added to <a href="http://adndevblog.typepad.com/autocad/2012/04/autoloader-the-startupcommand-parameter.html" target="_blank">Autoloader</a> in AutoCAD 2013 is support for adding Toolpalette support paths. Here’s how …</p>  <p><strong>Create your Tool palette</strong></p>  <p>Start the TOOLPALETTES command in AutoCAD.</p>  <p>Right click on the tool palettes control bar, and select New Palette.</p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01676577e1c6970b-pi"><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: block; float: none; border-top-width: 0px; border-bottom-width: 0px; margin-left: auto; border-left-width: 0px; margin-right: auto; padding-top: 0px" title="image" border="0" alt="image" src="/assets/image_820957.jpg" width="244" height="211" /></a></p>  <p>Give the new Palette a name and drag your content onto it. Here’s one I created called ‘Stephen’s Palette’.</p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b0168ea799abb970c-pi"><img style="background-image: none; border-bottom: 0px; border-left: 0px; padding-left: 0px; padding-right: 0px; display: block; float: none; margin-left: auto; border-top: 0px; margin-right: auto; border-right: 0px; padding-top: 0px" title="SNAGHTML24f382" border="0" alt="SNAGHTML24f382" src="/assets/image_630198.jpg" width="161" height="171" /></a></p>  <p><strong>Copy the Tool palette to your .bundle</strong></p>  <p>Check the Tool Palettes File Locations folder on your OPTIONS dialog –&gt; Files tab. The default will be something like %appdata%\Roaming\Autodesk\AutoCAD 2013 - English\R19.0\enu\Support\ToolPalette\Palettes.</p>  <p>Open that folder and look for the tool palette you just created.The file name will be &lt;PaletteName&gt;_&lt;GUID&gt;.atc, so the one I just created is called ‘Stephen's Palette_10E78C8E-A3D5-40DF-A284-2A61AC4E0E90.atc’</p>  <p>Move the ATC file into your bundle and delete palette from your Tool palettes window by right clicking on the control bar again. You might as well close down AutoCAD too, so it doesn’t try to load your bundle while you’re editing it.</p>  <p><strong>Add the ToolPalettePath to PackageContents.xml</strong></p>  <p>Your XML will look something like this – the ToolPalettePath highlighted in bold is the location within the bundle of the ATC file(s):</p>  <div style="font-family: courier new;
