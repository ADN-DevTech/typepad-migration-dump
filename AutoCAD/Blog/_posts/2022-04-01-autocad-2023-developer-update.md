<p>This year we have released many good features, for key features refer <a href="https://www.autodesk.com/products/autocad/features">https://www.autodesk.com/products/autocad/features</a>
	</p><p><img src="/assets/image_534798.jpg" alt=""/>
	</p><p>Important attraction on API side for this release is AutoCAD LISP API for AutoCAD web app.
</p><p>This API is exclusively for AutoCAD subscribers. Whether you are on the go, at a job site, or anywhere else, create your own customizations to automate sequences with LISP in the AutoCAD web app.  <a href="https://videos.autodesk.com/zencoder/content/dam/autodesk/www/products/autocad/fy23/features/videos/autocad-web-apis-video-1920x1080.mp4">(Video: 55 sec.)</a>
	</p><p>Some known limitations of Lisp API on web –
</p><p>- Any functionality that relies on a platform-specific technology (i.e., Visual Lisp and DCL).
</p><p>- Any functionality that is not supported in AutoCAD Web (AutoCAD Web does not support everything desktop does - 3d editing for example).
</p><p>- Any functionality that would violate browsers sandboxing rules (such as reading and writing to your local filesystem without going through normal browser restrictions.
