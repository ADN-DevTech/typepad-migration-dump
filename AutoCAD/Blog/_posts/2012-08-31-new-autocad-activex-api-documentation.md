<p>By <a href="http://adndevblog.typepad.com/autocad/stephen-preston.html">Stephen Preston</a></p>  <p>I received a pleasant email today from a good friend in our Tech Pubs team letting me know that a new set of ActiveX Developer Documentation just went live on the <a href="http://www.autodesk.com/developautocad" target="_blank">AutoCAD Developer Center</a>.&#160; It can be found&#160; Documentation section (where else):</p>  <p><a href="http://www.autodesk.com/developautocad" target="_blank"><img style="background-image: none; border-bottom: 0px; border-left: 0px; padding-left: 0px; padding-right: 0px; display: block; float: none; margin-left: auto; border-top: 0px; margin-right: auto; border-right: 0px; padding-top: 0px" title="image" border="0" alt="image" src="/assets/image_568570.jpg" width="438" height="300" /></a></p>  <p>This update includes or addresses the following:</p>  <ul>   <li>Updates to the Reference Guide to include missing classes, properties, and methods.</li>    <li>Fixes some of the existing VBA samples and introduces a large set of samples in Visual LISP.</li>    <li>Updates to the Developer’s Guide and makes it available now, since it was not available when AutoCAD 2013 shipped.</li>    <li>API History/Changes list has been added to the Developer’s Guide.</li> </ul>  <p>(Thanks Lee <img style="border-bottom-style: none; border-right-style: none; border-top-style: none; border-left-style: none" class="wlEmoticon wlEmoticon-smile" alt="Smile" src="/assets/image_884818.jpg" />).</p>
