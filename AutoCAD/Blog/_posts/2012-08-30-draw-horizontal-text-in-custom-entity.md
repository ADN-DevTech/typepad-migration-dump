Using the up vector one can easily calculate the flow direction for drawing the text horizontal to the current view.</p>  <p>During plot, one needs to take into account the plot rotation. During the plot, the drawing is rotated depending on the drawing orientation(Portrait or Landscape).</p>  <p>Of course, this can be implemented only in the viewportDraw() override of the custom entity, since this is where the viewport-specific graphics are drawn.
