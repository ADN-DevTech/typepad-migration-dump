<p>By <a href="http://adndevblog.typepad.com/autocad/balaji-ramamoorthy.html" target="_self">Balaji Ramamoorthy</a></p>
<p>The e-Trasmit API is exposed using COM (AcETransmit18.tlb) and so, you will first need to create an inter-op assembly for use in your .Net project. You can add reference from the COM tab and select "Transmittal 18.0 Type Library" in which case Visual Studio will create the inter-op assembly. The other way is to create it yourself using the <a href="http://msdn.microsoft.com/en-us/library/tt0cf3sx(v=vs.80).aspx">type library importer tool</a> and add reference to the generated inter-op assembly. Now, you are all set to use the e-Transmit API.</p>

<p>Here is a sample code :</p>
<p></p>
