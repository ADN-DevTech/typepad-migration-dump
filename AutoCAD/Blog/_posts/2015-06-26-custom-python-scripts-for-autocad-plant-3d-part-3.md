<p><strong>Script Testing</strong></p>

<p>While creating the script, you must be able to test it to see if your geometry can be generated.&#160; Because these scripts are designed to be processed within AutoCAD, currently there is no IDE that you can use to debug the shapes. The process of testing the script looks like this.</p>

<ol>
  <li>Create your script file at C:\AutoCAD Plant 3D 2016 Content\CPak Common\CustomScripts (or wherever your Shared Content folder is for plant (MODIFYSHAREDCONTENTFOLDER).</li>

  <li>Create your geometry functions, and metadata sections.</li>

  <li>Start Plant 3D</li>

  <li>Use PLANTREGISTERCUSTOMSCRIPTS. If there is an error in the script, you will get a warning at the command line with the line number. For most errors line of the script with the error will be listed.&#160; If no errors are shown, the script compiled.
    <br /><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b01bb08493c14970d-pi"><img title="04" style="border-top: 0px; border-right: 0px; border-bottom: 0px; border-left: 0px; display: inline" border="0" alt="04" src="/assets/image_923039.jpg" width="442" height="124" /></a> </li>

  <li>Load the PnP3dACPAdapter.arx. You can do this by pasting this at the command line (arxload &quot;PnP3dACPAdapter&quot;)</li>

  <li>Test the script. You can test with the default parameters by pasting this at the command line: (testacpscript &quot;EXPJOINT1&quot;). Fill in your script function where EXPJOINT1 is.
    <br />You can also test the script from one of the other following options:

    <br />&#160;&#160;&#160; (TESTACPSCRIPT &quot;CPFLR&quot;)

    <br />&#160;&#160;&#160; (TESTACPSCRIPT &quot;CPFLR&quot; &quot;D1&quot; &quot;300.5&quot;) 

    <br />&#160;&#160;&#160; (TESTACPSCRIPT &quot;CPFLR&quot; &quot;L&quot; &quot;40&quot; &quot;D1&quot; &quot;300.5&quot; &quot;D2&quot; &quot;88.9&quot;)

    <br /> Or 

    <br />&#160;&#160;&#160; (TESTACPSCRIPT1 &quot;CPFLR&quot;) 

    <br />&#160;&#160;&#160; (TESTACPSCRIPT1 &quot;CPFLR&quot; &quot;D1=300.5&quot;)

    <br />&#160;&#160;&#160; (TESTACPSCRIPT1 &quot;CPFLR&quot; &quot;L=40,D1=300.5,D2=88.9&quot;)</li>

  <li>If you need to make changes, make them before closing Plant. That was you can try to register the script again to see if you still have breaking code. </li>

  <li>If the script has been changed and registered, you must close Plant 3D in order to test the geometry again. Once the script is registered or run, the Python processor stores that script in memory until Plant 3D is closed, so starting a new session is the only way test new script changes.</li>
</ol>
