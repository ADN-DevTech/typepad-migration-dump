<p>By <a href="http://adndevblog.typepad.com/autocad/Virupaksha-Aithal.html" target="_self">Virupaksha Aithal</a></p>
<p>You need to iterate through the dictionary of plot style names in the Named Object Dictionary to find all plot styles available.&nbsp; In the dictionary, each entry is a type of System.Collections.DictionaryEntry. Each DictionaryEntry object has a Key/Value pair of properties. The Key property is the name of the plot style and the value stores the Object ID of a PlaceHolder object.</p>
