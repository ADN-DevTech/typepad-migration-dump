<p>by <a href="http://adndevblog.typepad.com/autocad/fenton-webb.html">Fenton Webb</a></p>
<p>In case you haven’t spotted it yet, AutoCAD and verticals have started adopting the ‘Configurable AutoCAD Structure’ for 2014.</p>
<p>What this means is that verticals no longer install to separate program files folders, but instead install off a child folder of AutoCAD… e.g.</p>
<p><strong>C:\Program Files\Autodesk\AutoCAD Plant3d 2013</strong></p>
<p>Is now</p>
<p><strong>C:\Program Files\Autodesk\AutoCAD 2014\PLNT3d</strong></p>
