<p>There was an issue with hideDisplay API of Body class in AecModeler in AutoCAD 2017. below is the updated module.</p>
<p><a href="http://adndevblog.typepad.com/autocad/AecModeler.zip">AecModeler for AutoCAD 2017 SP1</a></p>
<p><strong>To install this update</strong><br />1. Close all software applications.<br />2. Navigate to the folder where your Autodesk product is installed.<br /> C:\Program Files\Autodesk\AutoCAD 2017<br />3. Right-click AecModeler.dbx and rename it as AecModeler.bak.<br />4. Download the files to a location on your local computer.<br />5. Copy the downloaded file (AecModeler.dbx) (from the 32-bit or 64-bit sub-folder depending on whether you have 32-bit or 64-bit software) to your product installation folder <br /> <br /><strong>To remove this update</strong><br />1. Close all software applications.<br />2. Navigate to the folder where your Autodesk product is installed,<br /> C:\Program Files\Autodesk\AutoCAD 2017<br />3. Delete AecModeler.dbx (the file you downloaded and installed).<br />4. Rename AecModeler.bak to AecModeler. dbx</p>
