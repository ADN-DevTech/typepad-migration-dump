<p>By <a href="http://adndevblog.typepad.com/autocad/Madhukar-Moogala.html" target="_self">Madhukar Moogala</a></p><p>This is small utility file to create a zip file that ignores temporary, .vs and packages files. This can be applied to any VS project.<br>create a bat file with below contents and name it mkzip.bat<br>Replace this with attached file <code>C:\UtilStuff\bin\vsExclude.txt</code><p>or you download the zip package containing <code>mkzip.bat</code> and <code>vsExclude.txt</code><p>Note: make sure you have <a href="https://www.7-zip.org/download.html">7z</a> on your path<p>The Github project - <a title="https://github.com/MadhukarMoogala/mkzip" href="https://github.com/MadhukarMoogala/mkzip">https://github.com/MadhukarMoogala/mkzip</a><p><br>
