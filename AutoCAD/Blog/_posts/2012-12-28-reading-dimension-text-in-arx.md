<p>By <a href="http://adndevblog.typepad.com/autocad/gopinath-taget.html">Gopinath Taget</a></p>  <p>The dimension text cannot be read directly it is because the information is contained into an AdDbMtext entity owned by a block table record. From the AcDbDimension (or derived classes), you can get the block table record ID using the method dimBlockId(). Then you can iterate throught the AcDbBlockTableRecord object for the AcDbMText entity, and call the contents() method.
