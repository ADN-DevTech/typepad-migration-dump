<p>by <a href="http://adndevblog.typepad.com/autocad/fenton-webb.html" target="_blank">Fenton Webb</a></p>
<p>Following on from my previous post <a href="http://adndevblog.typepad.com/autocad/2012/03/dwg-inspector-for-autocad-2012-and-autocad-2013.html" target="_blank">DWG Inspector for AutoCAD 2012/2013</a>, here is the MgdDbg tool migrated to 2012 and 2013 bundled as an Exchange App – you can download it <a href="http://adndevblog.typepad.com/autocad/Downloads/AutodeskDWGMgdDBG.zip" target="_blank">here</a></p>
<p>One thing to note is that there are commands included in the app, however, because this app has startup code (it creates context menu entries) I have omitted all ComponentEntry initialization options, so LoadOnStartup is assumed. (I have not used the LoadOnCommandInvocation attribute)</p>
<p>&nbsp;</p>
