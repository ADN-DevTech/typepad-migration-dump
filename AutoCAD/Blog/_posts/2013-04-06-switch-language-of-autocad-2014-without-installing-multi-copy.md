<p>By <a href="http://adndevblog.typepad.com/autocad/daniel-du.html">Daniel Du</a></p>  <p>Are you bilingual, trilingual or even octalingual? Are you working languages other than English in AutoCAD? In previous release, you probably need to install multi copies of AutoCAD, like myself, I have to install AutoCAD English and Chinese on my laptop. A good news of AutoCAD 2014(AutoCAD 2013 actually) is that we can just install the language package rather than the whole product.</p>  <p>I first installed AutoCAD 2014 English on my laptop, if I examine the the short-cut, I can find that it is : </p>  <p>&quot;C:\Program Files\Autodesk\AutoCAD 2014\acad.exe&quot;&#160; /product ACAD <font style="background-color: #ffff00">/language &quot;en-US&quot;</font></p>  <p>Now I would like to use Chinese version as well, so I start AutoCAD 2014 and go to Help â€“&gt; Download Language Packs and download my favorite Simplified Chinese language pack and install it. A new shortcut will be created like this: </p>  <p>&quot;C:\Program Files\Autodesk\AutoCAD 2014\acad.exe&quot;&#160; /product ACAD<font style="background-color: #ffff00"> /language &quot;zh-CN&quot;</font></p>  <p><a href="http://adndevblog.typepad.com/.a/6a0167607c2431970b017eea0b96cd970d-pi"><img style="background-image: none; border-right-width: 0px; padding-left: 0px; padding-right: 0px; display: inline; border-top-width: 0px; border-bottom-width: 0px; border-left-width: 0px; padding-top: 0px" title="image" border="0" alt="image" src="/assets/image_678081.jpg" width="244" height="162" /></a></p>  <p>With this new shortcut, I can start Chinese version of AutoCAD! </p>  <p>Hope this helps.</p>
