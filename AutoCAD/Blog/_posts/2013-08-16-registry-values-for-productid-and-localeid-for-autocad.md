<p>By <a href="http://adndevblog.typepad.com/autocad/Virupaksha-Aithal.html" target="_self">Virupaksha Aithal</a></p>
<p>HKEY_LOCAL_MACHINE\Software\Autodesk\AutoCAD\R20.0\ACAD-E001:409 What is ACAD-E001:409?</p>
<p>R20.0 is the release number, the this corresponds to AutoCAD 2015</p>
<p>These digits like E000:409 are a ProductID (E000) and a LocaleID (409).</p>
<p>The ProductID structure is a 4-digit value. However, prior to AutoCAD 2006, the ProductID structure was only 3-digits with a single [p] value. Due to the number of products based on AutoCAD, in AutoCAD 2006 structure was changed to 4-digits.</p>
<p>From AutoCAD 2006 forward, the structure is configured as &quot;[n][r][pp]&quot;, where:<br />[n] = Number representing AutoCAD base release (see below)<br />[r] = Release number within AutoCAD base, reset to zero for every new n<br />[pp] = Product based on AutoCAD (see below)</p>
<p>ProductID Assignments (Assigned [n] Values)</p>
<p>F = AutoCAD 2016<br />E = AutoCAD 2015<br />D = AutoCAD 2014<br />B = AutoCAD 2013<br />A = AutoCAD 2012<br />9 = AutoCAD 2011<br />8 = AutoCAD 2010<br />7 = AutoCAD 2009<br />6 = AutoCAD 2008<br />5 = AutoCAD 2007<br />4 = AutoCAD 2006<br />3 = AutoCAD 2005<br />2 = AutoCAD 2004<br />1 = AutoCAD 2002</p>
<p>The LocaleID structure is a 4-character hex representation of the product&#39;s locale / language.</p>
